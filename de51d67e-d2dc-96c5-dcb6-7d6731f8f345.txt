        “寒哥，连星儿到底能不能动？”仲正浩弱弱地问道。
<br/>
<br/>        明司寒虽然交代了要孤立连星儿，但同时也交代了谁都不许动她连星儿。
<br/>
<br/>        可现在的问题是，这连星儿竟然欺负到他仲正浩的女朋友姚莹莹头上来了，这可是“是可忍孰不可忍”的事！
<br/>
<br/>        “谁动她了？”明司寒一脸不满地皱起了眉头。
<br/>
<br/>        仲正浩咧了咧嘴：“连星儿欺负我女朋友了，这账要怎么算？”
<br/>
<br/>        “谁欺负谁？”明司寒眉头紧锁，抬眸瞪向仲正浩。
<br/>
<br/>        他这一瞪，眼神犀利得将仲正浩吓了一跳，回话的气势也跟着弱了：“连星儿欺负我家莹莹了。”
<br/>
<br/>        “噢。”明司寒这才眉心舒展，继续若无其事地垂眸看着手中的英文词典。
<br/>
<br/>        “你这‘噢’是几个意思？”仲正浩顿时不高兴了。
<br/>
<br/>        躺在上铺的邢乐游一边翘着二郎腿拿着手机打游戏，一边忍俊不禁道：“正浩，你白痴哦？寒哥的意思就是，只要不是你家莹莹欺负连星儿，他就懒得管。”
<br/>
<br/>        “那我的女人被人欺负了，你们当兄弟的不管，我自己去管好了！”仲正浩有些恼火起来。
<br/>
<br/>        坐在一旁玩电脑游戏的张子晋，瞥了一眼仲正浩冷不丁地说道：“你最好别去，据我所搜查到的资料，连星儿曾得过国际散打少年组的蝉联冠军，超级能打，你去了只怕连男人的尊严都没了。连星儿不是那种会主动惹事的女生，十有八九是你那个不安分的女友先挑的事，所以，我劝你回个电话，叫你女朋友跟连星儿认个错。这事也就完了。”
<br/>
<br/>        “你们都当怂包，是不是？好！你们不去！我自己带人去！”仲正浩甩了鼠标，气呼呼地起身出了门。
<br/>
<br/>        明司寒这才合上手中的英文词典，从椅子上起身，淡淡地说道：“我们也去凑凑热闹。”
<br/>
<br/>        邢乐游立马从上铺直接跳了下来，双脚一落地，侧身对上张子晋的目光时，彼此心照不宣了。
<br/>
<br/>        还说不认识人家连星儿，这不，身体表现出来的倒是比嘴上说的要诚实。
<br/>
<br/>        明司寒跟着去，他们俩可不认为是他想替仲正浩出头，十有八九是怕仲正浩欺负连星儿。
<br/>
<br/>        这边没过多久，仲正浩就怒气冲冲地赶到了包厢里。
<br/>
<br/>        姚莹莹一看到男友仲正浩来了，立马小鸟依人地黏上去，哭诉道：“老公，连星儿她打我的姐妹们。”
<br/>
<br/>        下一秒，姚莹莹又纳闷地质问：“你怎么一个人来的啊？”
<br/>
<br/>        仲正浩搂着姚莹莹四下看了看，二话不说就拿起茶几上一瓶开动的饮料，大步流星地朝那边的连星儿走去。
<br/>
<br/>        连星儿不明所以然地看着仲正浩，就在他握着那瓶饮料挥手之际，一抹白影突然冲到她的跟前，将她的身子笼罩在他的阴影之中。
<br/>
<br/>        “哗”的一声，仲正浩手中的那瓶饮料全部泼到了一个白色的背影上。
<br/>
<br/>        明司寒面无表情地出现在大家的视野中，双手抄在裤袋中，冷冽的黑眸透着一丝慵懒，睨着面前个头只到自己胸口处的连星儿。
<br/>
<br/>        仲正浩见自己手中的饮料全泼到了明司寒的背上，顿时一点脾气也没有，反倒有些后怕。
<br/>
<br/>        姚莹莹和自己的那七个姐妹们也瞠目结舌得不敢出声。
<br/>
<br/>        “嗨！”连星儿却有些尴尬地向明司寒挥了挥手，“你终于认出我了？呵呵——我还以为你过了一个暑假就失忆了。”
<br/>
<br/>        “你先动手打人的吗？”明司寒明知故问。
<br/>
<br/>        连星儿还未开口，赵安彤就指控道：“是她们先动手的！若不是星儿，我们几个指不定就被她们甩耳光了！”
<br/>
<br/>        “姚莹莹，带着你的姐妹们跟星儿道歉。”明司寒淡漠地命令道。
<br/>
<br/>        姚莹莹顿时咬牙切齿道：“凭什么要我们道歉？受伤的人是我们，又不是她！”
<br/>
<br/>        连星儿一听姚莹莹这话，柳眉一皱，故作委屈地吸了吸鼻子，抬起手背，伸到明司寒的眼皮子底下，可怜兮兮地说道：“谁说我没受伤来着？！我这手背就是被她们抓伤的！”
<br/>
<br/>        明司寒垂了垂眸，看到连星儿那白皙的手背上出现的三道抓痕，莫名其妙地一阵揪心，不禁怒斥道：“仲正浩！”
<br/>
<br/>        仲正浩见状，皱着眉头反手就是一巴掌打在了姚莹莹的脸上。
<br/>
<br/>        几个女生被这气势给吓得出了一身冷汗。
<br/>
<br/>        赵安彤也不敢再多嘴了。
<br/>
<br/>        连星儿也是一脸不可思议地看向仲正浩和姚莹莹。
<br/>
<br/>        “你还真是个麻烦精。”明司寒伸手抓住连星儿的手腕，拉着她直接离开了包厢。
