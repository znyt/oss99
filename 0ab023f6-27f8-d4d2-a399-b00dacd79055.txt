        贺思思坐在副驾驶座里，一个劲地滔滔不绝起来：“那个连星儿有什么好？她很漂亮吗？也不见得有多漂亮！只不过是家里有钱而已！我家也很有钱啊！司寒哥哥为什么那么在乎她啊？”
<br/>
<br/>        “……”贺泽川继续开车，没吭声。
<br/>
<br/>        贺思思又接着说道：“哥，既然你没交女朋友，那么你去追那个连星儿，好不好？”
<br/>
<br/>        她这话一落，耳畔突然响起了一道刺耳的刹车声。
<br/>
<br/>        贺泽川踩了个急刹，贺思思的身子若不是系好了安全带，不然早就被一头撞到前面的挡风玻璃上了。
<br/>
<br/>        “哥，你干什么呀？”贺思思忍不住咆哮了起来。
<br/>
<br/>        贺泽川只是微微眯起了眸子，冷不丁地咧嘴笑了笑。
<br/>
<br/>        看到自家哥哥俊脸上那意味不明的笑意时，贺思思也跟着冷不丁地感到不寒而栗。
<br/>
<br/>        贺泽川没有回答贺思思的话，而是重新松开脚刹，踩下油门。
<br/>
<br/>        又是一道惯性，驱使贺思思的身子本能地往后仰。
<br/>
<br/>        “哥，你开车能不能稳妥一点？”车内，贺思思抱怨连连。
<br/>
<br/>        另一边，连星儿刚进寝室门，顾千凝和赵安彤还有路娅玟便凑了过来，一个个争先恐后地八卦道：
<br/>
<br/>        “你今天中午和明司寒去约会了吧？”
<br/>
<br/>        “星儿，你的男朋友就是明司寒吧？”
<br/>
<br/>        “明司寒可是我们学校的校草，你确定你不大方坦白承认恋情？”
<br/>
<br/>        三人一人一句。
<br/>
<br/>        连星儿一一否决后，胡诌了理由：“其实吧！我和明司寒是表兄妹关系。”
<br/>
<br/>        “啊？”三人面面相觑。
<br/>
<br/>        连星儿会心一笑，接着谎话连篇地说道：“我也是最近才知道的。原来他是我妈咪的姥姥的哥哥的后代。”
<br/>
<br/>        “……”顾千凝、路娅玟和赵安彤同时汗颜。
<br/>
<br/>        知道连星儿这家伙在故意撒谎后，三人便没再继续这个话题了。
<br/>
<br/>        不过，经过论坛上那件事情后，她连星儿的名声在临海大学里算是人尽皆知了。
<br/>
<br/>        “坊间”开始流传一句话叫——惹谁都行，就是不要惹连星儿。
<br/>
<br/>        否则，法院传票送上，警察局里喝茶，二选一。
<br/>
<br/>        这天，连星儿只是去上个洗手间，岂料她刚进洗手间的门，里面的女生一看到她，就像见了母老虎似的纷纷往门外窜。
<br/>
<br/>        上课的时候，除了顾千凝她们三个，其他无论男女同学皆离她两米远。
<br/>
<br/>        就连平常上课时爱叫她站起来回答问题的教授，也不敢再在课堂上点她的名了。
<br/>
<br/>        一个个生怕把她得罪了似的。
<br/>
<br/>        面对大家对自己的忌惮，连星儿心里虽然有些不舒服，但是很快就将这事抛到了九霄云外。
<br/>
<br/>        直到她上午上完课后，和顾千凝她们三个一起回寝室的途中，在寝室楼下遇到了一对中年夫妇，这件事情又再次突兀地闯入她的脑海中。
<br/>
<br/>        “请问，你是连星儿吗？”穿得珠光宝气却又面容憔悴的妇人，突然上前拉住连星儿的手。
<br/>
<br/>        连星儿和顾千凝她们面面相觑后，讷讷地应道：“我是，请问，您是？”
<br/>
<br/>        一旁穿着西装革履的中年男子立马解释道：“我们是朱小溪的爸爸妈妈。”
<br/>
<br/>        顾千凝一听这话，立即识趣地对连星儿说道：“星儿，我们就先上去了，你们慢慢聊。”
<br/>
<br/>        连星儿点了点头。
<br/>
<br/>        顾千凝随后拉着赵安彤和路娅玟一起先回了寝室。
<br/>
<br/>        妇人看了一眼顾千凝她们的背影，见她们走远后，又看向面前的连星儿，声音暗哑地接着说道：“你可否跟我们单独谈谈我女儿小溪的事情？”
<br/>
<br/>        “你们有什么话，就在这儿说吧！”连星儿淡漠道。
<br/>
<br/>        中年男子上前恳求道：“换个没人的地方可以吗？”
<br/>
<br/>        连星儿看了看这中年男子一眼，又看了看妇人一眼，咧着嘴虽然不乐意，但还是点头应许了他们的要求。
<br/>
<br/>        现在正是大家上午下课自由活动的时间段，所以学校里走哪，哪儿都有人。
<br/>
<br/>        于是，这对夫妻又提出一起去校外的咖啡馆坐坐。
<br/>
<br/>        连星儿倒是不怕这对夫妇对自己耍什么阴招，很爽快地就答应了。
<br/>
<br/>        朱氏夫妻俩在咖啡馆里开了个包厢，适合保护三个人谈话的隐私。
<br/>
<br/>        服务员将咖啡端上桌后，妇人便从手提包里拿出了一个牛皮纸信封，鼓鼓地一大包，递到了连星儿的面前。
<br/>
<br/>        “连同学，这里面有两万块，你看够不够？”妇人低声下气地问。
<br/>
<br/>        连星儿不解地看着妇人，反问：“阿姨，你这是什么意思？”
<br/>
<br/>        未等妇人回话，中年男子抢先说道：“孩子，这钱，你就收下。算是，我们对你的补偿。”
