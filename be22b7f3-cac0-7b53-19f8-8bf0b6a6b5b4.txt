　　第370章
<br/>　　钟伟虎回过头，看到程大雷打着哈欠立在自己身后。他的确刚睡醒的模样，脸颊浮肿，眼里布满血丝。
<br/>　　而钟伟虎眼中的怒火，任何一个人都可以感受得到，谁也不会怀疑，他随时会扑上撕咬程大雷。
<br/>　　当然，他并不会这么做，过了半晌，他咬着牙，从齿缝中蹦出一句话。
<br/>　　“你做得好事。”
<br/>　　“当然啦，可是不容易，昨天熬了一个大夜，你没看到我现在才睡醒么。”程大雷道：“当然，这也对得起宋大人对我的看重了。”
<br/>　　“程当家，我可记得你当初答应我的，人一定要活着？”钟伟虎咬牙切齿。
<br/>　　“我这个人最是言而有信，说过的话一口吐沫一口钉子，怎么能不作数。”程大雷一拉车帘，道：“诸位都看仔细了，这个人是活着还是死了。”
<br/>　　所有人都往里面望，不仅凉州城的兵，琴川关也有许多人不知道怎么会事。可在所有人都看过去的时候，钟伟虎却将头侧到一边。
<br/>　　因为车内的情景，他绝对不想再看第二眼。
<br/>　　只是一瞬，已足够令人心惊胆战。双方战士中，并不缺乏心狠手辣之辈，可在看到车内情形时，却有人脸色苍白，露出难受模样。
<br/>　　车内，有一口硕大的坛子。这坛子是用来腌咸菜用的，这时代没有什么保存食物的法子，就用大量的盐将蔬菜腌起来，几乎家家户户都有一口。不过，此刻坛子里并没有泡菜，而是装着一个人。
<br/>　　他的手和脚显然都被砍去了，就这样装进坛子里，只有一颗脑袋露在外面。张着口时断时续的呼吸，他很显然还活着，可和死了也差不多。甚至于说，他连决定自己去死的权力都没有，因为包括他的舌头都被割断了。
<br/>　　钟伟虎的愤怒，人人都可以感受到。他的目的是平安将崔弄海带出，可崔弄海现在这个样子，却是狠狠给了他一个耳光。如此模样带回去，怎么向宋伯康交差。
<br/>　　凉州城和琴川关两方的人剑拔弩张，气氛十分凝重，随时都可能擦枪走火。
<br/>　　“瞧一瞧。”程大雷的声音突然响起，显得有些格格不入：“看着没有，答应给你活人，人就一定要活着，我这个人做事，就是这么诚实可靠。”
<br/>　　钟伟虎怒发冲冠，恨不得当场就和程大雷拼命。还好，残存的理智提醒了他，这是琴川关，是程大雷的老巢，想来，程大雷是不介意找个机会杀死自己的。
<br/>　　他强压下心中的怒火，开口一字一顿道：“程当家果然诚实可靠，这件事我记下了。”
<br/>　　“好啦，好啦，我这个人施恩不忘报的。”
<br/>　　“我们走！”
<br/>　　钟伟虎一咬牙，带着马车上的崔弄海离去。程大雷驻足远送，不舍之意十分明显。
<br/>　　等人都走尽了，和珅才凑过来道：“大当家，接下来怎么办，怕凉州城那边不会善罢甘休？”
<br/>　　“还能怎么办，赶紧把粮食运进仓库里，难道你没看见天快下雨了。”
<br/>　　天空上果然彤云密布，一场大雨随时都可能会来。在程大雷的指挥下，大家加紧开始忙碌起来，几乎是刚刚把所有粮食存放好，秋雨便扑面而来。
<br/>　　包括程大雷在内的所有人，都被淋了个落汤鸡。
<br/>　　大家都躲在屋檐下避雨，嬉笑怒骂声一片，程大雷心情也是极其愉快，长久笼罩在头顶的乌云，似乎也将随着这场暴雨烟消云散。
<br/>　　忽然间，程大雷发现脑海中出现一个进度条状的东西，如今进度指向33%。
<br/>　　攻城模式。
<br/>　　程大雷心中一动，瞬间明白了这进度条的含义。莫非除掉这崔弄海，也是任务中的一环。什么时候进度条走慢，自己就可以开启攻城模式。
<br/>　　只是不知道下一环任务是什么。
<br/>　　……
<br/>　　大雨即落在不善的人身上，也落在善人身上。
<br/>　　还没回到凉州城的钟伟虎，就赶上了这场大雨，稀里哗啦被淋了个落汤鸡。
<br/>　　当然，如今的钟伟虎怒火中烧，全身燥热，感觉这场雨却也给他带来些许清凉。
<br/>　　“人还活着么？”钟伟虎问。
<br/>　　“嗯，还活着。”手下人回答。
<br/>　　钟伟虎说的，自然是马车中的崔弄海。钟伟虎暗暗咬牙，一个人怎么可能这么强的生命力，你都这个样子了，干脆死了拉倒，何必要活着呢。
<br/>　　现在崔弄海的确还活着，可他活着的状态实在是……手足被斩断，舌头被拔掉，身上大大小小的伤更是不计其数。
<br/>　　换做一般人，早就一命呜呼了，可崔弄海还活着，甚至求生欲望还很强。
<br/>　　死去的崔弄海要比活着的他更容易处理，如今钟伟虎是巴不得对方死翘翘。
<br/>　　“大人，不如干脆……”手下做出一个抹脖子的姿势。
<br/>　　“好啊，你去做，这个任务就交给你了。”
<br/>　　手下人讪讪离去，钟伟虎不是不想杀了崔弄海，手下的念头，他也动过。然而，他更明白顶头上司宋伯康的风格。
<br/>　　崔弄海这个样子，相府是肯定要派下人来惩罚的，如果自己杀了崔弄海，毫不怀疑，宋伯康会把自己推出去背黑锅。
<br/>　　钟伟虎可不愿意给任何人去背锅。
<br/>　　一队人返回凉州城，和钟伟虎预料中的一样，宋伯康勃然大怒。不仅没把崔弄海平安捞出来，反而损失了大量钱粮。
<br/>　　“人还活着么？”平静下来后宋伯康问。
<br/>　　“还活着，活的很结实。”钟伟虎也不知道怎么说，明明崔弄海就剩下一口气了，可他偏偏却还活着。
<br/>　　“大人，不如干脆杀了？”钟伟虎做出一个割喉的姿势。
<br/>　　“好啊，这件事就有你去办，你去杀了他。”
<br/>　　钟伟虎讪讪闭嘴。
<br/>　　这个时候，宋伯康和钟伟虎的心情差不多。谁都看得出来，崔弄海活着比死了更加痛苦，可杀人这件事，却绝不能由自己来做。
<br/>　　“先请大夫给他收拾一下，然后送回到长安城，接下来的事情，就由丞相大人做主吧。”宋伯康道。
<br/>　　“怕只怕，相府这次不会再放过程大雷。”钟伟虎。
<br/>　　“嗯，如果他让我们去打程大雷，就派人马过来吧。”宋伯康。
<br/>　　
<br/>(本章完)
