        而且结局的时候，女主没有跟男主在一起，而是跟明希哲所演的男三号在一起了。
<br/>
<br/>        对于这样的结局，本应该会被网友“吐槽”，谁知，观众全都拍手叫好。
<br/>
<br/>        因为女主最后结局和男三在一起的这个噱头，又吸引了一些好奇的观众，因此这部剧完结后也继续火了一年。
<br/>
<br/>        明希哲还因为这部剧获得了明星盛宴年度最具潜力的新人奖。
<br/>
<br/>        随后，他的团队为他选剧本，可谓是精挑细选，从电影到电视剧，他明希哲出品的必定都是精品。
<br/>
<br/>        而洛羽似乎就没他这么幸运，虽然刚出道时，在连星儿的帮助下拿了一部电影的男主角之位，也曾一夜爆红过。
<br/>
<br/>        但，似乎也就是所谓的“一夜”爆红，随后就又没落了一样。
<br/>
<br/>        学校的下课铃声一响，几乎是全校的女生，蜂拥似的直接往体育馆里跑。
<br/>
<br/>        因为大家听说，《酸甜初恋》这部剧的男女主角以及男二号，在体育馆里有粉丝互动会，只有两个小时的时间，可以给大家一个和自家爱豆拍合照要签名的机会。
<br/>
<br/>        所以，谁都不想错过这次机会。
<br/>
<br/>        连星儿其实无所谓，但是顾千凝则不一样，她太期待这次与洛羽的近距离接触。
<br/>
<br/>        然而，当她们赶到体育馆时才发现，队伍已经排到体育馆门外了。
<br/>
<br/>        不过，好在不是洛羽的队伍。
<br/>
<br/>        顾千凝很庆幸，连星儿却替洛羽感到心塞。
<br/>
<br/>        因为，那个队伍排到体育馆大门外的明星是明希哲。
<br/>
<br/>        顾千凝拉着连星儿默默地排在队伍后面，随着队伍缓慢地前行。
<br/>
<br/>        体育馆内可谓是人山人海，耳畔一片聒噪声。
<br/>
<br/>        连星儿和顾千凝站在体育馆球场边缘的台阶上，往下看就能看到最前面被围观的三个明星。
<br/>
<br/>        女主角和明希哲，她们不关心，至于洛羽……
<br/>
<br/>        “哎呀——你们别挤好不好！”就在这个时候，人群里忽然有女生尖叫着抱怨起来。
<br/>
<br/>        随后，连星儿和顾千凝就感觉到了背后有一股推力突然袭来。
<br/>
<br/>        顾千凝本能地往前一踩，结果踩了个空。
<br/>
<br/>        连星儿下意识地拉住顾千凝的手，背后随即又一阵推力顶了过来。
<br/>
<br/>        顾千凝抓着连星儿一起摔下了台阶。
<br/>
<br/>        下一秒，现场一片混乱。
<br/>
<br/>        摔下去后，连星儿本能地将顾千凝护在了身下，双脚的脚趾以及脚腕不停地被人踩踏着，痛得她硬生生地挤出了眼泪。
<br/>
<br/>        耳边传来哭喊的声音，头顶瞬间黑压压的一片。
<br/>
<br/>        不知过了多久，突然间，脚上那些被踩踏的疼痛感消失，连星儿只觉自己的身子腾空而起。
<br/>
<br/>        定睛一看后，明司寒的那张俊脸，闯入了她的眼帘。
<br/>
<br/>        而周遭被十个黑衣保镖围出一个安全圈。
<br/>
<br/>        明司寒将连星儿从地上抱起来后，在另外两个保镖的开道下直奔医务室。
<br/>
<br/>        顾千凝也被随同一起而来的张子晋拉了起来。
<br/>
<br/>        “你还好吗？”张子晋关问道。
<br/>
<br/>        顾千凝点了点头，一脸担忧地回答道：“我没事，星儿她……”
<br/>
<br/>        “那我陪你一起去医务室看她吧！”张子晋提议。
<br/>
<br/>        顾千凝连连点头：“好！”
<br/>
<br/>        医生说，顾千凝并无大碍，受伤比较严重的是连星儿。
<br/>
<br/>        不仅崴了脚，还被人给踩到脱臼。
<br/>
<br/>        所以，医生给连星儿的脚腕复位的时候，连星儿痛得咬了明司寒的手背一口。
<br/>
<br/>        就在这时，突然走进来两个保安，要求张子晋和顾千凝先行离开。
<br/>
<br/>        顾千凝和张子晋不得不照做。
<br/>
<br/>        医生给连星儿做完脱臼复位后，便开了药，给她包扎脚腕。
<br/>
<br/>        明司寒在一旁，手里拿着纸巾，一边替她擦眼泪，一边温柔地哄道：“乖，不哭了，哥哥我待会买糖给你吃。”
<br/>
<br/>        “……”连星儿虽然无语，但看到明司寒手背上那两排牙齿印，也哭笑不得。
<br/>
<br/>        医务室里的无关人员被“请出去”后，洛羽在经纪人林苏的陪同下走了进来。
<br/>
<br/>        “星儿，你有没有事？”洛羽大步上前，走到病床边后，完全当明司寒是空气，直接在床边坐下，并拉起了连星儿的手。
<br/>
<br/>        林苏走过来时，不禁皱了下眉头：“怎么还有一个人在这儿？”
<br/>
<br/>        “你们聊，我出去了。”明司寒冷冷地说道，压根就不把洛羽和林苏当回事，所以也就不会在乎林苏说的是什么话了。
<br/>
<br/>        当明司寒与林苏擦肩而过时，林苏莫名其妙地将这个男生和那个明希哲联系在了一起。
<br/>
<br/>        这只是她一个女人的直觉。
<br/>
<br/>        林苏不知道自己为什么会产生这种直觉，但她很快就回过了神来，将医务室里的窗帘一一拉上，然后对洛羽叮嘱道：“你有什么话快点跟星儿说，我先出去了。”
