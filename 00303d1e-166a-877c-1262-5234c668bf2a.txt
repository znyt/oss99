
<br/>　　<br/>　　    茅小草茫然的看着瞬间就被封上了的鬼门，呆愣了几秒钟之后，她突然就跟疯了一样，趴在洞口，用双手扣着岩壁：“王八蛋你骗我，你说你能出来的，你为什么要骗我啊·····”<br/>　　<br/>　　    几下之后，茅小草的两手就扣的血呼啦的了，指甲里全都夹着泥土和碎屑，十根手指全被抓破了。<br/>　　<br/>　　    旁边的陈三岁，沉默了半晌说道：“他明知道三个人不可能都出来的”<br/>　　<br/>　　    “滚！”茅小草眼圈通红的回头吼了一声，然后继续用手扣着岩壁。<br/>　　<br/>　　    鬼门外，出来就守着等王惊蛰出来的齐峥坤，常子敬和胡胡都皱着眉头看着已经被封死了的鬼门，胡胡说道：“他居然，没能出的来？”<br/>　　<br/>　　    齐峥坤冷笑着说道：“出不来就一辈子都留在阴曹地府里吧，鬼门一年才开一次，他在阴间肯定受不了被阴气给侵蚀一年，哼哼，就只能等着转世投胎了”<br/>　　<br/>　　    胡胡叹了口气，说道：“我是心疼他身上的连山和归藏啊······走吧”<br/>　　<br/>　　    王惊蛰没能从鬼门里走出来，按照常理他就只能留在阴曹地府了，这就相当于是将他给困在了一个没有任何方式能沟通逃脱的牢狱之内，除非等到明年鬼门再开的时候才能出来，但是阳间的人是没办法在阴间停留长达一年的时间的，阴曹地府中到处都充满着阴气，会无时无刻都洗刷着活人的生气，最终能将人给活活的耗死了，并且两大阴帅都已经过来了，他们断然不会轻易放王惊蛰离去的，肯定得将他捉拿回地府问罪，甚至于给镇压在炼狱里，所以王惊蛰想要再出来，基本没什么可能了。<br/>　　<br/>　　    胡胡他们走了，外面还剩下秋成子和姜冰啸还有陈小瑜，他们是为了茅小草的那盏引魂灯，秋成子刚要上前询问，陈小瑜忽然拦了他一下，摇了摇头。<br/>　　<br/>　　    “你看看她的这个状态，这时候你就算把满清十大酷刑全都用到她的身上，她也未必愿意交出来了，别白费力气了”<br/>　　<br/>　　    秋成子诧异的说道：“那我们就不要了？”<br/>　　<br/>　　    陈小瑜意味深长的说道：“再等等，人在灯就在，也不会跑了的，这次就没必要了”<br/>　　<br/>　　    人都走了以后，茅小草终于无助的堆坐在地上，泪眼婆娑迷离的呆呆的望着被封死的鬼门。<br/>　　<br/>　　    “真的就出不来了么？”茅小草喃喃的问道。<br/>　　<br/>　　    陈三岁看了眼离去的几人，忽然低头轻声说道：“其实，我觉得他就算出不来，可能留在阴曹地府也不会出什么大事”<br/>　　<br/>　　    茅小草茫然的抬起头问道：“为什么？”<br/>　　<br/>　　    陈三岁寻思着说道：“我们出来之前，都被阴帅的两把勾魂索给拘住了，那应该是阴差来阳间拘拿魂魄用的，不管是生人还是死人，只要有魂的就脱不了勾魂索，但你应该也看见了，对王惊蛰却似乎无用”<br/>　　<br/>　　    “他的命，受了天谴”茅小草瞬间通透了。<br/>　　<br/>　　    陈三岁愕然问道：“什么？”<br/>　　<br/>　　    “他被天谴了，活不过三十岁·······”<br/>　　<br/>　　    陈三岁无言了半天，才点头说道：“难怪了，原来是受了天谴？那他的魂也就只有老天能拘，哪怕是地府阴帅都不行了，所以那勾魂索拿他就不行了”<br/>　　<br/>　　    被提醒了一句，茅小草豁然起身，说道：“就算魂没不了，我也不能让他在阴间白白浪费一年的时间，三十岁前，少一年就少了一年的机会啊”<br/>　　<br/>　　    陈三岁说道：“先回去一趟，回头再商量，天要亮了······”<br/>　　<br/>　　    阴曹地府内，几个没出去的人被阴兵缉拿，王惊蛰扔掉手中的拘魂索毫无畏惧的望着鬼王和夜游：“你们不觉得尴尬么？阴曹地府不允许生人驻留，否则就是违反阴间条例，但你们却偏偏拘不出我的三魂七魄来，难不成你还能把我这人给押到炼狱，那不就是坏了规矩么？我就问你尴尬不？”<br/>　　<br/>　　    鬼王和夜游都直皱眉，明显王惊蛰的状况有点另类了，好像瞬间就成为了一个烫手的山芋，抓他吧，偏偏魂魄拘不出来，不抓吧，又不合乎规矩，这一下好像有点进退两难了。<br/>　　<br/>　　    “要不，咱就睁一只眼闭一只眼，就这么算了？你们就当没看见我，然后找个门缝给我送出去，是不就皆大欢喜了？”<br/>　　<br/>　　    鬼王阴着脸说道：“你想多了，我们拿你没办法，不见得阎王和判官拿你也不行，实在没辙还有十殿阎罗在，我还就不信了，整个地府都拘不出去你的三魂七魄？”<br/>　　<br/>　　    “死心眼！”王惊蛰撇嘴说道。<br/>　　<br/>　　    “给他带走，从奈何桥上走······”<br/>　　<br/>　　    忘川河上只有一座桥，那就是传说中的奈何桥，桥的那一头有个老婆婆就是孟婆，手里端着一碗忘魂汤，喝了孟婆汤过了奈何桥，就彻底的告别阳世进入阴曹地府，然后等待判官断命。<br/>　　<br/>　　    你生前若是平淡无常，既不是大奸大恶之人，也不是什么几世大善人的话，那就简单了，直接送往六道轮回转世投胎。<br/>　　<br/>　　    如果你生前作恶颇多，就得看判官怎么断了，小罪入忘川河禁锢几年，然后再投胎转世，如果作恶多端的话就会被送往十八层地狱中的炼狱镇压，待到身上罪孽消除然后再视情况而定，是进畜生道，还是饿鬼道，总之不可能再世为人了。<br/>　　<br/>　　    如果要是一世行善，或者世世行善的话，那就妥妥的了，你投胎的话必然是个大富大贵的人家。<br/>　　<br/>　　    也有人可能会问了，人活一辈子几十年，不可能没做过一件错事或者坏事吧？比如小的时候小偷小摸了，骂人脏话，泼谁家门口脏水了，长大了摸人屁股，黄赌毒三者中后一个犯的可能少，但前两者犯过的却很多，难不成这样的人最后都得入忘川河或者下地狱不成，谁也不敢保证自己一辈子没做过坏事吧？<br/>　　<br/>　　    其实呢则不然，曾经有人说过天下无真正的圣人，那意思是不管任何人都做过错事坏事，但只要弥补上，就能消除掉你曾经的孽障了。<br/>　　<br/>　　    所以，那些有钱人总是想方设法的搞慈善，捐道观盖庙宇，拿香火钱什么的，就是以此来抵消自己曾经犯下的过错，以求死前能积累阴德。
