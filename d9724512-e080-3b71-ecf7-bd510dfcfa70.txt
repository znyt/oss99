　　徐福道：“人家的目的，不是证道，是弄清自己心中的疑惑而已，证道对他来说，并没有什么吸引力。”
<br/>　　建文帝皱着眉头，喃喃道：“真的让人难以理解。”
<br/>　　道风道：“他不证道，不代表别人可以随便欺负他。”
<br/>　　徐福颔首笑道：“正是此理。他毕竟得到于人间，一向视自己为人间道尊，法术公会想要掌控人间，必然会引起他的反感，轩辕氏那帮人是很强大，不过青牛不是好惹的，贫道无意之中把他卷入民国，现在想来，虽然出人意料，却也不见得就是坏事，冥冥之中，或许注定如此。”
<br/>　　道风微微点头，说道：“你能不能把我也送去民国？”
<br/>　　“道风子，你在这边，尚有许多事要做，却是不能过去，否则也会阻挡叶少阳应劫。我故意不让他与那些好友相见，也正是为了这个原因，一切顺其自然，你我，却是做你我该做的事。轩辕氏那些家伙，似乎有很强的野心……”
<br/>　　说到这里，徐福盯着道风，认真打量了一会，说道：“道风子，你已斩去血肉、灵身，只差斩了元神，就能证得混元大道，你却是有眉目了没有？”
<br/>　　道风叹道：“血肉好斩，斩却灵身，却是有机缘做到，元神斩却，万念俱灭，如何还能重生？”
<br/>　　徐福郑重地点了点头，道：“也是机缘未到，天劫已至，还望道风子多加留意，早日斩却三尸，飞升无极。”
<br/>　　说完，朝左右看了一会，道：“阴司那边，能通过生死簿追踪到我的踪迹，我却是不能在这里久留，你我分头行动，以期将来合并一处……”
<br/>　　说完将封印着白起元神的铜镜收了起来，对道风拱了拱手要走。
<br/>　　“等一下，最后一句，”道风说道，望着徐福，冷冷说道：“虽说你有你的目的，但如果少阳在民国有什么三长两短，我绝不会放过你，不过跟我说什么大义，一屋不扫何以扫天下，连身边人都保护不了，更遑论天地人间！”
<br/>　　徐福捋着胡须，轻轻笑了笑，纵身遁走。
<br/>　　道风深吸了一口气，望着远处，喃喃说道：“事情越来越棘手了。”
<br/>　　建文帝上前一步，说道：“道风，有些事情，并不是一己之力能扛得住的，就算是你，也是一样。”
<br/>　　道风点点头，道：“我亦知之。”
<br/>　　随即说道：“你回风之谷，告诉大家，我要去人间一趟，不知什么时候回来，军务大事，你来掌控，但大小事宜，须同宫梓决断。同时放出我去人间的风声。”
<br/>　　建文帝皱眉道：“这是何意？”
<br/>　　“我私闯昭狱，救出一等鬼犯，罪无可赦，这一次，阴司一定不会善罢甘休，我若去人间，他们便不会为难风之谷，否则万一阴司发兵，与太阴山形成夹攻之势，十分危险。”
<br/>　　建文帝沉吟道：“阴司虽然盛怒，但是还要靠我们风之谷来牵制太阴山，一定会顾全大局，不大可能为了这件开战。”
<br/>　　道风道：“话虽如此，不过风之谷是我一手打造的基业，在最后一战之前，我不想冒任何风险……你去吧。”
<br/>　　建文帝也不说什么，对道风拱了拱手，朝风之谷的方向飞去了。
<br/>　　道风目视前方，说道：“还不出来。”
<br/>　　说完这句话不久，一道人影，从丛林深处缓缓走来，正是玉面锦衣的阿修罗王子南宫影。
<br/>　　“被你发现了。”南宫影耸了耸肩，“看来我这身法还是不够隐蔽。”
<br/>　　“不光我发现了，徐福也发现了，我对他暗中使了眼色，告诉他是自己人，他才没有介意。”
<br/>　　南宫影哈哈笑道：“我身上血滴子的气味太重，就知道瞒不过你。”
<br/>　　道风说道：“许久不见，你怎么会来这里？”
<br/>　　“我是追着你来的。”南宫影无奈苦笑，“道风，你可真是不要命了，去昭狱抢人，还惊动了地狱犼……”南宫影摸出一把折扇，冲道风点了点，“你不知道吧，我一直都在阴司石湖底修炼，离昭狱甚近，我当时听说有人闯了昭狱，大吃一惊，还在想谁有这么大的胆子，出来一看，居然是你……”
<br/>　　“你本意是去拦住劫狱之人的？”
<br/>　　“是。我阿修罗族皈依佛门，昭狱是佛门地盘，按说，我是该尽力的，道风，若我当时真的出手，联合地狱犼和两位尊者，你决计难以逃脱。”
<br/>　　道风有些不耐烦地看着他，“说了这么多，你不还是没动手。”
<br/>　　“当日败在你手上，我苦修至今，就是想要战胜你，不过跟别人一起捉拿你，却是没意思，而且你一旦被擒住，将来我想挑战你都难了，我为何要这么做？”
<br/>　　想打败自己，还不许别人抓住自己，这逻辑，道风也有点醉，说道：“你若是找我来说这些，你说完了，我也走了。”
<br/>　　南宫影道：“急什么，我方才偷听你们说话，对于天劫之类的，我可是没兴趣，不过我听说法术公会想要掌控人间法术界，这就不得了了，听说法术公会中有不少强者，我却是想会一会这帮家伙，我与你同行，如何？”
<br/>　　道风心中一动，道：“你怎么知道我要去探听法术公会的事？”
<br/>　　“不然你去人间还能有什么事。你我联手，我倒要看看，这些轩辕氏的子孙们，到底有什么能耐，敢于觊觎人间，我阿修罗族执掌修罗道那些久，也从不敢有此打算。”南宫影愤愤不平地说道。
<br/>　　道风沉吟起来，自己现在为了不拖累别人，打算孤身一人去找法术公会的麻烦，有南宫影帮忙，倒是真的不错。
<br/>　　南宫影的实力，自然是不用说的，更为重要的是，他有十分强大的背景：阿修罗族的王子。虽然阿修罗族一向盘踞在无量界，从来不跟人间有什么来往，但法术公会不可能不掂量这个强大的灵族。
<br/>　　不管怎么样，自己绝对不会拖累到南宫影，让他来当个帮手，似乎可以……
<br/>　　道风往南宫影身前走了几步。
