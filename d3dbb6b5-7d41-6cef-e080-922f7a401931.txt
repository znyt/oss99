　　陈家四合院。
<br/>　　此时已经是快接近深夜凌晨了，涂小安带着海棠站在了这栋充满历史气息的宅院大门前。
<br/>　　只见明艳动人的海棠娇躯呆滞住了，复杂的看着这栋历经数百年的老宅，恍如隔世。
<br/>　　因为她是在这栋宅院中生活最久最久的生物，就连陈家人都没一个比她住的久。
<br/>　　宅院中充满了一只猫上蹿下跳的痕迹，海棠没想到，今日，自己既然是已一个人的样子回来的。
<br/>　　如梦似幻，独独站在大门前，她还不断的在怀疑，一切都太过不可思议。
<br/>　　涂小安看了看发呆的海棠，没有出声，让她做点心理准备。
<br/>　　拿出手机，打开微信，里面全是李清照这丫头的信息，不断的询问他在哪，做什么，然后还自卖自夸的说自己今天拯救了很多的同学。
<br/>　　几乎成为了学校的女英雄。
<br/>　　涂小安敷衍的回复了她一句，说自己已经离开了学校，找一个宾馆睡觉。
<br/>　　小丫头又一大丢信息的来自卖自夸，非常的兴奋。
<br/>　　恐怕今晚校园十几万群蛇，人人自危宛如噩梦，就她兴奋了，跟吃了蜜蜂屎一样。
<br/>　　放下手机，涂小安看着还在大门前发呆的海棠道：“进去吧，他之前因为你消失，一个人漫无目的的在大街上喊着你的名字找，确实不容易”
<br/>　　让瞎子找人，本来就是一件非常难为他的事情，况且他要找的“人”还是一只猫精假扮。
<br/>　　已他的方式，恐怕一辈子都别想找到。
<br/>　　也不知道是不是下足了决心，海棠推开了宅院的门，大门没有关，没有反锁。
<br/>　　宅院中，一颗大槐树屹立，陈高阳呆坐在树下，整个环境除了月光，就什么都没有了，光线也显得有点昏暗。
<br/>　　但是昏暗，对于一个瞎子来说，从来不是什么问题。
<br/>　　因为他的世界早就一片黑暗了。
<br/>　　陈高阳每一天都会坐在院中，大门不锁，就是期盼着自己的女朋友会回来。
<br/>　　他知道这种期盼几乎是不可能实现了，自己的女友将所有的东西都带走了，连一件衣服都没留下（其实本来什么就没有）。
<br/>　　她是下定决定离开自己这个死瞎子。
<br/>　　“也好，她那么善良，那么美丽，应该找一个对她好，般配的人，自己给不了她幸福的”
<br/>　　几乎是每一个夜晚，每一个深夜，陈高阳都如此这般的安慰自己。
<br/>　　或许只有这样想，他心里会好受点。
<br/>　　今晚，对他又是一个无眠之夜，黑暗笼罩中，他充满了绝望。
<br/>　　不知是不是陈高阳想通了，他准备做一件早就该做的事情。
<br/>　　四合院中除了一颗大槐树，还有一口古井。
<br/>　　在古代，一般的大户人家，都会在自家的大院挖一口井，从风水学上讲水池水塘有聚气之势，水井亦有同样的功效，另外自己家有口水井用水也很方便。
<br/>　　古人建筑房间，宅院的，都是非常的讲究，必须要按照风水玄学。
<br/>　　宅院两口井，一为地井，二为天井，地井自然是地下挖的水井，而天井就是房子与房子或者房子和围墙之间所围成的长方形的露天空地，以及旧式房屋为了采光而在房顶上开的小洞，都叫做天井。
<br/>　　阳光从天井中射下，光线照在地上如井水一般，所以称其为天井。
<br/>　　无论地井还是天井，风水上说，都有“聚气养气”的功效。
<br/>　　古语有云:“水不宜直流，水不宜出门下，皆主耗散”
<br/>　　简单点，就是天井换风水，通天气，地井涌水源，凝财气。
<br/>　　这四合院数百年了，天地两口井，而此时心灰意冷的陈高阳，朝着那口地井走去。
<br/>　　那口地井已经早就荒废不用，由于现代污染严重，地下水早就变的不在干净。
<br/>　　所以早就用一块大理石将井口封闭。
<br/>　　再者陈高阳从小眼瞎，也是防止什么意外发生。
<br/>　　陈高阳步履蹒跚的走到了地井的旁边，将手中的拐杖丢弃，然后摸了摸，摸到大理石，奋力的掀开了。
<br/>　　瞬间，地井迎面涌上一股阴冷之气，让人有点浑身颤栗。
<br/>　　“这便是我最好的葬身之地了”
<br/>　　陈高阳凄凉的一笑，墨镜上英俊的眉有凝着化不开的悲凉。
<br/>　　这井口处不断的涌现一股阴冷之气，仿佛在像陈高阳挥手，召唤着他快点跳下去。
<br/>　　民间有传闻，人要是投井自杀，那么灵魂便会一直被困在井内，无法投胎转世。
<br/>　　但陈高阳不在乎，他出生就是一只井底之蛙，不对，他甚至不如一只井底之蛙，因为井底的蛙还能看到头顶一角的天。
<br/>　　而他，出生便什么都不曾看过。
<br/>　　当初奶奶去死，陈高阳一度想去死，要不是海棠的出现，他估计早就已经投井了。
<br/>　　原本以为老天可怜他，赐了他一位天使，改变他的命运，如此看来，自己的命运根本没有改变过。
<br/>　　这时，陈高阳重重的吸了一口凉气，站在了井口处，只要微微一跳，他就解脱了。
<br/>　　“高阳哥哥...”
<br/>　　也不知道是临死前的幻觉，还是陈高阳太思念自己的女友，她隐约听到了一声熟悉的声音。
<br/>　　徒然，他摇了摇头，这就是幻觉，海棠不可能回来了，于是乎双膝稍稍一歪，不在留恋这个人世间，直接跳了下去。
<br/>　　“呀，高阳哥哥，不要...”
<br/>　　刚跳下，陈高阳甚至还没被井口完全吞噬，他猛然震惊。
<br/>　　这道声音很尖，很响亮，这不是幻觉呀。
<br/>　　可...晚了，他跳了，马上就要沉到井底。
<br/>　　“给我回来”
<br/>　　涂小安出现在宅院，刚好看到了陈高阳往井底跳下去的一幕，顿时一惊，眼眸立刻浑然一眯，一股无形的意识力激荡出来，空间瞬间波动涟漪起来。
<br/>　　陈高阳跳下，本该出现落井“噗通”的声音，但井中病没有发出任何声响，静悄悄。
<br/>　　海棠已经吓得花容失色，本在宅院门前发呆的她，就想了很多种跟陈高阳团聚的温馨画面。
<br/>　　但她没想到，看到的却是陈高阳跳井的惊悚一幕。
<br/>　　几个呼吸，海棠倏然冲到井口，也准备跟着跳上去救人，可诡异的事情发生了。
<br/>　　只见陈高阳飘飘然的从井底悬浮了出来。
<br/>　　几乎霎时间，海棠转身看了涂小安一眼，只见他跟她微微的点了点头。
<br/>　　陈高阳在海棠的身边落地了。
<br/>　　海棠瞬间梨花带雨，泪流满脸，一把将陈高阳死死抱住，泣声：“高阳哥哥，你为什么要自杀，你不要海棠了吗”
<br/>　　“海棠...真的是你，你回来了”
<br/>　　陈高阳颤抖着，眼泪也无声的流了下来，这眼泪就好像是从这口地井当中涌出来似的，带着冰凉。
<br/>　　“高阳哥哥，对不起，对不起”
<br/>　　海棠紧紧的抱着陈高阳，全身都在轻微地颤动，眼泪宣泄而下，刚才看到他跳井的那一秒，海棠甚至觉得如度过了整个春夏秋冬一样。
<br/>　　漫长而可怕。
