　　“拍卖会？”
<br/>　　楚风眠眼前一亮，他现在拥有极为庞大的财富，这种拍卖会，他当然要去。
<br/>　　尤其是这拍卖会上，更是有着从如今远古洞府中产出的无数诸天奇物。
<br/>　　楚风眠大肆购买一番，正是足以提升他的实力。
<br/>　　“不错，这次的拍卖会，我们天商会也会前去，楚公子若是不介意的话，可以跟老夫一同前去。”
<br/>　　李长老看着楚风眠开口道。
<br/>　　“这次的拍卖会名额已经早早定下了，没有空余的名额了，不过以我们天商会的地位，带楚公子进入其中，倒也不难。”
<br/>　　楚风眠跟李长老的交易，给李长老也带来了极大的好处。
<br/>　　现在听到楚风眠打算去拍卖会，李长老直接便是邀请同行。
<br/>　　“那就多谢李长老了。。”
<br/>　　楚风眠点了点头道。
<br/>　　天商会在混元大陆中，毕竟是地头蛇，若是跟李长老一同前去，也足以减少许多麻烦。
<br/>　　“现在距离晚上已经不久了，我们先去吧。”
<br/>　　李长老也不废话，直接化为一道遁光离去。
<br/>　　“这次的拍卖会，是由如今混元大陆第一大商会，通天阁举办的，不过实际上，其实是由我们混元大陆中无数商会，一同联合举办的。”
<br/>　　一边走着，一边李长老业向着楚风眠介绍到。
<br/>　　“这拍卖会，每三年一度，规模虽然大，但是往届的名额，还都会有所空余的。”
<br/>　　“不过这一届，所有的名额却是早早订完了，因为六大古洞府开启，无数奇物出现，许许多多往日里不在虚空海中的大人物，都前来了，甚至是一些西域，中域的大人物，都前来了。”
<br/>　　听到这里，楚风眠暗自点了点头。
<br/>　　远古洞府开启的消息，的确是足以引动整个九域为之疯狂。
<br/>　　尤其像是颜古龙帝这样传说中的大人物，他的洞府，只怕是足以引来超越生死境的恐怖强者。
<br/>　　也幸好楚风眠是提前得知了消息，第一时间赶了过去，才可以收取那么多的宝物。
<br/>　　不然的话，真的遇到那种超越生死境的老古董，楚风眠根本没有跟他任何争夺的可能。
<br/>　　“楚公子，应该也在这古洞府中，得到了不小的好处吧。”
<br/>　　李长老悄然之间开口道，像是闲聊，隐约之间倒是有着几分试探之意。
<br/>　　“不错，在这古洞府中，我的确得到了一笔不小的财富。”
<br/>　　楚风眠毫不掩饰，直接开口道，活脱脱像是一个暴发户的样子。
<br/>　　他身上拥有如此之多的财富，任何人都明白，楚风眠肯定是得到了大机遇。
<br/>　　既然如此，倒不如直接大大方方展露出来。
<br/>　　以他现在的实力，也没有多少人，足以对楚风眠造成生命威胁。
<br/>　　楚风眠跟李长老一边交谈这，一边便是来到了天商会的城池门前，刚刚踏入天商会，就在周围，七八道身影，一下子显现了出来。
<br/>　　这些人，将楚风眠直接堵在了天商会之外的门前。
<br/>　　这几道身影，皆是年轻武者，一个个器宇轩昂，每个人的天赋都极高。
<br/>　　这些人的实力，几乎都不在北邙学院地榜前十的人物之下。
<br/>　　“是天盟的人！”
<br/>　　“天盟的人，怎么会突然堵在天商会的门口。”
<br/>　　“你们看，那是欧阳烈，他身上的气息，怎么会如此薄弱，居然被人抽取了身上的精血，这是谁如此疯狂，敢抽取欧阳烈的精血。”
<br/>　　“你们看那是，洪狱！天盟的副盟主之一，这天盟的大人物，居然都出动了。”
<br/>　　看着天商会前出现的无数天盟弟子，现场出现了一阵骚乱。
<br/>　　如今如此之多的天盟的人，聚集在了这里，甚至就连天盟大人物，副盟主洪狱都出现了。
<br/>　　“洪狱？”
<br/>　　楚风眠一眼望去，便是看向这洪狱，这洪狱的一尊长相狰狞的男子，
<br/>　　洪狱身上的气息，比起之前的欧阳烈，还要强大几分。
<br/>　　尤其是楚风眠更是足以感觉的出，洪狱的实力，所表现的，只是极少的一部分。
<br/>　　他的隐藏极深，只怕是留有无数手段。
<br/>　　“小子，你终于出来了。”
<br/>　　欧阳烈看向楚风眠，从天商会中走了出来，脸上露出几分狰狞道。
<br/>　　“洪盟主，就是这小子做的。”
<br/>　　“恩？就是他打伤了你，还抽取了你身上的圣者精血？”
<br/>　　洪狱听到欧阳烈的话，一双眼睛，猛然看向楚风眠，死死的盯着楚风眠道。
<br/>　　“就是你，打伤了欧阳烈，抽取了他的圣者精血？”
<br/>　　这洪狱的双眼中，仿佛是蕴含由着无尽杀机，跟他对视，都足以令人感觉到心神中的恐怖。
<br/>　　若是稍弱一些的武者，仅仅是跟洪狱对视一眼，就会落荒而逃。
<br/>　　不过楚风眠却是毫不在乎，他的心智坚定，在强大的武者，都不会让让他恐惧。
<br/>　　“不错。”
<br/>　　楚风眠淡淡的吐出两个字。
<br/>　　他做了，就是做了，从不需要任何隐瞒。
<br/>　　“你可知道，欧阳烈是我们天盟的人？”
<br/>　　洪狱听到楚风眠的话后，杀机涌现道。
<br/>　　“那你还敢抽取他的圣者精血？”
<br/>　　“现在你对着欧阳烈跪下道歉忏悔，在自废修为，将身上的空戒中的一切财富都交出来，我可以给你留一个全尸！”
<br/>　　听到洪狱的话，楚风眠心中都有些无语。
<br/>　　这洪狱的话，简直是嚣张到极点，让楚风眠跪下忏悔，自废修为交出一切，留个全尸？
<br/>　　别说到底有没有人可以让楚风眠做到这一点，就算是有人可以做到，也不是这洪狱。
<br/>　　“天盟的人，都是白痴吗？”
<br/>　　楚风眠冷冷一笑道。
<br/>　　“你以为你是谁？敢来这里命令我？也不怕说话闪了舌头。”
<br/>　　“现在我给你一个机会，你们全部跪下，将空戒中的所有财富都交出来，我可以不抽取你们的圣者精血，就算是你们冒犯我的代价如何？”
<br/>　　楚风眠的话一出，全场哗然。
<br/>　　天盟，可是混元大陆中最为庞大的势力之一。
<br/>　　而洪狱，可是天盟中的大人物，副盟主。
<br/>　　在混元大陆，向来都是天盟欺压别人，还从来没有被人欺压过的时候。
<br/>　　“找死？”
<br/>　　洪狱脸色骤然一变，眼神中露出熊熊怒火，猛然之间，一掌突然打出。
