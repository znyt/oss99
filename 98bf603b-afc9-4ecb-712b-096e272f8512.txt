　　楚风眠看着眼前的一幕，也都是愣住了。
<br/>　　谁也不可能想得到，在这颜古龙帝洞府的最深处，这片空间的最中心，居然是一片血海，血的海洋。
<br/>　　这血海，庞大无比，楚风眠一眼望去，都看不见尽头。
<br/>　　在这颜古龙帝洞府中，居然还隐藏这样一座庞大无比的血海。
<br/>　　这还不是令楚风眠最为震惊的，最令楚风眠震惊的一点是，眼前这血海中，楚风眠只能够感觉到一种灵力的存在。
<br/>　　这也就意味着，眼前这血海，居然就是由一尊强者的血，凝聚出来的。
<br/>　　“这是什么一尊恐怖存在，他的血，才可以化为这样一片血海。”
<br/>　　楚风眠前世也度过无数古籍，但是他无论如何也想不到，到底会是谁，才可以将鲜血，化为眼前这无边无际的血海。
<br/>　　“莫非这血海，是颜古龙帝的血凝聚而成的？”
<br/>　　能够做到这一切的，在加上祖龙至尊剑的指引，楚风眠突然想到一点。
<br/>　　只怕也只有曾经位于九域巅峰的恐怖存在，颜古龙帝，死后，他的鲜血流出，才会化为如此庞大的血海吧。
<br/>　　这颜古龙帝洞府中最大的宝藏，居然是以他的鲜血，化为的这一摊血海。
<br/>　　“这一滩血海中，不知道蕴含了多少精粹灵力！什么天地奇物，那龙血晶石，不过就是一些真龙血液凝聚的，这一滩血海，抵得上一百万，一千万个龙血晶石的力量！”
<br/>　　看着眼前的这摊血海，楚风眠简直就是看到了一件巨大无比的宝藏，不，这简直就是宝库，其中满是无穷无尽的宝藏，展现在的楚风眠的面前。
<br/>　　“走！”
<br/>　　楚风眠当机立断，瞬间清醒过来，怪不得战龙决会如此渴望。
<br/>　　在这一滩血海之中，楚风眠在其中大肆吞噬，他的灵力会达到一种多么恐怖的程度，他自己都无法想象。
<br/>　　楚风眠一瞬间钻入到了这血海之中，顿时血海中无数血液，都直接向着楚风眠涌了过来。
<br/>　　楚风眠也不客气，张开大口，疯狂的吞噬，无穷无尽的血液都被楚风眠吞噬。
<br/>　　这里的每一滴血液，都要比起所谓的圣者精血，甚至是比起生死境强者的精血，要精粹的多。
<br/>　　其中蕴含的龙力，更是最为适合楚风眠的力量。
<br/>　　这一摊血海，让一尊武者前来，可能他都根本无法吞噬，因为其中的龙力实在是太过庞大。
<br/>　　寻常武者吞噬其中的血液，只会受到龙力的影响，化为半龙半人的怪物。
<br/>　　但是楚风眠不同，楚风眠修炼战龙决，这些龙力，简直就是楚风眠最大的补品。
<br/>　　在楚风眠的身躯中，那远古战龙虚影都直接出现在了楚风眠的身后，疯狂的张开口，吞噬这血海中的无穷灵力。
<br/>　　如同一头饥饿的猛虎，冲入到了羊群一般，开始疯狂的吞噬这其中的无数血液。
<br/>　　一滴滴远古战龙精血，在楚风眠的身躯中疯狂的凝聚出来。
<br/>　　一百滴，两百滴，三百滴……
<br/>　　几乎是每时每刻，都在疯狂的凝聚这。
<br/>　　楚风眠也几乎动用了全力，疯狂的吞噬这其中的无数血液，将其中的灵力，尽数吞噬。
<br/>　　“轰！”
<br/>　　楚风眠身上的力量，很快达到临界点，突破了，一跃之间达到了神力境七重的地步。
<br/>　　并且楚风眠身上的灵力，还在节节攀升。
<br/>　　“好！战龙决只要没有遇到瓶颈，只要是灵力足够，就可以肆无忌惮的突破，这血海，简直就是我的天堂，吞噬！统统吞噬！”
<br/>　　楚风眠爆发出全力，直接化为了一条远古战龙，大口一张开，无数血液都被他直接吞噬炼化。
<br/>　　这其中的无数血液中，蕴含的灵力，简直是世间最为精粹的灵力，简直足以是跟皇品灵石中的灵力所媲美。
<br/>　　炼化起来，简单到了极点，根本没有任何需要化解其中的杂质，就可以全部转变为楚风眠的灵力。
<br/>　　“轰隆！”
<br/>　　就在这血海空中，一声巨响突然响起，那不死魔尊，终于也是追了过来。
<br/>　　不死魔尊一眼便是看见，楚风眠深处与这血海之中。
<br/>　　这血海中虽然蕴含无穷无尽的灵力，但是却也蕴含这一种独特的龙力，不死魔尊没有龙族血脉，根本无法靠近着血海。
<br/>　　但是这也难不倒不死魔尊这位地魔门的大人物。
<br/>　　他猛然出手，一把向着血海中的楚风眠抓了过来。
<br/>　　他的这一抓，有着移山填海之力，便是要将楚风眠，从这无尽血海中，直接抓出去。
<br/>　　“该死！这老魔头又来捣乱！”
<br/>　　楚风眠心中简直气愤到了极点，在这个关头，不死魔尊居然又再次追了上来了。
<br/>　　他现在正在凝聚远古战龙精血，提升实力，如今被人打扰，简直是气愤之极。
<br/>　　但是楚风眠心中也清楚，他的实力，还无法跟这不死魔尊抗衡，就算是一个重伤的不是魔尊，也是恐怖至极。
<br/>　　除非是楚风眠的实力，再进一步，现在楚风眠所需要的，便是拖延时间。
<br/>　　远古战龙虚影，已经是在血海中，源源不断的吞噬这其中的灵力，只要是在吞噬一会，楚风眠的境界，就可以再次突破。
<br/>　　“丰饶碑！”
<br/>　　楚风眠一声呼唤出丰饶碑，直接打了过去，跟不死魔尊的力量抗衡。
<br/>　　同时楚风眠，也是在源源不断的吞噬这血海中的无数灵力。
<br/>　　吞噬越多的灵力，楚风眠的实力也就越强，只要在拖延一阵，让楚风眠实力再进一步。
<br/>　　就算是面对这不死魔尊，楚风眠也不是没有一战之力。
<br/>　　“该死，这小子拥有龙族血脉，在源源不断的吞噬这血海中的力量，在让他吞噬，我都不是他的对手了。”
<br/>　　不死魔尊看着楚风眠抵挡，也是明白，楚风眠是在拖延时间。
<br/>　　楚风眠如今在源源不断的吞噬这血海中的力量，力量飞速暴涨，再让楚风眠实力暴涨，也许他都不再是楚风眠的对手。
<br/>　　“不行，拼了，今日我就算是燃烧本源，耗费寿命，也要斩杀了此子！”
<br/>　　不死魔尊狠狠的大吼一声，眼神中都露出了几分疯狂之色。
<br/>　　只见他的口中，足足是吐出了一大口精血，猛然燃烧起来。
