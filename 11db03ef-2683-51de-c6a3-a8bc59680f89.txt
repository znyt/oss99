　　叶尘并不知道，就在他修炼的这短短几个小时之内。
<br/>　　整个云丹城，因为天藏阁要拍卖完美品阶化婴丹的消息，已经彻底沸腾起来！
<br/>　　不但那些正处于金丹境巅峰的修真者，个个摩拳擦掌，就连云丹城各大势力，也破天荒的全部出动了。
<br/>　　叶尘所在的三楼包厢，是天藏阁拍卖场中，最为最贵的包厢之一，可以直接俯视整个场中，而且包厢周围，又法阵阻隔，外面的人却看不到里面的情景。
<br/>　　除了叶尘所在的这个包厢之外，还有几个类似的包厢，叶尘神念释放出了，可以隐隐察觉到，那几个包厢之中，有着几道连他都不敢轻视的气息。
<br/>　　“看来这些包厢里面的，应该就是云丹城的一些巨头大佬了……”
<br/>　　叶尘暗暗嘀咕了一句，便不再关注，转而将目光投向下方的场中。
<br/>　　此时，拍卖会在一名长袖老者的主持下，已经有条不紊的进行着。
<br/>　　各种圣器法宝，珍贵药材，乃至功法秘籍，轮番登场。
<br/>　　可惜场中的观众，似乎对这些东西，都已经有些兴致怏怏，主持人报出价格之后，很少有人跟拍，甚至有不少拍品，最终以流拍而告终。
<br/>　　终于，
<br/>　　就在叶尘都有些觉得无聊的时候，主持人的声音陡然拔高了几分，朗声道：
<br/>　　“诸位，接下来的这件拍品，相信很多朋友都是为此而来，老朽也就不绕弯子了！”
<br/>　　说到这里，主持人冲着后台一挥手，
<br/>　　一名打扮性感的女侍者，捧着一个玉盘，从后台缓步走了上来。
<br/>　　来到那主持人跟前，后者缓缓掀开玉盘上面的盖子，
<br/>　　哗！
<br/>　　所有人全都瞪大了双眼，满怀期待的望了过去。
<br/>　　只见那玉盘之中，赫然是三颗有金灿灿的灵丹，尤其是中间的那一颗，光芒最为璀璨，而且周身还有透着一股神秘的气息！
<br/>　　呼~~
<br/>　　一股浓郁的丹香，瞬间传遍整个天藏阁的每个角落。
<br/>　　“好香啊！”
<br/>　　“果然是宝丹，单单是闻上一口，都觉得神清气爽呢！”
<br/>　　“好宝贝！”
<br/>　　……
<br/>　　整个拍卖场，瞬间便炸开了锅。
<br/>　　甚至有些人，已经迫不及待的催促主持人，快点开始拍卖。
<br/>　　负责主持的那名老者微微一笑，双手朝四周虚按，那些躁动的人群，这才重新安静下来，
<br/>　　“诸位请看，眼前这三颗灵丹，皆为化婴丹，其中两颗为精品品阶，中间的一颗，乃是完美品阶的宝丹！”
<br/>　　轰！
<br/>　　听到这话，众人再次躁动起来。
<br/>　　主持人只好又重新将躁动的人群安抚下去，这才继续又道：
<br/>　　“既然大家如此热情，小老儿也就不卖关子了，按照雇主的要求，这三颗丹药，要放在一起拍卖，起拍价为一百万上品灵石！”
<br/>　　主持人这话一出，之前还躁乱的拍卖场，瞬间便安静了许多。
<br/>　　对于大多数人而言，一百万上品灵石，可不是一个小数目，更何况这还只是起拍价，待会被抬到多高，恐怕谁也说不了。
<br/>　　或许，大多人都是这么想的，因而过了好一会，竟然无人竞拍。
<br/>　　看到这一幕，不但那位主持人大为着急，就连在三楼包厢，密切注视着的叶尘，也有些愕然。
<br/>　　“难道是因为价格太高，把这些人给吓着了？早知道就把这三颗丹药分开拍卖了……”
<br/>　　就在叶尘有些郁闷之时，
<br/>　　“我出一百五十万！”
<br/>　　随着一道浑厚的声音，陡然在拍卖场中响起，犹如吹起了战斗的号角一般，很快便又有人开始纷纷响应，
<br/>　　“我出两百万！”
<br/>　　“两百五十万！”
<br/>　　“两百八十万！”
<br/>　　……
<br/>　　一转眼的功夫，价格就突破了四百万，直逼五百万而去。
<br/>　　坐在三楼包厢里的叶尘，顿时也暗暗松了一口长气。
<br/>　　很显然，云丹城作为炼丹师的聚集地，有钱人还是挺多。
<br/>　　当然，即便是再有钱，也会出现一个承受极限。
<br/>　　终于，
<br/>　　价格在突破到五百万之后，很多竞拍者，已经面如土色，无力在继续争夺下去。
<br/>　　最终，来自云丹城某一个大家族的老者，拍出六百二十万的高价之后，便无人再继续跟拍了。
<br/>　　“诸位，还有没有人竞拍了？如果没有的话，那老夫可就要落锤了！”
<br/>　　“六百二十万一次！”
<br/>　　“六百二十万两次！”
<br/>　　……
<br/>　　就在主持人准备落下巨锤的时候。
<br/>　　“七百万！”
<br/>　　三楼某个包厢之中，传出了一道冰冷的声音，
<br/>　　而且就在这道声音响起的同时，那处包厢的法阵也被人撤下，而在包厢的看台之上，出现了一名约莫三十岁上下的青年男子。
<br/>　　就连叶尘也不由得扭头一看，
<br/>　　“竟然是他！”
<br/>　　原来，这个突然半路杀出的竞拍者，居然就是叶尘在考核大会见过的，那位屠苏公子，屠苏元让！
<br/>　　“没想到连屠苏公子也来了！”
<br/>　　整个拍卖场，因为屠苏元让的出现，顿时再次躁动起来。
<br/>　　“诸位，这三颗化婴丹，我屠苏家要了，你们没有意见吧？”
<br/>　　屠苏元让扫视了一眼场下的众人，缓缓开口道，语气不容置疑。
<br/>　　听到这话，之前那位，眼看就要拍卖得手的老者，也脸色大变，不敢再继续出言竞拍。
<br/>　　屠苏家乃是云丹城五大家族之首，无论是拼实力，还是拼财力，他哪里是人家的对手？
<br/>　　然而，
<br/>　　就在场中众人一阵默然，都以为肯定没人再敢继续竞拍的时候，忽然又有一道冰冷的女子声音响起，
<br/>　　“我出八百万！！”
<br/>　　众人循着声音望去，
<br/>　　只见，同样是三楼贵宾包厢，开口竞价之人，是一位身穿素衣的年轻女子，
<br/>　　看到此人，整个拍卖中这才一阵躁动，
<br/>　　“没想到蝉衣小姐也来了！”
<br/>　　原来，这个和屠苏元让竞拍之人，赫然正是冷蝉衣。
<br/>　　屠苏元让死死盯着冷蝉衣片刻，眼眸发寒，
<br/>　　“冷蝉衣！这宝丹，可是我屠苏家先看上的，你敢和我屠苏家抢东西？”
<br/>　　冷蝉衣冷冷一笑，
<br/>　　“屠苏师兄，此言差矣！大家都是按规矩竞拍，价高者得，何来‘抢’字之说？你若是想要，尽管加价就是了！”
