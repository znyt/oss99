　　因为分出的神魂已经和金刚猿战魂融合，想要再和烈焰石战魂融合，困难成倍增加。
<br/>　　不过，苏莫也不着急，努力的控制着那缕神魂，再和烈焰石战魂二次融合。
<br/>　　这个过程无比的缓慢，神魂和烈焰石战魂即不排斥，也不相融。
<br/>　　但随着时间的流逝，两者逐渐的有了相融的趋势。
<br/>　　这让苏莫松了口气，只要能融合就好，耗费点时间不算什么！
<br/>　　时间匆匆而过，一日又一日。
<br/>　　转眼间便过去了一个月，一个月的时间，神魂终于和烈焰石战魂融合成功。
<br/>　　因为这两种战魂，本就已经是战魂，所以也不需要什么奥义了！
<br/>　　至此，这缕神魂完全与两种战魂相融，成为了一种特殊的双生战魂。
<br/>　　而后，在苏莫的控制之下，两种战魂进入了巫尘的体内，很快便主导了巫尘的身体。
<br/>　　随即，面前的巫尘向苏莫微微一笑，便运转了混天魔功，开始了修炼。
<br/>　　现在的巫尘，是苏莫的分身，神魂和苏莫的神魂彼此相连，心意相通。
<br/>　　这是用特殊的方法分化的神魂，虽然和苏莫的神魂彻底分开了，但却是能保持着联系。
<br/>　　换言之，就算苏莫死了，分身还在，他依旧存活。
<br/>　　“也不算难嘛！”苏莫面上露出了一丝笑容，摸了摸额头上的汗水，连续一个月的融合，让他心神颇为疲累。
<br/>　　望着沉浸在修炼之中的分身，苏莫心中突然冒出了一个大胆的想法，这个想法一出现，让他自己都心神一震。
<br/>　　他可以为分身造就战魂，能不能为别人造就战魂？
<br/>　　若是可以的话，他就能将苍穹神宫的人，全部变成绝世天才。
<br/>　　比如他的父亲苏洪，武魂只是人级武魂，实在是低的可怜。
<br/>　　就算有苍穹神宫大量的功法、资源，现在的修为依旧很低，才真罡境的修为。
<br/>　　他若是能让自己的父亲，拥有双生战魂，天赋则瞬间暴增无数倍。
<br/>　　这个想法，让得苏莫心绪起伏，不需要多，他只需将苍穹神宫万分之一的人，变成双生战魂的天才，对于苍穹神宫来说，简直是有无穷的好处。
<br/>　　只需数十年，苍穹神宫就能壮大无数倍，雄霸一方，而不是像现在一般，窝屈在一个小镇之上。
<br/>　　还有那些他曾经的朋友，牛小虎、洛千帆、天辰、等等，这些人天赋太低了，武道之路根本走不远。
<br/>　　比如牛小虎，以对方自身的天赋，就算有无尽的资源，一辈子都不可能达到武王之境。
<br/>　　再比如洛千帆，地级武魂，将来达到武王之境，已经是极限了。
<br/>　　但是，若是有了双生战魂，这些人将来达到武帝之境、武尊之境，轻而易举。
<br/>　　苏莫激动了，甚至现在就有离开巫族，返回古周星的冲动。
<br/>　　不过，他还是抑制住了心中的冲动，先不说这种想法能否实现，他这次来到巫族的目标，还没有完成呢！岂能半途而废。
<br/>　　不搞清楚青璇的情况，他无论如何也不能走。
<br/>　　少倾之后，苏莫不在多想，压下心中的思绪，也开始了修炼。
<br/>　　他本体目前的修为，才武帝境二重的修为，还不到二重的巅峰，必须要加快速度了。
<br/>　　修炼室内的阵法已经被激活，苏莫和分身包裹在较为浓郁的灵气之中，静静的修炼。
<br/>　　分身的身上魔气滚滚，威势骇人，虽然是修炼了圣级上品功法，修为也比苏莫要高一重，但并不比苏莫身上的气势强大。
<br/>　　五枚神丹一催动，苏莫体内玄力滚滚，如同江河咆哮，浩荡无比。
<br/>　　时间在修炼中飞逝，一日又一日，一月又一月。
<br/>　　眨眼间，便过去了十个月的时间。
<br/>　　分身的混天魔功越发的精深，威力越发的强大，十个月的时间，威力比以前强大了两倍。
<br/>　　而苏莫由于长时间的闭关，又不缺灵气，修为非常顺利的达到了武帝境三重的境界。
<br/>　　并且，因为是长时间闭关提升，他的修为较为稳固，根基也是比较扎实。
<br/>　　这一日，苏莫和分身同时停止了修炼，而后一同进入了玉镯空间之中。
<br/>　　距离巫族少司选拔，只有不足半个月的时间了，苏莫要趁着这半个月的时间，将分身的实力提升到最强大的地步。
<br/>　　巫族少司选拔非同小可，苏莫不敢有丝毫大意，因为他会遇到天冥星巫族支脉最顶级的天才，可不仅仅是黑耀城的巫族天才这么简单。
<br/>　　不说其它八座卫城，中央的天冥圣城之中，才是巫族天才的集中地，各种强大的天才数不胜数，他必须要将战力提升到极致。
<br/>　　因为这一次，他只能成功，不能失败！
<br/>　　混天魔功有很多强大的手段，苏莫还没来的及修炼，现在则要开始修炼了。
<br/>　　混天魔功共分七层境界，每修炼到一层境界，都会有一种特殊的能力。
<br/>　　分身现在已经将混天魔功第一层修炼的颇为精深，虽然还不到大圆满之境，但也算是彻底练成了第一层。
<br/>　　而练成第一层之后，便可以施展一种名为混天魔域的手段，这种手段类似于大日领域。
<br/>　　不过，混天魔域不能削弱对手的实力，但却能增强自身的实力。
<br/>　　在混天魔域之中，分身的战斗力会再次的暴增，达到新的高度。
<br/>　　玉镯空间中，苏莫的本体静立不动，目光望着远处的分身。
<br/>　　分身正在练习混天魔域，只见其全身魔气滚滚，简直如同绝世魔头，身上浩荡的魔气汹涌澎湃，扫荡四方，充斥身周方圆十数里。
<br/>　　滚滚的魔气将分身彻底淹没在了其中，形成了一处恐怖的魔域。
<br/>　　魔域之中，仿佛有鬼哭狼嚎之音，又似有魔头在其内搅动风云。
<br/>　　魔域不断的收缩、再膨胀，滚滚魔气涌动不休，气势极为惊人。
<br/>　　苏莫的本体静静看着，他能清晰的感受到，分身在这个魔域之中，所能发挥出的实力。
<br/>　　这让他有了很大的底气，如此战力，到时候再配合上他战魂的强大，少司之位近在眼前。
<br/>　　有了少司的身份，他在巫族地位崇高，搅风搅雨就容易多了。
<br/>　　“这是……巫尘！”
<br/>　　远处，巫娴和另外两名巫族青年，远远的看着如同绝世魔头的苏莫分身，满脸的骇然之色。
<br/>　　他们不知道巫尘为何会在这里？也不知道巫尘和苏莫什么关系？
<br/>　　只能暗自心惊，胡乱猜测。
