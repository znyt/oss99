<br/>　　白小飞摆出战斗的姿态，下意识的挡在众人身前，虽然敌人无比强大，但现在的情况，根本不容他退缩。
<br/>　　他身后，可是一群没有任何战斗能力的妇女孩童啊！
<br/>　　“不就是打架嘛，胖爷我还没怕过谁！”
<br/>　　小胖也主动站在了白小飞身边，刘哔向前迈了一步，没有任何话语，但却已经做好了战斗准备。
<br/>　　任由几人准备，背刀男子脸上露出一抹讥讽的笑容，对于自己的实力，他有信心。
<br/>　　在这个落后偏远的地方，还能有谁能够制裁的了他？
<br/>　　眼前这些人的动作，在他看来不过是垂死挣扎罢了。
<br/>　　“准备好了么？”转动了一下刀把，背刀男子眉头一挑，随后声音阴冷了下来：“准备好了……那就去死吧！”
<br/>　　话语刚落，一抹明晃晃的刀影就出现在了白小飞面前，突如其来的攻击，让白小飞心中一惊，但很快他就稳下了心神。
<br/>　　经历了这么多场战斗，他已经能激发自己大部分的战斗力了，在强化药剂的加成下，背刀男子虽强，却也并不是不可抵挡。
<br/>　　“铛！”
<br/>　　一声脆响，白小飞向左后方疾退几步，躲开了面前利刃。
<br/>　　在他惊骇的目光下，一柄碎裂的菜刀掉落在桥面上，仅接触了不到一秒的时间，他的武器，竟然已经折断了！
<br/>　　“嗖！”
<br/>　　背刀男子正欲追击，一柄简陋的自制长矛插向他的头颅。
<br/>　　“哼，雕虫小技，话说，你们这武器，是从原始人身上找出来的吗？”
<br/>　　嗤笑一声，背刀男子挥动了一下长刀，瞬间将长矛搅的粉碎，但如此一来，却也无法进行追击，白小飞总算是有了一口喘气的时间。
<br/>　　扔掉手中断裂的武器，小胖脸上露出一抹疯狂，拖动着二百多斤的身体，赤手向背刀男子冲去。
<br/>　　“妈的！”暗骂一声，白小飞趁乱跑到旁边的轿车前，拆下了主驾驶的车门，使出全身的力气向背刀男子撞去。
<br/>　　短短十几秒的时间里，小胖身上确实增添了许多伤口，背刀男子注意到了白小飞的动作，但被小胖拖住，根本抽不出手来进行阻拦。
<br/>　　像是发了情的母猪一般，小胖死死缠住背刀男子，即使已经浑身浴血，却也没有半点退缩。
<br/>　　他知道，自己需要做的，就是帮白小飞争取机会，只要自己后退，那这场战斗，可能就真的没有机会了。
<br/>　　输了，就代表死亡，没人认为这个血莲教的头目会放过自己。
<br/>　　“空气压缩，喷射！”
<br/>　　看着前方激战的场景，刚刚觉醒异能的晓佳也开始按耐不住了，只是，以她刚刚觉醒的力量，面对这样的战斗也帮不上太大的忙。
<br/>　　好无聊啊……
<br/>　　刘哔手中把玩着烧火棍，看着前方的战场，心中有些嘀咕。
<br/>　　他好像记得，这场战斗是一个蛮重要的剧情，所以才一直没有出手，虽然想要解决背刀男，也只是随手就能办到的事情。
<br/>　　什么嘛……这货比强化过的博士还要弱小，一家子人在这里围殴，都没能占到什么上风。
<br/>　　对于白小飞的能力，他是越来越持有怀疑态度了，怎么看，这都不像是一个主角。
<br/>　　“算了，帮一把吧，情况都已经开始不太对劲了。”
<br/>　　嘀咕了一声，刘哔抛了抛手中的烧火棍，精准的投向背刀男子。
<br/>　　瞬间，炙人的热浪从半空中炸开，黑色的棍体上覆盖了一层赤焰，迎风见长，最后棍棒顶端，固定成了一个诡异的形状。
<br/>　　像是骷髅，却又说不上是从何等生物身上取下来的头骨，体型不大，却是狰狞无比。
<br/>　　这是……什么招式！
<br/>　　这个地方，怎么会有如此强大的异能者！
<br/>　　就连ZF培养的特种部队身上，都没有如此恐怖的气息！
<br/>　　背刀男子满脸惊骇，面对这样的攻击，他根本提不起任何抵御的想法。
<br/>　　不可能抵挡的，如果不逃走……自己会死在这里！
<br/>　　绝望的眼神，在四周寻找着，但是所有退路已被封死，不管他往哪个方向逃跑，都会受到这股火焰的正面冲击。
<br/>　　不对，有机会的。
<br/>　　下面！下面有湖水，可以躲开这股火焰！
<br/>　　像是溺水的人抓住了稻草，背刀男子眼中泛出惊喜的光芒，咬牙向桥下跳去。
<br/>　　只有这种办法了，为了活命，他也不在乎湖内安全与否的问题了。

<br/>　　这东西……也会卡壳的？

<br/>　　心中带着疑惑，背刀男子掉入了湖中，庞大的冲击力，将湖面砸起一个硕大的水花。
<br/>　　“轰！”
<br/>　　紧随其后，火焰长棍落在了桥面上，瞬间整条桥梁从中间塌陷，受到火焰的灼烧，密密麻麻的汽车长龙也被接连引爆。
