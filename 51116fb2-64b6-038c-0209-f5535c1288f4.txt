　　至于“五行印”的来历，就有些奇妙了。
<br/>　　元释闭关期间，一个不自量力的混沌魔神想要闯入他闭关之地来打杀他，这是一个排名前一百的魔神——五行魔神！可惜，就算排名在靠前，遇到了元释，他也得死。
<br/>　　顺手杀死这个五行魔神之后，元释就将混沌魔神的魔神真身收入鸿蒙元始印之中，继续闭关参悟法则。可谁知，这时候鸿蒙元始印竟然开始提取并记录五行魔神体内的本命法则——五行法则！这可惊骇到元释了，鸿蒙元始印还有这样的用途，这是逼着他去打杀混沌魔神的节奏啊！
<br/>　　元释心动无比，想要去多杀一些混沌魔神，来多收集他们体内的本命大道法则。因为法则收集得越多，鸿蒙元始印的本源空间也就越完善，能够领悟的法则也就越多。
<br/>　　虽然没有被元始印记录的大道法则也可以通过它来领悟，可速度不知慢了多少倍，想要领悟一条法则，不知要多少岁月？“元心印”就杀元释通过元始印来领悟的，不过，时间花费的也比因果神通长了不知多少。
<br/>　　虽然杀魔神要让元释心动不已，不过，他也并没有付之行动。因为他知道，如今还不到混沌后期，一个两个还好，要是大量猎杀混沌魔神，众魔神及大道就会警觉，到时候，自己就麻烦了。
<br/>　　若是其他魔神知道了，倒也还无所谓，元释自信不惧任何魔神，就算最强大的盘古魔神想要杀自己，自己借助因果神通也可事先警觉而躲过。但，大道要是警觉了，自己的好日子也就到头了，大道可不是盘古能比的。
<br/>　　更何况，混沌后期，魔神越发强大，大道在鸿蒙破碎时受的伤借助魔神的发展也快恢复的差不多了。这时候混沌魔神的强大，对于他来说，却是一种威胁。他是多希望这些魔神此时能够死光。
<br/>　　所以，大道设计盘古开天灭杀众多魔神，而众魔神也为了成道而前去阻止盘古，一场屠杀，就此开始。至于魔神之间的相互厮杀，能解决魔神的威胁，大道高兴来来不及呢，到时候自己杀了多少混沌魔神，大道也不会管你丝毫。
<br/>　　当然，这些原因元释是不清楚的，他只是隐隐觉得内心有一个声音告诉他，不要这样做。到他如今这个境界，内心的感觉不会无缘无故出现，一旦出现了，那就意味着不同凡响了。是以，他不得不相信自己内心的感觉。
<br/>　　所以，基于众多不便的原因，元释也就忍住内心猎杀魔神的冲动，继续参悟大道法则，只不过，现在多了一条五行大道法则。
<br/>　　通过领悟五行大道，元释创出来包容五行的“五行印”。“五行印”又分为：金印、火印、水印、木印、土印；相互克制的水火印，五行相生的五行印及五行相克的逆五行印。
<br/>　　如今已经到了混沌后期，距离大劫已然不远了，自己的境界提升到了混元大罗金仙后期，想要更进一步，也不是那么容易了，神通等卫道手段也有了，是时候出去走走了。
<br/>　　混沌这么大，我也想去看看！
<br/>　　咦！等等？是不是遗忘了什么？元释如是想到。仔细想了想，看了看自己两手空空，全身上下只剩下自己了。灵宝也没有，灵根也没有，什么都没有，简直穷了不能再穷了。
<br/>　　作为一个修士，财侣法地，都不可缺少。自己一个混沌魔神，修炼的是大道给的法则，“法”是有了，“侣”暂且不缺，“地”也可以不要，废话，这里除了混沌，就是混沌魔神，那里不是一样的？自己可以随遇而安嘛。但“财”排在第一，不可以没有。自己什么都没有，这怎么可以，至少也得有一件称手的兵器啊。
<br/>　　想想人家盘古，不是混沌青莲，就是开天斧。而自己却什么都没有，这就是人与人之间的差距吗？于是，元释愈发的想要出去找些神铁、器材、灵根之类的东西了。
<br/>　　不过，在走之前，元释还是将就着用孕育自己的混沌蛋壳给自己炼了一枚因果莲台。到底是孕育了自己的混沌蛋壳，用灵魂之火锤炼之后，再加入自己的大道法则，用法则之力淬炼，最后形成了一朵黑白相间、闪耀着黑白两色因果之光的因果莲台，周围的混沌之气滚滚涌入莲台之中，莲台形成之后，与自己血肉相连，没有一丝不适，品级不算太高，仅仅达到极品先天灵宝的程度，连先天至宝都不到。
<br/>　　“好！虽然你只是先天灵宝，但你的潜力无限，只要找到合适的器材，却是可以将你升级，从今以后，你就是属于我元释自己的因果大道法宝——因果莲台！”元释感受到自己与莲台之间的联系，对着莲台说道。
<br/>　　“嗡嗡！”莲台轻鸣两声，之后继续吸收周围的混沌之气，好像周围的混沌之气就是他最好的食物一样。元释也不管，任由他吸收，自己在旁边找了一个位置，盘腿坐下，继续参悟法则，淬炼魔神真身，使之更加强大。
<br/>　　不知过了多久，因果莲台终于不再吸收混沌之气了，它停了下来。也就在此时，元释睁开眼睛，站了起来，左手一伸，原先还数百丈之大的莲台自动缩小，飘到元释手心，之后没入元释体内，继续温养。
<br/>　　“不错，虽然品质没有提高，但距离先天至宝已经只差一丝了，硬度却提升了不少，超越了先天至宝。而且莲台内还开辟了一个小型混沌空间，内蕴混沌，不错，真是不错！没想到还有这样的收获！”
<br/>　　元释对这因果莲台很满意，虽然它没有达到先天至宝的程度，但未必就没有机会晋级，最关键的是其内还有一个混沌空间，虽然小了一点，但也不错了。
<br/>　　一步踏出，就是数百万丈之远，一步又一步，眨眼间，元释那百万丈高的身影就消失在混沌苍茫之中。
<br/>　　……
<br/>　　不知道过了多少岁月，元释一直在混沌之中行走，混沌苍茫无垠，没有生命，没有黑暗，什么都没有。要不是多年的修道早已经习惯了寂寞，也不知道会不会发疯！
<br/>　　他那混元大罗金仙九重天的修为没有进步多少，倒是魔神真身在吸纳了无穷混沌气流的淬炼之下，泛起了道道古韵，隐隐有突破的趋势。
<br/>　　独自游历在这寂寥的苍茫之中，感悟着混沌的气息，倒也让他的道心得到洗涤，更加坚定了。
<br/>　　求道之路，本就是孤独的，没有谁能一辈子陪在你身边。能陪你的，唯有寂寞！感悟混沌，磨练意志，享受着这求道的寂寞，空虚！
<br/>　　元释的魔神真身现在有着数百万丈之高，轻轻一跨，就是数百万丈之远，在这漫长的岁月里，他不知道走了多少距离，但这些对于浩瀚无边的混沌，却只是沧海一粟而已！
<br/>　　在这漫长的时光里，元释还是遇到了几个混沌魔神，对于这种情况，那就只能算他们倒霉了。
<br/>　　结果不言而喻，只要是遇到的，全部被元释解决，并将之收入元始印之中，提取他们的本命大道法则。苍茫无限，能遇到几个魔神，真是不容易。当然，这也是元释没有目的性的游走混沌，能遇到也算是运气，要是他利用因果之道推算，他还是能算到一些的。
<br/>　　就这样，不知过了多久，混沌之中，厮杀愈演愈烈，最后元释也参与到了其中，他也遇到了几个强大的魔神，经过一番厮杀，终于将之击杀。其中就有梦魇魔神，掌控梦之法则，要不是元释对于梦之法则有一定了解，还真的阴沟里翻船了。
<br/>　　再如封印魔神，掌控封印法则，一手封印神通使得强大莫测，“混元封”直接将元释体内储存的混沌之气封印，“神魂封”封印元释的神魂，让他调动不了法则，而“神通封”却直接让元释使出的神通，威力降到最小。要不是最后元释利用强大的魔神真身消耗对方，还不一定杀得了他。
<br/>　　随着元释猎杀的魔神越多，元始印记载的法则也就越多，元始印也越强大。现如今已经有二十多条法则记录元释其中，也就意味着元释已经杀了二十多混沌魔神。
