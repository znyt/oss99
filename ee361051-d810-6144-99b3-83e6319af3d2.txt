　　第121章
<br/>　　骷髅岛二当家名叫耿星汉，他身子仰躺在船板上，像一只搁浅的章鱼。
<br/>　　这是他每次做事前的习惯，屏声静气，仔细感受身下海浪的涌动。他自幼在海边长大，从小就对大海有种莫名的亲近感，在很小的时候，凭直觉就能判断出风暴的来临时间。
<br/>　　后来成为一名海盗，就像是理所应当的事情。
<br/>　　粗糙的海风从他脸上扫过，他伸出舌头舔了舔空气，海风的味道有些咸。
<br/>　　和血类似。
<br/>　　血管内涌动的东西变得炙热起来，按计划，自己的女人如今已上船。那是一个很有味道的女人，耿星汉都忍不住想起与她在床上的滋味。这样的套路用过不止一次两次，每次都很好用。
<br/>　　接下来，自己的女人就会破坏掉船帆。而在大海上，一艘没帆的船，就像一个落在土匪窝里的黄花大闺女，等着人上她。
<br/>　　这里的上，是字面意思的上。
<br/>　　大海之上伸手不见五指，只有两艘船的灯火孤独靠近。而在他们前方，数十艘小船排成一个半圆形，像一只张开的巨口，等着猎物走进嘴里。
<br/>　　算时间该动手了。耿星汉抓住自己的鱼叉，准备起身，过程中顺手在海水中捞了一把，指头放进口中舔了舔。
<br/>　　这也是他的习惯之一。
<br/>　　海水的味道是咸的。
<br/>　　和血类似。
<br/>　　不，不是和血类似，这就是血。耿星汉吃惊的看向自己的手，手掌挂着淡薄的血痕。他再看向海面，一具浮尸缓缓向耿星汉飘来。
<br/>　　耿星汉一身冷汗。
<br/>　　身为海贼，刀刃添血，海面一具尸体自然不会令他害怕。关键是尸体的身份……
<br/>　　那媚人的桃花眼，馒头般饱满的脸颊，还有……还有那湿漉漉的长发，就像……
<br/>　　这是自己的女人呐，还是自己最爱的一个女人，尤其爱她海草般的长发。
<br/>　　『像一棵海草海草海草海草，随风飘摇，海草海草海草海草，浪花里舞蹈』
<br/>　　耿星汉血灌瞳仁，握紧鱼叉的手青筋爆现，谁他妈把这讨人厌的BGM关掉！
<br/>　　“动手！”
<br/>　　呼啦的起身声，多少柄兵器在此刻亮出，耿星汉恶毒的盯着渐近的货船，只等对方进入包围圈。
<br/>　　突然！
<br/>　　两艘船上的灯火同时熄灭，在黑暗的海面上，像隐入另外一个时空。
<br/>　　船头，程大雷挥手又抬头，口绽两个字：“执箭。”
<br/>　　两边船舷，有人张弓搭箭。
<br/>　　黑暗是公平的，耿星汉看不见程大雷的同时，程大雷也必定看不见对方。但双方都能察觉黑暗中潜藏的危险，如某种不知名的兽。
<br/>　　程大雷旁边立着赵子龙，张弓如满月。
<br/>　　“海上你能射多远？”
<br/>　　“百步之内，我有把握。”
<br/>　　“记得，射人先射马，擒贼先擒王。”程大雷低声吩咐。
<br/>　　“明白。”
<br/>　　船在这个时候，仍保持着全速前进，双方间的距离不断缩短。
<br/>　　在失去对方位置后，耿星汉便失了方寸，女人的死令他怒火中烧。
<br/>　　“冲！”
<br/>　　他下达命令，又第一个冲过去，哗哗水声响动，四周围的小船上往往是两人一组，一人握着兵器站在船头，另外一人卖力划桨。
<br/>　　不知有多少艘小船缩短着包围圈，距离渐近，在视线无法抵达的黑夜只能大概判断出个位置。
<br/>　　有人迫不及待亮起火把。
<br/>　　箭矢破空声，在对方刚举起火把的瞬间，羽箭便穿过他的喉咙，火把掉在海水里熄灭。
<br/>　　“漂亮！”程大雷赞了一声。
<br/>　　也就是火把亮起的一瞬间，程大雷看清楚了周围的情景，在如墨般浓稠的海面上，数十艘小船如蝗虫般扑过来。
<br/>　　乌压压人群。
<br/>　　“箭，准备。”
<br/>　　只要有人亮起火把，便立刻有人射箭，赵子龙的箭术干净到极致，百步之内，箭无虚发。
<br/>　　短短几个呼吸之内，便有十几人死在赵子龙箭下。
<br/>　　杀戮，安静漂亮。
<br/>　　耿星汉此刻已知道碰到硬茬，对方做出万全的准备，打得己方猝不及防。他们原先的目的是杀人抢粮，将粮船拖回骷髅岛。
<br/>　　但连死十几人，已经令他方寸大乱，而女人的死，更是让他怒发冲冠。
<br/>　　“放箭，放火箭。”
<br/>　　此刻耿星河宁肯烧船，也不想让对方逃走。
<br/>　　火箭射来，纷纷如火雨。
<br/>　　程大雷第一时间做出反应：“降帆，停船，把他们引过来。”
<br/>　　两艘船的白帆同时落下，有火箭落在船上，大家手忙脚乱的灭火，阵型完全被打乱。
<br/>　　看到这一幕，耿星汉得意笑，一旦船帆落下，这两艘船就是案板上的鱼肉，任自己宰割。
<br/>　　他命令手下停止射箭：“停手，给我上船。”
<br/>　　无数船只划水靠近，在接近货船上放出倒爪往上攀，更有人迫不及待，噗通跳入水中，潜水而来。
<br/>　　两艘船并排，船帮连在一起，从一艘船可以迅速跳到另外一艘船上。一只船上有程大雷，赵子龙，高飞豹，另外一只船上有张肥，小白狼等人。至于柳芷，苏樱，徐神机这等没有战斗力的，已经被安排到船舱的底部。
<br/>　　程大雷早已换上斧头，那剑虽然也是好剑，但终没有斧头来得顺手。多少人安静的趴在船上，只有眼睛雪亮。
<br/>　　“杀！”
<br/>　　程大雷暴跳而起，斧头干净的砍过一人，另外一人还没上船，就被程大雷劈下海水。
<br/>　　高飞豹耍着大刀，他每砍一刀，都要发出一声怪叫，也不知是怎样的习惯。
<br/>　　小白狼握着鸳鸯双刀，刀专门找人咽喉开刀。
<br/>　　而随船的柳家人也并不是泛泛之辈，他们曾经便在海上与水贼作战，人人都有好水性。
<br/>　　而张肥的表现最是抢眼，他在得到蛇矛后，实力似乎又上了一个等级。他本是一员猛将，悍不畏死，蛇矛挥过之处，无人能挡。
<br/>　　赵子龙一直记得程大雷的话，他站在船头，紧盯着周围，寻找着对方的头目。海贼中隐隐有命令呼喝声，但在黑暗中也听不真切。自己想要找的人，究竟在哪里。
<br/>　　嗖！
<br/>　　一只羽箭破空而出，射向一艘小船。黑暗中隐隐约约看不真切，但赵子龙的箭向来例不虚发。
<br/>　　而这个时候，杀戮才刚刚开始。
