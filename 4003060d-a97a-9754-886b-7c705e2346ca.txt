　　罗昊疑惑的看了眼赵天宇，随后拿起赵天宇仍在自己面前的那叠照片翻看起来。
<br/>　　出现在照片中的主要有三个，智天使、吴云、还有沙耶也就是SHADOW。沙耶这个身份是公开的，所以SHADOW也就继续在使用着这个身份，至于沙耶这张脸到底是不是SHADOW的真实面目，或许只有SHADOW自己知道了。
<br/>　　从照片上可以看出，三个人之间明显是认识的，而且是早就认识。其中有一张照片，智天使甚至向相机镜头这边看了眼，并且挥了挥手。
<br/>　　“我们的监视被发现了。”
<br/>　　罗昊把那张照片拿出来放在桌子上，手指在照片上轻轻敲了敲。
<br/>　　“他们没有想着隐瞒着我们。”赵天宇解释道：“这几天他们的行动很随意，根本没有一点要隐藏的意思，所以我们的跟踪拍摄也就没有掩藏。”
<br/>　　“或许，智天使这是故意的。”李然想了想，沉吟着说道：“他是故意告诉周围在监视的人，SHADOW跟‘暗月’有联系，不要试图轻举妄动。‘暗月’手里掌握着P病毒还有很多我们未知的，不了解的秘密武器，这就使得旁人不敢有所动作。P病毒释放，这样的毁灭性代价是任何一个国家都难以承担的。”
<br/>　　“妖瞳的想法很有道理。”赵天宇接过话，说道：“我们经过分析后，也同样这么认为。”
<br/>　　“‘暗月’的人也来到这个城市了。”罗昊皱眉，说道：“事情变得越来越复杂了。”
<br/>　　“不管他们有什么招，他们总是要亮招的，我们只要见招拆招就行了。”
<br/>　　赵天宇走到罗昊身边，在他肩膀上拍了拍说道：“代码小镇的事情我听说了，解决掉了毒天使，让‘暗月’又少了一个大将。就算‘暗月’再怎么财大气粗，这样的损失也是他们无法忽视的，而且‘十二主天使’这样的人不是说培养就能培养出来的。距离SHADOW约定的对决时间还有三天，你们先去休息，有最新消息我会通知你们。对了，洛菲克带走的P病毒融合配方已经拿到了，只是那家伙不愿意拿出来。”
<br/>　　“其实，他并不想要现在的生活。”罗昊叹了口气说道：“他有很多人羡慕都羡慕不来的东西，但如果给他选择的话，或许他并不想要那个东西。既然他不愿意交出来，就让他留着吧，反正他不会做出什么越线的事情，‘忠于祖国’是他父亲从他出生那一天起，就刻在他骨子里的东西，不管发生什么也不会将这四个字磨灭掉，否则当年琴魔也不会破例收他做自己的关门弟子。他之所以想要留着这个东西，或许……只是让某些顽固的家伙妥协吧。”
<br/>　　这一整栋写字楼目前暂时全都归龙隐部队使用，写字楼楼顶是直升机停机坪，供直升机起降；顶层是指挥中心；指挥中心下面那层是武器室，再往下那层是休息区。
<br/>　　这里原本就是供员工休息的地方，床、毯子、拖鞋、浴室，甚至就连一次性洗漱用具都是准备的很齐全，刚进入这层休息区的时候，罗昊差点以为自己来到了一个快捷便利酒店！
<br/>　　比起国内的白领写字楼，这里的工作条件的确是要优越太多太多了，完全不是一个层次的比较。
<br/>　　“你就去我的房间吧。”
<br/>　　云熙带着罗昊去了她休息的那间房间，十几平方米大的房间里一张单人床，一张写字桌，一把椅子就是房间里全部的家具了。这里所有的房间布置都一样，只是这个房间里多了一些……云熙身上特有的味道，这种味道，让罗昊不管在什么地方都有一种回家的感觉。
<br/>　　罗昊拿了一条新毛巾去了洗漱室，就算不洗澡，洗把脸也能舒服很多。回到房间的时候，发现云熙坐在桌子下，双手支着下巴出神想着什么事情。
<br/>　　把毛巾挂起，罗昊走到云熙背后，把双手搭在她肩膀上，问道：“在想什么呢，这么出神？”
<br/>　　“你信不信直觉？”云熙突兀的对罗昊问道。
<br/>　　罗昊先是一愣，随后点头回答道：“信。”
<br/>　　罗昊是唯物主义者，但是在战场上，他很相信自己的直觉，因为他始终相信赵天宇挂在嘴上那句话，而直觉也在关键时刻，好几次救了罗昊和他身边队员的命。
<br/>　　“都说女人的直觉比男人更灵敏。”云熙脸上写着忧虑，对罗昊说道：“我总觉得这次对决会出事情，而且这种感觉越来越强烈。”
<br/>　　罗昊伸手抓住云熙的手，揉捏着她纤细的手指，说道：“我在你身边的时候，你需要相信的，不是你的直觉，而是我！明白吗？”
<br/>　　云熙展颜一笑，说道：“我知道了，你赶紧休息吧，我去指挥中心。”
<br/>　　罗昊刚躺下就感觉到枕头下似乎藏在什么东西，伸手在枕头下摸了摸，竟然是摸出了一个女人使用的小玩具来。
<br/>　　罗昊瞪大着眼睛，看着在自己面前来回晃动的粉红色椭圆小球，叫住正准备开门离开的云熙，“云小熙，你就算再寂寞也不至于用这个东西吧？”
<br/>　　云熙扭头看到罗昊拿在手里的东西，一张俏脸顿时红得跟红布一样，就连说话都是变得支吾起来，“这、这、这不是我的！”
<br/>　　“在你枕头下发现的，不是你的？”罗昊问道。
<br/>　　“这东西真不是我的！”云熙急的直跺脚，说道：“我这几天累了都在指挥中心休息，根本没回这个房间睡觉，怎么可能会是我的东西！？”
<br/>　　“云小熙，你总算是说实话了。”罗昊翻身而起，一把将云熙给抓了回来，用脚跟把门关上，说道：“不好好休息还要去指挥中心？难道那里少了你还不行？现在给我乖乖睡觉！”
<br/>　　“那你放开我，我去隔壁房间睡！”云熙挣扎了一下，说道。
<br/>　　“你当我傻吗？”罗昊在云熙屁股上打了一下，说道：“我智商虽然没有你那么高，但是你也别拿我当小孩子骗，出了这个房间，你还会去隔壁房间睡觉？骗谁呢？”
