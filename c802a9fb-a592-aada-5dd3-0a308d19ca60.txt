　　对于云悠悠而言，她一直想知道自己的过往、自己的以前，她不想活的不明不白，但是没有人知道。
<br/>　　所以，她一直便想着，等到日后修为高深了，达到武圣、甚至是虚神之境，恢复以前的记忆，从而知晓自己得身世。
<br/>　　而现在，突然有人来到她的面前，告诉她自己原来的名字，这让云悠悠即意外、又惊喜、同时还有些不安。
<br/>　　意外是因为完全没有料到，惊喜是因为可以知道自己的过往，不安是担心自己的从前，不堪回首。
<br/>　　“你此言当真？”深深的吸了口气，云悠悠凝声问道，她并没有完全相信苏莫。
<br/>　　因为她没有以前记忆的事情，也有极少数的人知晓，不能保证没有泄露出去，而遭到有心人的利用。
<br/>　　“当然，我们是多年的朋友，我对你的身份自然相当了解！”苏莫重重的点了点头。
<br/>　　“那你说说，关于你知晓的我的一切。”云悠悠面色郑重，凝声说道。
<br/>　　“好！”
<br/>　　苏莫微微点头，随即，转头看向山峰之下，那一望无际的山林，心中陷入了回忆之中。
<br/>　　“我们相识于二十年前，那时候，我是风凌岛的弟子，而你是……！”
<br/>　　接下来，苏莫徐徐开口，将他与对方何时相识，所发生过哪些经历，以及宏青璇的身份等等，一切都说了出来。
<br/>　　当然，关于两人之间的那一丝感情，他只字未提。
<br/>　　直到两个时辰之后，苏莫才说完了一切，目光静静的望着云悠悠。
<br/>　　听完苏莫所说的一切，云悠悠的面色很平静，伫立在山巅之上，目光望向天际的尽头。
<br/>　　清风拂过，她的秀发随风飘动，身上的白衣猎猎作响。
<br/>　　“你如何能证明，你所说的一切全部为真！”云悠悠轻声说道，虽然言语如此，但她的心中，已经有五分相信了，因为苏莫的这些话，听起来没有任何的纰漏。
<br/>　　而且，从苏莫眼眸之中，他能看到深深的回忆，写好像很难做假。
<br/>　　“很多办法可以证明！”苏莫沉声说道，言罢，他略一沉吟，便手臂一挥，一具透明的棺醇突然初一在了他的身前。
<br/>　　这具透明的棺醇，正是存放芊寻月的玄天灵柩。
<br/>　　透过棺醇，可以清晰的看到，芊寻月正静静的躺在其中，安详而又宁静。
<br/>　　“这是你的师尊，芊寻月！”苏莫沉声说道，此刻，望着棺醇中的芊寻月，苏莫心中即有些落寞，也有些感慨。
<br/>　　落寞的是，芊寻月已经躺了很久，真的不知道什么时候才能救活。
<br/>　　感慨的是，此时此刻，时隔多年，芊寻月和宏青璇师徒两人再次相见，早已物是人非。
<br/>　　当初，在帝玄山脉，苏莫打败了上官昊之后，黄泉魔宗之人对他出手，这师徒两人不顾一切的去相救。
<br/>　　因而，一人被擒，一人几乎算是陨落，均是遭受大难。
<br/>　　现在师徒两人再一次相见，已是十几年之后了。
<br/>　　云悠悠同样在望着棺醇，柳眉微皱，静静的看着其内的芊寻月，但是，任他如何回想，也没有丝毫的印象。
<br/>　　“你的父母和亲人，都在古灵星河！”苏莫再次说道，当初他称霸苍穹世界之后，等待巫族试炼者到来的时间里，他曾派人去了宏域，将青璇的家族，天宏皇室的人都接到了苍穹神宫。
<br/>　　“古灵星河？”云悠悠闻言，轻喃一声，面色波澜不惊，但是她的心中，却是如同大海浪涛，汹涌澎湃。
<br/>　　她的心中，已经有七分相信了苏莫的话语。
<br/>　　“那我是如何来到太阴族的？”云悠悠沉声问道，苏莫刚才并未说出她为何会来到太阴族。
<br/>　　“你被巫族擒住以后，天冥星巫族一脉的大祭司，将你卖给了巫族，一名太阴族的女性强者抹去了你以往的全部记忆！”苏莫说道。
<br/>　　云悠悠闻言心中一震，她的记忆，是被太阴族之人抹去的？
<br/>　　“哪位强者？”云悠悠问道。
<br/>　　“我并不认识，她身穿宫装，气质高贵，面目……！”苏莫将搜魂得来的记忆中，带走青璇的那名太阴族女性强者的身形外貌，都详细的诉说了一番。
<br/>　　云悠悠闻言，轻轻的点了点头，通过苏莫的描述，她已经知道了是何人。
<br/>　　那名宫装女性，正是太阴族的一名长老，武圣境强者。
<br/>　　此刻，云悠悠的心中很乱，她已经非常相信苏莫的话了，但也不是绝对的相信。
<br/>　　最主要的是，听闻了苏莫所说的一切，她并不能感同身受，仿佛只是在听闻一个故事。
<br/>　　因为记忆的缺失，她完全无法将‘宏青璇’的一切，代入到自己的身上。
<br/>　　但是，无论怎样，听闻了苏莫所说的一切，她的心中都有极大的触动。
<br/>　　毕竟，这可能就是她的过往。
<br/>　　“谢谢你告知我一切！”
<br/>　　少倾之后，云悠悠微微欠身，向苏莫道谢。
<br/>　　“青璇，我之所以来到明虚星河，便是为了找你！”苏莫沉声说道。
<br/>　　“多谢！”
<br/>　　云悠悠闻言，再次道谢，旋即略作沉吟，道：“今日你告诉我的一切，我会去求证！”
<br/>　　虽然心中已经很相信苏莫的话，但云悠悠不可能听信苏莫一面之词，她还会想办法去求证。
<br/>　　毕竟，太阴族给予了他一切，让她无忧无虑的生活在太阴族，一切的资源都不吝啬。
<br/>　　而且，普阳神长老对她视如己出，倾心教授，她不可能仅凭苏莫一面之词，就生出对太阴族的怨恨之心。
<br/>　　“你如何求证？”苏莫皱眉问道，他说的已经相当详细，对方貌似并没有完全相信。
<br/>　　闻听苏莫之言，云悠悠却是并没有回答，她的心中很乱，急需要静一静，稳定心绪。
<br/>　　“苏莫，我要回族了，改日再见！”云悠悠说了一句，便身形一身，破空离去。
<br/>　　“这……！”苏莫见此，神情一怔，他没想到，事情还没说清楚，对方就迫不及待的离开了。
<br/>　　暗叹一声，苏莫有些无奈，看来想要和青璇彻底恢复如初，根本不是短时间内能解决的事情。
<br/>　　最主要的事情，仅凭他的言语，很难消除两人之间的隔阂，除非，恢复对方的记忆！
