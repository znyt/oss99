br/>　　    姚巍山笑了一下，说：“也谈不要找他们的麻烦了，是这些家伙现在因为垂直管理了，拿市委市政府不当一回事了，想让他们支持一下市里面的工作推三阻四的，不想办法整治他们一下是不行的。诶，陈书记，这些举报信当有没有能够达到采取措施的程度的？”<br/>　　<br/>　　    陈昌荣笑了笑说：“肯定是有的了，不过姚市长你总该有个目标吧，不会是想把他们一打尽吧？”<br/>　　<br/>　　    姚巍山笑了，他知道是不可能将海川市各大银行的行长都抓起来的，如果海川市几大行的行长全军皆没的话，恐怕将会成为一个轰动全国的大新闻了，这对海川市来说可并不是什么好事。<br/>　　<br/>　　    姚巍山笑了笑说：“那当然不会，你看有没有关于海川市建行的行长李成华的举报资料啊？”<br/>　　<br/>　　    姚巍山选择李成华作为打击目标，不仅是因为李成华是这些银行当第一个不给他面子的行长，还因为他觉得后面那些银行行长们之所以会应对他的时候口径一致，肯定是李成华将他们那一次的会面通报给了其他银行的行长。<br/>　　<br/>　　    这是姚巍山觉得李成华最可恶的地方，这家伙自己不肯支持他也罢了，居然还串通其他银行一起不支持他，如果对这样一个家伙都不给一点教训的话，他心的这口恶气出不来的。他想要李成华知道，不管怎么说也轮不到李成华来给他姚巍山定规矩。<br/>　　<br/>　　    陈昌荣打了电话给下面的监察室，让他们把搜集的有关举报李成华的资料送了过来，然后对姚巍山说：“资料是有的，不过要对他采取行动的话，最好您还是跟市委孙书记说一声。”<br/>　　<br/>　　    纪委是在市委的权限范围之内的，想要让纪委对付李成华这种有相当身份的人，是不能不让市委书记知道的。姚巍山知道这件事情是无法回避孙守义的，笑了笑说：“那是自然，你把这些资料给我一份，我去跟孙书记汇报一下。”<br/>　　<br/>　　    陈昌荣给了姚巍山一份李成华的举报资料，这些资料大多是举报李成华在批准贷款的时候受贿，和李成华生活腐化堕落跟一些女性存在有不正当关系的。<br/>　　<br/>　　    姚巍山拿着资料去找到了孙守义，把他的想法跟孙守义作了汇报，然后请求孙守义支持一下他的工作。孙守义听完，沉吟了一会儿，他在考虑这么做的利弊。<br/>　　<br/>　　    孙守义首先觉得姚巍山这么做是手段有点过于狠辣了，这又不是你死我活的权力斗争，应该还不至于一定要将李成华送绝路。孙守义虽然没详细看资料，但也很清楚现在这些领导们的行为都是经不起查的，一经纪委调查，李成华这辈子也完了。<br/>　　<br/>　　    姚巍山这么做是有些乱了规矩的，官场是很少有人会主动去做这种毁人一生的事情的，通常这么去做的人都是不会有什么好下场的。你不给人留余地，别人也不会给你留余地的。所以古往今来，这样做的一些酷吏的下场都不是很好的。<br/>　　<br/>　　    不过孙守义其实也是不反对姚巍山这么做的，他心早想给这些目无人的行长们一个教训的。尤其是这个李成华，是其最滑头的一个，孙守义也曾跟他打过招呼，想要他在贷款方面支持一下海川市的企业，但李成华都是当面答应的挺好，背后却是想尽办法拖着不办。<br/>　　<br/>　　    虽然这些银行什么的都是垂直管理的，但是总还是在海川市地面的，如果任由这些行长们对他这个主政海川市的一把手这么不尊重，对他的威信来说，也是一个很大的损害的。<br/>　　<br/>　　    不过不反对姚巍山的做法，也不代表着孙守义愿意公开站出来表态支持这么做。孙守义知道这样做也是有一些弊端的。李成华能够做到市建行的行长的位置，下下肯定是有一些人在支持他的。<br/>　　<br/>　　    这其很可能也包括一些他的级别还高的官员，因此要动李成华的话，很可能是要得罪一些很有权势的人物的。如果说为了他自身的利益非要得罪这些人的话，孙守义倒也不会因为怕这些人而缩手。不过为了姚巍山这么去做，那实在没必要了。<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
