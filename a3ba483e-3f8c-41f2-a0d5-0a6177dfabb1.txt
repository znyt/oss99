　　所谓的主神计划，即是混沌道君用来收取此界资源的计划。
<br/>　　遮天界中，并没有什么太过值得混沌道君在意的地方。此界中，最强者也就是几尊红尘仙。红尘仙也就相当于洪荒大罗金仙，混沌道君在此界就相当于无敌了。
<br/>　　但是，从原著中可以看出，此界此纪元之前，或者之后，都不简单，有仙帝级的高手，也就是混元大罗金仙级的强者，而且还不止一个，更何况还有上苍之上这样的更高位面。
<br/>　　道君不得不为以后做准备。他不可能只靠自己一人便去收拾那么仙帝强者吧。总需要几个伙伴，亦或者……战友。
<br/>　　而遮天界中，各种宝物都比较低级，道君不需要，他在意的只有此界的大道道则，以及各种资源。而此界中让他最在意的便是人才资源。
<br/>　　别的修炼资源，此界发展到如今，已经有些贫乏了，各种资源他也看不上。但是此界中的人才资源却是最不缺少的。
<br/>　　大世之争，各种天骄多如细雨。
<br/>　　有“旷古绝伦，盖世圣体”之称的荒古圣体，有圣体的宿敌苍天霸体、有神王体、有天妖体、有人王体、有太阴圣体、有太阳圣体、有冥王体、有先天道胎体、有先天圣体道胎，有与法则亲和的元灵体、有广寒灵体、有光明体、有暗黑之体、有混沌体、有太上仙体、有梵天战体、有仙灵体，有虚空体，等等。
<br/>　　遮天世界中，特殊体质太多了，这样的体质，放在其他世界中，每一个都有绝世之资，若是给他们足够的机缘，他们绝对会一飞冲天。他们都有问鼎至高的资质。若是浪费了，那就太可惜了。
<br/>　　可惜此界中，只有一人能够成帝，一人成帝，就绝了天下人的路。
<br/>　　混沌道君的想法很简单，就是要培养这些人，打破一人成帝的法则，将他们培养成大帝，仙，甚至仙王，这样自己以后也就不会缺少手下，或者战友了。
<br/>　　这样的事情，原著中叶梵就做过了，道君自然也可以做到。
<br/>　　同时，道君若是需要什么，可以让他们去收集，自己静心参悟此界道则即可。
<br/>　　而主神计划中最重要的就是主神空间，道君第一时间就想到了九龙拉棺！
<br/>　　九龙拉棺，这是一件堪比仙器的特殊灵宝，它贯穿了仙古、乱古、神话、荒古等纪元，就是一部活化石。里面还有乱古时代荒天帝演化的小仙界。
<br/>　　而且，最重要的是，它的形迹飘忽不定，一直在星空之中流浪，若是隐匿起来，谁也发现不了。
<br/>　　主神空间要的就是神秘，九龙拉棺中的棺内空间，再合适不过了。
<br/>　　想到此处，道君闭目不语，双手十指连转，一股玄之又玄的天机奥妙在他十指上显现。道光流转，因果隐现，一股隐藏的天机，顿时显现在道君手中。
<br/>　　“找到了。”
<br/>　　不知过了多久，道君睁开眼睛，抬头看向星空，轻语道。
<br/>　　“走！”
<br/>　　混沌道君化作一道流光，向着幽暗深邃的星空飞去。
<br/>　　原地人来人往，没有一人能看见道君，只因为，他不想让人看见，就没人能够看到他。
<br/>　　道君漫步在星空之中，无边宇宙虚空自他身后倒退，一个刹那，不知多少路，他便来到了紫微星域。
<br/>　　再几步他出，一个不知名的仙域显现在他面前。
<br/>　　此时，他看到冰冷与黑暗并存的虚空中，九条庞然大物一动不动，仿佛亘古就已横在那里，让人感觉到无尽的苍凉与久远，那就是九条龙尸。
<br/>　　与传说中的龙一般无二！
<br/>　　每具龙尸都长达百米，犹如铁水浇铸而成，充满了震撼性的力感。
<br/>　　九条龙尸都是五爪黑龙，除却龙角晶莹剔透，紫光闪闪外，龙身通体呈黑色，乌光闪闪，鳞片在黑暗中闪烁着点点神秘的光华。
<br/>　　在龙尸的尾端，劫绑缚这碗口大的黑色锁链，连接着一口长达二十米的青铜棺。
<br/>　　青铜巨棺古朴无华，上面一些模糊的古老图案，充满岁月的沧桑，也不知在宇宙中漂浮了多少年。
<br/>　　九龙拉棺！
<br/>　　正是道君此行的目的。
<br/>　　他一步踏出，来到青铜棺上，看着眼前的巨大铜棺，静静打量很长时间。突然，他向着铜棺打出几道印诀。
<br/>　　顿时，光华流转，铜棺之上刻印的数千古字全部烁烁放光，漂浮了起来，宛若有了生命一般。整个铜棺顿时流转出一股柔和的光芒。
<br/>　　道君眼中神光爆闪，神眼睁开，看向铜棺。
<br/>　　古字光华流转，渐渐凝聚成一个巨大的八卦，笼罩在九具庞大的龙尸以及青铜巨棺之上，一股神秘的力量震动而出。
<br/>　　紧接着，半空中的八卦图中央有浮现出一个太极图，两个阴阳鱼抱中而居。
<br/>　　四周时空扭曲，光线迷蒙，与乾、坤、兑、坎等对应的八卦符号先后发出光芒，像是一组古老而神秘的密码。
<br/>　　太极图两条阴阳鱼宛若两道门户，不断颤动，缓缓打开一道缝隙。
<br/>　　光华不断闪耀，八卦的八种符号明灭不定，最终按照繁复的顺序变换多次后，终于全部亮起，发出夺目的光彩。
<br/>　　轰！
<br/>　　阴阳鱼完全打开，露出一道神秘而又漆黑的通道，不知连向何处。
<br/>　　“这个世界的道则，很好，我越来越喜欢了。”
<br/>　　道君说了这么一句话。他打出几道印诀，顿时，虚空之中出现一个同样的巨大八卦图案，道光流转，道韵自生。
<br/>　　仅仅几个呼吸的时间，他便领悟了这八卦图的奥妙，八卦玄之又玄，没有人能够完全领悟，混沌道君也不敢说自己完全懂。
<br/>　　他也只是懂其中很大一部分，但仅仅是一部分，就让他领悟了众多法则奥妙。这简直是不可思议的事情。若是在洪荒，这简直不可能。就算是顿悟，恐怕也不过如此吧。
<br/>　　混沌道君化作一道流光，钻入黑洞之中。之后，通道关闭，青铜巨棺颤动，开始向着远处驶去。
<br/>　　铜棺之中，漆黑一片，看不到半点光彩。但这对于的道君来说，自己的神觉早已经达到了不用肉眼看的地步了。在他眼中，黑暗不再是黑暗，再黑暗的东西，在他眼前也没有神秘感。
<br/>　　在仔细打量了青铜棺内部空间后，道君悠悠说道：
<br/>　　“唔！这里太黑了，不行，主神空间需要重新改造。”
<br/>　　……
