　　“嗯，我现在和你说说祖龙塔的情况！”
<br/>　　龙腾点了点头，随即沉声道：“此塔就在吞虚龙府附近，共有十层，是洪荒祖龙对于龙族后辈的一个考验之地，其内，有一条阵法凝聚的龙灵，无比的强大，就连龙帝都不是其对手。”
<br/>　　“这么夸张？”司空炎惊讶的张大了嘴巴，比龙族现在的掌控着吞虚龙帝还要强大，这未免太夸张了吧！
<br/>　　“不夸张！”
<br/>　　龙腾摇了摇头，面色凝重的道：“据说这龙灵的力量之源，便是这祖龙精血，进入此塔，就要和同阶的龙灵交手，第一层龙灵只有一成实力，第二层龙灵有二成实力，第三层有三成实力，以此类推，登上第十层，龙灵的实力就是巅峰的十成实力！”
<br/>　　苏莫和司空炎闻言，默默的点了点头，看来只有打败了十成实力的龙灵，才能取得祖龙精血。
<br/>　　“你进入过此塔吗？”苏莫沉声问道。
<br/>　　“当然！”
<br/>　　龙腾点了点头，道：“我刚来到龙族之时，便进入了一次，在第七层败北，前段时间又进入了一次，在第八层败北！”
<br/>　　“第八层？”苏莫闻言凝重，龙腾的天赋，现在可是远超万古天骄了，估计比之那些无上神王当年，也丝毫不弱了，居然连第八层都闯不过去！
<br/>　　“这一段时间，我在龙族地位大涨，名声响亮，就是因为闯进了第八层，因为，从古至今，从来没有人能闯入第九层，而闯进第八层之人，全部加起来，也只有一手之数。”龙腾道。
<br/>　　“哪几人闯进了第八层？”司空炎好奇的问道，龙族之中，居然还有几人天赋堪比龙腾？
<br/>　　“首先是吞虚龙帝，其次是龙族的无上神王龙裴霖，以及龙族的两位万古天骄龙羽和龙双双，最后就是我了！”龙腾道。
<br/>　　“龙双双这么厉害？”苏莫闻言惊讶，此女比较低调，没想到如此厉害。
<br/>　　“她确实厉害，不过是闯了十几次，才勉强登上第八层，而且在第八层根本没有反抗之力，立刻就被淘汰，我在第八层能坚持十几个呼吸的时间！”龙腾解释道。
<br/>　　言罢，龙腾继续道：“苏莫，我相信以你的战力，有五成的可能性，取得祖龙精血。”
<br/>　　“的确是有可能！”
<br/>　　苏莫闻言点了点头，他对于自己的战力，还是比较有自信的，但是，他还是有一些隐忧。
<br/>　　“龙腾，我若是在塔中施展全部底牌，外界能否发现？”苏莫问出了第一个隐忧。
<br/>　　“不会发现，外面的人只能发现你在第几层，根本不可能发现你的具体情况，就算是龙帝都不行！”龙腾十分确定的说道。
<br/>　　“那就好，还有第二个问题！”
<br/>　　苏莫点了点头，略一沉吟，道：“假如，我得到了祖龙精血，并且也成功让你融合了，那你会不会有危险，毕竟，你会威胁到吞虚龙帝的地位？”
<br/>　　“你这个问题，之前我也考虑过了！”
<br/>　　龙腾的面色，无比的凝重，道：“龙帝的确是想得到祖龙精血，不过，祖龙精血一旦被我融合之后，等于是彻底的消失了，他就算将我杀死，也炼化不出来；再者，洪荒祖龙是龙族的祖先，在龙族拥有神圣而崇高的地位，我若是融合了祖龙精血，全族尊崇，龙帝都不敢有异议。”
<br/>　　对于此事，龙腾早有思量，虽然说有些风险，但是风险并不大，造化远胜风险十倍。
<br/>　　唯一要注意的是，苏莫得到祖龙精血之后，他要第一时间融合，不然肯定会被人抢夺。
<br/>　　“既然如此，这祖龙塔，我便去闯上一闯！”苏莫深深的吸了口气，沉声说道。
<br/>　　“我进入你的空间宝器之中，只要你夺得了祖龙精血，我就能第一时间融合！”龙腾道，这是最安全的办法。
<br/>　　当然，在苏莫闯塔期间，他觉得不能出来，不然，龙灵就不是一条，而是两条了。
<br/>　　“嗯，那事不宜迟，现在就去吧！”苏莫有些迫不及待了起来，若是真如龙腾所言，对方融合了祖龙精血之后，能在龙族拥有崇高地位，那他就可以无所畏惧了。
<br/>　　只要有龙族撑腰，终极神王殿、天王宫、地王宫，都不足为惧了。
<br/>　　“还有，若是事不可为，你千万不要勉强，不然有陨落的风险！”龙腾再次提醒了一句。
<br/>　　“明白了！”
<br/>　　苏莫点了点头，道：“我和司空炎同去，你先进入我空间宝器之中吧！”
<br/>　　他带着龙腾进入祖龙塔，还是先不要泄露出去为好。
<br/>　　“嗯！”龙腾点头。
<br/>　　随后，苏莫意念一动，裹挟着龙腾进入了他的空间戒指之中。
<br/>　　“司空炎，我们走！”苏莫招呼司空炎一声，便大步向外走去。
<br/>　　“嘿嘿，说的我都心痒了！”司空炎嘿嘿一笑，大步跟上，他也向进入祖龙塔看一看。
<br/>　　当然，他可没有信心取得祖龙精血，只是要看看自己的水平比之龙腾如何。
<br/>　　苏莫和司空炎，走出府邸，向龙都城的中央位置走去。
<br/>　　龙都城的塔楼不少，但是，苏莫和司空炎还是能轻易的分辨，哪一座是祖龙塔。
<br/>　　因为此塔，与所有的塔楼都不相同，首先，这是唯一一座十层的塔楼，其次，此塔仿佛是由青铜打造，每一层塔身之上，都雕刻着一缕栩栩如生的神龙。
<br/>　　这些神龙各不相同，第一层刻画的是黄龙，第二层刻画的是独角银龙，第三层刻画的是银角金龙，第四层是黑暗魔龙，第五层是八翼飞龙，第六层是太虚雷龙，第七层是紫血天龙，第八层是五爪金龙，第九层和第十层刻画的神龙，毕竟特殊，见所未见。
<br/>　　苏莫和司空炎来到了祖龙塔之前，此塔坐落在一片宽敞的广场之上，整个广场上除了祖龙塔之外，别无它物。
<br/>　　而祖龙塔并不是清静之地，有不少龙族之人进进出出，还有不少人聚拢在塔门之外。
<br/>　　那些进入的龙族之人，个个精神熠熠，神情坚定，而出来之人就狼狈了，全部是从塔门处抛飞出来，脸色惨白。
<br/>　　苏莫和司空炎在龙族也算是有很大的名气，两人联袂来到广场，立刻就引起了不少人的关注。
<br/>　　“屠霄和炎空怎么来了？”
<br/>　　“他们两人额头之上的奴印居然消失了？”
<br/>　　“听说是龙帝让他们晋升为外族族人了！”
<br/>　　“看来他们两人，也是要在祖龙塔中证明自己的天赋实力了！”
<br/>　　不少人望着苏莫和司空炎，低声议论，祖龙塔是不允许奴仆和龙族之外的人进入的，但是却允许外族族人进入。
<br/>　　因为只有通过祖龙塔第十层的考验，才能得到祖龙精血，而这无数年来，连第九层都无人进入，更遑论第十层了。
<br/>　　所以，现在的祖龙塔，已经成为龙族之人，磨炼战斗经验、或者是证明自己天赋实力的地方了。
<br/>　　而苏莫和司空炎的到来，倒是没有人会联想到，两人是为了祖龙精血。
<br/>　　毕竟，现如今龙族之中，所有进入祖龙塔的人，都不是为了祖龙精血，因为压根就没有得到的可能。
<br/>　　“炎空，屠霄，恭喜你们成为我族的外族族人！”
<br/>　　“可喜可贺啊！”
<br/>　　“你们也算是实至名归！”
<br/>　　见苏莫和司空炎大步向祖龙塔走来，不少龙族之人，纷纷抱拳，客气的说道。
<br/>　　虽然苏莫和司空炎只是外族族人，地位不高，但是，天赋和实力，还是让很多人打心底里佩服的。
