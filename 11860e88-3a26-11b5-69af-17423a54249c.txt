　　“太苍、凤晴，只有我们所有人联手，才有击败那两人的实力，到时候那两人的宝物，你们两个自然是拿大头。”巫铨面色郑重的说道。
<br/>　　当然，这一切都只是说辞而已，若是他们真的胜了，宝物怎么分配，还要再行商量。
<br/>　　“既然如此，那我们还等什么？”太苍淡然一笑道。
<br/>　　巫铨闻言，顿时眼眸一亮，看来对方是同意了，随即她有看向凤晴。
<br/>　　凤晴虽然没有多少表示，但却是此战的关键人物。
<br/>　　“走吧！”凤晴点了点头。
<br/>　　“好，我来为大家带路！”巫铨朗声一笑，随即大步而行，带着众人向出口处而去。
<br/>　　出口虽然在移动，但是移动的速度和规律是不变的，所有他能轻易找到出口。
<br/>　　此时此刻，迷宫城之外，所有人都期待了起来，关键的一战终于要来了。
<br/>　　龙腾的两位强大的奴仆，即将和凤晴、太苍等一百多人交手，算是巅峰对决了。
<br/>　　若是凤晴和太苍等人落败，那么，迷宫城还剩下的那一小撮人，已经翻不起什么大浪了。
<br/>　　不过，看目前的局势，双方的胜负不好说。
<br/>　　龙腾的两位奴仆，虽然都是万古天骄级别，战力无比强大，但是，凤晴和太苍两人，也是接近万古天骄的存在。
<br/>　　最主要的是，还有一百多个天才奴仆帮助，胜负很难预料。
<br/>　　“龙腾，你认为你的两个奴仆，这次还能胜吗？”龙洌眼珠子一转，微笑着问道。
<br/>　　“必胜！”龙腾面色淡淡，言语无比的笃定。
<br/>　　龙腾的态度，让无数人为之侧目，居然如此有信心。
<br/>　　“既然你如此有信心，那我们赌一次如何？”龙洌嘴角带着笑意，继续说道：“要是你那两位奴仆输了，就让一个给我。”
<br/>　　众人闻言无语，这龙洌的企图，简直是昭然若揭啊。
<br/>　　“那要是胜了呢？”龙腾反问道。
<br/>　　“你若是胜了的话，冰儿和凝雪，我便都给你了！”龙洌指了指身后的两位女奴。
<br/>　　冰儿和凝雪，早已回到了龙洌身后，两人闻听龙洌之言，面色并无波澜，但是，两人的眼眸深处，均是闪过一道异色。
<br/>　　不过，她们也别无它法，身为奴仆，便只能遵从主人的意志。
<br/>　　“呵呵，我有了炎空和屠霄，你认为我还需要其它的奴仆吗？”龙腾耸了耸肩，面带嗤笑之色。
<br/>　　“此言诧异！”龙洌摇了摇头，沉声道：“冰儿和凝雪，不仅天赋很高，姿色也是一等一的存在，最主要的是，他们目前还是处子，你可以好好调教！”
<br/>　　龙洌毫不避讳，直言冰儿和凝雪的优点。
<br/>　　“不好意思，我们志趣不同！”龙腾摇了摇头说道。
<br/>　　“源石、宝物，你还有什么条件，尽管开价！”龙洌还是不死心。
<br/>　　不过，龙腾却是不在理会对方，沉默便是最好的态度。
<br/>　　龙洌见此，脸色一沉，不过，他也没有动怒，以后有的是手段。
<br/>　　“感觉这一战，炎空和屠霄有点悬啊！”
<br/>　　“的确有点悬，双拳难敌四手！”
<br/>　　“不过，他们两人就算不敌，也能随时从出口逃出来！”
<br/>　　周围有不少人，在低声议论，探讨此战的结果。
<br/>　　迷宫城，出口处。
<br/>　　苏莫大马金刀的伫立在那里，而司空炎，则是在低头查看着宝物。
<br/>　　他的手中，不时的出现一枚枚神果、神草、兵器、丹药等等之物。
<br/>　　“苏莫，我总共得到了三十一件宝物，这其中应该只有二十二件宝物是迷宫城之中的宝物，其它的宝物都不是！”司空炎向苏莫说道。
<br/>　　他得到的宝物很多，但大多数的宝物，都是其它奴仆的宝器，或者一些奴仆随身携带的宝物。
<br/>　　虽然大多数进入迷宫城的人，都不带任何宝物，但还是有些人，带了不少宝物。
<br/>　　只不过，这些奴仆自身携带的宝物，肯定不能与迷宫城中的宝物相比，完全不在一个层次。
<br/>　　“二十二件宝物之中，有没有特别逆天的宝物？”苏莫转头问道。
<br/>　　“这些宝物都很特殊，不过，具体的效用，还要回去再研究！”司空炎道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“你呢？得到了多少宝物？”司空炎好奇的问道。
<br/>　　“我得到……！”
<br/>　　苏莫正欲回答，突然顿住了，目光望向通道的尽头，沉声道：“猎物来了！”
<br/>　　司空炎闻言，便也抬头望向远处，通道的尽头，渐渐的传来阵阵的破空之声，显然有大量的人，在急速飞掠而来。
<br/>　　“人数好像还不少！”司空炎舔了舔嘴.唇，冷冷一笑。
<br/>　　少倾，一道道人影，纷纷出现在了视线之中。
<br/>　　一个！
<br/>　　两个！
<br/>　　三个！
<br/>　　一百一十四个！
<br/>　　总共有一百一十四个人，来到了这处通道之中，个个气息强大。
<br/>　　苏莫眸光一扫，看到了巫铨，心中有些好笑，这个家伙的能耐倒真是不小。
<br/>　　“凤晴、太苍，就是他们两人！”通道的尽头，巫铨指了指苏莫和司空炎，面色凝重的说道。
<br/>　　言语之间，他稍稍后退，退向太苍和凤晴身后，不敢做出头之鸟。
<br/>　　他领略过苏莫和司空炎的强大，深知不能大意，安全起见，必须躲在后面。
<br/>　　凤晴和太苍两人，沉默不语，均是在打量这司空炎和苏莫。
<br/>　　他们自然认识司空炎和苏莫，在未进迷宫城之前，就在龙腾身边看到了。
<br/>　　只不过，那时候，并不能看出苏莫和司空炎的不凡。
<br/>　　“你们这帮家伙，只会人多欺负人少，活该做奴仆的命！”司空炎望着众人，面带嘲弄之色，大声的说道。
<br/>　　“做奴仆怎么了？龙族的奴仆也是一种身份！”
<br/>　　巫铨冷笑一声，讥讽道：“你这么厉害，不也是做奴仆的命！”
<br/>　　对于他来说，奴仆的身份虽然有些耻辱，但是，作为龙族的奴仆，在整个天龙神朝同样风光。
<br/>　　毕竟，天龙神朝这么大，龙族之人很少，大部分都是人族和妖族、异族。
<br/>　　他是巫族之人，在神域这片大地之上，根本没有什么地位。
<br/>　　而作为龙族的奴仆，在天龙神朝，还是很有地位的。
<br/>　　“呃……你这个巫族之人，做奴仆居然还做出优越感了！”司空炎闻言愕然，他简直不敢相信，这个家伙居然还认为做奴仆是一种身份！
<br/>　　“哼！你们两个若是现在逃出去，还有机会！”巫铨冷冷的说道，他是在拖延时间，因为，只要拖延片刻，出口就会移动到另一处通道，他逃出去的机会就能更大。
<br/>　　踏！
<br/>　　就在此刻，苏莫向前踏出了一步，面色淡漠的道：“还是那句话，你们想要出去，那就留下所有的宝物！”
