<br/>　　这武皇出手笼罩的巨大手掌，在这无上剑气的面前，直接是被切开了一道裂痕，被当场打碎。

<br/>　　武皇都没有想到，这无上剑气居然蕴含这如此锋利的力量，他的手掌，在这无上剑气面前，都被无情的切开，这虽然只是他以灵力凝聚出来的一道手掌，但是代表着的，可依然是生死台阶第七重，天命之境的力量。
<br/>　　无论如何他都想不到，这居然非但是没有将楚风眠给擒下来，反而是被楚风眠这样轻易的打破。

<br/>　　“小辈，速速住手！”
<br/>　　风皇，阳皇这两尊创始者，也是大声的喊了起来。
<br/>　　同时猛然之间出手，又是两股灵力，同时向着楚风眠笼罩而来，依然是要阻止楚风眠，要将楚风眠给彻底镇压。
<br/>　　“没人可以阻止我，统统给我破！”
<br/>　　楚风眠冰冷长啸一声，突然一滴精血，从他的手指尖中飞了出去，直接落入到了战龙宝车之上，这战龙宝车上的八尊远古战龙上的气息，都是在疯狂之间，节节攀升起来。
<br/>　　一声声的龙吟，响彻而起，在这战龙宝车上面的罗伞，都是出现了八道代表着远古战龙的纹路，突然打开，化为七彩霞光，在这霞光之中，更是还蕴含有着一声声的龙吟。

<br/>　　就算是这两尊天命之境的创始者，一时之间都无法打破。
