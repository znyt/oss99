<br/>    一时之间，全城惊动，他们手中的剑在鞘中嘶鸣，颤动。
<br/>    
<br/>        如同是害怕，如同是敬畏。
<br/>    
<br/>        雾飞花看向身前身子单薄的卓青瑶，卓青瑶鞘中的玉女剑缓缓滑出。
<br/>    
<br/>        雾飞花的身体上感受得到那剑中凌厉的寒意，在那鞘中，到底藏着什么？
<br/>    
<br/>        说实话，卓青瑶的剑道天赋，好的让他都有些嫉妒，怎么说呢，如同为剑而生。
<br/>    
<br/>        阴月天狼将目光落在卓青瑶的身上，他并不相信是卓青瑶领悟了天人合一，在十万大山之中，只有大君才真正踏足了这个境界。
<br/>    
<br/>        他相信这是陈明搞的鬼。
<br/>    
<br/>        同样的，没有人相信，这毕竟只是一个十四五岁的小姑娘，小姑娘还只是闻道境。
<br/>    
<br/>        我这一剑，不问苍生，不问鬼神，不问是非，不问对错，只问四个字，天下第一！
<br/>    
<br/>        然后他们就看到了那如一泓秋水的剑光披靡四野，一剑出鞘，恍如割开天地，不像是一个小姑娘使出的剑，因为这一剑，威猛霸道，寒冷孤寂。
<br/>    
<br/>        一剑祭出，大势无边。
<br/>    
<br/>        沧澜紫金山，浑然变色，剑气狂卷数十丈，卓青瑶手握玉女剑，一剑天光乍泄，数十丈长的剑光落下。
<br/>    
<br/>        雾飞花的双眼之中露出一抹焦急之色，她能够感受到这一剑的强大，甚至在这一剑的强大之下恍然失神，她不敢相信，这只是一个闻道境施展出来的。
<br/>    
<br/>        额间，一缕青丝落下。
<br/>    
<br/>        卓青瑶的剑停在她的肩膀上，卓青瑶淡然收剑转身，“我还以为有多厉害呢，连一剑都接不住。”
<br/>    
<br/>        卓青瑶走向陈明，在陈明的背后静静站着，仿佛刚刚做的一切都只是微不足道而已。
<br/>    
<br/>        凌仙和李素衣对视一眼，终于是明白了为什么卓青瑶才是大师姐，亲娘叻，大师姐越来越恐怖了。
<br/>    
<br/>        一剑，就已经决出了胜负。
<br/>    
<br/>        他们猜得到卓青瑶会赢，但是却永远没有想到卓青瑶会赢得这么干脆。
<br/>    
<br/>        陈明仔细感受了一番卓青瑶身上的剑气，那是，他曾经在那木舟上的尸骨之上感受到的剑意。
<br/>    
<br/>        场面一时之间寂静地可怕。
<br/>    
<br/>        阴月天狼看向卓青瑶，杀意顿起，寒冷肃杀的意念蔓延四方，风雪随着他的意念开始飞扬，周围的大妖们纷纷闪开，不敢继续留在阴月天狼的身旁。
<br/>    
<br/>        阴月天狼将目光转向卓青瑶身前的陈明，滚滚的杀意终于是消散。
<br/>    
<br/>        陈明，他看不穿，在他的眼中，陈明只是入道境。
<br/>    
<br/>        是的，入道境在他的面前，只是蝼蚁一般的存在，可就在刚刚，那个闻道境的小姑娘，一剑天人合一，直接击败了十万大山之中天赋最为可怕的雾飞花，甚至于，雾飞花在这一剑面前，连出手的勇气都没有。
<br/>    
<br/>        阴月天狼伸手扶起雾飞花，“公主，我们该走了。”
<br/>    
<br/>        搀扶着雾飞花，两人离去。
<br/>    
<br/>        离开紫金山之后，阴月天狼问道：“公主，刚刚那一剑，你到底看到了什么？”
<br/>    
<br/>        雾飞花深吸一口气，然后道：“我看到了那个小姑娘的道心，她的道心，只有四个字，天下第一。”
<br/>    
<br/>        阴月天狼道：“这样的人，不该出现在这里，我们该去回禀大君了，一统十万大山，留给我们的时间不多了。下一次，我希望你有勇气对她出手。”
<br/>    
<br/>        雾飞花点点头，道：“不杀了她，我的道心便无法坚定，而这，将会成为我修行路上的魔障。”
<br/>    
<br/>        卓青瑶已经收剑站在陈明的身旁，长河的儿子，小王爷的双腿还在颤抖，他刚刚，招惹的就是这样的一个女魔鬼吗？
<br/>    
<br/>        场面一时寂静的可怕。
<br/>    
<br/>        周围无论是大妖还是妖王，都心中还留有一丝害怕。
<br/>    
<br/>        陈明呵呵一笑，打破了这寂静，“劣徒不知轻重，哈哈哈，献丑了，献丑了！”
<br/>    
<br/>        长河直到现在才缓过神来，从今天起，这十万大山第一天才的称号，可就换了人了，而且还没有任何争议，就是他，也没有想到，只是一剑，就已经决出了胜负。
<br/>    
<br/>        长河看向陈明，能够教出如此天骄的徒弟，怪不得能够斩了妖月。
<br/>    
<br/>        长河笑道：“名师出高徒啊，贤弟可是为我们大妖王涨了脸了！”
<br/>    
<br/>        陈明拱手笑道：“哪里哪里，现在，我们来讨论一下劣徒刚刚赢了一百万灵石的事！”
<br/>    
<br/>        凭本事赢得灵石，怎么可以不要回来？
<br/>    
<br/>        长河呵呵一笑道：“贤弟，你我现在同属一番，共同对抗大君，不过一百万灵石而已，算了吧。”
<br/>    
<br/>        陈明道：“要是你我兄弟之间，那自当无事，但是，这是属于小徒的，长河大哥，是要欺负晚辈吗？”
<br/>    
<br/>        长河怒道：“就欺负了，怎么着吧？”
<br/>    
<br/>        陈明仔细看了一眼长河，合着这就是大妖王，都这么不要脸的吗？
<br/>    
<br/>        长河毫不心虚，“看我做啥？”
<br/>    
<br/>        陈明道：“没啥，只是你刚刚的样子，和还没死的妖月有几分相似。”
<br/>    
<br/>        长河的脸皮抖了抖，“好吧，你们四个小家伙，跟我来......”
<br/>    
<br/>        长河终于是想起了陈明的手下，可是死过一位大妖王的。
<br/>    
<br/>        四个小家伙跟着长河去领了他们赢的灵石，陈明现在也是不太放心，毕竟大君的女儿，还有大君麾下第一忠犬，都已经到了，现在形势可是已经变得不太对了起来。
<br/>    
<br/>        而燕山，自这一战之后，名声更是彻底响彻了十万大山，十万大山第一天才的名头易主，这可不是小事。
<br/>    
<br/>        几乎整个十万大山，都知道了这个消息，更为离奇的，是这个新晋的十万大山第一天才，不过才是闻道境。
<br/>    
<br/>        加上燕山又是刚刚兴起的势力，不为人知，关于燕山的各种猜测层出不穷。
<br/>    
<br/>        陈明亲自带着四个小家伙在到处溜达，凌仙看向前方的一处赌坊，“三师弟，大师姐，四师妹，要不我们再进去赌两把？”
<br/>    
<br/>        陈鱼完全不是怕事的主儿，“嗯，倒不是不行，反正输不了。”
<br/>    
<br/>        陈明横了四个小家伙一眼，你们四个安生点不行吗？凌仙马上改口了，“啊，今晚的阳光真是明媚啊！”
<br/>    
<br/>        李素衣煞有其事的点头，“温暖的阳光，让我都觉得浑身暖洋洋的。”
<br/>    
<br/>        陈明道：“别去惹事了，到处看看有没有什么你们需要的东西。”
