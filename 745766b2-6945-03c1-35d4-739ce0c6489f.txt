　　苏莫和姜神夜暗暗震惊，好奇那位界主，到底是何方神圣？
<br/>　　这时，真贤道圣突然手指连点，点向苏莫和姜神夜的额头，两道淡淡的指芒瞬间没入了两人的额头之中。
<br/>　　而后，两人便感觉到一股信息涌进了脑海中，这是一种法诀的信息。
<br/>　　“这法诀，可以无视地域的限制，开启真圣界的通道，你们可以通过真圣碑来到万界山！”真贤道圣说道。
<br/>　　苏莫两人闻言点头，看来对方要让他们离开万界山了。
<br/>　　苏莫心中略一沉吟，沉声道：“真贤师兄，我在真圣界外，有一仇人，这一出去怕是凶多吉少！”
<br/>　　“怎么回事？”真贤圣师闻言眉头一皱。
<br/>　　“是这样的……！”苏莫简略的将和巫族的仇怨，以及大祭司巫仑等在真圣界外，欲杀他的事情说了一遍。
<br/>　　“还有这种事！”真贤道圣脸色沉了下来。
<br/>　　“是啊！我若是出去，必然会被对方杀死，所以想请真贤师兄帮助！”苏莫点头道，自从来到万界山，他心中也松了口气，想必这位真贤道圣不会不管他。
<br/>　　“那巫族的巫仑大祭司是什么修为？”真贤道圣问道，他虽然对于巫族还算熟悉，但巫族支脉的大祭司，他并不了解。
<br/>　　“好像是武圣境中期！”苏莫说道。
<br/>　　“武圣境中期？那倒不足为惧，本道将他赶走就是！”真贤道圣说道。
<br/>　　苏莫面色凝重，道：“真贤师兄，我与巫族仇深似海，就算您这次保护了我，但总不可能一直保护我，巫族强者随时随地可以对我出手！”
<br/>　　“苏兄，我们三神宗可以保护你，去我们三神宗吧！”姜神夜说道，现在他和苏莫同时加入了万界山，也算是同门了，自然愿意伸出援手。
<br/>　　“三神宗也不能保护我一辈子！”苏莫摇了摇头，除非他一辈子不离开三神宗，不然的话，巫族想要杀他实在是太容易了。
<br/>　　“苏莫，你有什么想法？”真贤道圣问道，他知道苏莫既然如此说，必然有自己的想法。
<br/>　　此言的确在理，他这次可以帮助苏莫赶走巫族大祭司，但不可能将对方击杀，因为巫族也不是小势力。
<br/>　　如此深仇大恨，日后巫族想暗中击杀苏莫，会防不胜防。
<br/>　　苏莫沉吟了片刻，凝声道：“真贤师兄，我有一个办法，或许可以彻底的化去危机，但不知道是否可行？”
<br/>　　“你说。”真贤道圣说道。
<br/>　　“我有一种炼制分身的方法，可以炼制成分身，我的想法是将分身变化为本体的模样，然后让巫族大祭司杀死，而我就可以彻底摆脱巫族的杀机！”
<br/>　　苏莫面色凝重，继续说道：“但是，我无法瞒过大祭司，不知真贤师兄是否有办法，能让我瞒天过海？”
<br/>　　“瞒天过海？”
<br/>　　真贤道圣闻言，沉默了片刻，便点头道：“若那巫仑大祭司只有武圣境中期的修为，本道的确有办法帮助你瞒过他，不过，不可能瞒太久，最多片刻的功夫。”
<br/>　　“片刻就够了！”苏莫眸光一亮，顿时心中大喜，只要能瞒过大祭司，到时候大祭司必然将他的分身击杀。
<br/>　　到时候，巫族就认为他死了，再也不会找他的麻烦，而他就能安心崛起，将来再去报仇。
<br/>　　“真贤师兄、姜兄，你们等我片刻！”
<br/>　　苏莫向两人说了一句，便立刻进入了玉镯空间之中，在地面上留下了一只玉镯。
<br/>　　他也没有什么避讳，空间宝器也不是多珍贵的宝物，没有什么值得避讳的地方。
<br/>　　玉镯空间中。
<br/>　　巫人王正盘膝坐在一片草地上疗伤，他全身经脉破碎，几乎快成了废人。
<br/>　　这些时日他一直在恢复破碎的经脉，以他的修为，这并没有多难，只需两个月的时间，就能恢复如初。
<br/>　　在巫人王的不远处，还漂浮着两只战魂，一只为冥马兽战魂，一只是玉石战魂。
<br/>　　这两种战魂正是巫殇的战魂，只不过这两种战魂极弱，非常的虚幻，几乎快到了湮灭的边缘。
<br/>　　巫殇的两种战魂，本就被苏莫击伤，找受了重创，来到玉镯空间之后，居然还想夺舍巫人王。
<br/>　　但他失败了，巫人王的战魂并没有受伤，精神力强大，根本不是巫殇的战魂能够夺舍的。
<br/>　　最后时刻，巫殇的战魂从巫人王的体内逃了出来，但已经快要油尽灯枯了。
<br/>　　“巫人王，我的战魂需要肉身的滋养，不然坚持不了多久就会消散！”冥马兽战魂的的口中传出巫殇的声音。
<br/>　　“哼！你消散是你的事，别想占据我的肉身！”巫人王连眼眸都没有睁开，冷冷的哼了一声。
<br/>　　对方想占据他的肉身，简直是痴人说梦。
<br/>　　若不是他现在发挥不多什么修为实力，已经将对方灭了。
<br/>　　“现在我们被苏莫所擒，只有联手之下，才有逃生的希望！”巫殇说道。
<br/>　　“放心吧！苏莫只要离开真圣界，大祭司必然会出手！”巫人王说道，他也没有过多的担心。
<br/>　　只是，他心中有些疑惑，按照真圣界的考验规则，应该早已结束了吧？
<br/>　　怎么到现在还没有任何反应？
<br/>　　难道苏莫还没有离开真圣界？
<br/>　　嗡~~
<br/>　　就在此刻，空间一阵轻微的波动，苏莫的身影突兀的出现在了巫人王的面前。
<br/>　　“苏莫！”
<br/>　　巫人王和巫殇见此，顿时吓得急忙后退，想要和苏莫拉开距离。
<br/>　　“巫殇，你可以死了！”
<br/>　　苏莫望向巫殇的两种战魂，吞噬战魂直接释放而出，强大的吞噬之力瞬间笼罩住了对方。
<br/>　　“不！”
<br/>　　在一阵惊恐的叫声之中，巫殇的冥马兽战魂和玉石战魂，瞬间被吞噬旋涡吞噬了。
<br/>　　而另一边，在快速后退的巫人王，见到巫殇的战魂被苏莫吞噬，顿时被这一幕吓的肝胆俱裂，他哪里见到过这种情况？
<br/>　　“你也过来吧！”苏莫又看向巫人王，强大的吞噬之力立刻也笼罩了对方。
<br/>　　巫人王身上有伤，全身经脉恢复了不到两成，实力大损，哪里能抵挡苏莫的吞噬之力，顿时就被强行拉扯了过来。
<br/>　　“苏莫，你想干什么？”巫人王怒吼出声，眸中尽是惊恐之色。
