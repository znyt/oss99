　　　　接下来，李清照陪着涂小安继续在校园内逛了逛。
<br/>　　此时大家都在议论蛇的这个话题上，想必蛇的话题在一个月之内，都会成为温城大学的热门话题。
<br/>　　“小安哥哥，晚上我们一起吃饭吧”
<br/>　　李清照甜美一笑，做了邀请。
<br/>　　涂小安一看现在的时间，平静的说：“现在不过是下午三点，距离吃晚饭还早着”
<br/>　　“不早不早，我带你去参观一下我们女生宿舍，我也要收拾一番，到时候也差不多饭点了”
<br/>　　“你要收拾二个多小时？”
<br/>　　李清照傲娇道：“女生收拾，自然要时间久点，况且这新学期入住，还不都要整理跟打扫吗”
<br/>　　“那你去吧，我自己出去逛逛，到时候你整理好了，给我发微信”
<br/>　　涂小安可没兴趣等这个丫头。
<br/>　　“真的不去吗，我宿舍可是整个学校出了名的美女多，这个时间点，应该有不少人回来了”李清照冲着涂小安挤眉弄眼。
<br/>　　涂小安笃定一声：“不去”
<br/>　　“那这蛇...”李清照看了看肩膀上的太阴蛇，小心机又暴露出来。
<br/>　　“跟着你”
<br/>　　涂小安摇了摇头，转身便走。
<br/>　　“小安哥哥帅帅哒”
<br/>　　李清照兴奋的喊了一声，欢天喜地，朝着宿舍的方向蹦跶。
<br/>　　涂小安朝着校门外走去，脑海中还在思忖校园的群蛇事件。
<br/>　　蛇虽然都游走了，但是问题的关键点他到现在还没找到。
<br/>　　这让他很郁闷。
<br/>　　出了校门，迎对面就是一条繁华的步行街，人来人往，就算是大下午，街上也都是人。
<br/>　　暴露了这座城市繁华跟人流量。
<br/>　　闲来无事，涂小安熟悉熟悉周边的环境，朝着步行街抬腿而去，刚到步行街，突然从拐角处蹦出一个年轻人，没注意一头撞到了涂小安的身上。
<br/>　　啪...
<br/>　　一副墨镜掉到了地上。
<br/>　　而那个年轻人也一屁股坐到了地上。
<br/>　　“你没事吧”涂小安见状，赶忙出手去扶。
<br/>　　年轻人灰头土脸的被扶起来，嘴巴却道：“我没事，我没事，我的眼镜掉了，能麻烦你帮我找找吗”
<br/>　　涂小安一楞，这还用找吗，就在他脚下。
<br/>　　旋即，他才发现这个年轻人的眼睛是紧紧闭着的，头发凌乱不堪，但是长像却非常的出众，英挺的眉和鼻梁，脸颊消瘦但轮廓很好，脸色稍稍有点差，带着丝病态。
<br/>　　这人居然是个瞎子。
<br/>　　他的一双手好像无处安放，这时涂小安才看到地上有一根棍子，想必是他的指路拐杖。
<br/>　　涂小安连忙将墨迹跟拐杖都给捡了起来，抬手把他的刘海撩过一旁，然后将墨镜帮他带好，拐杖放到他手上。
<br/>　　在认真的审视着这位带上墨迹的年轻人，很帅，很酷，脑海中不仅想象了一下这男子挑眉笑的样子，因为能迷倒不少少女。
<br/>　　可惜，他的眼睛是紧紧闭着的，睁不开，睁开了，也想不到这嘹亮的光。
<br/>　　“谢谢你，太感谢你了”年轻人连忙冲着涂小安鞠躬，真挚的表示感谢。
<br/>　　“不用谢”
<br/>　　涂小安随意的回了句，忽然觉得有点奇怪，一个瞎子一个人碰碰撞撞的来到这人山人海的步行街干嘛。
<br/>　　他应该是跟别人一起来的吧。
<br/>　　“你朋友呢，怎么把你一个人丢在这里”
<br/>　　年轻人有点含蓄的抿了下嘴：“我是自己来的，我来找人，我想跟您打听一下，你见过一个叫海棠的女人吗”
<br/>　　“海棠？”
<br/>　　一个瞎子来找人，他家里人怎么心那么大呀，不怕这个也走丢？
<br/>　　“不认识”涂小安摇了摇头：“我也是刚来到温城，你说的海棠是一个小女孩吗，她走丢了？”
<br/>　　“嗯，走丢了，但不是小女孩，是我的女朋友”
<br/>　　涂小安有点听糊涂了，成年人怎么走丢呀。
<br/>　　“你有照片吗，我看看”
<br/>　　年轻人摇了摇头：“我没有她的照片”
<br/>　　“那她又多高，大概什么样子”
<br/>　　一个看不到的人，在大街上乱窜，也不容易，涂小安不禁多问了两句。
<br/>　　年轻人呆滞了下，没有在说，然后再次跟涂小安说了一声谢谢，就转身就走。
<br/>　　他手中的拐杖就跟打草惊蛇一样，在前面的路敲几下，然后走几步。
<br/>　　不小心敲到了一个人，就立刻问道：“你好，我想问一下，你见过一个叫海棠的女人吗”
<br/>　　“不认识”被问的人看到他是个瞎子，淡漠的回了一句，就向前走。
<br/>　　“你好，请问你见过一个叫海棠的女人吗”
<br/>　　这个带墨迹的年轻人，跌跌撞撞的询问路人，逮到一个问一个。
<br/>　　有时候甚至巧合的将一个人逮到连续问几遍。
<br/>　　这一幕幕，将后面的涂小安都给看懵了。
<br/>　　如果是找人，拜托，你起码拿张当事人的照片出来吧。
<br/>　　这就光一个名字向路人问，怎么可能问的出答案。
<br/>　　天底下叫同个名字的人也不在少数。
<br/>　　这简直比大海捞针还要难。
<br/>　　这时，年轻人跌跌撞撞，不小心碰到了一个正在接电话的中年大汉，他一头撞到了大汉的后背上。
<br/>　　“卧靠，你是不是瞎呀，走路不看人”
<br/>　　这位中年大汉原本应该心情就不好，面带怒色，扭头瞪着大眼看着他。
<br/>　　“对不起，对不起”墨镜的年轻人连忙道歉，然后又问：“你好，你见过一个叫海棠的女人吗”
<br/>　　中年大汉见他大白天戴墨镜，用手在年轻人的眼前晃悠了两下，恍然大悟：“卧靠，还真是一个瞎子呀”
<br/>　　“你一个瞎子还出来找人，要是迷路了，还要别人来找你吧”大汉脸上透着讥笑。
<br/>　　年轻人默不作声，拐杖敲敲打打，准备绕路。
<br/>　　中年大汉眼珠子转动，挡住了他的去路：“瞎子，你刚才撞到了我，是不是应该补偿我一下”
<br/>　　年轻人微微一楞，从身上摸，摸了五六百钱币出来，抽出一张：“这位大哥实在对不起，这一百就当是我给你的补偿”
<br/>　　大汉将一百顺手接过来，眼中一抹狡黠：“你个瞎子，这分明是一张十块的，你当一百给我，你当我也瞎啊”
<br/>　　年轻人立刻摇头：“大哥，我刚才给你的那张，就是一百的，我虽然眼睛看不见，但是我的手却认得钱币”
<br/>　　其实对于钱币，是很好摸出来的，十块，五十，一百，质感都是有所不同。
<br/>　　这中年大汉见对方是瞎子，故意将一百说成十块，还想贪的更多。
