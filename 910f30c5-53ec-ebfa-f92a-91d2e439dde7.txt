夏夕绾万万没有想到会是陆子羡，当时她只顾着救人，并没有将注意力放在陆子羡的身上，后来回了海城，她也将这件事给忘了。
<br/>
<br/>        这世界真是太小了，原来两年前他就跟她见过了，还合作了一场惊世骇俗的手术。
<br/>
<br/>        夏夕绾勾起红唇，“那我们还真是有缘分…”
<br/>
<br/>        陆子羡的唇角也勾出了淡淡的弧线，是啊…
<br/>
<br/>        这时就听夏夕绾又加了一句，“你看我现在又成了你的嫂嫂了，这缘分真是不浅。”
<br/>
<br/>        当然夏夕绾在心里又道，这嫂嫂当的挺尴尬的，陆先生跟这个弟弟是同父异母，关键是陆先生还爱吃醋，莫名其妙的就怀疑她跟这个弟弟。
<br/>
<br/>        陆子羡唇角那抹淡淡的弧线迅速消失了，他拿了外套，转身就走了。
<br/>
<br/>        他就这样走了？
<br/>
<br/>        夏夕绾觉得陆家这两兄弟脾气都怪怪的，不过走了也好，虽然陆子羡帮过她，但是她和他关系实在尴尬，还是保持点距离，免得陆先生又生气。
<br/>
<br/>        ……
<br/>
<br/>        天色越来越晚了，外面已经没人了，夏夕绾怕有人来，直接将药房的门给锁上了，然后她拿出一个试管，将细长的针眼推进了自己手臂的血管里。
<br/>
<br/>        眼见着那滴鲜丽到近乎妖冶的曼陀罗花毒就推进了她的体内。
<br/>
<br/>        陆先生的病已经不能再拖了，她知道陆先生很在意自己这个病，他怕自己发病，更怕自己发病的时候会伤害她。
<br/>
<br/>        所以奶奶才说他很想将她紧紧的攥在手心里，但又怕将她给攥疼了。
<br/>
<br/>        夏夕绾放下了针管，现在耳畔寂静无声，她能清晰的感觉到那滴花毒已经快速的蔓延到了全身。
<br/>
<br/>        很快，她觉得浑身的血管里爬满了那种小蚂蚁，那些小蚂蚁在吞噬啃咬着她，锥心刺骨的疼痛。
<br/>
<br/>        夏夕绾额头疼出了一层冷汗，她拿出了银针刺进自己的穴道，那滴花毒从她的指尖慢慢的排了出来。
<br/>
<br/>        这时耳畔传来了声响，有人来了。
<br/>
<br/>        夏夕绾一惊，“谁？”
<br/>
<br/>        “是我。”
<br/>
<br/>        陆子羡的声音。
<br/>
<br/>        他怎么又回来了？
<br/>
<br/>        夏夕绾想起身，但是这时耳膜里突然呼啸而过了那种尖锐刺耳的嗓音，她纤柔的身体直接摔在了地上。
<br/>
<br/>        两只小手捂住了自己的耳朵，她痛苦的蜷缩了起来。
<br/>
<br/>        外面的陆子羡已经听到了里面的异响，本来她将门给反锁了就很异常了，“夏夕绾，你在里面干什么，快开门，不然我踹门了。”
<br/>
<br/>        陆子羡抬脚，一脚踹开了门。
<br/>
<br/>        夏夕绾摔在地上，额头上全是冷汗。
<br/>
<br/>        “你怎么了？”陆子羡迅速上前单膝蹲地，他握住了夏夕绾的小手，将她滴血的指尖放在鼻翼下嗅了嗅，“曼陀罗花毒？你哪来的曼陀罗花，你提炼花毒试毒干什么，夏夕绾，你是不是不要命了？”
<br/>
<br/>        夏夕绾两眼发黑，虚弱道，“我没事…”
<br/>
<br/>        陆子羡拿了一把锋利的刀，将她指尖划出了一个小小的“十”字，然后按住她的穴位帮她加速排毒。
<br/>
<br/>        最后一滴毒排尽的时候，夏夕绾双眼一闭，直接往地上倒去。
<br/>
<br/>        “夏夕绾！”
<br/>
<br/>        陆子羡迅速伸手，夏夕绾软绵绵的倒在了他的怀里。
