　　住所内，此时有一股烤肉的气味在空气中弥漫。
<br/>　　咳咳...
<br/>　　被印在墙壁上的狼人罗子雄不断的咳出血，生命气息急速下降，非常的虚弱。
<br/>　　连续的重击让他几乎要奄奄一息，尤其是刚才那一记拳，蕴含火焰，将他胸口那一片皮肤全部烧焦，疼痛感不是常人所能忍受。
<br/>　　“你别过来，你别过来”满是惊恐之状的罗子雄就像是一片可怜的小纸，被暴风雨随便吹打和蹂埔。
<br/>　　他看来涂小安朝他走来，那脸上的表情，就好像是恶魔在微笑，让人不寒而栗。
<br/>　　忽然，妖人状态的罗子雄全身发生了变化，一阵闪烁光芒起来。
<br/>　　霎时间，他的体型就地缩小，变成了正常的形态。。
<br/>　　嗷...
<br/>　　一道狼嚎声冲天而起。
<br/>　　涂小安脚步一滞，只见一头通体血红毛发的巨狼，携带着恐怖的气息，两只眼睛里发出幽幽的凶光，朝着他通灵摄厉的扑了过来，威势赫赫。
<br/>　　“血月，咬死他”罗子雄像是用尽了全身力气大喝一声。
<br/>　　嗯？
<br/>　　“叮...血月巨狼，银月巨狼的战斗状态，等级二十三级，狼族一脉的王者，血统高贵，高等生物”
<br/>　　“好一匹凶狼，血统高贵，品相极佳，我喜欢”
<br/>　　涂小安见此，不闪不躲，妖治的眼眸浑然一眯，前方空间激荡涟漪起来。
<br/>　　凶煞扑来的血月巨狼直接被禁锢在了半空中，嚎叫连连，满目凶恶。
<br/>　　“你的等级太低了，二十三级可破不了我的意识力束缚”涂小安淡淡一笑，这血月巨狼跟罗子雄脱离了合体的状态，出来护主。
<br/>　　狼是草原上的王者，狼张望的眼神中总让人感到隐隐的杀气和恐怖。
<br/>　　这显然是一头狼王了，虽然距离证道，成为真正的狼王还有很长的一条路走，但也绝对是一方领袖了。
<br/>　　可惜，这头狼王，是光杆司令，没有一群手下，一只狼可以捕获一只羊，一百只狼却可以屠杀一万只羊，双拳难敌四手，恶虎敌不过群狼。
<br/>　　任由它达到精怪级别，但面对的却是二十八级的涂小安，战斗力比一般的三阶妖灵还要强，两者差距宛如鸿沟。
<br/>　　“你还不错，但你这个主人就不怎么样，配不上你，给你换个主人可好”
<br/>　　涂小安眼中闪过一丝玩味，看着禁锢空中的血月巨狼，轻挑眉目，浅笑吟吟起来。
<br/>　　此话一出，血月巨狼满目凶光，引颈长嚎，声震四野，可无奈破不了涂小安的意识苏束缚。
<br/>　　还烙印在墙壁中的罗子雄听到涂小安说这句话，忽然觉得有点耳熟，这话自己好像对那只孔雀说过。
<br/>　　他想干什么，还想抢自己的血月巨狼不成？
<br/>　　“来，让我帮帮你”涂小安根本不理会罗子雄，眼睛再次狠狠一眯，一股意识威压释放出来。
<br/>　　血月巨狼顿时嚎叫连连，半晌之后，它跟罗子雄同时一口重血喷了出来。
<br/>　　涂小安用强大的意识力破解了这头巨狼跟罗子雄的血之契约。
<br/>　　“怎么回事，你做了什么，为什么我跟血月没有灵魂契约存在了”
<br/>　　罗子雄惊慌失措了起来，瞬间整个人懵逼，他感受不到自己跟战宠之间的亲密联系了。
<br/>　　也就是说，这头血月巨狼，基本上，跟他没有一毛钱关系了。
<br/>　　他是怎么做到的，可以能破解自己跟战宠的血之契约，变态，恶魔，罗子雄要疯了。
<br/>　　其实这并不是涂小安第一次破解血之契约，相对别人的大惊小怪，他可风轻云淡了。
<br/>　　“现在，臣服于我，认我为主吧”
<br/>　　涂小安现在对这匹血月巨狼施行强制性的收服，意识威压排山倒海的侵袭着它。
<br/>　　“嗷...”
<br/>　　血月巨狼仰天长啸，两只眼睛依然还隐隐凶光，拼死反抗。
<br/>　　“叮...强制性收服二十三级血月巨狼失败”
<br/>　　嗯？
<br/>　　涂小安蹙眉，他是高等级压制，如果是同族的蛇，那么一个眨眼就能收服，不能反抗。
<br/>　　可惜对方是一匹高等血脉的狼王，不同的种族，让他没那么好收服。
<br/>　　另外罗子雄虽然被涂小安打个半死，但这匹冲出来的巨狼，无论是精气神，或是生命力都还非常的旺盛。
<br/>　　它扛住了涂小安的意识威压。
<br/>　　哼，既然这样，那就别怪我不客气了。
<br/>　　涂小安看着禁锢在空中的血月巨狼，体内雷电之力运转。
<br/>　　“惊雷...”
<br/>　　劈...
<br/>　　一道白光乍起，毫无预兆的劈在了血月巨狼的身上，它被禁锢的身体，就是一个活靶子。
<br/>　　劈...劈...劈...
<br/>　　上空连续五六道雷电劈在了血月巨狼的身上，硬生生将它满是凶光的眼神劈的惊恐之状起来。
<br/>　　“臣服与我，否则你就是死”
<br/>　　涂小安大喝一声，强大的蛇类气息笼罩在血月巨狼的身上，意识威压再次强制收服。
<br/>　　不将它劈个半死不活，还真不好强制收服。
<br/>　　呜呜...
<br/>　　血月巨狼绝望了，实力上的巨大差距，让它只能任人宰割。
<br/>　　它充满凶性跟野性，可在涂小安面前，是条龙也要盘着，是头老虎，也要给趴着。
<br/>　　“给我收”
<br/>　　只见禁锢在空中的血月巨狼，再也无法挣扎跟反抗，瞬间骤然消失。
<br/>　　下一秒，等它再次出现的时候，就已经站在了涂小安的面前。
<br/>　　它通体血红毛发慢慢的淡化了，变成了银白，发出了一道阴森而低沉的声音：“银狼拜见主人”
<br/>　　银月巨狼低下了高傲的头颅，趴在了涂小安的脚下。
<br/>　　“银狼，我才是你的主人，你给我回来，混账东西，孽畜...”
<br/>　　墙壁上的罗子雄发出一种受伤狮子般的怒吼声。
<br/>　　这什么情况，他瞪大眼睛，看着自己的战宠趴在了蛇妖的脚下，摇尾乞怜起来，好无凶性，变的非常乖巧。
<br/>　　罗子雄的咆哮，让银月巨狼置若罔闻，好无反应。
<br/>　　“别叫了，他现在是我的战宠了”涂小安目光重新投放在罗子雄身上。
<br/>　　罗子雄浑身颤栗起来，像在梦中被惊醒似地，道：“好，这银狼我送与你，就当我向你战宠下手，给你赔礼道歉，你现在放我一马”
<br/>　　什么叫偷鸡不成蚀把米，这便是，罗子雄现在是肠子都悔青了。
<br/>　　他怎会想到，堂堂最强新生，既然是一只三阶妖灵，他还作死的想将对方孔雀抓来，送给大小姐邪七七。
<br/>　　现在他什么都可以不管不顾，只要能保命，留的青山在，不怕没柴烧。
