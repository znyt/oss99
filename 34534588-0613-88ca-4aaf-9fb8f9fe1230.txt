　　话虽如此，但慕清霄也只是吓一吓她，就凭他身为蜀山长老，徐长卿是蜀山的弟子，他就不可能下手。
<br/>　　更何况，徐长卿上面有个清微老头子罩着。
<br/>　　他有自己原则，不可能为了一颗水灵珠，如此丧心病狂，当然，这也成不了让他放弃水灵珠的理由。
<br/>　　紫萱下意识的看了一眼自己的小腹，深深的瞥了眼慕清霄。
<br/>　　此刻，她才认识到什么样的人最可怕，是将你秘密摸的一清二楚，而且完全不守原则的人最可怕。
<br/>　　因为，谁也不知道，在下一秒，他会做出怎样疯狂的事情来。
<br/>　　慕清霄见她举棋不定的模样，并没有在出口威胁，而是给她充足的时间来思考。
<br/>　　紫萱是一个聪明的女子，同时也是一个用情至深的痴情女子，为了爱甘愿一次次飞蛾扑火，甚至不顾一切的为一个人逆转生死。
<br/>　　所以，不用慕清霄开口，她也会做出最正确的决定。
<br/>　　沉吟片刻，紫萱徐徐开口道：“就算不将水灵珠给你，你也会强行拿走，对吗？”
<br/>　　“不错，水灵珠我会拿走。”
<br/>　　不置可否，慕清霄确实会拿走，但会在不伤及她腹中小生命的同时，将水灵珠取走。
<br/>　　“你说的不错，我有女娲嫡系传承，就算没有水灵珠也不会死，境界没了可以重修，我将水灵珠给你，但我也有一个要求。”
<br/>　　虽说是第一次接触，但紫萱何等聪慧，完全能看出慕清霄并非恶人。
<br/>　　否则，根本无需在这里与她说一堆废话，直接强行取走她的水灵珠岂不是更好，他拥有这样的能力。
<br/>　　但她也能看出，后者对水灵珠是志在必得，与其让他动手，倒不如用水灵珠换取一个条件。
<br/>　　慕清霄诧异的看了一眼紫萱，道：“果然没让我失望，你是个聪明的女子，知道如何取舍，虽然没必要，但我答应你。”
<br/>　　“难道你能猜出我的要求？”
<br/>　　紫萱有些惊讶，她什么都没说，后者居然就答应了她的要求。
<br/>　　“无非就是徐长卿。”
<br/>　　话语刚落，紫萱嫣然一笑。
<br/>　　“虽然他此刻记不起我们的过往，但我相信，终有一天他会想起来，我希望他能坐上蜀山掌门的位置，你刚才已经答应了我。”
<br/>　　慕清霄微微一愣，现也没想，道：“我答应你。”
<br/>　　他没理由不答应，因为在不久的将来，徐长卿可不就是蜀山掌门么，根本就不用他做任何的事情。
<br/>　　“好，我需要散功，请你去外面稍等。”
<br/>　　慕清霄深深的看了一眼紫萱，为了一个男人做出这样的决定，这样一个女子值得尊敬。
<br/>　　刚走出竹楼，就看到圣姑靠在围栏上，脸色极其苍白，并且一脸不善的望着他。
<br/>　　慕清霄脸上扬起一抹坏笑，道：“怎么，圣姑的职责除了保护女娲后裔，原来还有听墙角的习惯？”
<br/>　　“哼，今日的事情，我迟早会找你报回来。”冷哼一声，圣姑瞥了一眼竹楼，转身径直离开。
<br/>　　“咻……”
<br/>　　玉手抓住袭来之物，圣姑柳眉一皱，摊开手心，其中正躺着一颗圆润如玉，龙眼大小的丹药，香气扑鼻。
<br/>　　虽然不知几阶丹药，但圣姑心底却升起一抹异样，回头看了一眼，哪里还有慕清霄的身影。
<br/>　　紫萱大乘期修为，想要散功，保住胎儿的同时，顺利取出水灵珠，需要一定的时间。
<br/>　　慕清霄并不在意的短短几天，出了灵溪谷的结界后，并没有离开，而是在盘坐在河边，打坐静心修炼起来。
<br/>　　平心静神，五心向天。
<br/>　　运转心法后，慕清霄上空就出现一个小型旋窝，四周灵气以极快的速度汇聚过来，最后灌入他的体内。
<br/>　　刚才与圣姑一战，让他对自身的实力有了一个判断。
<br/>　　在大乘九重巅峰就施展仙品剑诀，他能够重创圣姑，如果后者运气差，会被一剑击杀也不算奇怪。
<br/>　　但圣姑是毒修，战斗进展太快，她根本没来得及施展就被慕清霄吓到了。
<br/>　　正是因此，慕清霄占据主动权，才能一剑将她重创，若是对战渡劫九重的剑修，可能情况就不一定了。
<br/>　　况且，施展仙品剑诀，需要浩瀚的灵气，消耗根本不是一般修士能够承受的。
<br/>　　如果普通的大乘九重巅峰修士，暂且不说能不能施展，就算能施展出来，也会在一瞬间导致灵海枯竭。
<br/>　　到时候根基受损是小，重则可能直接成为废人。
<br/>　　慕清霄能够在大乘期就施展仙品剑诀，完全是因为他修炼的心法等级过高的缘故，他的灵气本就比别人精纯十倍，甚至百倍。
<br/>　　饶是如此，只是施展了一次《御剑伏魔》，阴阳湖中的阴阳灵气居然被抽掉了五分之一。
<br/>　　这五分之一的灵气，需要慕清霄连续吸收三天灵气才能填补回去。
<br/>　　随着时间的推移，三天的时间转眼即逝。
<br/>　　此时，慕清霄阴阳湖中的灵气已经填满，但体内灵气依旧是饱和状态，哪怕一丝都不可能在吸收。
<br/>　　而且，阴阳玉中，魂剑已经暂时停止炼化焚寂剑。
<br/>　　慕清霄知道，这是瓶颈的缘故，除非他冲破瓶颈，顺利迈入渡劫期，肉身再度蜕变，方能继续继续吸收。
<br/>　　对此，他也是相当的无奈，如果一天不能迈入渡劫，哪怕不断的打坐也相当于白费时间。
<br/>　　正如清微老头子所言，渡劫是凡修的最后一个境界，哪怕是命运之子也不是想破就破的，需要时间，更需要悟性。
<br/>　　历练对于慕清霄来说，是最适合的办法。
<br/>　　不过，也并非没有办法，清微老头子说机缘一到可能就会突破，所以，慕清霄要去寻找机缘。
<br/>　　在仙剑三位面修炼个百年，那绝对是不可能的。
<br/>　　所以，慕清霄打算拿到水灵珠后，直接前往斗破位面，里面任务没完成是其一，最主要的是里面有突破渡劫的办法。
<br/>　　陀舍古帝中有突破渡劫的办法，只要将所有异火集齐即可。
<br/>　　慕清霄现在有这样的实力，而且很久没回去，他也有些不放心，毕竟小医仙的厄难毒体一直都是个问题。
