<br/>　　江行泽闻言，将下巴微微抬了下，面目可憎，淡淡的发出如同野兽的低沉声，道：“你在说一遍，我刚没听清楚”

<br/>　　门卫吞吞吐吐的还未说完，江行泽暴戾的一只手倏然的掐住了门卫的脖子，举重若轻的将他提到了半空中，用沙哑阴冷的声音，道：“你知不知道，我要是不进去，我就会死，你是不是想我死”
<br/>　　“江...少...饶..命...”门卫脖子被掐住，一种窒息感压的他喘不过气，一张脸瞬间涨红起来。
<br/>　　黑袍下的江行泽面露疯狂之色，配上他那张千疮百孔，面目可憎的脸，简直让人看一眼，能做好几日的噩梦。
<br/>　　“你是不是想我死，你说，你是不是想我死”
<br/>　　门卫拼命的挣扎，可那只手宛如铁钳，让他半点难以挣脱。
<br/>　　江行泽狂妄猖獗，眼中满是杀气：“谁想我死，我就会先让他死”
<br/>　　“救...救..命..救命...”
<br/>　　门卫已经开始在翻白眼了，死亡的气息笼罩着他。
<br/>　　“轰...”
<br/>　　江行泽重重的一把将门卫砸到了地上，顿时门卫口吐鲜血，翻了翻白眼，直接晕了过去。

<br/>　　“师父，你可不是逼我，你不让我进去，是不是后悔救我了”
<br/>　　他站在原地呢喃自语，露出不屑的讥笑。
<br/>　　涅槃重生捡回一条命的江行泽，换来的却是内心的仇恨跟疯狂，他现在就像是一位行走在地狱边缘的死徒，死过一次，变成不人不鬼，反而通灵摄厉起来。
<br/>　　曾经的江行泽是一位帅气的年轻小伙，纵然心高气傲，目中无人，但还是活在阳光下，可现如今，他不但外形凶恶狰狞，内心更是宛如一条致命的毒蛇，随时要咬人，夺人性命。
<br/>　　俗话说的好：傻的怕楞的，楞的怕不要命的。
<br/>　　他现在就是一位疯狂而不要命的赌徒，可以六亲不认。
<br/>　　冷漠的斜睨了一眼晕迷的门卫，江行泽朝着深处牢笼走去，刚没走两步，他停滞了，又返回到门卫的面前，嘴角一拉，一抹阴冷的笑，徒手抓住门卫的一只脚。
<br/>　　拖着望里走。
<br/>　　一扇铁门被打开，迎面便是空气中一股潮湿跟血腥味道，让人不禁作呕。

<br/>　　“吼...”
<br/>　　江行泽刚踏进牢笼，一声如同深渊地狱的野兽嘶吼声便冲天而起，那一片片火光流动了起来，愈演愈烈，看上去如同葬身火海。
<br/>　　“宝贝，接着”
<br/>　　江行泽眼神一凝，用力一抛，将晕迷的门卫顺着那片一窜一跳的火光抛去。

<br/>　　门卫被这突然来临的震动转醒了，掀开眼，一颗狰狞可怖的偌大蛇头散发贪婪的光，盯着他看，不断吐出一条长长的舌头。
<br/>　　此一眼，门卫三魂不见了七魄，顿时惊恐的大叫起来。
<br/>　　这一叫，转眼便笼罩在一张巨大的血盆大口中，霎时，生机全无。
<br/>　　接下来便是一幅非常残忍了的蛇吃人画面，而那窜跳的火光，不过是此蟒身上的火红鳞片，如同火焰在沸腾燃烧。

<br/>　　而山野集团近日谣言中吃人的怪兽，恐怕也正是这条吸血蛇无疑。
<br/>　　变成不人不鬼的江行泽就是它的帮凶。

<br/>　　江行泽含着笑，如同在欣赏一幅至美的画卷，看的津津乐道，他口中的肉，指的却是人肉，而不是牲畜的肉。

<br/>　　江行泽目光投放在一身火红鳞片而进食的吸血蛇身上，发出沙哑的声音：“宝贝，现在也该轮到我享受一下了”
