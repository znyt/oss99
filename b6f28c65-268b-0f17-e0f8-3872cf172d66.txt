　　不过众人走到一半的时候就遇到了一大堆丧尸。
<br/>　　足足上百头。
<br/>　　“下车！速度解决战斗！”
<br/>　　“得，要开始忙咯！”
<br/>　　几个干事立即下车去解决战斗。他们倒不害怕，毕竟都是干事，杀过的丧尸没五百也有一百了。
<br/>　　几个干事最弱的都是五阶了，大部分都已经六阶了。众人把车子并称一排，形成一道铁皮防线。
<br/>　　几百个丧尸虽然够多。
<br/>　　不过凭借着经验丰富一刻钟时间全部给他解决掉。
<br/>　　林峰也是第一次看到韩风出手。
<br/>　　这个家伙算是这些干事之中最厉害的，据说之前是个武馆教练。
<br/>　　对方的武器居然是个双节棍，就好像李小龙一样，一棍子扫去就是十几个丧尸脑袋被砸爆了。
<br/>　　其余的干事都是些砍刀水管这类常见的武器。
<br/>　　不过一刻钟，所有挡路丧尸被干掉。
<br/>　　“要是能够把军部的那个丧尸战车给借来就好了。”
<br/>　　刘峰道。
<br/>　　“丧尸战车？”林峰好奇道。
<br/>　　“那是什么鬼。”
<br/>　　“一个装甲车改装的，浑身都是钢板，还加装了炮筒口，可以发射燃烧弹照明弹之类的武器，前面还加装了铲子，是军部新研发出来的专门用来开路的战车。反正特别牛逼。不过军部目前也就一共五辆。”刘峰道。
<br/>　　林峰闻言厉害啊，这军部的黑科技还真是够多的。
<br/>　　干掉了丧尸一行人继续朝前推进。
<br/>　　不断出现陆陆续续的丧尸，不过都不算多。
<br/>　　车队干脆就笔直一路碾压过去，时不时就能听到扑出噗嗤就好像西瓜被砸烂的声音，坐在车内的每个人都知道，那是丧尸的脑袋被轮胎压爆了。
<br/>　　很快。
<br/>　　众人抵达了黑龙镇。
<br/>　　“我去，这么多丧尸？”
<br/>　　每个人都从车子上跳下来，他们距离黑龙镇还有几百米距离。
<br/>　　但隔着老远就可以看到一个死气沉沉的小镇子，许多丧尸漫无目的的四处闲逛，偶然可以看到几头变异狗变异猫在镇子内啃着丧尸或者爬在屋顶上休息。
<br/>　　一个个战士会之人都从车子上跳下来。
<br/>　　看着黑龙镇内也都是表情有些沉重。
<br/>　　“这丧尸少说五六千个了吧。”
<br/>　　刘峰扫了眼喃喃道。
<br/>　　“这就怕了？”
<br/>　　寒鸦从车上跳下来。
<br/>　　“现在，每个人都去找到自己的队伍。然后开始行动。”
<br/>　　每个干事立即开始行动，来都来了，怕有个毛用，要是真的怕了，那么当初还不如在躺在被窝里美滋滋的抱着女模特睡觉，何必来这里冒险，一群人立即去后面车队带队伍去了。
<br/>　　“林峰，你确定要单独行动吗？”
<br/>　　寒鸦道。
<br/>　　“嗯，我一个人习惯了，人太多，我不好管理。”
<br/>　　林峰道。
<br/>　　寒鸦点了点头没有多说什么。
<br/>　　每个人都手心里冒汗，他们虽然都杀过丧尸，但是这么多的，还是很少见到。
<br/>　　“出发。”
<br/>　　上千人直接全部冲了过去。
<br/>　　一辆卡车在前面带头冲锋，冲到一半的时候，司机就利索的从驾驶室内跳出来。
<br/>　　然后那卡车继续慢悠悠的朝前冲。
<br/>　　寒鸦点了根烟，火柴朝那卡车货箱内一丢。
<br/>　　顿时噼里啪啦。
<br/>　　居然响起了鞭炮声。
<br/>　　鞭炮声特别的刺耳。
<br/>　　顿时一群丧尸目光都被卡车吸引纷纷怒吼着朝卡车撕咬而来。很快这个卡车就被包围的里三层外三层。每个丧尸都伸着双手要抓车厢内的东西。
<br/>　　砰！
<br/>　　下一刻。
<br/>　　卡车直接爆炸成为一道火光顿时几百只丧尸被火焰吞噬。
<br/>　　然后就是第二辆。
<br/>　　依然是这样的办法。
<br/>　　砰！
<br/>　　第二辆卡车爆炸再次吞没了几百个丧尸。
<br/>　　然而第三辆第四辆。
<br/>　　足足事先准备的四辆充当诱饵的卡车全部引爆，直接消灭了上千只丧尸。一切进行的十分顺利。
<br/>　　不过爆炸声也彻底的把黑龙镇内的丧尸和怪物们惊醒了，一道道怒吼犹如雷鸣一样响起。
<br/>　　寒鸦拿出了信号枪就是咻的一下朝天发射。
<br/>　　顿时一群丧尸抬头看着这个信号弹，然后朝信号弹掉落的地方追去，尤其是一些变异狗变异猫。他们的动作极为的迅捷。
<br/>　　噗噗噗
<br/>　　一瞬间，寒鸦就是朝不同方位射出去了好几个信号弹，他这么做的目的，就是要把聚集在一起的丧尸给分散开来。化整为零。
<br/>　　顿时原本黑压压一片的丧尸群开始四分五裂起来，当然了，其中最多的一股丧尸群，朝战士会这边杀来。
<br/>　　因为他们除了可以看到可以听到，最主要的，他们还可以闻到活人的气息。
<br/>　　“杀！”
<br/>　　韩风刘峰等干事纷纷深吸一口气，，握紧了手中的武器，然后带队冲锋。
<br/>　　寒鸦和罗敏也紧随其后而去，林峰自然没有打酱油，也是左手武士刀，右手斩灵刀。
<br/>　　狂龙出海！
<br/>　　青莲剑歌！
<br/>　　林峰一刀下去十几个丧尸头颅飞去，一脚下去，可以把丧尸踹飞个几米远顺带着还砸到七八个前赴后继的丧尸。
<br/>　　其余干事也都颇为凶猛，在干事的带头下，战士会的普通成员们也都拿出了自己的实力和战士大杀特杀起来。
<br/>　　时不时有丧尸被砸爆头颅。
<br/>　　但时不时也有人被丧尸扑倒。
<br/>　　“妈的，去死吧！你们这群肮脏丑陋的东西！”
<br/>　　一个普通成员一棍子砸爆了一个丧尸脑袋。
<br/>　　还没反应过来，旁边冲来了一个三阶丧尸直接把他扑倒在地，然后七八个丧尸全部扑了上来。
<br/>　　“啊啊啊，我不想死，你们这群杂碎，滚开，啊啊，奶奶，我喜欢吃你的饺子，啊啊啊爸爸我错了，我不该和你吵架…………”
<br/>　　惨叫声逐渐消失，只剩下了丧尸的咀嚼声。
<br/>　　“啊啊，救我啊，谁来救我啊，我不该来的，我为什么要来，啊啊啊啊。”
<br/>　　“老子王者荣耀最强王者，你们这群丧尸去死吧！”
<br/>　　“我不想死，我不想死。”
<br/>　　“太多了，丧尸太多了，我们要死的！！！”
<br/>　　也有正式成员。看到同伴被扑倒，心中被畏惧充斥，失去了勇气，掉头就跑，但没跑几步就被丧尸追上扑倒在地。
<br/>　　“求求你们别杀我，啊啊啊啊，王云我喜欢你啊啊啊啊。”
<br/>　　……
<br/>　　随着战士会众人和丧尸的全面接触。
<br/>　　各式各样的惨叫不断响起。这是一场背水一战的战斗。
<br/>　　林峰耳畔已经听不到人的声音了，他只能够听到丧尸的怒吼，但是他还是可以看到寒鸦射出去的信号弹。
<br/>　　不过此时此刻，信号弹已经没有多大的诱饵作用了。许多跑到一半的丧尸群全部都调头跑了回来。
<br/>　　林峰一刀劈掉前面围过来的十几个丧尸然而又是十几个丧尸前赴后继扑上来。
<br/>　　“妈的，这里到底多少丧尸！”
<br/>　　林峰直接拿出了火龙符箓。
<br/>　　顿时。
<br/>　　轰的一下。
<br/>　　火焰爆发直接吞没了林峰身边的丧尸，一个个丧尸被烧的皮开肉绽，但是这群玩意生命力太强大了，只要脑子还在，就不会死，哪怕烧成木炭都可以动。
