　　第352章
<br/>　　凉州城，城主府。
<br/>　　钟武才胆战心惊的站在大厅，便是抬头看上一眼不肯。今天在大厅坐的，都是整个西北的大人物。甚至在整个帝国，也算得上大人物。
<br/>　　凉州牧宋伯康，年轻时曾中过状元，也是帝国文官势力的代表人物。也有人说，他将是当今帝国丞相百年之后，最有可能接任相位的人。
<br/>　　西北宣武将军戴白，统十万凉州铁骑，他出身相府，娶的是相府的千金，按辈分算下来，是薛问婷的姑父。
<br/>　　参军长史陆洛，出身江南陆家。陆氏一族乃是江南巨富，民间有陆氏半天下的说法。
<br/>　　……
<br/>　　包括钟武才的父亲也在其内，众人沉着脸，表情都不是很好看。
<br/>　　“你怎么回来的，仔细再说一遍？”宋伯康问。
<br/>　　回来的路上钟武才已经在不停琢磨该怎么说，刚才已说了一遍，此刻更加熟练。
<br/>　　“姓程的一知道我们身份就怕了，尤其是听到宋大人的名字，赶紧陪不是，特意送我回来，现在只要我们去个人，就可以把人领回来……”
<br/>　　众人听罢之后面面相觑，半晌，陆洛问：“你说他怕了？”
<br/>　　“当然，吓得他差点跪在地上，这不一刻也不敢耽搁，就让我回来了，还要我向宋大人好好解释。”
<br/>　　陆洛略倾身子，问：“他既然怕了，为何不遣人送你回来，又为何不把所有人一起送回来？”
<br/>　　“这……”钟武才试探道：“会不会是他吓怂了，没想到这个。”
<br/>　　“混账！”钟武才的父亲钟伟虎终于忍不住了：“赶紧滚下去，不要在这里丢人现眼！”
<br/>　　“父亲，我到底哪里说错了？”
<br/>　　“滚！”钟伟虎手中的茶杯丢了出去。
<br/>　　待将钟武才赶出去后，钟伟虎余怒未消，他向宋伯康道：“属下教子无方，还请大人降罪。”
<br/>　　宋伯康挥挥手，道：“还是想办法把人救回来要紧，如今他们落在程大雷手中，时间拖得越久越危险。”
<br/>　　“大人，程大雷既然放人回来传信，游渠几人便应该不会有性命之忧，无非就是想扣下人勒索钱财而已。”戴白道。
<br/>　　宋伯康沉思片刻，道：“还要派人去一趟，看他想要什么，先将人救回来要紧。诸位，谁敢走一趟琴川关？”
<br/>　　以程大雷的人性，不能不令众人畏惧，一言谈不拢，或许就死在琴川关了。千金之子，坐不垂堂，诸人心里不得不掂量一番。
<br/>　　“两国交兵，不斩来使，便是戎族也知道这规矩，这程大雷该不会不懂这些吧？”戴白。
<br/>　　“若是戎族还是好办的，毕竟戎族的信用无问题，可程大雷……”陆洛叹口气：“没人说他是个讲规矩的人呐。”
<br/>　　诸人暗暗赞同，当着帝国天子的面，他敢杀杨龙停。对于这个人，很难寄希望于他懂规矩，讲礼貌。
<br/>　　“谁敢去一趟琴川关，会一会那程大雷？”宋伯康问。
<br/>　　这声音落地，大厅上雅雀无声，戴白看向陆洛，陆洛看向天空。
<br/>　　“戴将军……”宋伯康道。
<br/>　　“咳咳咳，属下军务缠身，实在脱不开。”
<br/>　　“陆先生？”
<br/>　　“呃，咳咳，属下昨夜着了风寒，今日还头疼不止，实在难堪大人。”
<br/>　　“属下刚得高人算了一卦，近日不宜远行，否则必有血光之灾。”
<br/>　　你一言我一语，与刚才的安静截然不同，此刻乱哄哄一片，都是推脱的声音。
<br/>　　“混账！”宋伯康怒拍桌案：“程大雷不过是个小小的山贼而已，有那么可怕么，难道偌大凉州，就没个不怕他的。”
<br/>　　众人眼观心，心观口，闭口不言。性命是大，绝不可乱开玩笑。
<br/>　　“宋大人！”钟武才起身，深施一礼：“属下愿走一遭，救回公子。”
<br/>　　如果这件事和钟武才没关系，钟伟虎也不愿趟这次浑水。天子尚且避醉汉，自己再怎么说也是个体面人，不要和程大雷这种混账计较。可没办法，其他人都扣着，只有钟武才一人跑了回来。而且还在大家面前丢这么大人，为了自己的前程，为了钟武才的前程，他都必须走这一遭。
<br/>　　儿子闯出来的祸，老子就该站起来扛，没办法，钟伟虎都得去一趟。
<br/>　　……
<br/>　　琴川。
<br/>　　河水滔滔，有一人影精赤着膀子，立在河中央。
<br/>　　他手中握着一柄大斧，当水浪袭来，便以大斧劈砍过去。
<br/>　　浪花分开，冰凉的河水溅在他身上，顺着干净的身体流下来。
<br/>　　一次又一次，这个过程不断重复，他脸上已经呈现出疲态，可双目中的凶光却越来越瘆人。
<br/>　　这几天，程大雷一有时间便在河中磨练斧技。刑天三斧的威力太过恐怖，不但伤人，而且伤主。也就在河中，水流的助力可以限制程大雷的力量，使他不至于陷入疯狂。
<br/>　　每在筋疲力尽时，程大雷尝试着去控制这恐怖的力量，一次又一次，乐此不疲。
<br/>　　经常程大雷都会伤痕累累，皮肤渗出血水来，这是因为肌肉承受了太大力量。
<br/>　　而在忍受痛苦和疲惫的同时，程大雷的实力也在与日俱增，他渐渐已经习惯控制这股力量。
<br/>　　这不是一朝顿悟，而是水滴石穿，循序渐进，终究到了这一日。程大雷立于河中，双手握着大斧，一个浪头袭来，程大雷闭目，吸气，猛地发力，大斧狠狠抡下。
<br/>　　大斧宛若凝了一团白光，裹夹的磅礴力量施放出来，浑浊的河水眼睁睁分开一条线，露出下面的沙石。
<br/>　　程大雷倒抽了一口凉气，脸上浮现难以置信的表情。
<br/>　　这就是绝世的力量么……常听人说千人敌万人敌，以为是书生戏言。可眼前这一幕却令程大雷不得不相信：或许在这个世界，并不缺万人敌的虎将。
<br/>　　幸好，自己也是其中一员。
<br/>　　程大雷看着手中的巨斧，露出沾沾自喜的表情。
<br/>　　忽地，一个浪头袭来，将毫无防备的程大雷冲倒。他连滚带爬，好险是抓住一颗石头，从河底爬到岸边。
<br/>　　浑身泥泞不堪，身体筋疲力尽，他往黑牛背上一趴，一拍牛头：走着！
<br/>　　
<br/>(本章完)
