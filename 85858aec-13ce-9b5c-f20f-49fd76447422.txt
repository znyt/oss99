　　楚风眠隐隐预约的感觉到，这玉佩之中也隐藏着什么，但是想要参悟，可没有那般的容易。
<br/>　　“你还要买吗？”
<br/>　　老者看着楚风眠将玉佩收了起来，突然对着楚风眠开口道。
<br/>　　这让众人又是吃了一惊。
<br/>　　这老者虽然是一个老骗子，可是他从来不会主动找人说话的，一般就算是有人问他买东西，都是爱答不理的。
<br/>　　现在居然是主动来找楚风眠说话？
<br/>　　“这老骗子今天是怎么了？”
<br/>　　“看来这老骗子，也打算在坑一坑这冤大头了。”
<br/>　　“这小子的财力惊人，没看拿出圣品灵石来，都不心疼吗？”
<br/>　　众人都有些惊奇，一群人围在旁边，倒是看着热闹，这样冤大头被坑的事，他们平日里想要看还看不着呢。
<br/>　　“买。”
<br/>　　楚风眠看着老者，随意笑了笑道。
<br/>　　“还有什么东西？”
<br/>　　楚风眠的话一出，人群不少武者的眼皮都是不由的跳了一下。
<br/>　　还要买？
<br/>　　这就算是楚风眠出身自什么大宗门，是什么大宗门的少爷，也没有这样败家灵石的呀。
<br/>　　“这件。”
<br/>　　老者听到楚风眠的话，也不墨迹，手心一转，又是一件东西出现在了他的手心中。
<br/>　　“这是，剑柄？”
<br/>　　一道道目光望了过去，每个人的脸色都有些奇怪，仔细的认了一下，一些人才看了出来。
<br/>　　这一次老者拿出来的，似乎是一件剑柄，似乎是一把灵器的一部分。
<br/>　　那古籍，玉佩，还可以说是完整的东西，这一次的剑柄，可就连一件完整的灵器都算不上了。
<br/>　　楚风眠没有开口。
<br/>　　他的目光已经是死死的落在这了剑柄上，自从是这剑柄拿出来的一刻，楚风眠的目光就有些无法离开这剑柄。
<br/>　　“多少灵石？”
<br/>　　楚风眠看了一眼，沉声说道。
<br/>　　这剑柄，比起那玉符，那古籍，对楚风眠带来的吸引力还要大。
<br/>　　“还要买？”
<br/>　　“这小子是真的疯了？”
<br/>　　“疯子，这小子绝对是一个疯子……”
<br/>　　听到楚风眠的话，一群人大惊失色道。
<br/>　　“怎么回事？前面怎么了？这么多人？”
<br/>　　就在这人群后面，突然传来了一声冷傲的声音，只见一名年轻男子，走了过来。
<br/>　　在他的面前，还站着几名武者，一路上为他开路。
<br/>　　“都让开，都让开，让本少看看，到底是什么东西？”
<br/>　　那几名武者，立刻挤开了人群，打开了一条通路来。
<br/>　　那些被挤开的武者，本是脸色都不怎么好看，刚想要呵斥，当看到这年轻男子的时候，都是脸色一变。
<br/>　　“是战魔宗的少主，纣绝阴！”
<br/>　　战魔宗，六道魔门麾下的六宗之一。
<br/>　　现在六道魔门崛起，六宗也随之崛起，在这归墟城内，少有不认识这纣绝阴的人。
<br/>　　看来纣绝阴走过来，几名武者更是都不敢看热闹了，直接走了，至于其他人也都是退后几步。
<br/>　　六道魔门的人，手段凶残无比，这纣绝阴更是手段残忍到极致，只要是稍微得罪过他的人，可都没有好下场。
<br/>　　“老家伙，是什么宝物啊，拿出来看看。”
<br/>　　那纣绝阴没有理会人群，而是直接向着那老者走了过去。
<br/>　　老者没有说话，只是挥了挥手中的剑柄。
<br/>　　“什么？就这一把剑柄，就是所谓的宝物？”
<br/>　　纣绝阴看到这剑柄的一刻，目光闪烁，露出几分凶意道。
<br/>　　“老家伙，你耍我？”
<br/>　　“不卖就走。”
<br/>　　老者冰冷的回答道，随即将剑柄收了起来，态度一如既往的冰冷，丝毫没有因为这纣绝阴的身份而改变。
<br/>　　“糟了。”
<br/>　　“老骗子要完了。”
<br/>　　这句话一出，众人都是心中一沉。
<br/>　　果然那纣绝阴的表情，在听到老者的这一句话后，当即变的阴沉了起来，眼神之中都露出了几分杀机。
<br/>　　“好一个老不死的，敢这么跟本少说话？找死？给我去死！”
<br/>　　纣绝阴凝聚灵力，即将出手的一刻。
<br/>　　在他旁边的一位老者，急忙开口道。
<br/>　　“少主请住手，这毕竟是归墟城内，您出手，也会对主人有所影响，还有陛下的口谕……”
<br/>　　“陛下的口谕……”
<br/>　　纣绝阴听到这话，目光闪烁，眼神之中的杀机，缓缓的压了下去。
<br/>　　“很好，老家伙，今日我给你一个机会，不杀你。”
<br/>　　这纣绝阴说完，倒是没有走的打算，反而是手一挥，叫人在旁边，搬了一把椅子过来，就坐在了那老者的摊位前。
<br/>　　“不过我今日倒是要看看，谁敢从你这里买东西！”
<br/>　　说着，这纣绝阴的目光扫视了所有人。
<br/>　　众人一下子也都明白，这纣绝阴是想要断了老者的财路。
<br/>　　毕竟在这里摆摊的，都是求财的，这纣绝阴坐在这里，摆明了是要逼着老者离开。
<br/>　　毕竟可没有几个人，敢为了这老者，去得罪纣绝阴。
<br/>　　现在九域之中的共识，都是宁可得罪其他六大宗门，也不可得罪六道魔门。
<br/>　　毕竟其他六大宗门，做起事来，还算是有些底线，可六道魔门做事，却是从来没有底线这二字。
<br/>　　老者倒是没有因为这纣绝阴坐在面前，而有着任何的变化，他的目光之中，甚至都没有将这纣绝阴放在眼中，而是看了一眼楚风眠，开口道。

<br/>　　纣绝阴阴狠的开口道。
<br/>　　在归墟城里他不好出手，但是一旦这老者只要是敢离开归墟城，他在出手可就没有任何的顾忌了。

<br/>　　这纣绝阴心中的算计还没有结束，突然看见一道人影，走了过来。
<br/>　　“买，当然要买？什么价格？”
<br/>　　楚风眠根本没有去理会纣绝阴的威胁，而是径直走了过去，对着老者开口道。
<br/>　　“小子，你没有听到本少的话？”
<br/>　　那纣绝阴看着楚风眠不理会他的话，更是将他的威胁瞥到一旁，心中的怒火更盛。
<br/>　　今日这一个老者不给他面子也就算了，他的威胁都说出来了，楚风眠都毫不在乎？
<br/>　　在纣绝阴身边的一名六道魔门的弟子，也是不客气的说道。
<br/>　　“小子，我家少主的话你没有听见吗？快滚！”
