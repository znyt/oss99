        &lt;h3&gt;第三卷第二百七十一章  我只想与你携手同游人间&lt;/h3&gt;
<br/>
<br/>        宁雅在饭桌上很少发言，一直在默默的吃着东西，在家里基本也轮不上她发言。
<br/>
<br/>        听着他们谈笑风生，讲着一些趣事，她也会跟着笑一下，晚上的飞机，她晚上得回北京，下午就让律师直接过去签合同就好了。
<br/>
<br/>        “小雅。”宁少天终于想起了自己的女儿。
<br/>
<br/>        “嗯，爸。”宁雅抬起头看着宁少天。
<br/>
<br/>        “今天的合作我很开心，谢谢你，我代表宁家上上下下敬你一杯。”宁少天举起了酒杯。
<br/>
<br/>        “爸，不用，我也没有为家里做什么，陈子昂既然让我来，肯定会有她的想法，我觉得公平是对我们合作最好的诠释。”宁雅解释了一下，毕竟宁少天是她的爸爸，她希望她的爸爸能够体谅她。
<br/>
<br/>        “哈哈哈哈，我们难道不公平吗？对你们公司我们已经做了很大的让步，要不是因为你，我怕是不会同意的，不过既然你已经出了宁家的门，我希望你以后好自为之，就好好的为陈子昂谋福利吧，做人呢，最主要的是要忠诚，对父母，对上司。”宁少天将杯中酒一饮而尽。
<br/>
<br/>        “爸？”宁雅声音颤抖，她的父亲这是要放弃她，永远的放弃她，出了宁家的门？他怎么就不了解她的苦心？陈子昂的公司目前无论是在国内还是在国际都占有一席之地，他们和陈子昂合作哪里会受了委屈？而且她也要马上离开陈子昂的公司了，给了他们这样一个好的合作机会，他竟然过河拆桥想要放弃她？这就是她的好父亲吗？
<br/>
<br/>        “宁伯伯，不知道小雅哪里做得不对，要让你如此说她？她能在别的公司得到重用，可想而知是有能力的，可是为什么她在你们宁家就是一个无用之人，为什么她在宁家就会受到排挤？现在你竟然还想将她赶出宁家？”凌辰站起来了。
<br/>
<br/>        “辰哥哥，你激动什么嘛，和你有什么关系啊，快坐下。”宁玥伸手拽凌辰的手。
<br/>
<br/>        “凌辰！”凌思永语气严厉，这孩子没大没小，还没有和宁玥结婚呢。
<br/>
<br/>        “放开你的手！”凌辰一巴掌拍开了宁玥的手。
<br/>
<br/>        “你......”宁玥一副楚楚可怜的样子，眼泪跃跃欲滴。
<br/>
<br/>        “凌辰，你应该叫小雅姐姐，而不是小雅，不管怎么说，宁雅是我们宁家人，这个是她和她父亲之间的问题，也是我们宁家的事情，你和宁玥还没有结婚，现在轮不到你在这里说三道四，你的父亲今天来提亲，我们是答应了，但宁玥能不能嫁给你，还要看你的表现。”兰乔脸上挂起了一层寒霜。
<br/>
<br/>        “你说什么？向宁玥提亲？谁要向宁玥提亲？”凌辰语气森冷。
<br/>
<br/>        “你和宁玥都已经生米煮成熟饭了，你还不向她提亲？你还要等到什么时候？”凌思永痛心疾首，这孩子怎么就成了这样，难道不应该为宁玥负责吗？
<br/>
<br/>        宁雅看着一桌子的人，看着凌辰，看着他们的嘴一张一合，耳边只是嗡嗡作响，她拧紧了眉头。
<br/>
<br/>        “你胡说，我什么时候和她生米煮成熟饭了？”凌辰气急，小雅你可千万别误会啊。
<br/>
<br/>        “辰哥哥，在山西的那些天，我们.......我们不是一直在一起吗？你......你.......我.......”宁玥已经泣不成声了。
<br/>
<br/>        “姐姐，我和辰哥哥已经在一起了，你不要再将他抢走好吗？从小只要是好东西，都是先着你的，这一次你就让我一次好吗？”宁玥梨花带雨的看着宁雅。
<br/>
<br/>        “成何体统，成何体统，宁雅，你看看你做的好事情。”宁少天捶胸顿足。
<br/>
<br/>        “亲家，别生气，都怪我们这个臭小子，不过，我从来没有听凌辰说过宁雅，你们是不是误会了？”凌思永一脸懵，今天的这个饭吃得好奇怪啊。
<br/>
<br/>        凌思永接到宁玥的电话，宁玥说凌辰这几天在杭州，准备向她提亲，希望凌思永能来向她爸爸提亲，而不是凌辰提亲，这样显得正式一些。
<br/>
<br/>        凌思永听了心中大喜，凌辰终于是开了窍，春节带着宁玥回家，他就觉得有戏，儿子的终生大事他得出面。
<br/>
<br/>        凌思永被喜悦冲昏了头脑，喜滋滋的按照宁玥提供的地址上了宁家的门，兰乔在家里等着凌思永，在家里聊了半个小时，带着凌思永到了宁家公司。
<br/>
<br/>        现在他感觉有些不对劲，宁家俩个女儿，一个高冷端庄，一个小家碧玉哭哭啼啼，好似和他儿子都有关系。
<br/>
<br/>        “煮豆燃豆萁，豆在釜中泣，本是同根生，相煎何太急？”宁雅冷冷的看着面前的几个人，心里凉透了。
<br/>
<br/>        “既然我在宁家让父亲感觉丢脸了，那我现在就离开，祝愿父亲今后幸福安康!”宁雅站起来拿起酒杯：“第一杯，敬父亲的生养之恩，第二杯，敬父亲的教养之恩，第三杯，敬父亲的不责之恩。”
<br/>
<br/>        宁雅喝下三杯酒，拿起包，头也不回的离开了。
<br/>
<br/>        凌辰站起来，准备去追宁雅，被宁玥拉住了：“放手，我不想动手，我怕脏了我的手。”
<br/>
<br/>        可是宁玥死死的拽住他的衣袖，凌辰“撕”将整个袖子撕下来扔在了宁玥的脸上：“渣女！”
<br/>
<br/>        “我拜托你做事情前，先过过脑子，别别人说什么你就信什么，你好歹还是凌家的老大。”凌辰冷冰冰的对他的父亲说完转身去追宁雅去了。
<br/>
<br/>        什么荣华富贵？什么门当户对，什么家世股份，统统的都滚蛋吧，他只想要愿得一人心白首不离分。
<br/>
<br/>        “小雅，小雅。”凌辰追着宁雅到了宁雅住着的酒店。
<br/>
<br/>        “你来干什么？”宁雅看着凌辰，她现在已经无心想别的，她的心里一直有个声音在说：“宁雅，努力吧，努力做你自己吧！”
<br/>
<br/>        “我来陪着你，无论你是什么样的宁雅，我都会一直陪在你身边，无论贫穷贵贱！”凌辰怕宁雅将他关在门外，双手撑在门框上。
<br/>
<br/>        “你的衣服袖子呢？”宁雅看着凌辰缺掉的一只手袖，显得有些滑稽。
<br/>
<br/>        “割袍断义你知道吗？tui，没有义，是特别的生气。”凌辰真的是生气，宁玥在山西的那几天，都有董亚子一起陪着，他哪有和她生米煮成熟饭？简直是毁他清白，他还要脸呢。
<br/>
<br/>        “噗嗤，我陪你去商场买衣服吧。”宁雅看着凌辰那气呼呼的样子，感觉很好笑。
<br/>
<br/>        “你还笑得出来？”
<br/>
<br/>        “我怎么就笑不出来了？没有希望也就没有失望，什么样的结局我都会接受。”宁雅微微一笑，人真的是要改变自己才能接受外界带给你所有的一切苦难及折磨。
<br/>
<br/>        “那你对我呢？”凌辰的安全感降到了零。
<br/>
<br/>        “你去不去买衣服？”宁雅嗔怪的看着凌辰。
<br/>
<br/>        “去，去、去。”
<br/>
<br/>        凌辰和宁雅此时不会去顾及什么家庭了，只要俩个人能够在一起，他们就是幸福的，俩个人淡然的手拉着手逛商城去了。
<br/>
<br/>        至于以后，他们既然心有所属便不相弃亦不相离，他们知道面临着他们的将会是狂风暴雨，他们十指相扣已经在心理上做好了准备。
<br/>
<br/>        我于万千人海中与你相遇，只是想携手与你同游人间，紧握双手将不会松开。
