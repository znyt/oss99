　　“我没有监护入门弟子的经验，我不也不想出差错。”
<br/>　　知道百里屠苏推脱，但陵端阴险狡诈，自然不肯让他推脱掉，否则自己如何实施接下来的计划。
<br/>　　“你是怕难当重任吧，也是，除了大师兄给你的那件法宝外，你还有什么本事？”
<br/>　　陵端咄咄逼人，冷笑一声，接着讽刺道：“只怕到时候，监护弟子不成，自己倒出了丑，那丢的不仅仅是天墉城的人，更是丢了你师傅，执剑长老的脸。”
<br/>　　话语刚落，百里屠苏脸色一变，正色道：“不会有这样的事情，我去便是了。”
<br/>　　说到底，百里屠苏只是一个十六七岁的少年，常年待在天墉城，自然是应付不了陵端这种阴险狡诈之辈。
<br/>　　...
<br/>　　...
<br/>　　随着时间的推移，夕阳西下，夜幕降临。
<br/>　　翡翠谷，是天墉城右边的一座深山幽谷，其中妖魔都被尽数斩杀，但不乏有妖灵存活了下来。
<br/>　　数百年来，这里就作为新进弟子的考核地点，也是让这些新进弟子锻炼胆魄的好地方。
<br/>　　夜晚，在百里屠苏的带领下，一众新进弟子来到翡翠谷，翡翠谷是天墉城胜景之一，刚踏入此地，众人就被这里的风景给迷住。
<br/>　　“这仙境一样的地方，那来什么妖怪啊？”
<br/>　　“听说妖怪都是晚上出没，咱们还是去林中找些干柴来，好应付这一晚。”
<br/>　　有了一名弟子领头，其余弟子也纷纷跟了上去，很快找来几堆干柴，升起火堆，围成一圈，开始谈天说地。
<br/>　　有的弟子梦想着修成仙术，下山行侠仗义，有的希望与天同寿，有的希望娇妻成群......
<br/>　　当然，梦想是梦想，现实是现实，根本不能混为一谈。
<br/>　　以他们的根骨，或许努力一点能够成为内门精英弟子，但想要与天同寿却是完全不可能的。
<br/>　　新进弟子谈天说地，百里屠苏则抱着长剑，坐在树林一旁，面无表情，一副生人勿近的模样。
<br/>　　由于身具焚寂煞气的缘故，百里屠苏这些年来，几乎一直待在执剑堂后山修炼，整个天墉城与他接触的也就寥寥几人而已。
<br/>　　正是因此，他不敢与旁人多接触，性格也逐渐变得沉默寡言，冷漠异常。
<br/>　　树林火堆旁边，风晴雪双手撑着下巴，时不时看向百里屠苏，眼眸中闪过一抹思索，但随后又摇了摇头。
<br/>　　察觉到风晴雪的模样，欧阳少恭面露微笑，瞥了眼百里屠苏，道：“晴雪，你要找的人，该不会就是这位师兄啊？”
<br/>　　风晴雪闻言，语气坚定道：“不，不会是他的，他不会这么一副冷冰冰的模样。”
<br/>　　“是吗？我倒是觉得他似曾相识。”
<br/>　　欧阳少恭眼底闪过一抹精芒，其实在看到百里屠苏的第一眼，他就知道，所谓的百里屠苏，正是他要找的韩云溪。
<br/>　　化名能够骗得了风晴雪，却骗不了他，他能清晰的感觉到，自己的另一半仙魂，正是被封印在了百里屠苏的体内。
<br/>　　所以，现在的百里屠苏，绝对是当年的韩云溪。
<br/>　　若非这里是天墉城，他暂时打不过紫胤真人，他早就将百里屠苏抹杀掉，抽出自己的另一半仙魂。
<br/>　　想到此处，欧阳少恭知道，无论是自己的仙魂，或者是焚寂剑都需要智取。
<br/>　　“你等等，我去套套他的话。”
<br/>　　说罢，欧阳少恭几步就来到百里屠苏身旁坐下，道：“师兄，在下欧阳少恭，我觉得师兄您特别面熟，咋们以前是不是见过？”
<br/>　　闻言，百里屠苏依旧面无表情。
<br/>　　关于儿时的记忆有些模糊不清，但师傅说他从小就是在天墉城长大，自然不可能认识面前的俊美青年，以及所谓的风晴雪。
<br/>　　与他熟悉的，整个天墉城也就大师兄陵越与芙蕖，所以立刻将欧阳少恭当成了攀关系的那一类人，偏过头不想搭理。
<br/>　　见他的模样，欧阳少恭也不在意，继续问道：“风晴雪，您认识吗？”
<br/>　　见欧阳少恭似乎打算死缠烂打，百里屠苏露出一副似笑非笑的模样。
<br/>　　“可能是我记错了，打扰了。”
<br/>　　见他这幅模样，欧阳少恭清楚，在继续询问下去也是无济于事。
<br/>　　正所谓，眼睛是通往心灵的一扇窗，刚才问话的时候，百里屠苏眼中闪过思索，想必是想不起来以前的事情。
<br/>　　有两种可能，第一是百里屠苏失忆，第二种就是记忆被强行封印了起来，完全找不到以前的痕迹。
<br/>　　天墉城擅长封印之术，由紫胤真人施展，封印百里屠苏的记忆，绝对轻松无比。
<br/>　　虽然清楚百里屠苏就是他要找的韩云溪，但欧阳少恭却不可能点破，反倒是开始策划接下来的计谋。
<br/>　　抽离仙魂夺走焚寂的同时，他又不想与紫胤真人正面交锋。
<br/>　　欧阳少恭礼貌的离开，令得百里屠苏有些诧异，脑海中盘旋着风晴雪三个字，但他真的听都没听说过。
<br/>　　见欧阳少恭走过来，风晴雪眼中闪过一丝期待，急忙问道：“怎么样？那个师兄，是你的故人吗？”
<br/>　　“他一个字都没说，确实挺冷漠的，估计心里是在想，我可能是在攀关系吧。”
<br/>　　欧阳少恭没有半点迟疑，轻轻摇了摇头，韩云溪的身份由他暴露，可能会引起怀疑，他自然不可能告诉风晴雪。
<br/>　　“别着急，过了今晚，等我们进入天墉城，一定还有机会。”
<br/>　　随着时间的推移，夜色深沉，树林中的气氛也变得阴冷起来。
<br/>　　由于生着火堆，也可能是人多，又有师兄陪护的关系，新进弟子们都一脸轻松，早就将妖灵抛到了九霄云外。
<br/>　　这时，其中一名弟子指了指百里屠苏，道：“你们看那位师兄，坐那里快两个时辰了吧，一动不动，跟块木头似得。”
<br/>　　“师兄，他该不会是在练功吧？”
<br/>　　“我觉得不是，万一妖灵真的出现了，他能保护的了我们吗？”
<br/>　　其中一名弟子早就看百里屠苏不顺眼，冷笑一声，道：“一副生人勿近，而且跟个哑巴似得，一句话都不说，你说哑巴会斩妖术吗？”
<br/>　　“呵，谁知道呢？”
