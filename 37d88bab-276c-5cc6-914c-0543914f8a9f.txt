　　哐！哐！哐！
<br/>　　三枚美制M84闪光震撼弹发出相当于2400万支烛火的强烈闪光和540分贝左右的巨大噪声。
<br/>　　巨大的噪声对Derrick的耳膜造成了不可逆的伤害，他的耳道中渗出了鲜血；强烈的闪光将他掀翻在地上，眼前白茫茫的一片，完全看不到任何东西。
<br/>　　等到Derrick视力有所恢复的时候，恍惚间他看到的是一个个荷枪实弹的DEA行动队员从大门和工厂屋顶索降进工厂内。
<br/>　　“来的还真及时。”
<br/>　　Derrick嘴里嘀咕了一句，但是此时他根本听不到任何东西，即便是他自己说得话也一样听不清楚。
<br/>　　刀疤熊很快就意识到发生了什么，这次交易是Derrick临时通知他来进行的，没有任何的计划，所以知道的人非常有限，如果有人会出卖这次交易的话，那么就只有Derrick；只是刀疤熊根本来不及对Derrick采取什么动作，他就被两个DEA队员控制了起来。
<br/>　　工厂中有零星的枪声响起，DEA和刀疤熊手下的保镖发生了交火，但是交火很快就结束了。刀疤熊、跟着刀疤熊一起来的那个女人，以及他手下的那些保镖，还有Derrick都被DEA带走了。
<br/>　　当Derrick被DEA队员带出工厂，押上汽车的时候，他看到了停在工厂外的防弹路虎，Derrick认出了那是“维伦恩”生物制药公司保安队的车，如果DEA行动迟到十分钟，那么他现在被带上的汽车就不是DEA的押解车，而是“维伦恩”生物制药公司保安队的防弹路虎，Derrick看向坐在防弹路虎中的克罗恩，嘴角勾起，露出一抹笑容。
<br/>　　克罗恩看着Derrick被DEA带走，但是他却无可奈何。克罗恩拨通了霍斯曼德办公室的电话。
<br/>　　“人找到了吗？”霍斯曼德着急的开口问道。
<br/>　　“找到了。”克罗恩回答道。
<br/>　　“找到了就立刻带回来！”霍斯曼德用命令的口吻对克罗恩说道。
<br/>　　“抱歉，我想我没办法把他带回来。”克罗恩回答道。
<br/>　　“为什么！？”
<br/>　　霍斯曼德“噌”的一下从办公桌背后的椅子上站起来。
<br/>　　“因为他被DEA的人带走了。”
<br/>　　“什么！？”霍斯曼德诧异地问道。
<br/>　　“他被DEA的人带走了。”克罗恩回答道：“就在两分钟前，他和刀疤熊一起被DEA的人带走了，就从我面前经过。”
<br/>　　“你应该把他拦下来，应该把他拦下来！”霍斯曼德几乎是在对着克罗恩怒吼了。
<br/>　　“拜托，你在跟我开玩笑吗？”克罗恩摊了摊手，说道：“你让我带着人去跟DEA开战，然后把人从DEA手上抢下来吗？我拿钱是给你们解决那些你们解决不了的麻烦，不是从你们手里拿钱然后去送死的。如果我跟DEA开战，很快我跟我的人就会被军方的卫星锁定，然后无人机会出现在我们的头顶，只要得到许可，一颗‘地狱火’就能送我们去见上帝，这些事情我比你更了解。”
<br/>　　说完，克罗恩就挂断了电话，把手机扔在中控台上，骂了一句“白痴”，然后离开从废弃食品加工厂外离开。
<br/>　　……
<br/>　　“维伦恩”生物制药公司，总裁办公室。
<br/>　　霍斯曼德没有把情况隐瞒下来，立刻就跟玛丽科娃取得了联系。视频中，玛丽科娃抚摸着抱在手中的波斯猫，那双能令男人迷失在其中的眼睛轻轻眯起，突然笑了起来，说道：“果然很聪明，我没有看错他，没有枉费我费了那么大的功夫把他从监狱里弄出来，但是我既然能把他从监狱里弄出来一次，他难道认为躲在监狱里就会绝对安全吗？这件事情你不用管了，我会去处理。”
<br/>　　说完，玛丽科娃就中断了跟霍斯曼德的视频通讯，然后拿起桌子上的电话拨通了一个号码。
<br/>　　……
<br/>　　杭山市。
<br/>　　钱鑫在失踪后不久，钱鑫的父母就报了案，由于钱鑫父母保安的时候，距离钱鑫失踪还不到24个小时，按照程序还不能立案。
<br/>　　钱鑫父母回去后，继续等待着消息，因为以前钱鑫也有过在外面瞎混不回家的事情，但是从来没有一次像这次这么长时间不跟家里联系。焦急之下，钱鑫父母再次去了公安局。
<br/>　　钱鑫失踪正式被立案调查，而在钱鑫失踪被立案后不久，也有人来报案陆岑失踪。
<br/>　　陆岑是外来人员，所以来报案的是住在陆岑旁边那套公寓里的小姐妹。她们从事的都是相同的行当，她们经常会带男人回去过夜，但是好几个晚上都没看到陆岑回公寓，打她电话也没有人接，陆岑的小姐妹觉得情况不对，才到公安局报案。
<br/>　　杭山市，刑侦队会议室。
<br/>　　钱鑫和陆岑的失踪案件已经被正式立案并且并案调查，刑侦队也成立了专案小组，负责专案小组的是刑侦队队长周宋捷。
<br/>　　周宋捷用激光笔在投影布的两张照片上画着圈，对坐在会议桌旁边的刑侦队员说道：“钱鑫，小区保安，九天前在跟家人吃过晚饭后去上班途中失踪；陆岑，夜总会小姐，最后一次被人目击也是九天前，然后不知所踪。现在基本可以认为，这两个人的失踪存在着必然联系。”
<br/>　　“说不定是私奔了呢。”
<br/>　　会议桌边，今年刚从警校毕业分配到刑侦支队的吴晓飞小声嘀咕。
<br/>　　“这不可能。”
<br/>　　坐在吴晓飞旁边，比吴晓飞早两年进入刑侦队的陆昕说道：“陆岑这样的女人就算是要私奔，也不会跟钱鑫这样的人私奔，钱鑫根本养不起陆岑。”
<br/>　　陆昕拿起一张在陆岑公寓取证时拍下来的照片，递给吴晓飞说道：“知道这是什么吗？”
<br/>　　吴晓飞拿过照片看了眼，把照片放在桌子上，很随意的说道：“只不过是一支口红而已。”
<br/>　　“想要找女朋友首先先得了解女人们喜欢什么。”陆昕拿起照片说道：“这支萝卜丁口红售价在1200块，而陆岑那个化妆桌上，总共有超过十支不同颜色的萝卜丁口红，就这么一支口红的价格就相当于钱鑫工资的三分之一，而且还不包括陆岑所使用的名牌皮包和名牌衣服，你觉得以钱鑫的经济实力能承担得起陆岑平时的消费吗？”
