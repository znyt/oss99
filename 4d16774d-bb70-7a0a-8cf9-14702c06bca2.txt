　　看上去四五岁大的一个小孩，头上顶着一串好像草藤一样的东西，圆头圆脑的，看上去带着几分滑稽。
<br/>　　美华看到他的时候，他也看到了美华，四目直视，小家伙怔了怔，转身想逃，但是想到外面的环境更加可怕，又折返回来，对美华说道：“外面有人要杀我，我借这个地方躲一躲。”
<br/>　　美华愣了半天，想到什么，问道：“谁追杀你，你是什么人？”
<br/>　　“就是那些臭道士。”小男孩也想到了什么，歪头盯着美华，说道：“你就是他那个仆从吧？”
<br/>　　“你是什么人。”说话间美华已经站了起来。
<br/>　　小男孩有点警觉，往后退了退，道：“我看你似乎也是鬼域之魂，为何甘愿给一个道士做仆人？”
<br/>　　美华不理会，问道：“你到底是谁，怎么会到这里来？”
<br/>　　“我……我与你同样都是异类，切不可自相残杀……”
<br/>　　美华冷冷一笑，“我已认主，心中只有主人，哪里还有什么同类，你既然是被他逼进来的，那我也不能放过你！”
<br/>　　说着凌空冲了过去。
<br/>　　小男孩咬了咬牙，也冲过来，两只手突然化作青藤，朝美华卷过来，不过也就是看着诡异点，美华几下就解决了他，一掌将他拍飞，自己也是放下心来，这种实力……在这里肯定是跑不出去了。
<br/>　　美华没有着急过去，而是冲他戏弄地笑了笑，道：“毛孩子，还打么？”
<br/>　　小男孩挣扎着站起来，四下观察了一下，觉得自己不可能有机会逃得出去，也不是面前这个美女阿姨的对手，犹豫了一下，摆起手道：“罢了，不打了，你那个主人端的厉害，没想到你也这么厉害，不过这里是虚空化境，我的一手本事施展不出来，我认输了，不打了。”
<br/>　　“那你老实点，让我把你绑起来。”
<br/>　　“绑起来，为什么要绑起来？”小男孩一脸无辜。
<br/>　　“你不乖一点，也就是多费点周折，自己选。”
<br/>　　小男孩犹豫了一下，伸出双手，恨恨说道：“你舍得的话就来吧！”
<br/>　　美华笑道：“你又不是我儿子，我有什么舍不得，何况只是绑住你而已。”
<br/>　　小男孩眼前一亮，冲她喊道：“那我认你做干娘吧，干娘！”
<br/>　　美华身子晃了晃，狠狠瞪了他一眼，“好不害臊！”
<br/>　　走了过去，口中吐出一道水汽，将他浑身缠绕，裹得像个粽子，这才伸手摸了摸他的脑袋，“这下子好了，你在这呆着，我出去见老大。”
<br/>　　叶少阳和众人一起烧了通灵佛子花，正要越过花坛继续往前走，阴阳镜中突然飞出一道人影，落在他面前，叫道：“老大。”
<br/>　　叶少阳怔了一下道：“你怎么出来了？”
<br/>　　“有一个家伙，刚才闯进阴阳镜里，被我抓住了，来问问你怎么处置。”
<br/>　　什么东西？
<br/>　　叶少阳怔了好半天，惊道：“不可能吧，什么时候的事，我怎么不知道。是什么人？”
<br/>　　美华看了看周围几人好奇的目光，说道：“要在这里说吗？”
<br/>　　叶少阳心中一动，道：“这样，你不是抓住他了吗，先给他捆好，我现在没工夫，等我从古墓里出去，到时候再细问。”
<br/>　　美华答应下来，再度钻进了阴阳镜里。
<br/>　　“她刚才说的，什么意思。”妙心第一个问道，“什么人去了你的阴阳镜里？”
<br/>　　“我也不知道，真不知道。”叶少阳耸了耸肩，仔细回忆，自己的确一点也想不起来这档子事，难道有什么东西偷偷钻进了阴阳镜，为什么要这么做？
<br/>　　“那你不去检查一下？”妙心追问。
<br/>　　“反正被抓住了，出去再说吧，我不想在这里多呆下去，让你们等着。”
<br/>　　听他这么说，妙心也隐约明白了什么，瞟了一眼陈晓宇等人，知道叶少阳是在防备着他们，于是也也不再问下去，点了点头，表示自己明白了。
<br/>　　她不问，别人就算再好奇也不好开口了，而且他们现在的主要任务是下到地宫里，寻找炼尸缸，从而判断出峡谷闹鬼的原因。
<br/>　　约过花坛，再往前走，没走多远，果然是经过了一个岔路口，大伙钻进岔路，不一会就来到了一堵石门的前头。
<br/>　　跟幻境中看过的完全一模一样。
<br/>　　叶少阳走到门前，伸手抠了一下门锁附近的石头，纹丝不动……
<br/>　　“看来我们幻境中遇到的这一块松动的情况，是被佛子花修改了。”
<br/>　　“当时感觉跟真的一样。”梧桐喃喃道。
<br/>　　叶少阳道：“是啊，就是跟真的一样，这也是佛子花最厉害的地方了，如果直接弄出一个破洞之类的，太不切实际，我们很容易产生怀疑。”
<br/>　　他的一席话，让众人又回忆起了幻境中的经历，还是感到有些不寒而栗。
<br/>　　“那之前也说了，这里的财宝是真有的了，现在怎么办？”陈晓宇面朝妙心和梧桐问道。他跟叶少阳暂时和解，但是不会对话，说话也不会看着对方。
<br/>　　妙心道：“能怎么办，我们现在没有能打开这座石门的工具，一切也只有等回去再说了。先把重要的事做了。”
<br/>　　都知道进不去，大伙也没意见，继续往前走。
<br/>　　叶少阳不想跟陈晓宇走太近，跟毛小方一起走在队伍的后面，梧桐也故意放慢了脚步，跟他们一起走，突然小声问叶少阳：“你手上那块，是真的阴阳镜？”
<br/>　　“嗯？当然是了。”叶少阳反应过来之后说道。
<br/>　　“茅山的阴阳镜？”梧桐的表情很是不可思议，“我听说，阴阳镜是茅山至宝之一，你一个外门弟子，这东西怎么会在你手上？”
<br/>　　“我……我师父给我的。”叶少阳选择了实话实说。
<br/>　　梧桐哼了一声道：“不要骗我，这些法器，不可能给内门弟子的，所以要么你手上的是假的，要么……你就是偷来的。”
<br/>　　说到这，她表情很严肃地盯着叶少阳，“我不想跟一个偷东西的人交朋友。”
<br/>　　朋友？她把自己当朋友了？
<br/>　　叶少阳怔了一下，随即对她笑了笑，“你放心，不是偷的。这个……一言难尽，现在不能说，但是我保证将来一定会告诉你。”
