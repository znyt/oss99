        “知道她家地址吗？”季运恒心情极其不爽地接着问。
<br/>
<br/>        小梅抹了抹眼泪，微微摇头。
<br/>
<br/>        她连小谢是什么来头都不知道，又怎么可能知道小谢的家庭住址。
<br/>
<br/>        更何况，她平日里跟小谢压根就没什么交集。
<br/>
<br/>        季运恒随之将烟头摁灭在烟灰缸中，起身穿好衣服后，从钱包里抽了几百块钱出来，甩在了小梅的脸上，然后只身离开了。
<br/>
<br/>        小梅并未再看季运恒一眼，而是跪坐在床上，一张张地将那几百块钱给捡了起来。
<br/>
<br/>        季运恒这一晚上找了五六个女人试做，所有的前戏都做足了，找的女人也是一个比一个漂亮，身材一个比一个比好，而他的下面却没有任何生理反应，这是怎么回事？
<br/>
<br/>        他可是个血气方刚的男人！
<br/>
<br/>        如果，下面抬不起头来，以后他还怎么做男人？
<br/>
<br/>        季运恒气得回到家后猛砸东西发泄自己的不满与恼火。
<br/>
<br/>        这半夜三更，哥哥不睡觉，在家里打砸，吵得隔壁房间里的季思妍也睡不着后，兄妹两隔着一堵墙都能大吵一架。
<br/>
<br/>        季家一整晚都不得安宁。
<br/>
<br/>        滨江城，傅夜沉的私宅里，倒是格外静谧。
<br/>
<br/>        第二天，清晨。
<br/>
<br/>        傅夜沉坐在餐桌边吃早餐的时候，看到连华生那一脸坏笑地盯着自己，总觉得他没好事要说。
<br/>
<br/>        “夜沉，没想到，你这么腹黑啊！是不是，以后，只要是跟你抢女人的男人，都没什么好下场？嗯？”连华生扬了扬眉，眼睛里透着雪一样的眸光。
<br/>
<br/>        他最近听以前的战友，也就是他傅夜沉现在的保镖队队长韩剑锋说，之前滨江城市长的儿子郭裕给傅夜沉的未婚妻“苏漫雪”下了药，差点强上了“苏漫雪”。
<br/>
<br/>        事后，他傅夜沉二话不说，直接派人把郭裕给抓了，直到现在让那个郭裕还蹲在监狱里未出来。
<br/>
<br/>        而郭裕他老爸，身为市长，在他傅夜沉面前一个屁都不敢放。
<br/>
<br/>        连华生想想这事儿，就觉得好笑。
<br/>
<br/>        傅夜沉突然想起季运恒的事情来后，一边优雅地切着盘子里的食物，一边略带好奇地答非所问：“那个季家的大少爷，你是怎么处理他的？”
<br/>
<br/>        “我最近了，研究了一种新型药！就是能让一个男人硬不起来！小白鼠试验倒是成功了！也不知道在那位季大少爷身上成功了没有！”连华生唉声叹气地说。
<br/>
<br/>        “有解药吗？”傅夜沉淡定地问。
<br/>
<br/>        连华生立即眉开眼笑：“当然有！不过，价格很昂贵，你要买吗？”
<br/>
<br/>        “你可以留着去找那位季大少爷。”傅夜沉端起桌上的早茶，轻啜了一口，便放下了手里的刀叉，准备启程去公司。
<br/>
<br/>        连华生打了个响指，咧嘴一笑：“你这主意不错！到时候，你就再引荐引荐他过来找我，正好我可以大赚一笔！”
<br/>
<br/>        语毕后，连华生还不忘向傅夜沉眨了眨右眼。
<br/>
<br/>        傅夜沉会心一笑，不语，起身离去。
<br/>
<br/>        就在童筱颖以为那条陨石项链可以卖个好价钱，替傅昼景还债的时候，费尔南的一通电话，犹如晴天霹雳，让她所有的幻想都破灭了。
<br/>
<br/>        “小谢啊！你这条陨石项链是偷的？还是抢的？还是捡的啊？”费尔南的声音从听筒里传来时，语调忽高忽低，像似在嘲讽，又像似在质疑。
<br/>
<br/>        童筱颖抿了抿唇，咬牙切齿地回答：“我都说过了，是一个男人送我的！我救了他的命，所以他送了这条陨石项链给我！”
<br/>
<br/>        “小谢，你这话说出去，真不会有人信。这条陨石项链是编了号的，无法在黑市里拍卖贩卖。现在，你涉及非法占有他人财物。是我找关系，替你将这件事情给压下来了。只说是我自己无意间自己的会所里捡到的！你到现在，还不肯跟我说实话吗？”费尔南不禁有些恼火起来。
<br/>
<br/>        “我说的就是实话！既然你不信，把陨石项链还给我！”童筱颖不禁握紧了拳头。
<br/>
<br/>        费尔南顿时就没好气了：“诶？我说你这妮子！好心没好报了是不是？”
<br/>
<br/>        “我看是你私吞了我的陨石项链！”童筱颖气急败坏地说。
<br/>
<br/>        费尔南也因为她的话而气呼呼地怒吼了起来：“我费尔南是贪财，但不会贪别人的救命之财！你说你拍卖陨石项链是因为急需用钱！我信了！好心替你张罗，结果你那条陨石项链捅了这么大的篓子，如果不是我罩着你，你早就吃牢饭去了！”
<br/>
<br/>        “我……”童筱颖欲言又止，努力让自己平静下来后，压低了语气，问道，“我说的是真的，也是事实！真的是一个男人为了报答我对他的救命之恩，才送给我那条陨石项链的。所以，这到底是怎么回事？”
<br/>
<br/>        “怎么回事，就是，你那条陨石项链，被鉴定出价值非凡，而且已经入了全球系统编号，名花有主。而那位主儿，不是你小谢‘童筱颖’的大名，听说好像是叫什么‘雪’来着。总而言之，再说简单点，就是那条陨石项链的主人名字不是你！所以，那条陨石项链被没收了！”费尔南忍不住唉声叹气起来。
<br/>
<br/>        童筱颖顿时眸光黯然地垂下了眼帘，她以为，她找到了可以帮昼景的法子。
<br/>
<br/>        结果，竹篮打水一场空……
<br/>
<br/>        费尔南其实打心底地还是相信了童筱颖的话，知道她得知那条陨石项链没了后，心情一定不好，于是念在她是唐倩兮的小姑子份上，换了种语气，安慰道：“小谢，你还真是遇人不淑。说不定，你救的那个男人，也不是什么好人。总而言之，这件事情，就这么算了吧！以后，你也不要再说自己曾经拥有过一条什么陨石项链了。嗯，就这样吧！”
<br/>
<br/>        童筱颖默默无闻地听着费尔南的话，沉默不语。
<br/>
<br/>        费尔南见她不吭声，索性直接将电话给挂了。
<br/>
<br/>        童筱颖张开双臂，往后一躺，身子摔在了大床上后，双目无神地看着天花板发呆。
<br/>
<br/>        到头来，她真的一点用也没有。
<br/>
<br/>        她以为，她可以帮到昼景的。
<br/>
<br/>        她以为，她会是昼景的贤内助……
