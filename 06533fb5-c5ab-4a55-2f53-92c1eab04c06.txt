　　三仙岛的三仙中的另一尊，巫毒子看着楚风眠一招化解了漫天风斩，都是有些震惊。
<br/>　　风生子这出手，可是不留余力，全力出手。
<br/>　　真正的巅峰圣者，全力出手，足以轻易灭杀寻常圣者，更何况楚风眠的境界还没有踏入神力境，根本就是一个连圣者都不是的小子。
<br/>　　可如今楚风眠居然是化解了风生子的攻势，就算这不是风生子的必杀之术，也令他震惊无比。
<br/>　　“厉害，今日不杀了此人，来日必成大患！”
<br/>　　另一尊易剑子，也是缓缓开口道，眉宇中献出杀机。
<br/>　　以御风境抗衡寻常圣者，在普通人眼中也许是毫无可能的一件事，但是以他们的身份，自然知晓。
<br/>　　像是北邙学院中，真正的一些天之骄子，的确是足以御风境抗衡圣者。
<br/>　　但是这也却只能是抗衡寻常的圣者，抗衡巅峰圣者，这在北邙学院中都只有真正的超级天才才能做到。
<br/>　　楚风眠如今表现出来的资质甚至是不比北邙学院的超级天才要差，这让他们如何相信。
<br/>　　在放任楚风眠成长下去，将来不可限量，如今既然已经是跟楚风眠为敌，便是就要彻底的斩草除根！
<br/>　　巫毒子跟易剑子对视了一眼，都从对方的眼神中看出来了对方的意思，便是在缓缓的等待着机会。
<br/>　　“楚风眠，你的实力的确很不错。”
<br/>　　风生子看着楚风眠，目光闪烁，一招被破，最为惊讶的人莫过于他。
<br/>　　不过风生子这次出手，虽然是杀招，但是并非是真正的全力，他们这些巅峰圣者，自然不缺少底牌。
<br/>　　“但是你以为这样，就可以抗衡巅峰圣者，还是你太自大了！风魔出！”
<br/>　　风生子猛然一声怒喝，在他的身躯之中，仿佛是有着一股强大到极点的力量被唤醒。
<br/>　　令风生子如今的身躯上，居然是长出了一道道的鳞片。
<br/>　　风魔！
<br/>　　这风生子，居然是一道魔修，化为了一尊风魔，一身鳞片，令他现在看起来完全不像是一尊圣者，倒像是一尊魔头。
<br/>　　“这风魔之力！老夫看你如何可以抵挡！”
<br/>　　化身风魔，风生子的实力比起刚刚还要强大了数倍，一招出手，便是化为了无数飓风，这一道道飓风中，更是带有着魔性。
<br/>　　每一道飓风，仿佛都是化为了一尊尊小风魔一样，向着楚风眠轰杀过来。
<br/>　　“我当是什么来历，原来是九魔城的杂碎吗？”
<br/>　　楚风眠冷哼一声，这三仙，居然是三尊魔头，这九魔城的手，伸的倒是长，在这偏远的武胜国度居然都有着九魔城的人。
<br/>　　“哦？你居然知道我们的来历？”
<br/>　　听到楚风眠喊出九魔城的名字，风生子更是冷笑连连。
<br/>　　“看来你知道的真不少，既然如此，那你就化为我们的养料吧！我们会杀了你，分食了你的血肉的！你身上的无数奇遇，都将属于我们！”
<br/>　　风生子狰狞大笑道，哪里还有之前的风范，完全是一尊食人魔头。
<br/>　　九魔城虽然位列七大宗门之一，但是其中门下无数弟子还是魔头，这风生子应该就是其中的一尊小魔头，来到了这武胜国度称王称霸。
<br/>　　“不只是你，还有寒月漓，无数寒雪阁的弟子，都将化为我们的血食，杀了你们，我们便是可以踏入到生死境！
<br/>　　“到时候就可以将整个武胜国度全部强者斩杀！化为我们的血食！”
<br/>　　风生子狰狞大笑，张牙舞爪向着楚风眠冲了过来。
<br/>　　无尽疯狂，化为了两道巨大无比的利爪，冲着楚风眠顿时抓了过来。
<br/>　　就在这风生子出手的一刻。
<br/>　　那一直不动寻找机会的巫毒子，易剑子同时出手了。
<br/>　　一道巫术，一抹剑锋，同时向着楚风眠轰杀过来。
<br/>　　这二人等待许久，便是打算趁着这个机会，要将楚风眠彻底灭杀。
<br/>　　“玄寒千里！”
<br/>　　一声娇喝，在楚风眠身边的寒月漓猛然出手，手持灵剑，一剑斩出，化为了层层玄冰，抵挡这那巫毒子跟易剑子的偷袭。
<br/>　　“玄寒之体又如何？一个神力境七重的小家伙还敢对我们出手？给我破！”
<br/>　　易剑子看着寒月漓出手，冷哼一声，一剑便是对着寒月漓斩了过去。
<br/>　　顿时层层玄冰，顿时破碎，巅峰圣者的实力，还是要远远能超越了寒月漓。
<br/>　　同时那巫毒子的一道巫术，也是已经来到了楚风眠的面前，这巫术中蕴含无数古怪玄妙之力，将楚风眠逼迫到了极点。
<br/>　　“小子，死吧！”
<br/>　　风生子狰狞笑道，脸色得意无比，他们三人如今一齐出手，就是要将楚风眠彻底灭杀。
<br/>　　杀了楚风眠，他们吞食了楚风眠的血肉，实力便是可以再进一步，渡过生死大劫，指日可待。
<br/>　　那巫毒子，易剑子一开始不出手，便是为了麻痹楚风眠，如今突然出手，偷袭的威力，比起他们三人一齐围攻还要有着更大的威胁。
<br/>　　“糟了！”
<br/>　　“这三仙岛的老家伙，居然是不顾身份，一齐出手甚至还是偷袭！”
<br/>　　下方的无数寒雪阁弟子，都是不由的一惊，心中紧急万分。
<br/>　　这样的攻击，就算是让一尊同为巅峰圣者的人物来，都是必死的局面。

<br/>　　从楚风眠的身上，仿佛是压抑已久的力量，彻底的爆发了出来。

<br/>　　这一道战龙之爪中，甚至还蕴含了几分造化之力，简直是如今楚风眠的最强手段。
<br/>　　远古战龙的血脉，跟造化之力融合，化为了更加强大的战龙之爪，这战龙之爪上的每一个鳞片，都蕴含有着诡异的纹路。
<br/>　　天地，因为这一爪而震动。
<br/>　　变化在瞬息之间。
<br/>　　战龙之爪落下，顿时轰隆一声，这风生子，巫毒子，易剑子三人，居然是一下子都被打飞出去，身上被打出无数裂痕来。
