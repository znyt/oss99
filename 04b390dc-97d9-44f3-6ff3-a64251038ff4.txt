　　巨大的魔渊坐落在大地之上，其内一片漆黑，如同无底洞一般。
<br/>　　不时的有魔物从魔渊中飞出，或单独一只，或三五成群。
<br/>　　一旦有魔物出现，那群二十多名天虚宗的弟子，便轮幡上阵，将魔物一一击杀。
<br/>　　虽然有些魔物很强，但最终还是难挡天虚宗弟子，纷纷陨落。
<br/>　　“吕罡，这魔渊即然有如此多的魔物，为何没有超级强者来将其毁灭？”苏莫一边等待，一边和吕罡闲聊。
<br/>　　“这魔渊就算是武圣境的强者，也毁灭不了，而且这么好的历练之地，为何要毁灭呢！”吕罡摇了摇头说道。
<br/>　　“也是！”苏莫微微点头，的确是很好的历练之地。
<br/>　　“这魔渊之下，是否是一个地下世界？”苏莫再次问道，在他想来，这里肯定是一个魔窟，其内魔物无数。
<br/>　　“并不是！”
<br/>　　吕罡摇头，沉声道：“据说这魔渊通向另一个星河，魔陀星河，那里是魔之祖地，邪魔诞生的地方！”
<br/>　　“魔陀星河？”苏莫闻言惊讶，这无尽星空中星河无数，真是各种各样的星河都有。
<br/>　　“传闻无数万年前，魔陀星河的魔族，曾通过这魔渊大举进攻明虚星河，不过后来，这魔渊中被大量的虚神境大能布置了封印，实力稍强一些的魔物，就再也来不了了，只有这些普通的魔物能通过魔渊来到这里！”吕罡徐徐说道。
<br/>　　苏莫心中了然，难怪魔渊中出来的魔物，没有太过恐怖的存在。
<br/>　　两人一边闲聊，一边能待，可是，足足等了十几个时辰，到了第二日的时候，那些天虚宗的弟子，居然完全没有要离开的迹象。
<br/>　　“这些人要猎杀到什么时候？”苏莫眉头皱了起来，心中郁闷不已，这简直就是在浪费时间。
<br/>　　“等吧！等到天虚宗的人走了，还要等御剑谷的人！”吕罡摇了摇头。
<br/>　　“什么？还要等御剑谷的人？”苏莫闻言一怔。
<br/>　　“是啊！”
<br/>　　吕罡点了点头，指向另一个方向，道：“你看，那里有十几名御剑谷的弟子，其中也有御剑谷的真传弟子，待天虚宗之人走了，他们就会把持魔渊！”吕罡说道。
<br/>　　苏莫闻言，顺着吕罡手指的方向看了过去，果然在另一边有十几名御剑谷的弟子。
<br/>　　这些人个个背负长剑，虽然没有刻意显露气息，但身上的剑威已经弥漫而出。
<br/>　　“尼玛！”苏莫的脸色彻底黑了下来，等完天虚宗的人还要等御剑谷的人，是不是御剑谷的人杀完了还要等其它势力的人？
<br/>　　那他还杀个屁，等到猴年马月也轮不到他啊！
<br/>　　“吕罡，你就说吧，什么时候能排到我们？”苏莫黑着脸问道。
<br/>　　“这个……我们道宗没有真传弟子来此，可能要等很久！”
<br/>　　吕罡叹了口气，继续道：“天虚宗之后是御剑谷，然后啸天狼族也有真传弟子，等他们三大势力的人猎杀完之后，我们这些其它势力的人，才有机会。”
<br/>　　“狗屁吧！”苏莫面带无语之色，等到这三大势力的弟子猎杀完，那要等到什么时候了！
<br/>　　再说了，就算三大势力的弟子都走了，剩下的这些人，也是要争夺猎杀魔物的优先权吧！
<br/>　　还有就是，三大势力猎杀魔物的这个时间会很长，若是期间还有其它势力的强者来此，他们岂不是还是要继续排到后面！
<br/>　　说白了，这猎杀魔物，也是谁的拳头大谁优先。
<br/>　　“吕罡，我们道宗有多少人在此？”苏莫沉声问道，他不准备继续等下去了。
<br/>　　吕罡打量了一番四方，道：“除了我们两人之外，还有五人！”
<br/>　　“把他们都喊过来，我们不能等了，要夺得优先权！”苏莫沉声说道，他也没有自己独自行动，毕竟其他几大势力，都有真传弟子在此，他也没有把握，所以还是把道宗弟子聚集起来，齐心合力才好。
<br/>　　“啊！”
<br/>　　吕罡闻言，顿时惊讶的张大了嘴巴，不可置信的问道：“苏莫，你是说我们要和天虚宗的人抢？”
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“这……！”吕罡顿时心中暴汗，满脸无语的说道：“苏莫，天虚宗人多势众，还有真传弟子在此，我们还是算了吧！”
<br/>　　吕罡心中很是无语，和天虚宗的人抢，那不是找虐吗？
<br/>　　天虚宗不仅有真传弟子在此，其中武尊境一重修为的弟子，都有六、七人，谁敢和他们抢？
<br/>　　连御剑谷的弟子都不敢抢，只能乖乖的排在后面，苏莫居然还想去抢？
<br/>　　此时，吕罡不仅暗想，这苏莫天赋是够妖孽了，但未免太自大了。
<br/>　　他也看出来了，苏莫突破到了武帝境九重，但即便突破到了武帝境九重，也不可能是天虚宗真传弟子的对手啊！
<br/>　　“怎么？你不想提前猎杀魔物？还想一直干等下去？”苏莫见对方不愿意，皱眉问道。
<br/>　　“我当然不想等，可是……！”吕罡面带迟疑之色。
<br/>　　“没什么可是，你把其他人都喊过来吧！”苏莫打断了对方的话语，他虽然也没有绝对的把握，但总要一试才知晓。
<br/>　　“好吧！”吕罡无奈的点了点头，不过，他不认为其他人会跟着苏莫一起动手。
<br/>　　旋即，吕罡飞身离开，将魔渊附近的五名道宗弟子聚拢在了一起，带到了苏莫的身边。
<br/>　　苏莫打量了一番，这五人之中，一名矮小的蓝衣青年修为最低，武帝境八重境界，一名银袍青年修为最高，武尊境一重境界，其他三人都是武帝境九重。
<br/>　　“苏莫，你找我们有何事？”三名武帝境九重的道宗弟子之中，其中一名身材最为高大的青年笑着问道。
<br/>　　这名青年面带和气之色，他参加过七大势力的会武，认识苏莫，也知道苏莫的厉害，所以非常的客气。
<br/>　　“我们不能继续等了，现在就和天虚宗的弟子，抢夺魔物！”苏莫开门见山的说道。
<br/>　　“啊！”
<br/>　　五人听闻苏莫之言，反应和吕罡一般无二，顿时瞪大了眼眸，张大了嘴巴。
<br/>　　他们着实被苏莫的话吓到了，和天虚宗弟子争夺猎杀魔物的优先权，这是脑子进水了吗？
