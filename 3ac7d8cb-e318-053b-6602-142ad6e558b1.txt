　　玄天神殿，大厅之中。
<br/>　　紫袍中年人在凝望着苏莫的分身，分身同样也在凝望着对方。
<br/>　　“拜见玄天长老！”
<br/>　　少倾，分身双手抱拳，向对方郑重一拜。
<br/>　　虽然对方看起来，没有多么的可怕，但是他丝毫不敢大意。
<br/>　　中年人面无表情，少倾之后，淡淡的开口道：“你神魂不全，乃是一具分身，你是何人？”
<br/>　　分身闻言，心中并未有什么意外，对方乃是神王，看出他是分身，再正常不过了。
<br/>　　若是他的本体在此，对方能直接认出他是轮回之身，也不无可能。
<br/>　　“禀长老，弟子名为段惊天！”分身沉声说道。
<br/>　　“段惊天？”
<br/>　　中年人微微点头，随即转身在主座上大马金刀的坐了下来，道：“本长老听过你的名字，是一个天才，不知你今日，找本长老有何事？”
<br/>　　他能看的出来，这个段惊天，非常的不简单，体内的元神有两种，都是极度强大的元神。
<br/>　　只不过，具体是什么元神，他并不能轻易的窥探而出。
<br/>　　若是普通的弟子来找他，他懒得多言，但是，这个段惊天貌似有些诡异。
<br/>　　“玄天长老，不知您对于帝一魂和拓跋青云的争斗，有什么看法？”分身沉声问道。
<br/>　　玄天长老闻听此言，眸中精光摄人，眼眸微微的眯了下来。
<br/>　　“小子，此事，你没有必要知晓吧？”
<br/>　　玄天长老目光紧紧的凝视着分身，仿佛要将他看透，继续道：“说出你的来意，本长老没有时间与你闲聊！”
<br/>　　“我是想说，除了帝一魂和拓跋青云之外，还有第三具轮回之身！”
<br/>　　分身深深的吸了口气，一字一句的说道。
<br/>　　“你说什么？”
<br/>　　闻听此言，玄天长老立刻又站了起来，眸中射出璀璨如剑的精芒。
<br/>　　在这股精芒之下，分身全身一震，立刻蹬蹬蹬的后退了三步。
<br/>　　他心中骇然不已，神王果然是神王，不是普通的真神可比。
<br/>　　“你就是第三具轮回之身？或者说你的本体就是第三具轮回之身？”玄天长老面色郑重的问道。
<br/>　　眼前的段惊天既然如此说，那肯定是说自己也是轮回之身了，不然，这种事情，不可能告知外人。
<br/>　　众所周知，轮回天尊的六具轮回之身，目前只剩下了帝一魂和拓跋青云。
<br/>　　但是，眼前的人，居然说还有第三位。
<br/>　　“不错，我就是第三位！”分身笃定的说道，他已经做好了迎接一切变故的准备。
<br/>　　今日，要么完成计划，要么这具分身惨死。
<br/>　　当然，他还是有些底气的，毕竟，他是综合了各方面的情况，才敢来到此地。
<br/>　　闻听分身之言，玄天长老沉默不语，整个人仿佛陷入了深沉的思索之中。
<br/>　　分身没有打扰，静静的望着玄天神王，等待对方的态度。
<br/>　　“你如何证明，你就是第三具轮回之身？”
<br/>　　少倾，玄天神王声音低沉的问道。
<br/>　　从眼前的这具分身之上，并不能看出，与轮回之身有关。
<br/>　　“如何证明？”
<br/>　　分身闻言，沉默不语，他好像真的无法证明。
<br/>　　毕竟，他只是一具分身，万化神剑元神和不败战气元神，也不能说明什么?
<br/>　　沉默了片刻，分身道：“长老想必见过轮回天尊吧？”
<br/>　　“这是自然！”玄天长老微微点头，他作为神王，活了几百万年，自然见过天尊。
<br/>　　“好，那么想必长老能认识这个！”
<br/>　　分身手掌一伸，掌中一丝淡淡的光芒出现了。
<br/>　　这丝光芒，极为细小，细如毫毛，这是苏莫的一缕神魂。
<br/>　　分身从元神之中，剥离出的一缕细微的神魂。
<br/>　　随即，分身手掌一抛，这一缕神魂向玄天长老飞了过去。
<br/>　　玄天长老接住了这一缕神魂，仔细的查看了起来。
<br/>　　随着查看，他不免动容，眼眸深处闪过深邃莫测的光芒。
<br/>　　少倾，玄天长老将这一缕神魂，又还给了分身。
<br/>　　分身收起神魂，再次融入了体内。
<br/>　　他目光凝视着玄天长老，静静的观看着对方的态度。
<br/>　　玄天长老又重新坐了下去，双手扶着紫色座椅的龙头扶手，整个人陷入了沉默之中。
<br/>　　“长老，不知道您是否辩出真伪？”分身沉声问道。
<br/>　　他相信，只要对方熟悉轮回天尊，那就一定能辨认的出来。
<br/>　　毕竟，他是轮回之身，神魂的本质，并不会改变。
<br/>　　玄天长老闻言，抬头凝视着分身，声音低沉的问道：“你就不怕，我靠你这具分身，将你的本体擒来，然后交给帝一魂？”
<br/>　　分身闻言，心中暗凛，对方能不能靠这分身，找到他的本体，他并不清楚。
<br/>　　不过，对方没有立刻动手，显然不会这么做。
<br/>　　“长老，人王宫和鬼王宫、兽王宫、魔王宫，没有帮助帝一魂对付拓跋青云，显然你们并不是很希望轮回天尊复生！”分身沉声说道。
<br/>　　“你很聪明，不过，你只说对了一半，我们都希望天尊大人复生！”玄天长老沉声道。
<br/>　　“是吗？那为何？”
<br/>　　分身闻言，顿时眼眸微眯了下来，他有些不理解，既然希望轮回天尊复生，那为何不帮助帝一魂呢？
<br/>　　“此事你以后自然会知晓！”
<br/>　　玄天长老声音低沉，继续道：“你来到人王宫，是想要人王宫庇护于你，对抗帝一魂和拓跋青云吧？”
<br/>　　“正是！”分身点了点头，对方能轻易猜到他的意图，这也不用他浪费口舌了。
<br/>　　“你什么时候加入的人王宫？”玄天长老没有告知具体态度，而是问起了苏莫的情况。
<br/>　　他知道，苏莫肯定已经加入了人王宫，不然不会来到此地。
<br/>　　“几年之前！”分身沉声说道。
<br/>　　“这样吧！你先回去吧，明日再来找我！”玄天长老道。
<br/>　　“明日？”
<br/>　　分身深深的吸了口气，他知道，对方可能还不能决定人王宫的态度。
<br/>　　他心中疑惑甚多，对于六大王宫和轮回之身，其中到底有什么隐秘，他很是疑惑。
<br/>　　不过，对方没有对他出手，那么，局势还是比较理想的。
<br/>　　“好，明日再来叨扰长老！”
<br/>　　分身点了点头，随即向玄天长老抱拳一拜，转身打开殿门，大步的走了出去。
<br/>　　既然今日不能得到结果，那就明日再来就是。
<br/>　　望着分身离去的目光，玄天长老的眸中，闪过复杂的之色，随即，他的身形无声的消失了。
