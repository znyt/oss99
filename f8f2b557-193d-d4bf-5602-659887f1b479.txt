<br/>　　到那时，这个小镇内的吃喝玩乐一切的一切，都是这个小镇管理者所有，军部都不得有任何插手，谁凭本事打下来的，就是谁的私人地盘。

<br/>　　捡了两天的空投。两个C级空投
<br/>　　咔擦。

<br/>　　“别废话了，这个地方对我们来说意义重大，这个老龙口虽然面积不如黑龙镇大，但是我看过地图，这附近有个矿山，之前有人在这里采矿，我们若是把这个地方给霸占了，那日后这矿山不就是我们的了？”林峰道。
<br/>　　“握草，这么说，我们要家里有矿了？”陈勾道。
<br/>　　“所以说，这个老龙口无论如何，也要拿下，开干！”
<br/>　　没有任何废话，陈勾直接肉弹战车冲了过去瞬间就压爆了几十头丧尸。

<br/>　　这个自动火炮塔可以自己开枪，不需要人去管，专门打这种密集丧尸群特别有效就是神器的存在。
<br/>　　唯一缺点就是特别消耗子弹。
<br/>　　不过这两天开的弹药到也足够消耗。
<br/>　　而林峰则没有去管这些喽啰，他去找那个九阶的存在。

<br/>　　林峰看到这穿山甲身上的盔甲心中骂娘，这玩意真的无敌。
<br/>　　变异穿山甲二话不说蜷缩成一个球就朝林峰滚来。若是胖子那是肉弹战车，那么这个就是全金属战车了。

<br/>　　林峰直接往这家伙身上丢汽油弹。十个十个的丢。他现在汽油弹背包里不少，上次开B级空投就直接得到了100发汽油弹。
<br/>　　啪啪啪啪
