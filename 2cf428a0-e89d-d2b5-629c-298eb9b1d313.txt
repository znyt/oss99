　　第14章
<br/>　　羊肠小路，一线天峡谷。
<br/>　　“大当家，来了！”林少羽道。
<br/>　　程大雷一激灵翻身站起，这几天他都是睡在峡谷上，心中也是忐忑不安，生怕军师算无一策的属性出现意外。所以他才会让徐神机走遍青牛山各处山寨，诱那些山贼在官道之上埋伏，而真正的杀招还是在一线天峡谷。
<br/>　　这峡谷十分狭窄，只能容两马并行，四人并肩，当队伍来到峡谷前时，骑兵先进入，步兵紧跟其后。
<br/>　　黎明天未亮，黑压压一片，耳边还能听见盔甲摩擦的声响。程大雷从峡谷上往下望，立刻出了一身冷汗。
<br/>　　“大当家，这……这足有上千人呐！”徐神机吃惊道。
<br/>　　程大雷咬得牙齿嘎吱嘎吱作响，自己山寨总共才五个人，尼玛派一千人来，吃饱了撑得嘛！
<br/>　　“大当家，撤吧，不行了。”徐神机道。
<br/>　　“撤……已经不行了，按原先的计划打。”程大雷捧着一根十米长，一抱粗细的原木，道：“把人放进山谷，我丢下原木的时候，林少侠点燃埋在谷口的炸药，徐军师、秦大哥、灵儿你们三个往下面丢炸药箱……”
<br/>　　这就是程大雷的计划，几人守在峡谷上方，看着一个个人走进峡谷。
<br/>　　徐神机小声问：“咱们为何只炸塌一面谷口，不把两边山谷都炸塌……”
<br/>　　“围城必缺，在绝境中，人会发挥出想象不到的战斗力，而只有在知道有退路的时候，他们才不会拼命。”说到这里，程大雷忽然叹口气：“我们和他们不一样，我们已没有退路。”
<br/>　　队伍如同长蛇之中涌入山谷，事情突然发生一点意外，峡谷不过五六百米，队伍拖拖拉拉要超过这个长度，现在有骑兵走出山谷，有人还没走出山谷……
<br/>　　“大当家，怎么办？”徐神机紧张得浑身冒汗。
<br/>　　“等一等，等一等。”程大雷声音出口，才发现嗓音已经变了。他现在的每个决定，可能都会影响最后的结局。
<br/>　　有骑兵不停走出峡谷，可现在队伍最后方那辆马车还未进入峡谷，而走出峡谷的十几个骑兵已经足以收拾自己这伙人，不能再等了。
<br/>　　“杀！”
<br/>　　程大雷忽然发力，将手中的原木推了下去，轰隆一声，硬生生挡在峡谷出口，一骑兵噗通撞在上面，人马分离。
<br/>　　“林少羽，点火！”
<br/>　　程大雷大喊一声，秦蛮和徐神机抱起炸药箱，就等着林少羽炸塌峡谷出口后就往下砸。
<br/>　　可好久没有动静，程大雷猛地回头，就见林少羽拿着火石，脸上挂满汗珠。
<br/>　　“你该不会第一次杀人吧？”
<br/>　　“我……我……下面人太多，我……我……”林少羽结结巴巴。
<br/>　　事情终于出现意外，事实不可能和计划一模一样，就在这个时候，又有十几个骑兵离开峡谷。
<br/>　　“妈的！”程大雷恨恨一个耳光砸在林少羽脸上：“这是战争，不是小孩过家家，战争没来得那么体面，没那么文质彬彬，战争就是你想杀死我，我想杀死你！”
<br/>　　程大雷穷凶极恶，绝眦欲裂，林少羽看到他牙关轻轻打颤，原来不止自己害怕，他也在害怕。
<br/>　　程大雷一把夺过林少羽手中火石，锵的一声点燃引线，火顺着浸油的棉线往下转，所有人都看着，生怕这星星之火因为意外熄灭。
<br/>　　轰！
<br/>　　事先埋在峡谷出口的火药被点燃，山石纷飞，滚滚而落，将山谷出口堵死，可这时，已经有一百骑兵逃出峡谷。
<br/>　　秦蛮和徐神机配合，一个点燃炸药箱引线，一个往下砸，炸药箱在空中炸开，里面装着的石子速度不弱子弹。
<br/>　　程大雷也加入他们，一个个炸药箱被丢下去。
<br/>　　山谷下方已经乱成一团，爆炸声如同雷震，纷纷而落的石头可以轻易夺走他们的生命。很多人都不知道火药的存在，看到这一幕还以为是神仙显灵，山鬼作祟，有人甚至直接跪在地上，磕头祷告。
<br/>　　骑兵，永远是一只部队的精锐，高速的机动能力至少在二战时都还发挥着作用。而在这个时代，家里能养得起一匹马就证明不是普通人家。
<br/>　　“在山上，冲过去！”骑兵头领发号施令。
<br/>　　“杀，杀，杀！”
<br/>　　长枪扬起，向山谷上方冲来，他们还有极强的战斗意志，突然而来的变故，并不会令他们畏惧，反而激起他们的杀敌之心。
<br/>　　这一幕在程大雷的计划之外，事情并不会一直按照他所想的发展。毕竟有一千人，只要手指缝落下来几个人，就足以收拾程大雷几个。若是被他们冲到山上，所以计划都是一场空。
<br/>　　“大当家，我去挡住他们。”秦蛮手持铁枪，翻身跨马。
<br/>　　“好！”程大雷大吼，手中点燃的炸药箱狠狠砸下去。
<br/>　　从山下到峡谷上方只有一条路，此刻一人往下冲，一队人往上冲，双方狭路相逢。
<br/>　　一百骑兵，各个都是精锐儿郎，骑兵头领更是身经百战。他们行至半山坡上，就见一大汉横枪立马拦在路上。
<br/>　　“我秦蛮在此，你们不许过。”秦蛮枪尖直指对方眉心。
<br/>　　“杀！”
<br/>　　一声大喊，骑兵头领一马当先，手中枪冲着秦蛮心脏扎来。秦蛮眉心拧起，一甩手腕，黑色枪身如同毒蛇一般钻了过去。只是一个回合便将骑兵头领挑落马下。
<br/>　　“说了不许过，便是不许过。”
<br/>　　一百轻骑一拥而上，秦蛮手中大枪或是抡，或是甩，或是砸，或是捅，一甩手腕，梅花枪抖出五朵枪花，朵朵都是杀人花。
<br/>　　程大雷不知道秦蛮能撑多久，心脏不停跳动，只能发疯一般的将炸药箱丢下去。徐灵儿点燃填了火药的棉被，顺着峡谷丢下来，棉被在空中剧烈燃烧，宛若火云一般。
<br/>　　峡谷内，士兵的战斗意志很快被摧毁，慌慌张张乱成一团。重甲兵东倒西歪，长枪兵枪砸在弓兵身上，大刀兵七零八落，弓兵抱头鼠窜。
<br/>　　“逃呀，逃……”
<br/>　　不知谁喊了一声，一群人宛若找到方向，向山谷入口冲去。可不巧的是，韩玄之那辆豪华马车正好堵在峡谷入口。
<br/>　　韩玄之也不知道怎么会事，刚从马车内钻出来。
<br/>　　“少城主……有妖怪啊……逃啊……”
<br/>　　韩玄之心脏砰砰跳动，忽然拔出佩剑，高喝一声：“不许逃，过此线者死！”
<br/>　　这一刻，他又感觉某位名将附身，自己将力挽狂澜，反败为胜。
<br/>　　一群士兵眼神闪烁，身后的爆炸声，哀嚎声响成一团。
<br/>　　“去你娘的吧。”
<br/>　　有人大喊一声，一拥而上将马车掀翻，人群疯一般向外逃去。
<br/>　　
<br/>(本章完)
