　　人与人之间的缘份是一种很奇怪的东西，你永远也不知道你会在什么时候遇到一个什么样人的，然后就会觉得很投缘，进而义无反顾的决定帮助他（她）。
<br/>　　称心便是在一次机缘巧合之下从一个人贩子手里救下小钰的，当时的小钰正被锁在地窖里面，饿的奄奄一息，不过这个倔强的小姑娘似乎宁可自己被饿死，也不愿意屈从于人贩子。
<br/>　　这一行为勾起了称心内心深处最痛苦的一段回忆，所以他放弃了暗中追查一个人口贩卖团伙的行动，直接现身对那个人贩子严刑逼供，问出想要知道的东西之后，将人丢给了情报科，之后便带着小钰开始了自己的隐居生活。
<br/>　　也正是从那一天开始，称心将小钰收为弟子，不过这也只是名义上的，因为在任何人看来，他对小钰所作的一切，都跟一个父亲没有什么区别。
<br/>　　甚至为了小钰这个丫头，称心还放弃了在情报科的一切，虽然他很清楚自己如果在情报科继续干下去，小钰会得到更好的教育。但这样作同样也会使小钰落处情报科的眼中，将来有一天很有可能这个小丫头也会走上他的老路，这是称心十分不想看到的。
<br/>　　可任何事情都不会按照个人的意愿去发展，虽然称心不想让小钰走上自己的老路，但是如何生存下去却成了他需要解决的问题。
<br/>　　他没有钱，也不知道如何赚钱，可以说除了杀人之外，他什么都不会。
<br/>　　当年存下的那些钱已经被花的差不多了，眼看着如果再没有什么收入的话，他与小钰两个就只能去喝风，到底去哪里弄钱成了眼下最重要的事情。
<br/>　　……
<br/>　　“梆梆梆”一阵轻微的敲击声隐隐传来，听上去就像是更夫打更的声音。
<br/>　　可是原本正躺在榻上的称心却一个翻身坐了起来，一双凤目死死的盯着窗户。
<br/>　　“梆梆梆”，声音依旧在持续，不远不近，但却一直在响。
<br/>　　“该死的，竟然敢跟踪老子！”称心的眼中射出一缕寒光，手指也不由抽动了一下，很久不杀人了，看来那些家伙已经忘了‘红衣’的威名了吧！
<br/>　　“梆梆梆……”的声音还在契而不舍的响着，一直不断刺激着称心的神经。
<br/>　　终于，当年那个让倭人闻风丧胆的‘红衣’坐不住了，探手将窗子一推，整个人便如同一条鱼般游了出去，来到院子里之后略顿了顿，便一个翻身来到了屋顶之上，几个纵跃便消失在了夜色之中。
<br/>　　……
<br/>　　距离称心家小院百米之外的一个街口，杨天手中拿着更夫常用的一套家什，有一下没一下的敲着，而打更的更夫就倒在他的脚边，从其还在起伏的胸口来看，应该是被打晕了。
<br/>　　“杨天，我都已经躲到这里来了，为什么你还是不肯放过我？！”不知何时，杨天的身后已经多出一个人影，不是称心又是何人。
<br/>　　“红衣，你真以为我是刚刚跟着你来的？你真的以为你躲在这里就没有人知道？”杨天转过身，正视着身后那个与他在倭国并肩作战了十余年的‘战友’，笑着问道。
<br/>　　称心的瞳孔微微一缩，双手微微一抖，两根三棱军刺已经消然出现，同时冷声问道：“你到底想说什么？我都已经跟黑子说了，从今以后与情报科没有任何关系。”
<br/>　　“没什么，只是想来看看你。”杨天微微扫了一眼称心手中的军刺，淡淡说道：“你也可以把这看成是一场道别！”
<br/>　　“你觉得我会相信么？”称心反问道。
<br/>　　“我杨天什么时候骗过你？而且就算你不相信我，陛下你总该相信吧？”杨天说道。
<br/>　　称心闻言微微一愣：“陛下？陛下怎么会知道？！”
<br/>　　“陛下一直都很关心咱们这批‘老人儿’，你不想再继续在这个行当里混陛下也知道，而且陛下尊重你的意思，特地嘱咐我过来为的就是跟你道别！”杨天叹了口气，似乎有些惋惜。
<br/>　　称心这个时候也不知道应不应该相信杨天，狡兔死走狗烹的典故他听过不是一次两次了，而且他也知道当年那个李元昌是如何对付最信任的手下的。
<br/>　　所以在他看来，李承乾应该不大可能会放过他这只走狗，毕竟他知道太多东西了，能够让他闭嘴最好的办法就是杀了他灭口。
<br/>　　但是现在杨天竟然说李承乾会放他离开，这对他来说简直就是最冷的冷笑话。
<br/>　　想到这里，称心紧了紧手中的军刺：“杨天，我还是那句话，你到底想怎么样？”
<br/>　　在他说出这句话的同时，他的心中已经有了必死的准备，只等杨天出手或者四下里‘埋伏’的那些枪手下他发起进攻。
<br/>　　然而杨天的举动再次让他大感意外，只见他从怀里摸了半天，掏出一个精致的小酒壶出来：“红衣，既然你也肯相信我，那么就把这酒喝了吧，喝了之后你我从此便是路人，你走你的阳关道，我走我的独木桥，此前种种尽与你没有关系了。”
<br/>　　酒中有毒，这是称心的第一个想法，不过很快他就想通了这一切。
<br/>　　该来的总会来，如果李承乾真的想要杀他，一杯毒酒的确是最好的方法，只是……自己死了之后，小钰估计……，算了，想那么多干什么！
<br/>　　人一旦心中有了某个念头之后，就很难再消除，称心也是人，自然也不会例外。
<br/>　　不过，他最终还是没有反抗，只是苦笑着收起了两把军刺，然后从杨天的手中将酒接了过来，狠命的灌进了自己的喉咙里面。
<br/>　　……
<br/>　　一个呼吸过去了，两个呼吸过去了，直到最后一刻钟过去了，想像中的剧痛还是没有传来，只有一阵阵的眩晕感冲击着他的神经。
<br/>　　这是醉酒的症状，并不是中毒，称心很清楚这一点。
<br/>　　疑惑中的他看了杨天一眼：“为什么……？！”
<br/>　　不过，杨天却并没有理他，只是淡淡的看着他，就像在看一个路人，片刻之后转身向远处的黑暗走去。
