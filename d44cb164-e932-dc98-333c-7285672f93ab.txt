　　　　嗖。
<br/>　　萧逸听闻萧离火的话，立刻飞跃至高空。
<br/>　　速度飙升到极致。
<br/>　　刚欲去追那两个蒙面的破玄境武者。
<br/>　　却发现他们早已没了踪影，甚至于，连气息都没有半分残留。
<br/>　　“该死。”萧逸咬了咬牙，只能从高空中落下。
<br/>　　“逸儿，怎么不追了？”萧离火等长老围上来，急问道。
<br/>　　“追不了。”萧逸脸色难看地摇了摇头。
<br/>　　“抹气息，隐踪迹，他们不是普通的武者。”萧逸沉声道，“他们是杀手。”
<br/>　　“杀手，暗影楼？”萧家众长老惊呼一声。
<br/>　　“应该不是。”萧逸摇摇头。
<br/>　　观刚才那两个蒙面武者逃跑的速度，修为起码在破玄六重以上。
<br/>　　而且，隐匿气息的手段，甚是高明。
<br/>　　萧逸见识过暗影楼的杀手，也曾在东荒兽潮时，见识过副楼主的手段。
<br/>　　很显然，刚才那两人的手段，更高。
<br/>　　自己不过慢了几秒才去追，他们就消失得无影无踪了。
<br/>　　“到底怎么回事？”萧逸沉声问道，“刚才那两人逃跑时，两手空空，并不见萧重叔叔被掳去。”
<br/>　　要是萧逸早知萧重被掳，根本不会降落萧家，而是会立刻去追那两人。
<br/>　　萧离火着急道，“十数分钟前，一个强者御空而来。”
<br/>　　“一拳便破了你的灵气屏障。”
<br/>　　“我见强敌来袭，便打算使用你留下的五火傀儡球。”
<br/>　　“孰料，才刚拿出来，还未来得及使用，便被那强者夺了去。”
<br/>　　“而后，那强者准备出手时，猎妖殿的人便来了。”
<br/>　　“他似乎很是忌惮猎妖殿的人，只掳走了萧重，留下一道灭了我们萧家的命令，便充满离去了。”
<br/>　　萧离火无比着急地快速说着。
<br/>　　“再之后的事，你也看到了。”
<br/>　　“在两个蒙面武者的带领下，那群黑衣人对我们萧家发起了攻击。”
<br/>　　“原来如此。”萧逸点点头，难怪自己不见萧重被掳。
<br/>　　原来竟还有另外一位强者，掳走了萧重后就离开了。
<br/>　　现在的萧逸，其实无比着急。
<br/>　　只不过，他知道自己现在不能慌，必须冷静。
<br/>　　“能一拳破了我的灵气屏障，这个强者的实力，起码在破玄九重以上。”萧逸暗暗思索着。
<br/>　　这时，猎妖殿执法队的两个队长，走了过来。
<br/>　　刚才，他们正对地面上的黑衣人尸体做着检查。
<br/>　　“不知几位是？”两个队长对着萧逸拱了拱手。
<br/>　　刚才萧逸一出现，便能吓退两个破玄境武者，两个队长自知面前年轻人绝非泛泛之辈。
<br/>　　萧逸不语，还在思索着。
<br/>　　玉如龙回答道，“我是裂天剑派，兽堂首席，玉如龙。”
<br/>　　木妙妙也道，“裂天剑派，万木堂首席，木妙妙。”
<br/>　　“原来是裂天剑派的高徒。”两位队长点点头。
<br/>　　“此事，我们要立刻赶回猎妖殿，上报总执事。”
<br/>　　“今日袭击萧家的势力，我们也会尽快查出来。”
<br/>　　“告辞。”两位队长，对着萧家大长老拱了拱手，而后转身离去。
<br/>　　受猎妖令保护的家族，若被灭，猎妖殿将倾力为其报仇。
<br/>　　若只是受袭击，则不会太过重视。
<br/>　　毕竟受猎妖令保护的家族或者个人，多了去了。
<br/>　　今日萧家之事，这些执法队，应该会上报总执事，而后再商议后续的对策。
<br/>　　执法队离去。
<br/>　　萧逸没有管他们，自顾地走到黑衣人尸体前，细致地检查了起来。
<br/>　　“逸儿，如何了？”萧家众长老问道。
<br/>　　萧逸回答道，“这些黑衣人，都是猎妖师。”
<br/>　　萧逸从他们身上，找到了猎妖殿的令牌。
<br/>　　“猎妖师？猎妖师怎么会对付我们萧家？”萧离火疑惑问道。
<br/>　　“收人钱财，替人消灾罢了。”萧逸回答道。
<br/>　　“猎妖师，可不止猎妖；只要给钱，谁都能雇佣他们办事。”
<br/>　　“猎妖师，漂泊无踪，现在又全死了，很难找到是谁雇佣的他们。”
<br/>　　萧逸暗暗咬牙，早知道刚才就留下活口。
<br/>　　萧离火担忧道，“刚才那两个为首的蒙面人，可是自信连猎妖殿都查不出他们是哪个势力，这下麻烦了。”
<br/>　　“我猜，他们肯定是北山慕容家的人。”
<br/>　　“会对付我们萧家的，也就他们了。”
<br/>　　其余萧家长老说道。
<br/>　　萧逸自语道，“他们明明可以杀了萧重叔叔，却选择掳走。”
<br/>　　“想来，萧重叔叔一时半会不会有生命之虞。”
<br/>　　“只要我尽快查到是哪个势力在对付我们萧家便可。”
<br/>　　萧逸说罢，转过身，准备离去。
<br/>　　“逸儿，你去哪？”众长老问道。
<br/>　　“去调查。”萧逸回答道。
<br/>　　“如何调查？”萧离火问道。
<br/>　　萧离火也是人精，定下心神后，越想越是觉得今日之事不对劲。
<br/>　　“这个势力，不用自己的武者，反而多此一举的雇佣猎妖师。”
<br/>　　“必是为了不想留下蛛丝马迹。”
<br/>　　“这分明就是一场针对萧家的早有预谋的阴谋。”
<br/>　　“他们如此小心翼翼，是忌惮猎妖殿查出他们。”
<br/>　　“我甚至相信，刚才逸儿你就算不杀光那些黑衣人。”
<br/>　　“那两个蒙面武者，也会先杀光他们，再逃离，不会留下任何线索。”
<br/>　　萧逸点点头，冷笑一声，“他们能雇佣猎妖师，我也能驱使猎妖师。”
<br/>　　“各位长老等着，逸儿去去就回，必能将萧重叔叔救回来。”
<br/>　　萧逸打算，以易霄的身份，去一趟北山城，请分殿主帮忙。
<br/>　　猎妖师，遍布各地。
<br/>　　只要让各大猎妖殿帮忙，请动所有猎妖师提供情报。
<br/>　　要查出今日袭击萧家的，是哪个势力，绝非难事。
<br/>　　“嗯？”萧逸刚想离去，脚步却猛地一滞。
<br/>　　“怎么了？”萧离火问道。
<br/>　　“大长老，我记得我说过，若萧家有强敌来袭，可捏碎我给你的子令。”
<br/>　　“我会立刻感知到，并赶回来。”
<br/>　　“为何不早早捏碎？可是子令不在你身上？”萧逸问道。
<br/>　　他一直都有留意身上母令的情况，根本不知萧家有事。
<br/>　　直到回来紫云城，看到冲天火光，才发现萧家被袭。
<br/>　　萧离火先是一愣，而后回答道，“子令我给萧重了。”
<br/>　　“你以前告诉我，这子令有护身之效。“
<br/>　　“我想着，既然你把五火傀儡球给我了；我便把子令给萧重护身去了。”
<br/>　　“可是，刚才萧重被掳，也没有见那子令有什么护身功效阿？”
<br/>　　萧逸瞬间一惊，而后蓦地一笑。
<br/>　　子令所谓的护身功效，是指威慑性，而并非子令本身有什么强大的护身功效。
<br/>　　“原来子令在萧重叔叔身上，那就好办了。”萧逸身影一闪，瞬间离去。
<br/>　　猎妖令，分子母令。
<br/>　　两令，彼此相连。
<br/>　　子令不可以感知到母令的情况。
<br/>　　但母令，却可以追寻到子令在何处，也可得到子令上反馈回来的气息。
<br/>　　高空中，萧逸往母令内输入真气。
<br/>　　母令瞬间生效，发出一阵光芒。
<br/>　　代表着，已经与子令产生联系。
<br/>　　“萧重叔叔，等着我。”萧逸脸色一冷，速度飙升到极致。
<br/>　　……
<br/>　　另一边，萧家内。
<br/>　　玉如龙和木妙妙见萧逸瞬间离去，顿时一愣。
<br/>　　他们可跟不上萧逸的速度。
<br/>　　“怎么办？”玉如龙问道。
<br/>　　“立刻发信号回剑派。”木妙妙沉声道，“萧逸如今是剑主，不能出事。”
<br/>　　“那股势力，也不知是何势力，到底难不难对付。”
<br/>　　“还是立刻通知长老们吧。”
<br/>　　两人立刻发信号，却不知抽什么疯，竟发了八个字回剑派。
<br/>　　那八个字，正是‘萧逸有危，速来救助’。
<br/>　　他们甚至想不到，就因为这八个字。
<br/>　　整个北山郡的大人物，都在一瞬间动了起来。
