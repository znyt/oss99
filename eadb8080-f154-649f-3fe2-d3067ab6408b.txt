　　第283章
<br/>　　徐神机眼皮微抬，轻描淡写一句话，却给赵三壮带来巨大的压力。
<br/>　　“是，是。”赵三壮点头哈腰，赶忙答应。
<br/>　　“诸位……”徐神机忽然起身，把所有人注意力吸引到自己身上：“诸位都是附近的乡亲父老，以后我们蛤蟆寨要在此地安家立寨，大家都是邻居。古语有云：远亲不如近邻，蛤蟆寨免不得有用得着各位的，当然也不会让大家白帮忙。而我们兄弟既然来了，自然也保一方太平，附近若有那个为非作歹，作威作福，自有我们兄弟照应。诸位，且看……”
<br/>　　徐神机突然高声，把周围人都吓了一跳。大家顺他手指的方向看去，只见在蛤蟆寨的大旗旁，还立着一根旗杆，上面四个大字飞舞：
<br/>　　替天行道。
<br/>　　两千壮丁拿着工钱离开，这一次的事情对他们来说是大落大起，，大惊大喜，最后还是喜大于惊。本以为这次是有去无回，没想到最后能拿着工钱欢天喜地回家。
<br/>　　而这个时候，程大雷也收到了系统提示。
<br/>　　嘟，山寨翻修任务完成，山寨所有人水性+1，拥有者水性+2 。
<br/>　　嘟，二级山寨建设完成，点将台功能开启。
<br/>　　嘟，二级山寨建设完成，随机奖励山寨技能:江湖传言
<br/>　　江湖传言：秀才不出门，便知天下事，江湖传言将通过各种渠道传到你耳中。
<br/>　　嘟，二级山寨建设完成，拥有者力量提升。
<br/>　　……
<br/>　　上次建设成二级山寨，程大雷直接提了一级，不过他现在已经是顶级实力，所以这次获得的只是力量方面的提升。
<br/>　　如果力量，敏捷，体力提升到一个足够的境界，也就是他突破顶级，进入绝世的时候。
<br/>　　程大雷打开系统界面，分别查看山寨和自己的信息。
<br/>　　山寨：蛤蟆寨（二级山寨）
<br/>　　拥有者：程大雷
<br/>　　人口：113
<br/>　　技能：建筑精通，江湖传言，招募（大旗附带技能）
<br/>　　姓名：程大雷（名扬四海的顶级山贼）
<br/>　　年龄：20
<br/>　　技能：大魔王三板斧，雄辩，阿飞快剑，身无彩凤双飞翼，心有灵犀一点通
<br/>　　隐藏属性：一个好人
<br/>　　“大当家，大当家……”
<br/>　　程大雷正查看着系统信息，徐神机这个时候跑过来，道：“大当家，我刚得到小道消息，说是窦竹仝联合十八家土匪，择日就要攻打我们。”
<br/>　　小道消息？这是江湖传言发挥的效果。这个时代信息滞后，想要秀才不出门，便知天下事几乎相当于扯淡，而有『江湖传言』这个技能，程大雷就能提前得到各方各面的情报。当然，既然是江湖传言，准确度就不会太高，还需要甄别分析。
<br/>　　“小道消息，你是从哪里来的？”
<br/>　　“是出去买东西的兄弟回来后说的。”
<br/>　　“哦，都说了些什么啊？”
<br/>　　“据说，莫鸣米把手下十八家山寨都联合了起来，总兵力超过一万，这几天随时都会杀过来。”
<br/>　　说罢，徐神机又补充一句：“大当家，咱们可是不得不准备啊，对方要真有一万人，凭咱们这百十个兄弟，无论如何也挡不住啊。”
<br/>　　“我知道了，你去查查莫鸣米留下的铁箭有多少支，不够一万的话，让弟兄们去外面找关系买些回来。”
<br/>　　“可人的事……”
<br/>　　“人的事不用你担心，到时候……”程大雷轻咳一声，忽然高声道：“自有神兵天降！”
<br/>　　徐神机一头雾水，摸不准程大雷的套路，反正这也不是第一次了。他离开后，便去安排手下筹集弓箭，随时准备战斗。
<br/>　　程大雷坐在大厅内，口中默念了两个字。
<br/>　　“抽奖。”
<br/>　　脑海中的扭蛋机出现，再一次开始旋转。
<br/>　　程大雷很久之前就获得一次抽奖机会，只不过因为没有什么需求，所以他也一直没有抽。这次大战来临之，多一些准备总不会错的。当然，以窦竹仝为首的十八路乌合之众，在程大雷看来也算不得什么大战。
<br/>　　莫说窦竹仝，便是莫鸣米亲自，只要二级山寨建设完成，程大雷就没有怕的道理。
<br/>　　骨碌碌旋转的声音，从扭蛋机中掉出一物，程大雷收到了系统提示。
<br/>　　嘟，恭喜获得顶级武奖……
<br/>　　后面缓缓出现三个字：樊梨花。
<br/>　　咦，程大雷可以说是又惊又喜，惊喜交加，这还是第一次抽到一位女将，还是那名震大唐的西凉第一女将。
<br/>　　系统终于改过自新了么，还是自己已经脱非入欧，人品开始变好了。
<br/>　　“大当家，大当家……”高飞豹跑过来：“出事了？”
<br/>　　“什么大事？”程大雷思绪被打断有几分不痛快：“能不能有几分大将风度，泰山崩于前而不乱，麋鹿兴于左而不变？”
<br/>　　“外面来了个姓樊的女将……”
<br/>　　“什么，我家樊娘子来了！”程大雷腾地坐起，大步流星向外跑去。
<br/>　　“不是说要有大将风度么。”高飞豹撇撇嘴：“我还没说完呢，人家是点名道姓要来杀你的。”
<br/>　　程大雷跑到码头上，见码头上已经聚集了不少人，秦蛮，赵子龙，关鱼，张肥等人都在。
<br/>　　在码头旁，拴着一艘小船，岸上有匹胭脂马，马上一穿红挂绿的红袍女将，秀目圆睁，腮边挂红，当真是俊得一逼。
<br/>　　程大雷阅读对方信息。
<br/>　　姓名：樊梨花（初出茅庐的顶级武将）
<br/>　　年龄：26
<br/>　　技能：梨花刀，骑术精通。
<br/>　　隐藏属性：绝世之质
<br/>　　这便是哪位师从黎山老母，官封大唐征西元帅的樊家梨花，当然，这是原先设定。就像林冲一样，到这个世界，人物背景肯定是不一样的。
<br/>　　接下来看看人设是什么。
<br/>　　程大雷深施一礼，款款言道：“敢问姑娘家住何方，芳龄几许，是否婚配，从哪里来，又要到何处去？”
<br/>　　“呔，你就是那姓程的狗贼，姑奶奶有话说与你听，劝你从哪里来，回哪里去，琴川不是你撒野的地方，若敢有半个不字，就与姑奶奶的刀说话。”
<br/>　　呃……程大雷被噎得够呛，看来这次招募任务难度不低。
<br/>　　“秦蛮，抓了。”程大雷一挥衣袖，直起腰杆。
<br/>　　“大当家，已经动过手了。”秦蛮。
<br/>　　“然后呢？”
<br/>　　“输啦。”
