　　“一枪——生世界！”
<br/>　　纣王一枪向前轰出，于枪尖一点亮光闪现，紧接着，光点变大，汇聚成一条真龙，整杆枪都仿佛化成了真龙，真龙向前一声长吼，腾跃而起，光亮散去，一个崭新的世界出现在元始面前。
<br/>　　世界之内，日月星辰，无尽星河，山川江河，草木森林……应有尽有，但元始却感受到，这个世界之内的锋芒。此界之内，全是枪芒的锋锐，万物皆由枪道构成，所谓一枪生世界，便是如此。
<br/>　　元始脚步向旁边一跨，两者之间的空间距离，瞬间便拉得很长，看似一步之遥，实际上却是不止亿万公里，这便是空间大道的妙用，咫尺天涯。
<br/>　　随后，元始一手伸出，先前抓去，空间迅速缩小，那一方世界，也随着元始的大手逐渐缩小，元始将手捏起，那世界便化为一道拳头大小的光球，在元始手中不断挣扎，看似是一只手掌，实际上却是一个世界。
<br/>　　世界迅速缩小，元始张口一吞，便将整个枪道世界吞入腹中，如此，元始对枪道的领悟，在飞速提升着，那一方世界，落入元始体内一个玄窍之中，化为一方真实世界。
<br/>　　“味道也不过如此！”未了，元始还淡淡评价一句。
<br/>　　“嘶！”
<br/>　　见此，纣王等人倒吸一口气，都傻眼了，这是什么鬼，竟然将一道世界攻击就这么吞下去……吃了！这怎么可能，世界也能吃？你就算将它毁灭，别人只会感叹你实力强大，但你吃了又是要干什么，吓人也不是这样吧！
<br/>　　“再来！”
<br/>　　纣王脸色沉重，不甘心就此输掉，他继续操控大阵，无论如何也要在元始身上找回点自信。
<br/>　　“时间——静止！空间——冻结！”
<br/>　　显然，借助衍生界，纣王一样能够做到时间静止与空间冻结。两者都是时空一道上最为厉害的神通之一，理论上只要足够强，完全可以无敌。
<br/>　　时间静止，便是让你周遭的时间完全处于静止状态，身体不能动，甚至就连思维都会停止不动，你的一切都保持原样。而空间冻结，那是将一处空间本身定住，这一处空间不再有波动，更不会与其他地方有联系，处于这种环境下，圣人亦不可瞬移。
<br/>　　思维不能动，身体不能动，那么，等待你的便只有死亡。
<br/>　　瞬间，元始便感觉自己身边的时间被无限制的拉长，在他的感念之中，时间变得极慢，平常一秒钟此刻需要数百上千秒，才可能走完。而且，周边的空间也在向他压来，空间变得坚固无比，每走一步，都须耗费极大的精力。
<br/>　　无数时间与空间的力量向着四面八方疯狂地攻击元始，每一击都带着无与伦比的力量，轰天动地。
<br/>　　只是，这些攻击虽强，但落在元始眼里，还是不够看。毕竟，修为的差距在那里摆着，你想让一只蚂蚁去将大象定住，可能吗？。
<br/>　　“你能凭借准圣修为做到如此地步，就算是借助衍生界的力量，也足够自傲了，只可惜，你面对的是本座，你永远不知道圣主的恐怖！”
<br/>　　元始开口说道，抬手便是一道元始印轰出，此地立刻有万道法则波动，万法缭绕，道光璀璨，道韵绵延，元始被绚烂的法则秩序神链围绕，他就是法则的化身，光辉如雨洒落，光束漫天。
<br/>　　此刻天地法则化成了有形的锁链，五彩缤纷，十色齐出，流霞浪漫，瑞彩飞舞，氤氲弥漫，看起来极为灿烂与梦幻。
<br/>　　这些秩序神链最终化为一道烘炉，如天地烘炉般广阔无涯，一炉便包含了天地，将元始包裹在内。
<br/>　　纣王借助衍生界施展出的时空大道，不论是时间法则，还是空间法则，此刻全部被法则烘炉吸收，被元始大道同化，化为元始大道的一部分。
<br/>　　“什么！”
<br/>　　纣王骤然色变，他几乎全力一击，竟然被对方以如此方法破除，他不但没有占到半分便宜，竟然还送给对方机缘。
<br/>　　对方没有以时空大道破解，也没有以强大的力量轰碎一切，却是以如此方法将他的攻击给吸收了。
<br/>　　这超乎他的想象，不可思议。
<br/>　　“六道——轮回！”
<br/>　　纣王不服输，发动最后一击，他调动全部大阵力量，汇聚起来，大地的力量、黄泉的力量、死亡的力量等聚集，纣王施展轮回神通，整片衍生界化成了一方冥界，六道巨大的黑洞滚滚而来，蕴含着轮回的力量。
<br/>　　滚滚道韵形成，一种铺天盖地的天地大道气息，犹如混沌初开炸裂般滚滚而下。
<br/>　　浩瀚的天地大道在轮回之中衍生，这一刻世界在返源，不，这一刻，世界在聚拢。
<br/>　　天地从混沌而来，一切规则慢慢衍生，衍生出来的规则在慢慢聚拢。
<br/>　　浩瀚宏大的轮回道韵在天地间轰鸣，道韵越来越磅礴，轮回的气息，包裹着空间与岁月一次次轮转，岁月开始疯狂的流逝。
<br/>　　天地间的一切生命，天地间的一切生灵在这一刻被轮回所掌控，他们都要在这一刻重新回归混沌，也在这一刻于混沌之中灭亡。
<br/>　　元始感觉自己就要随着这轮回不断转轮，回归混沌的怀抱，他知道，这是一种错觉，以纣王如今的实力，还不足以让他轮回，但这种轮回大道，却是极为可怕的。
<br/>　　元始周身元始大道在轰鸣，秩序神链所化天地烘炉在不断吸收着这种轮回的气息与道韵，不断被元始大道同化。
<br/>　　元始感觉自身对轮回一道的理解正在以一种不可思议的速度提升着，他不禁沉醉在这种美妙的感觉中。
<br/>　　数个呼吸后，元始睁开眼睛，他看到整个衍生界都已经开始倒塌，天地在轰鸣，欲重演地水火风，归于混沌。
<br/>　　“好了，不再浪费时间了，纣王，本座对你的秘密十分感兴趣，不过，想必你也不会亲手交于本座，那么，本座只好自己来取了。”
<br/>　　元始收起元始大道，看着就要准备离开的纣王淡淡道。
<br/>　　“可恶，真一定会回来的。”纣王恨恨地看了元始一样，心中正准备默念离开。但就在此时，元始那淡淡的声音传来：
<br/>　　“你想要去哪里啊，本座早已猜到你会离开此界，所以特意为你准备了一份大礼，请收好！”
<br/>　　接着，就在纣王那逐渐淡化的身影要消失之际，一个巨大的圆印从天而降，周边虚空尽皆被定住，甚至连时间也没了意义，纣王维持着那离开的状态，丝毫不能动弹。
<br/>　　在他那不可置信的目光之中，圆印连着他与那片虚空一同收了进去。
<br/>　　原地，元始看着手中缩小无数倍的圆印，淡淡笑道：
<br/>　　“知道你有至宝，能够穿梭万界，是以，要对付你，也只有同样的至宝方才管用。本座这鸿蒙元始印的本源空间内，却不在万界之中，你倒是给本座穿个看看，看看能否穿越回去？”
<br/>(本章完)
