　　“斩星剑道。”
<br/>　　萧逸停下身影，看着面前的剑碑，面露喜色。
<br/>　　偌大个万剑碑林内，除了那剑帝碑参天般高大无比外，其余剑碑，大多在差不多大小。
<br/>　　大概在十数米高，宽数米。
<br/>　　剑碑之上，没有关于这位前辈的事迹；有的，仅仅是一些武道方面的文字记载。
<br/>　　武者通过这些文字记载，便可知晓这位前辈生前修的是什么剑道。
<br/>　　而自己参悟这块剑碑，又能得到什么剑道方面的参悟。
<br/>　　很显然，萧逸面前这块剑碑的主人，生前便是一位修习斩星剑道的前辈。
<br/>　　萧逸盘膝坐下，准备开始参悟。
<br/>　　这里的一块块剑碑，显然都层次极高。
<br/>　　对于云渊长老二人来说，他们的剑道之路，已经走到一个极其前的地步。
<br/>　　他们的剑道，很大程度上，代表的便是他们自己。
<br/>　　故，即便这里的剑碑层次极高，但对他们二人来说，也仅仅是存在借鉴的效果。
<br/>　　可萧逸的剑道之路，还远着呢。
<br/>　　这里的任何一块剑碑，都对他的剑道提升，大有裨益。
<br/>　　刚要开始闭目参悟。
<br/>　　丁秋月的身影，却缓缓走来，随后在他身旁的剑碑前坐下。
<br/>　　“哦？”萧逸饶有兴致地看了眼身旁石碑。
<br/>　　“清风揽明月，尽在飘渺间。”
<br/>　　“是飘渺剑道。”
<br/>　　丁秋月看了眼萧逸，又看了眼剑碑，歪了歪脑袋，道，“大哥哥，我觉得这剑碑上的剑道，很适合我呢。”
<br/>　　“当然适合。”萧逸笑道，“这飘渺剑道位列顶尖剑道之一。”
<br/>　　“不狂不躁，不猛不烈，但却玄奥莫测。”
<br/>　　“最是适合你这种心清如水，心境平和的武者修炼。”
<br/>　　事实上，飘渺剑道是极其难得的剑道之一。
<br/>　　修习起来，亦极难。
<br/>　　怕是放眼这数万剑碑，这飘渺剑碑都是其中最厉害的剑碑之一。
<br/>　　丁秋月这丫头，修习这个剑碑，确实是极好的选择。
<br/>　　“那便听大哥哥的。”丁秋月盈盈一笑。
<br/>　　不过，丁秋月的目光，也注视了碑林中间那巨大剑帝碑一眼。
<br/>　　那里，子风师弟与桌万宗，已开始盘膝坐下。
<br/>　　参悟剑帝碑的名额，只有两个，而且只有剑域本土的剑者能够去参悟。
<br/>　　萧逸虽然也修炼出了魂剑力，能够参悟，但他并没有名额。
<br/>　　而丁秋月，自也是面露羡慕目光。
<br/>　　“难怪你如此看重那场比拼，明知不敌，却非要再战。”萧逸摇了摇头。
<br/>　　事关剑帝碑的参悟名额，自然不仅丁秋月，连丁家族长还有器宗老宗主，都不愿放弃那场比拼。
<br/>　　丁秋月略微失落地点了点头，“剑帝碑，可是我们剑域的圣物。”
<br/>　　“任何一个剑者，一生的愿望，便是在剑帝碑前参悟一次。”
<br/>　　丁秋月瘪着嘴，“就算不能参悟，去摸一回，看也回也是好的。”
<br/>　　或许，在丁秋月心头，剑帝碑，已不单单是一个可以参悟且提升剑道的剑碑。
<br/>　　而是一份信仰。
<br/>　　一个在剑域中长大的纯粹剑者，心头的一份信仰。
<br/>　　“别多想了，好好参悟吧。”萧逸笑笑。
<br/>　　“嗯。”丁秋月点了点头，散去了脸上失落，恢复了一如既往的心境平和，开始闭目参悟。
<br/>　　萧逸淡淡一笑，看回自己的剑碑，也开始闭目参悟。
<br/>　　“摸一回，看一回吗？”萧逸心头暗暗想着。
<br/>　　这不过是丁秋月的一个简单愿望。
<br/>　　若是可以的话，盛事结束之后，拜托林夜一番，重新带丁秋月来一趟，应当不是什么难事。
<br/>　　当然了，参悟或许不行，毕竟剑域有剑域的规矩。
<br/>　　但仅仅是去观摩一番，近距离细细感受，却不是什么大事。
<br/>　　下一秒，萧逸收敛了杂念，快速参悟起来。
<br/>　　这斩星剑道，他听说过，剑道狂猛霸道。
<br/>　　一来，这种狂猛的霸道，与他的霸道剑道比较契合。
<br/>　　二来，斩星剑道，亦有着星光一道的武道意味；故对他的星幻剑道一途，同样有着可借鉴的效果。
<br/>　　这一剑道，倒是无比适合他参悟。
<br/>　　另外一点，事实上，单纯一条剑道的话，本身的剑道知识，并不算太多。
<br/>　　如萧逸的霸道剑道，早就完整了。
<br/>　　但，这些剑碑上的剑道知识，乃是一位剑道前辈的一生剑道知识。
<br/>　　更简单来说，这位前辈，乃是以‘斩星剑道’为基，作为第一条剑道。
<br/>　　随后，不断融合别的剑道，不断走着、完善着自己的剑道。
<br/>　　当然了，这位前辈的剑道之路，已经走得极其远了。
<br/>　　萧逸真正参悟的，便是这位前辈的整条剑道之路。
<br/>　　这位前辈，以斩星剑道为基，融合了什么剑道在里头，又融合了多少条剑道在其中，最终走出一条滔天剑道之路。
<br/>　　萧逸对此，充满了期待。
<br/>　　萧逸快速参悟着。
<br/>　　一分钟后。
<br/>　　轰…萧逸的脑海，忽然产生一声炸响。
<br/>　　脑海中，一阵剧烈疼痛。
<br/>　　但这种疼痛，来得快，消失得也快。
<br/>　　当萧逸反应过来时，猛地睁开双眼，却发现，周遭一切，早就变了。
<br/>　　眼前剑碑，身旁丁秋月等等，早便消失。
<br/>　　出现于眼前的，是一片漆黑。
<br/>　　“怎么回事。”萧逸心头暗暗惊骇。
<br/>　　周遭，暗无天日，伸手难见五指。
<br/>　　他仿佛已不在万剑碑林，而是身处别的空间。
<br/>　　萧逸皱了皱眉，脚步开始挪动。
<br/>　　停留原地，绝不是一个好的选择。
<br/>　　又是数分钟后，萧逸的眉头，皱得更紧。
<br/>　　他忽然发现，自己置身这一漆黑无比的空间，不断走着，却如同原地踏步。
<br/>　　“嗯？”萧逸忽然双眼一眯。
<br/>　　向来敏锐的他，忽然察觉到，这漆黑的空间内，似存在着莫名的武道意味。
<br/>　　萧逸闭目感知了一下。
<br/>　　“武道意味狂猛而虚幻，或势如破竹，又带着莫名的虚幻星光。”
<br/>　　“是斩星剑道。”
<br/>　　萧逸瞬间反应了过来。
<br/>　　“原来如此。”
<br/>　　萧逸开始参悟着这里头的斩星剑道。
<br/>　　边参悟，脚步，却也开始挪动着。
<br/>　　他明显发现，这里的剑道意味，并非单纯停留，而是不断移动。
<br/>　　不过，他能否追寻这些移动的剑道力量，却要看他能否参悟掌控这些剑道力量了。
