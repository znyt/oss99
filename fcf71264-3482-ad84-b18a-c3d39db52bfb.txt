　　苏莫离开了明岳城之后，便径直向终极剑域而去。
<br/>　　拥有天幻神丹，他还有有些底气的。
<br/>　　一边飞行，苏莫一边参悟吞噬法则，他的吞噬法则，即将进入三阶的地步，不能懈怠。
<br/>　　一旦晋升三阶，对于他的总体实力，也能提升一些。
<br/>　　本已晋阶在即的苏莫，一个多月之后，吞噬法则便达到了三阶初期，算是迈进了一个更高的层次。
<br/>　　“不知道如今吞噬法则，具体有多强？”
<br/>　　一边飞行，苏莫一边暗想，去时空之城之前，他的吞噬法则才一阶圆满，现在三阶初期，可谓是天差地别。
<br/>　　三阶的吞噬法则，若是配合剑道施展，定然威力极大。
<br/>　　苏莫没有尝试，不再多想，静心赶路。
<br/>　　数月之后，他再次回到了终极剑域的地界，进入终极剑域之后，他格外的小心。
<br/>　　不再高空飞行，而是低空飞掠，以免引起别人的注意。
<br/>　　随后，苏莫进入了一座城池之中，暗暗打探了一番万道剑冢的情况。
<br/>　　这一打探之下，发现这万道剑冢，当真是一处宝地。
<br/>　　万道剑冢，在距离神剑山不是很远的地方，是一处禁地，连神王级别的强者，都不敢轻易踏足。
<br/>　　只有百年一次开启的时候，才能顺利进入。
<br/>　　当然，这万道剑冢也是一处考验之地，只对妖孽天才开放，非妖孽不可入。
<br/>　　千岁之内的妖孽，均可进入，但是，实力越强，危险越小，实力越弱，危险越高。
<br/>　　只要是能从万道剑尊中走出来之人，必然实力大增，迈入更高的层次。
<br/>　　具体万道剑冢之内，是什么情况，苏莫也没有打听出来，不过，这都不重要了，他进去之后自然能知晓。
<br/>　　“应该不会有神王级别的强者来吧！”
<br/>　　苏莫行走在苍茫的大地之上，心中暗道，万道剑冢，应该只会吸引附近几个界域的天才，不太可能有神王级别的强者出现。
<br/>　　这样的话，他并没有多大的危险。
<br/>　　唯一有风险的是，如何将元神给予绫裳，毕竟，他并不能传讯给对方。
<br/>　　距离万道剑冢开启，还有两个多月，苏莫想趁着这段时间，将元神给予绫裳。
<br/>　　他现在行径的方向，正是神剑山所在的方向。
<br/>　　数日之后，苏莫距离神剑山，已经不远了，他极目远眺，已经能看到神剑山和山脚下城池的轮廓。
<br/>　　他停了下来，不再前进，蛰伏在隐秘之处。
<br/>　　他在等待，等待有终极神王殿的弟子路过，最好是绫裳剑宫的弟子路过，好帮他通知绫裳。
<br/>　　这里距离神剑山不是很远，不时便有弟子路过，很快，苏莫便锁定了一个人。
<br/>　　此人是一个女子，容貌一般，穿着一身翡翠色的长裙。
<br/>　　女子修为不高，只有二品低级，身上悬着腰牌，正是终极神王殿弟子特有的身份牌。
<br/>　　女子从外边回来，很显然是要回神剑山。
<br/>　　嗖！
<br/>　　苏莫冲天而起，立刻将女子拦住了。
<br/>　　“什么人？”
<br/>　　女子见到突然出现的苏莫，顿时一惊，取出长剑，一副警惕的模样。
<br/>　　“姑娘莫慌，在下有事相求！”苏莫身形停在了对方前方，面带微笑的说道。
<br/>　　“什么事？”女子一边打量苏莫一边问道，她完全看不透苏莫，但是却能看出苏莫的修为比她高。
<br/>　　“姑娘是哪一宫的弟子？”苏莫沉声问道。
<br/>　　“绫裳剑宫。”女子淡淡的说道，面带警惕之色，不过，她并不是很担心，毕竟此地距离神剑山很近，有大的动静，神剑山上就能发现。
<br/>　　“正好，我有一事，需要姑娘帮忙，十万中品源石作为酬劳！”苏莫道。
<br/>　　“什么忙？”女子面色惊疑不定，十万中品源石，对于她来说，数量不算少，但是，也要看具体是什么忙？
<br/>　　“很简单，你告诉绫裳宫主，我公乘太皓在这里等他！”苏莫微笑着道。
<br/>　　为了安全起见，他没有告诉对方秦陨之名，也没有告诉对方苏莫之名。
<br/>　　而是说了公乘太皓这个名字，因为只有绫裳，才知晓这个名字。
<br/>　　“就这么简单？”女子闻言愕然，这么简单之事，对方居然出十万中品源石？
<br/>　　“就这么简单，只要你通传一声就好！”苏莫点了点头。
<br/>　　“那好，我可以为你通传！”女子没有过多考虑，便点了点头。
<br/>　　此事，之事举手之劳而已。
<br/>　　她猜测，苏莫可能是有什么难言之隐，才不便自己去神剑山，但是，这都于她无关。
<br/>　　通传一声，就能得到十万中品源石，天下间没有比这再轻松的交易了。
<br/>　　“这是十万水属性中品源石！”
<br/>　　苏莫手掌一番，掌中出现了一枚储物戒，他没有立刻给予对方，而是叮嘱道：“还请姑娘保密，此事不能让任何人知晓！”
<br/>　　“哦？”女子闻言，面带好奇之色，难道眼前之人，和绫裳宫主之间，有什么秘密不成？
<br/>　　不过，她也没有多问，因为不可能问出结果，随即，她便点头道：“我答应你，不会让其他人知晓！”
<br/>　　“那就好！”苏莫微微一笑，随即手掌一抛，便将储物戒交到了对方的手中。
<br/>　　女子接住储物戒，稍一检查，便确定无误，其内的确是十万水属性的中品源石。
<br/>　　“告辞了！”
<br/>　　女子没有多留，抱了抱拳，便闪身向神剑山飞去。
<br/>　　苏莫深深的吸了口气，对方想必不会骗他，毕竟，通报一声而已，太简单了。
<br/>　　随即，苏莫闪身离开，到附近躲藏了起来。
<br/>　　他很小心，万一这个女子暴露了他的身份，他也好逃走。
<br/>　　没有多久，仅仅是过了半刻钟，苏莫便看到绫裳来了，到达了他之前拦截那名女子之地。
<br/>　　“太好了！”
<br/>　　苏莫心中暗喜，随即立刻现身，向绫裳飞了过去。
<br/>　　绫裳远远的看到苏莫，随着苏莫展露一丝真实的气息，她立刻认了出来。
<br/>　　“怎么回事？”待苏莫来到近前，绫裳柳眉微皱的问道。
<br/>　　她受到宫内弟子的禀报，立刻就过来了。
<br/>　　“这里不是说话之地，我们走！”苏莫不由分说，闪身向神剑山相反的方向而去。
<br/>　　绫裳见此，心中虽然疑惑，但也立刻跟了上去。
<br/>　　飞行了半个时辰，两人彻底的远离了神剑山，便在一处荒山之中降落了下来。
<br/>　　“苏……秦陨，到底是什么情况？”
<br/>　　荒林深处，绫裳不解的凝视着苏莫，什么事情这么神秘。
<br/>　　“紫霄神王不会跟踪你吧？”苏莫没有回答，眸光扫视四方。
<br/>　　“不会，紫霄闭关了！”绫裳摇了摇头道。
<br/>　　“那就好！”苏莫闻言松了口气，随即轻笑一声，道：“这一次来，是要送你一件大礼。”
<br/>　　“你为何不敢回神剑山？”绫裳没有询问是什么大礼，而是问起了苏莫的情况。
<br/>　　她基本上都在闭关，对于苏莫没有关注，并不知道苏莫最近的情况。
<br/>　　“我杀了几名终极神王殿的弟子，以后应该都不会回神剑山了。”苏莫叹了口气，许焱以死的消息，估计也早已经传到神剑山了。
<br/>　　“杀了几名弟子？”绫裳微微皱眉，最近几年，神剑山平安无事，并没有传出此事。
<br/>　　“你难道不知道此事？”苏莫疑惑，这么大的事情，难道对方一点都不知晓？
<br/>　　“从未听闻！”绫裳摇了摇头。
<br/>　　苏莫闻言讶然，这倒是怪了，他可是杀了四个人，绫裳居然都不知晓，难道此事没有在神剑山传开？
