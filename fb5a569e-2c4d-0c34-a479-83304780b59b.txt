　　很显然，这蛇沟中的毒蛇都非常的与众不同。
<br/>　　越是这样，涂小安越对那天材地宝充满了好奇心，跟贪婪。
<br/>　　毫无疑问，那是宝贝，能提升生物的血脉，突破身体桎梏，这种宝贝他还必须要得到。
<br/>　　不过吸血蛇已经抢先一步，肯定在蛇沟深处了。
<br/>　　涂小安继续跟着大部队赶路，快速飞行，眼看就要超越了那条十米长的黑曼巴。
<br/>　　这黑曼巴本身就是蛇类中爬行最快的毒蛇了，但体型变大，也丝毫没有影响它的行动速度，反而有点一日千里的意思。
<br/>　　绝对能跟跑车赛跑了。
<br/>　　所谓天下武功唯快不破，这家伙攻击速度快，爬行速度更是毫不逊色。
<br/>　　此时，涂小安这条小蛇飞到黑曼巴的头顶，只见此蛇蛇瞳中流露出诧异之色，一条蛇飞到空中，这一幕搁谁看到都会惊奇。
<br/>　　任它成长到十米，已它的灵智也完全想不明白，这小蛇如何能做到飞行。
<br/>　　那么吊炸天？它都要怀疑自己的蛇人生了。
<br/>　　涂小安却悄然思忖着，这些蛇类都在朝着蛇沟深处赶路，想必知道些什么，何不让它们给自己带路。
<br/>　　于是乎，飞行的金银蛇朝着黑曼巴的头顶落去，就让这个大家伙先当当自己的坐骑吧。
<br/>　　“吼...”
<br/>　　黑曼巴岂能答应，仰天咆哮起来，人有人的尊严，蛇也有蛇的尊严，尤其还是变成十米大蟒的黑曼巴。
<br/>　　这条十米大的黑曼巴要是放出去，可是价值连城。
<br/>　　绝对不逊色一条珍稀蛇种。
<br/>　　“哼，当我蛇王的坐骑，是你的荣幸”
<br/>　　涂小安冷淡无波的看了一眼发狂的黑曼巴，蛇眸浑然一眯，一股王者之气激荡出来。
<br/>　　霎时间，黑曼巴一脸震惊，旋即诚惶诚恐了起来，直接匍匐在地，低下了高傲的头颅，长沟中的毒蛇全都停止前进，瑟瑟发抖起来，就好像要迎接自己的帝王。
<br/>　　涂小安轻飘飘的落到了黑曼巴的头顶，这大家伙现在安分起来了，不敢有一点忤逆。
<br/>　　“走”
<br/>　　涂小安一声令下，黑曼巴再次开始赶路，周围的毒蛇也恢复了正常。
<br/>　　大自然的生物等级高了，都会有一些灵智，就算没有，也有本能，不管什么生物，低下头颅的意思就是在跟强者臣服。
<br/>　　涂小安的气息何等强大，岂能是这些还没觉醒的蛇类可以抗衡。
<br/>　　只要他想，一个念头，就能将黑曼巴收服了。
<br/>　　不过隐隐间，涂小安有预感，蛇沟深处会有更加极品的蛇种等着他。
<br/>　　这黑曼巴现在也就勉强先当当坐骑。
<br/>　　顺着蜿蜒的长沟，涂小安趴在黑曼巴的身上，好生惬意，这种毒蛇爬行速度是真的快，几个眨眼，就是一公里外。
<br/>　　很快，前方的地势面貌变了，迎来了一片开阔地，倏然地，涂小安错愕了起来，因为短短不过十几个呼吸，他居然看到四面八方涌出了很多毒蛇，密密麻麻，成千上万，不，说是几十万都不过分。
<br/>　　太多毒蛇了，小的大的，将他这条蛇王都给吓到了，如此规模的蛇群如潮水涌现，目前是一致的，蛇沟深处，都被空气中的一股异香所牵动吸引。
<br/>　　你难以想象毒蛇的数量有多少，这开阔地放眼望去，四面八方全是涌现的毒蛇，品种就更杂乱看，都说佘山有不下一百种蛇品种，已经非常夸张。
<br/>　　但此时，开阔地的蛇品种才叫真的多，随便一眼，恐怕都能看到几百种不同品种的毒蛇。
<br/>　　有不过几十公分的小蛇，有几米的大蛇，更有几十条体型达到十米以上的巨蟒，它们浩浩荡荡，足足可以去袭击人类城市了。
<br/>　　我的乖乖，要不要那么夸张呀。
<br/>　　涂小安看的目瞪口呆，一时半刻都难以回神，蛇沟的恐怖之处，还是外人所不能设想的。
<br/>　　嗯？
<br/>　　涂小安趴在黑曼巴的头顶，眼观六路，一个余光下，他既然发现了一条珍稀蛇种，混迹在浩浩荡荡的蛇群中。
<br/>　　别的毒蛇涂小安可以不屑一顾，但是珍稀蛇种他是不可能放过的，那是行走的RMB呀。
<br/>　　他这条白镇蛇王，可还肩负这养家糊口的重任。
<br/>　　此蛇通体蓝色条纹，鲜艳瞩目，在蛇群之中让人一眼就能看到它，难以忽略，就宛如婀娜多姿的美女，太扎眼。
<br/>　　别的毒蛇在它的面前，简直是丑陋不堪，不堪入目。
<br/>　　“小家伙，被我看上了，你就别跑了“
<br/>　　“意识锁定，给我收”
<br/>　　涂小安的蛇眸浑然一眯，强大的意识力骤然笼罩在那条珍稀蛇种身上。
<br/>　　呯...
<br/>　　仅仅一个念头，这条珍稀蛇种就出现在了涂小安的系统之中。
<br/>　　此珍稀蛇种叫蓝血，本身颜值非常的高，要在以前，涂小安身为捕蛇人看到这种蛇，恐怕要激动的几天几夜睡不着觉。
<br/>　　它的价值等同一辆兰博基尼的价格，非常珍贵。
<br/>　　佘山基本上不可能出现这种毒蛇，没想到却在蛇沟之中出现，有鸡窝里出凤凰的赶脚。
<br/>　　漂亮，真的很漂亮，颜值基本上只在涂小安这条金银蛇之下了。
<br/>　　赶路中，闲着也是闲着，就让我在找找还有没有别的珍稀蛇种。
<br/>　　涂小安可谓大浪淘沙，一双蛇眸闪闪发光，这种机会不多见啊，四面八方密密麻麻的毒蛇，整整几十万有余。

<br/>　　涂小安笑了起来，能觉醒的蛇，价值已经超过了珍稀蛇种了，这些蛇，都是他的，全都是他的。

<br/>　　此时开阔地出现的毒蛇数量已经恐怖如斯，如果涂小安将这些毒蛇都能收服，才称的上真正意义上的蛇王。
