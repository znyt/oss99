　　巨大的刀影斩向苏莫。
<br/>　　不过，苏莫的体外有大日领域，这便是极强的防御屏障。
<br/>　　刀影斩进大日领域之中，立刻遭到恐怖高温的炙烤，威力不断的减弱。
<br/>　　“碎！”苏莫身形微微后退，同时再出一剑，立刻将刀影击碎，化为浩大的气浪，随即被大日领域蒸发一空。
<br/>　　“还有什么绝招，都使出来吧！”苏莫朗声向冷江喊道，他现在已经没有什么担心了。
<br/>　　对方的实力虽强，但并没有强过他太多，不足为惧！
<br/>　　若是对方短时间内不能击败他，那他就能反败为胜。
<br/>　　冷江脸色阴沉如水，眸中寒光闪动，看来不展现巅峰实力，根本无法击败苏莫了。
<br/>　　嗡~~
<br/>　　就在此时，冷江背后的战魂虚影，一阵颤动，而后居然钻进了他手中的长刀之中。
<br/>　　下一息，淡金色的长刀高高举起，爆发出夺目的光芒，一股凌厉恐怖的气息，从长刀之中暴冲而起，浩荡九天十地。
<br/>　　这股气息威势惊天，乃是战魂和冷江修为实力的完美结合。
<br/>　　苏莫见此，顿时眼眸一凝，原来兵器战魂，还有这种用法。
<br/>　　那其他的战魂，能不能也这么用，毕竟，他可是也吞噬了几个战魂。
<br/>　　比如巫森的地狱三头狼战魂，途家家主的岩石战魂，廖云剑的融灵兽战魂！
<br/>　　不过，苏莫没有时间多想，他心中亦是凝重了起来，因为对方长刀中的气息太恐怖了，让他都感觉心中发寒。
<br/>　　“冷江这是要一击定胜负了啊！”
<br/>　　“他可真是不计后果，居然直接动用战魂攻击，若是战魂受创，那可不是小事！”
<br/>　　“估计他有必胜的把握吧！”
<br/>　　“也是，苏莫的实力虽然不弱，但还不足以伤到冷江的战魂！”
<br/>　　观战的人议论纷纷，用战魂直接攻击，虽然威力巨大，但若是战魂受创，想恢复过来可不是易事。
<br/>　　不过，众人也理解冷江的想法，苏莫的实力虽强，但还没有强到离谱的地步，肯定挡不住冷江战魂的攻击。
<br/>　　如此，冷江一击便能决定胜负。
<br/>　　所有人都睁大了眼眸，决出胜负的时刻，就要到了！
<br/>　　苏莫，已经必败无疑了！
<br/>　　聂乾和韩盈两人，均是紧张了起来，若是苏莫挡不住，极有可能会当场陨落。
<br/>　　何传岩亦是紧张了起来，若是记下来冷江再击不败苏莫，那他的仇就很难报了。
<br/>　　木璃深吸了口气，清冷的眼眸紧紧的凝视着虚空中的两人，她也是很期待，苏莫到底能不能挡住冷江的绝杀。
<br/>　　嗖！
<br/>　　下一刻，冷江身形如电，直接向苏莫冲了过来。
<br/>　　他的速度奇快无比，瞬间便冲进了大日领域之中。
<br/>　　大日领域虽强，但冷江却是丝毫不惧，因为仅凭大日领域，根本不能对他造成多少的伤害，最多影响一些他的实力而已。
<br/>　　他的体外，布下了一层浑厚的护体玄力，足以保证他短时间内的安全。
<br/>　　“斩！”
<br/>　　冷江冲进大日领域之后，急速临近了苏莫，手中长刀一挥，直劈而出。
<br/>　　刹那之间，巨大的刀影再次出现，这刀影呈纯金之色，威势恐怖骇人，气势如怒海狂涛。
<br/>　　这金色刀影，仿佛就是冷江的战魂放大了数十倍一般，比之前的两刀强大了近一倍。
<br/>　　刀影一出，大日领域仿佛都承受不住了一般，开始不断的溃散。
<br/>　　嗤！
<br/>　　金色刀影直接撕开了大日领域，以闪电般的速度向苏莫身上袭杀而去，蕴含惊天动地的杀机。
<br/>　　“杀！”
<br/>　　苏莫厉喝一声，他早已做好了接招的准备，对方一击刚出，他便催动了二十八柄本命灵剑，瞬间形成了星宿剑爆，向对方绞杀而去。
<br/>　　轰隆隆！
<br/>　　浩大的剑阵风暴，仿佛万千长剑搅动，形成了一个巨大的剑气漩涡，瞬间便和金色的刀影撞击在了一起。
<br/>　　轰轰轰！！
<br/>　　一阵惊天动地的爆响，剑阵风暴直接被强势撕开，二十八柄本命灵剑，全部崩飞了出去。
<br/>　　这没有让苏莫有什么意外，因为他也没指望以剑阵挡住对方，但只要能削弱对方攻击的威力，就足够了。
<br/>　　他手中的长剑早已高高举起，不管三七二十一，地狱三头狼战魂、岩石战魂和融灵兽战魂，全部被他沟通，而后随着玄力一起注入了手中长剑之中。
<br/>　　刹那之间，他手中之剑，变成了一柄擎天巨剑，三色玄力澎湃如海。
<br/>　　巨剑之内，仿佛有兽吼狼嚎，威势同样恐怖无比。
<br/>　　这三种战魂，全部是苏莫吞噬而来，他可不会有什么顾忌，只要能增加他的实力便可。
<br/>　　效果的确不错，苏莫感觉他手中之剑，仿佛变成了一柄强大无比的尊器，拥有毁天灭地之威。
<br/>　　“碎！”
<br/>　　眼见着金色刀影已经来到了身前，苏莫一声怒喝，手中擎天之剑猛然斩下。
<br/>　　下一息，擎天之剑便和金色刀影碰撞在了一起。
<br/>　　轰！
<br/>　　一声惊天爆响，整动了整个战界。
<br/>　　擎天之剑和金色的刀影，如同两颗核弹，同时爆炸，升起毁天灭地的冲击波。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声，突然从爆炸之中响起，但仅仅是一瞬间，便戛然而止。
<br/>　　嗖！
<br/>　　苏莫的身形急速暴退，但爆炸的威力实在太大，猛烈的冲击波击在他的护体玄力上，立刻冲碎了他的护体玄力。
<br/>　　噗！
<br/>　　苏莫身形剧震，口中喷出一口鲜血，身形暴退数千里，才堪堪止住了身形。
<br/>　　浩荡的冲击波，席卷方圆万里，破碎一方空间，才缓缓四散而开。
<br/>　　“这是……！”
<br/>　　抬眼看去，苏莫顿时眼眸一凝，因为他发现，冷江的身体居然从高空跌落了，向下方大海中坠去。
<br/>　　“死了！”苏莫面露诧异之色，因为冷江的身上，已经没有了任何的生命气息。
<br/>　　转念一想，他便立刻恍然，对方动用战魂直接攻击与他。
<br/>　　而对方的战魂，在刚才的一击中，已经彻底被毁灭了。
<br/>　　战魂中包含对方的神魂，战魂毁灭，对方的肉身也就只是一个空壳了，自然也就死了。
<br/>　　而他动用的三种战魂，虽然也毁灭了，但那根本不是他的战魂，毁灭了对他也没有丝毫的影响。
<br/>　　苏莫不由的心中有些无语，对方这死的也太悲催了。
<br/>　　本来，对方的实力比他强，就算很难胜他，也足以完全占据上风。
<br/>　　但对方居然为了杀他，不惜用战魂直接攻击，这正是自己找死啊！
<br/>　　整个战界，一片寂静，仿佛连远处海浪的声音都消失了。
<br/>　　十几座小岛上观战的人，俱是不可思议的看着这一幕，仿佛不敢相信一般。
<br/>　　冷江，居然死了！
