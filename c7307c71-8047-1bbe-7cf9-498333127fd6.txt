　　此时，不光是苏莫的目光凝视在了白衣女子的身上，拍卖会场门外，所有人的目光，都凝视在了白衣女子的身上。
<br/>　　虽然此女面目被遮盖，但其气质却是如同天山雪莲，圣洁而又高贵，仿佛散发着无形的光芒。
<br/>　　所有人都是惊讶不已，拍卖会场中，居然还有如此气质的女子。
<br/>　　扬天成和杨天威的目光，同样落在了白衣女子的身上。
<br/>　　两人的眸中均是露出了一丝疑惑之色，因为这个气质，他们有些熟悉。
<br/>　　尤其是扬天成，几乎转瞬之间，便认出了白衣女子的身份。
<br/>　　“神女？”扬天成瞪圆了眼眸，满脸惊讶之色的盯着白衣女子，这白衣女子的气质、身姿，不是神女吗？
<br/>　　在无数人的注视之下，白衣女子缓缓的走出了拍卖会场的大门。
<br/>　　见到眼前的场景，白衣女子柳眉微微皱了起来，她之所以出来的这么晚，就是想不引起任何人的注意。
<br/>　　没想到这些人，居然都堵在门口，看来已经认出了她的身份。
<br/>　　“神女，是你吗？你怎么会在这里？”
<br/>　　扬天成回过神来，顿时激动不已，一步冲出，瞬间便来到了白衣女子的身边。
<br/>　　看到神女云悠悠，扬天成立刻将苏莫抛到了九霄云外，显然在他心中，云悠悠才是重中之中。
<br/>　　而且，对付苏莫根本不是他的事情，自然有杨天威出手。
<br/>　　众人闻听扬天成之言，顿时惊讶不已，原来这白衣女子，居然是太阴族神女云悠悠！
<br/>　　云悠悠居然隐藏了身份，来到了拍卖会，难道之前三号包厢中的女子就是她。
<br/>　　“来买些东西！”
<br/>　　白衣女子轻声说道，她没有辩解，显然承认了自己就是云悠悠，而且她的声音恢复了原貌。
<br/>　　言罢，云悠悠又道：“我要回族了，不好意思！”
<br/>　　“呵呵！此地距太阴城路途遥远，我愿意护送神女回族！”扬天成微笑着说道。
<br/>　　后方，苏莫听闻扬天成和云悠悠的谈话，顿时眸中精光一闪，果然是青璇。
<br/>　　随即，苏莫立刻加快苏莫，几步之间，便走出了拍卖会场的大门。
<br/>　　“苏莫，哪里走？”
<br/>　　杨天威见此，以为苏莫要逃走，立刻冲身而上，直接拦在了苏莫的身前。
<br/>　　虽然他没有见过苏莫，但是和吕罡走在一起，修为又是武尊境一重，那就一定是苏莫无疑了。
<br/>　　所有围观之人见此，再次惊讶了起来，原来此人是苏莫？道宗的妖孽苏莫？
<br/>　　“让开！”苏莫见此，顿时厉喝一声，眼眸直视前方的杨天威，眸中杀机爆射，此人真是找死！
<br/>　　“让开？”杨天威见此不屑一笑，道：“苏莫，今日你若是束手就擒，我也不为难你，你若是还想要反抗，那就怨不得我了！”
<br/>　　“你是要找死吗？”苏莫冰冷的目光直视杨天威，如同冷厉的剑光。
<br/>　　“呵呵，看来你是要反抗了？”扬天威冷笑一声，手中光芒一闪，出现了一柄金色的长刀。
<br/>　　“找死！”苏莫彻底怒了，手中同样剑光一闪，石剑在手，气势大变。
<br/>　　杨天威和苏莫两人，陷入了对峙之中，两人的身上，气势不断的暴涨，直线的攀升。
<br/>　　哗啦啦！！
<br/>　　众人见此，立刻纷纷后退，不敢距离太近，免得遭池鱼之殃。
<br/>　　云悠悠见此，亦是眉头微皱，身形向一边飘退而去。
<br/>　　而扬天成，没有去给杨天威助阵，则是紧跟着云悠悠飞了过去。
<br/>　　轰隆隆！
<br/>　　苏莫和杨天威的气势，如同天地狼烟，浩荡不休，冲天而起，顿时让得大地不断的龟裂，拍卖场的建筑都剧烈的颤抖了起来。
<br/>　　“不得在此放肆！”
<br/>　　就在此时，一声暴喝突然响起，一股庞大的威压随之而来，顿时压迫在了苏莫和杨天威的身上。
<br/>　　霎时之间，苏莫两人感觉身体一震，身上的气势瞬间被压制住了。
<br/>　　一道苍老的身影，从拍卖场中走了出来，正是龚忠。
<br/>　　“你们两人有何恩怨，老夫不管，但你们不能在此出手，免得破坏了拍卖会场！”龚忠走到苏莫和扬天成的身前，声音淡漠的说道。
<br/>　　龚忠不管苏莫和杨天威是何身份，想要破坏拍卖会场，他必须要阻止。
<br/>　　杨天威闻言，转头看向苏莫，冷声道：“苏莫，可敢与我到星空一战？”
<br/>　　苏莫闻听杨天威之言，并没有立刻回答，他的面部肌肉蠕动，很快便恢复了原来的相貌。
<br/>　　而后，他转头看向远处的云悠悠，朗声道：“在下有要事和神女一叙，不知神女是否愿意等我一柱香的时间？”
<br/>　　云悠悠闻言一怔，不知道苏莫有什么事要和他说！
<br/>　　略一沉吟，云悠悠摇了摇头，轻声道：“不好意思，我急等着回族！”
<br/>　　云悠悠拒绝了苏莫的要求，不是他讨厌苏莫，而是她必须要尽快回去了。
<br/>　　因为她的师尊，不让她离开太阴族，这次是她私自离开的。
<br/>　　虽然普阳神长老没有限制她的自由，但她不想让师尊知晓，自己私自离开了太阴族。
<br/>　　“半柱香！神女只需等我半柱香的时间，我就能解决此人！”苏莫大声说道，同时他手指指向前方的杨天威。
<br/>　　云悠悠闻言惊讶，苏莫半柱香就能打败杨天威？这好像不太可能吧？
<br/>　　杨天威的实力，云悠悠也有所听闻，那绝对是武尊境三重武者之中，最顶尖的存在。
<br/>　　“什么？”
<br/>　　杨天威闻听此言，顿时脸色一黑，怒急而笑，道：“好你个苏莫，到了这个时候，你居然还敢狂妄自大！”
<br/>　　杨天威心中怒气升腾，半柱香的时间解决他，这个苏莫是膨胀到了何种地步？
<br/>　　不光是杨天威和云悠悠不信，周围所有的武者，听闻此言，均是有些不信。
<br/>　　“这苏莫太自大了吧！”
<br/>　　“他居然自信半柱香打败杨天威！”
<br/>　　“这不太可能吧！他的战斗力虽然逆天，但杨天威同样恐怖啊！”
<br/>　　众人低声议论，心中多是抱着怀疑的态度。
<br/>　　“扬天威，跟我去星空！”苏莫厉喝一声，脚下一踏，身形冲天而起，如同一道利剑，瞬间冲上了九霄。
<br/>　　苏莫知道，不解决杨天威的话，他今日根本没法和青璇详谈。
<br/>　　所以，他必须要一战，而且是速战速决，以最快的速度结束战斗。
<br/>　　“哪里走！”杨天威见此，很怕苏莫私自逃了，同样身形冲天而起，立刻向苏莫追了过去。
