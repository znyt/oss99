　　听闻帝释天的话语，怀灭沉吟片刻道：“主人，此人能够信任吗？”
<br/>　　“呵呵，慕清霄实力强的出奇，天境六层的实力，比你高出不少，但他贪名好利，只要给他足够的权势，他就绝不会背叛。”
<br/>　　说到这里，帝释天阴险的笑，仿佛一切尽在掌控中。
<br/>　　“而且，我早已在他食用的饭菜当中，撒进罗厄粉，无色无味，哪怕是天境强者也无法发现，他要是敢背叛本座，定要他化作一摊血水。”
<br/>　　“原来如此，主人英明。”
<br/>　　“怀灭，其实你也不必杞人忧天，这半年来慕清霄为天门做事，尽心尽力，也未曾和任何人接触过，江湖上也没有他的名声，来到天门或许是个意外。”
<br/>　　这时，帝释天眼中闪过一丝困惑，暗道：“本座在他身上，还察觉到一股熟悉的气息，难道他和无名一样，都是本座的后代，这种天赋，极有可能。”
<br/>　　也是这股熟悉的气息，让他对慕清霄的怀疑，减轻不少。
<br/>　　“只有获得龙元，我帝释天才是真正的不死不灭，无所不能的神！”
<br/>　　......
<br/>　　......
<br/>　　时间飞速流逝，转眼又过去半个月，已经是前往神龙岛的日子。
<br/>　　风云位面的大海，并未有什么不同，波澜壮阔，一望无际看不到尽头，海水卷起一层层浪涛，拍打在天门所准备的船只上。
<br/>　　此次出海，帝释天仅仅带着几位高手，神将，神母，怀空，怀灭，皇影，步惊云，慕清霄，而破军，剑晨和断浪三人，早已在帆船旁等待。
<br/>　　“恭迎主人大驾。”
<br/>　　帝释天满意的点头道：“本座于惊瑞之期实行屠龙大计，乃是旷古绝今的头等大事，你们必须同心同德，合作无间，方可有成，听明白了吗？”
<br/>　　“出发！”
<br/>　　帝释天话语刚落，慕清霄等人陆续登船，扬帆起航，船只破开重重海浪，直奔苍龙所在的神龙岛。
<br/>　　四日后的清晨，大海上弥漫着浓郁的白雾，天门船只穿过海域，终于临近神龙岛，急速前行。
<br/>　　神龙岛上，海族，神龙祠堂中，数十名青年男子，朝着上方的神龙石像不断祭拜。
<br/>　　“神龙庇佑，海族平安。”
<br/>　　“神龙庇佑，海族平安。”
<br/>　　“神龙庇佑，海族平安。”
<br/>　　......
<br/>　　忽然，整个祠堂开始剧烈震动起来，海族族长站起身来，道：“这肯定是神龙显灵了，神龙显灵了啊。”
<br/>　　这时，一名中年男子走上前来道：“族长，今日海啸地震相继出现，难道是有大事发生吗？”
<br/>　　“天有异象，必有奇事，看来我们老祖宗立下的预言，已经迫在眉睫了。”
<br/>　　“我们世代得神龙庇佑，才能繁衍至今，老祖宗曾立下重誓，必须世世代代守护在这里，保护神龙。”
<br/>　　说到这里，海族族长神色坚定道：“你们听着，不准任何外人接近神龙岛，如果有人来犯，不管他有多凶悍，我们一定要把他们赶出去，保护神龙！”
<br/>　　“保护神龙！”
<br/>　　“保护神龙！”
<br/>　　“传我命令，各族中弟子，加紧巡逻，注意行驶船只，如有异常，立即采取行动。”
<br/>　　海族族长话语刚落，所有弟子视死如归。
<br/>　　忽然，一名气喘吁吁的男子闯入祠堂中，急忙道：“族长，族长大事不好了，族长，有艘大船正向神龙岛驶来。”
<br/>　　闻言，水族族长神色凝重道：“无论是谁，一个都不能让他们上岸。”
<br/>　　......
<br/>　　......
<br/>　　此时，一望无际的海面上，一艘黑色的船只快速行驶着，宛如沧海一粟，紧接着一座充满迷雾的岛屿，渐渐出现在所有人的视线中。
<br/>　　船只上各怀心思，就在这时，船外吹起号角，船只内所有人纷纷快速走出船舱。
<br/>　　帝释天望着隐隐可见的岛屿，笑道：“本座终于等到这一天了。”
<br/>　　随着船只的前行，岛屿已经清晰可见，就在这时，岸边突然传来一道厉喝：“船上的人听着，神龙岛不欢迎外人，请你们赶快回航。”
<br/>　　闻言，怀空等人面面相觑，神母奇怪的问道：“主人，这些人......”
<br/>　　帝释天寒冰面具下，看不清楚任何表情，语气淡然道：“他们是世代隐居于神龙岛的海族人，历代保护神龙，给我传令下去，别管他们，继续前进。”
<br/>　　“是，继续前进。”
<br/>　　在数十年前，帝释天就已经开始谋划屠龙，对于神龙岛上的情况，自然是调查的很清楚，更不会让任何阻碍到他的计划。
<br/>　　看到远处的巨大帆船，根本不理会自己的警告，海族族长愤怒至极，他知道敌人来犯：“勇士们，敌人来犯，誓死保卫神龙，给我出击。”
<br/>　　随着海族族长的话语声落下，预警号角吹响，岛上的海族弟子，握着手弩，奋不顾身的跃入海中，没有惊起一丝波澜，反而如鱼得水。
<br/>　　大海深处，数百到黑影，灵活无比的接近着帆船，很快就抵达慕清霄所乘坐的船只底部。
<br/>　　紧接着无数弩箭所组成的箭幕，自海底狂涌而上，帆船上的天门弟子还未反应过来，就便弩箭夺取生命。
<br/>　　对于死去的天门弟子，帝释天完全不在意，笑道：“真是好玩，好玩啊。”
<br/>　　慕清霄则控制着灵气，让弩箭偏离，海族能世代守护着神龙岛，并非没有理由，这利用《神水诀》所发射出的弩箭，带着惊涛骇浪般的力量，常人根本难以抵御。
<br/>　　当然，对于帝释天他们来说，根本没有任何的效果。
<br/>　　断浪冷哼一声，一掌拍出，金丹二层所参杂的恐怖力量，直接在海中掀起巨浪狂潮，数名海族弟子，直接葬送在这一掌下。
<br/>　　“哈哈哈，好，真是好玩，继续，继续别停下。”
<br/>　　有断浪开头，破军和剑晨几人也不甘落后，纷纷动起手来，慕清霄依旧悠闲的靠早甲板上，看着热闹，
<br/>　　对此，帝释天也并未说什么，而是乐此不疲的看着海族弟子，各各死去的模样，这就是阻拦他长生不老的下场。
