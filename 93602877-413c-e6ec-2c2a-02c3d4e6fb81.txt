        “我……”落以柔欲言又止。
<br/>
<br/>        因为，傅夜沉的态度已经很明确了。
<br/>
<br/>        “小落，我只是赏识你的工作能力、工作态度，每个公司的老板都会珍惜对自己公司有用的人才。如果，之前，我让你产生了什么误解，我跟你道歉。倘若以后，你再针对筱颖，别怪我对你不客气。”抬眸瞟向欲要吭声又无法把话说完的落以柔，傅夜沉的眸光冷凝，声音严厉到令人不寒而栗。
<br/>
<br/>        “我明白了！以后，我会把精力放在工作上。”落以柔很识趣地颔首，再看到傅夜沉那一双锐利的黑眸时，就算她还有其他想说的话，此刻也不敢再说了。
<br/>
<br/>        她是个聪明的女人，所以，傅夜沉那些话的话外音，她心知肚明。
<br/>
<br/>        他已经很给足了她颜面了不是吗？
<br/>
<br/>        拒绝她的同时，也在告诉她，让她不要对他有非分之想。
<br/>
<br/>        难道，他心里除了那个童筱颖，就真的再也装不下其他女人了吗？
<br/>
<br/>        既然如此，那么他为什么还要跟那个童筱颖离婚？
<br/>
<br/>        他傅夜沉真的还爱着那个童筱颖吗？
<br/>
<br/>        明明他们已经离婚了！
<br/>
<br/>        落以柔其实在心底又在狐疑，因为她看不透傅夜沉的心思，这个男人总是把他自己的心思隐藏得太深。
<br/>
<br/>        以至于，她在童筱颖之前，完全看不出来他傅夜沉到底对什么样的女人感兴趣。
<br/>
<br/>        对于爱情与事业双双失意的童筱颖来说，这座城市似乎已经没有什么值得她去留恋的了。
<br/>
<br/>        当初跟大哥离婚，离开私宅，她选择继续留在这里，似乎是想等大哥对她回心转意。
<br/>
<br/>        但是，她想错了。
<br/>
<br/>        现在一切的一切，都是她在自作多情。
<br/>
<br/>        童筱颖打包收拾了行李后，发了条简讯向夏凌焰辞职，然后又叩响了夏凌焕家的门，跟他道个别，顺便谢谢他这段日子对她的关照。
<br/>
<br/>        夏凌焕见童筱颖要走，并未挽留，只是释然地说道：“珍重。”
<br/>
<br/>        这个女人于他而言，就跟昙花一现差不多。
<br/>
<br/>        并没有过多的不舍，也没有太过的留恋。
<br/>
<br/>        只是隐隐间，对她有一种说不清道不明的情愫在心里。
<br/>
<br/>        所以，夏凌焕并未问童筱颖接下来的去向。
<br/>
<br/>        离开滨江城之前，童筱颖为自己做了最后一件“犯贱”的事情。
<br/>
<br/>        她去办理了一张手机卡，然后给傅夜沉打了一通电话，只可惜，这最后一通电话，她没有打通，但语音提示可以留言。
<br/>
<br/>        说来也奇怪，她以前从不记任何人的电话号码，现在她居然能在潜意识里背出大哥的私人号码。
<br/>
<br/>        “嘟”声之后，童筱颖将手机的话筒孔，举到了自己的嘴边，明明有很多话想说，而到了此刻真正要说的时候，她却一句话也说不出来。
<br/>
<br/>        结果时间超时，电话被自动挂断。
<br/>
<br/>        犹豫了许久，她才再次拨通电话，只是她没料到，电话竟然被接通了。
<br/>
<br/>        “喂，你好。”充斥着磁性的嗓音，透过助听器传入耳中，显得格外缥缈。
<br/>
<br/>        童筱颖怔愣着垂下眼帘，讷讷地唤了声：“大哥，是我。”
<br/>
<br/>        “有事吗？”他声音清冷地问，就好像是在询问一个无关紧要的人，为什么要打电话过来一样。
<br/>
<br/>        这一时间，竟让童筱颖不知道自己应该要跟他说什么是好。
<br/>
<br/>        她好像没事，但是又有事，却找不到话题可以跟他说。
<br/>
<br/>        如果，她说，大哥，我喜欢的人是你，他会信吗？
<br/>
<br/>        如果，她说，大哥，你可不可以相信我，他能做到吗？
<br/>
<br/>        如果，她说，大哥，我想和你在一起，他会同意吗？
<br/>
<br/>        明明有太多话要说，结果，她脑子里一片混乱，慌不择言起来：“我……我要离开滨江城了。”
<br/>
<br/>        但她这话刚落，他就迫不及待地接着问：“还有其他事吗？”
<br/>
<br/>        他这次的语气里，稍稍透着少许不耐烦。
<br/>
<br/>        童筱颖鼻子一酸，眼泪瞬间在眼眶里打转，声音忽然被哽咽在喉咙里，一句话也说不出来。
<br/>
<br/>        或许是因为她这边太久没声音，他淡淡地开口说：“你若没别的事情，我挂电话了。”
<br/>
<br/>        “……”豆大的泪珠，扑簌簌地从她的眼角往下掉，她抬起另一只手紧紧地捂住了自己的嘴巴，不让自己哭出声来。
<br/>
<br/>        结果眼泪顺着指缝滑入到嘴角，又咸又苦，甚至灼痛了脸上的肌肤。
<br/>
<br/>        下一秒，他不等她的回复，真的就将电话给挂断了。
<br/>
<br/>        男人若是狠起来，真的比女人还要狠。
<br/>
<br/>        傅夜沉对她的绝情，已经做到了不闻不问、不管不顾、不理不睬、甚至不留余地！
<br/>
<br/>        爱有多久，情有多深，男人狠心起来便会是加倍的绝情。
