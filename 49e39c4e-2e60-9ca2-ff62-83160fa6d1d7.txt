        童筱颖如此冷静到异常，让傅昼景心里十分不安。
<br/>
<br/>        他一点都不否认，看到她未跟他的大哥在一起，他心里确实很解气，也很得意。
<br/>
<br/>        同时，他也笃定，筱颍对大哥，一定爱得不深。
<br/>
<br/>        童筱颖离开临海城后，回到了滨江城，一下车，她便刻不容缓地去了“夜惑”。
<br/>
<br/>        “夜惑”里的执事费尔南，在看到童筱颖闯进办公室里时，眨巴了一下眼睛后，咧着嘴，笑脸相迎：“哟！这不是我们‘夜惑’里出去的小谢嘛！三年不见，你这又变漂亮了。”
<br/>
<br/>        “我要找十爷。”童筱颖不打算跟他绕弯子，直接开门见山，直奔主题。
<br/>
<br/>        费尔南双手一拍，两眼眯得像月牙，回答道：“小谢你可是来得太巧了！今晚，咱们夜惑里的十大爷将会齐聚一堂，召开重大会议，到时候，你就能见到十爷了。”
<br/>
<br/>        “你安排一下。”童筱颖接着说道。
<br/>
<br/>        费尔南咧嘴道：“小谢就这么相信我？”
<br/>
<br/>        “有十爷罩着我，你不敢拿我怎么样。”童筱颖淡定道。
<br/>
<br/>        费尔南“呵呵”地笑了起来：“就算小谢不搬出十爷，念在你曾经是我的摇钱树和镇店福的份上，我都一定会帮你安排见十爷。”
<br/>
<br/>        “镇店福是什么意思？”童筱颖凝了凝眉。
<br/>
<br/>        费尔南解释道：“我这店好几次都被警察围剿了，不是停业整顿，就是各种原因让我这店无法顺利营业，所以，我就以你的名义去求了傅氏集团那位有权有势的傅大总裁出面帮忙。每次，他一出面，我这店必能安然无恙。你说你是不是我的镇店福？”
<br/>
<br/>        “费老板还真是一如既往的……不要脸！”童筱颖一脸不悦地呵斥。
<br/>
<br/>        费尔南无所谓地耸了耸肩，笑着说道：“我要是要脸的话，就当不了这店的执事了，是不是？”
<br/>
<br/>        “也对。”童筱颖淡淡地应了声，心里却若有所思起来。
<br/>
<br/>        她笃定，大哥的死，一定跟夜惑里那“十位大爷”有关。
<br/>
<br/>        而这费尔南，看起来倒像是一副“中立”的姿态。
<br/>
<br/>        “我听说，那位傅氏集团的傅大总裁，死于非命。不知道是否当真？”费尔南看着童筱颖，八卦了起来。
<br/>
<br/>        童筱颖的心口上一直在隐隐作痛，偏偏这费尔南还要继续往她心口上扎针：“不知道，没听过。”
<br/>
<br/>        “那傅大总裁，还挺中意你的。”费尔南感慨。
<br/>
<br/>        童筱颖若无其事地岔开话题：“我去夜惑里随便逛逛，联系上十爷了，你给我电话。”
<br/>
<br/>        她说着，拿起费尔南办公桌上的笔和纸，“唰唰唰”地写下了自己的联系方式。
<br/>
<br/>        费尔南拿起童筱颖写下了手机号码的纸，瞄了一眼后，咧嘴提醒：“小谢，还是那句话。来咱们夜惑，可不要多管闲事噢！”
<br/>
<br/>        “知道，谢谢提醒。”童筱颖应了声，淡然地转身走出了费尔南的办公室。
<br/>
<br/>        其实，她心里也很迷茫。
<br/>
<br/>        她现在在做什么？
<br/>
<br/>        以身犯险吗？
<br/>
<br/>        可是，她已经别无选择了。
<br/>
<br/>        或许，她其实还有一种选择，那就是跟落以柔一样，为大哥殉情。
<br/>
<br/>        但不是现在。
<br/>
<br/>        她得找到十爷，然后去接近那个八爷，如果真是八爷派人杀的大哥，她一定会找机会杀了八爷然后自己再自杀。
<br/>
<br/>        与其让她孤独的活着，还不如去找杀害大哥的真凶。
<br/>
<br/>        童筱颖走着走着，不知不觉中，走到了自己曾经在这里待过的休息室。
<br/>
<br/>        这间休息室是当初嫂嫂唐倩兮为她准备的，不知道现在她还能不能进去。
<br/>
<br/>        童筱颖下意识地从手提包里翻出一串钥匙来，谁知，这钥匙竟然还能打开这张休息室的门。
<br/>
<br/>        推门而入，里面依旧很干净，架子上挂满了演出服，梳妆台上还摆满了各种化妆品。镜子旁，有各种颜色的羽毛假面眼镜。
<br/>
<br/>        童筱颖挑了一款白色的羽毛假面眼镜，戴在了自己的脸上，这让她回想起了，当初，大哥来夜惑里听她唱歌，虽然三番五次地救了她，但是也三番五次地对她动手动脚。
<br/>
<br/>        “你怎么还没换衣服啊？赶紧换衣服，要去见八爷了！”身后忽然响起一个女人的催促声。
<br/>
<br/>        童筱颖怔了怔，索性将错就错，应了声：“我马上换衣服。”
<br/>
<br/>        上天还真是给了她一个好机会，她绞尽脑汁地想要去找那位八爷，没想到现在那八爷倒是自己送上门来了。
<br/>
<br/>        童筱颖以最快的速度，替自己换了装，她特意挑了一件比较性感的礼服。
<br/>
<br/>        而领她去见八爷的女人，是当初和唐倩兮平起平坐的晚霞姐。
