        &lt;h3&gt;第三卷第三百三十章  无论你悲伤欢喜我都一直会陪着你&lt;/h3&gt;
<br/>
<br/>        陈子昂和江俞轩又一同飞回了西城，这一段时间陈子昂马不停蹄地的跑来跑去，从酒店出来的时候，陈子昂再也没有了那种坚强，冯睿的离去给她的冲击很大。
<br/>
<br/>        “江俞轩，你为什么昨天不告诉我？”陈子昂坐在车上，带着哭腔。
<br/>
<br/>        “我是上午才知道的。那个时候你正在忙，我就没有告诉你。”江俞轩知道陈子昂心里难受。
<br/>
<br/>        “可是，她是我的二嫂啊！再忙你也得告诉我不是？”这一会的陈子昂显得有些无理取闹。
<br/>
<br/>        “子昂，我和你一样难过，但是人死不能复生，我们活着的人更应该要学会爱自己，子昂，想哭就哭出来吧?”
<br/>
<br/>        “江俞轩，你知道吗？我知道我二嫂做过的那些事情，我心里是曾经责怪过她，但是我从来都没有恨过她，我想换成绩的事情她肯定是不知道的，还有她和张函妈妈来往，无非就是想有个靠山，为陈伟冯谋一份好的工作，说来说去，其实她活得挺卑微的。她算计了一生，可是最后什么都没有。”
<br/>
<br/>        “子昂，人活着有很多的活法，但是总是去踩压和利用别人是不对的，对于二嫂之前的过错我们不要去纠结了好不好？人生苦短，我们更要珍惜眼前人。”
<br/>
<br/>        “我会珍惜你们的。”陈子昂低声说，她极力的忍住哭泣。
<br/>
<br/>        “这一段时间你也太累了，靠着眯一会吧，等到了我叫你。”江俞轩找不出来安慰陈子昂的话。
<br/>
<br/>        陈子昂闭上眼睛，靠在座位上，江俞轩定定的看着陈子昂，脸上还有泪痕，他拿出纸巾轻轻的替陈子昂擦去泪痕，陈子昂一动不动，她显得孤独而冷寂，周身都包围在一种悲伤的气息里，那个风风火火的女人不见了，剩下的似乎是一只弱小无助的小猫咪在独自疗伤。
<br/>
<br/>        江俞轩心里一疼，他伸出胳膊，让陈子昂靠在他的肩膀上，然后想去碰碰陈子昂的额头，陈子昂突然睁开眼睛，江俞轩心里一慌。
<br/>
<br/>        陈子昂拨开江俞轩的胳膊：“你这里不属于我，我还是靠在座位上舒服。”然后又闭上了眼睛。
<br/>
<br/>        江俞轩有些尴尬，嘴巴动了动，什么都说不出来，他现在说什么都不对，什么都不能说。
<br/>
<br/>        此时的陈子昂像一只刺猬，将自己深深的裹在自己那坚硬的毛发里，谁也走不进她的心里。
<br/>
<br/>        沉默了两个多小时，江俞轩多希望陈子昂像以前那样在他的面前可以哭，可以闹，多希望陈子昂说：“俞轩，我怎么怎么了。”可是陈子昂全程一句话也不说。
<br/>
<br/>        快到机场了，陈子昂睁开了眼睛，原来她并没有睡，只是不想说话而已。
<br/>
<br/>        “子昂，来，慢点！”江俞轩伸出手，想扶着陈子昂。
<br/>
<br/>        陈子昂厌烦的拍了一下江俞轩的手：“起开！”
<br/>
<br/>        公务舱里的人看了一眼江俞轩和陈子昂，江俞轩尴尬的一笑：“好的，你慢点。”
<br/>
<br/>        陈子昂挺了挺身子，长舒一口气，目不斜视的朝外走去，她又走出了那种雷霆万钧的气势。
<br/>
<br/>        “子昂，我会一直在你身边的，一生一世！你别这样！”江俞轩知道陈子昂伤心，可是看着这样的陈子昂他更心疼。
<br/>
<br/>        “以前喜欢张倩楠，后来喜欢陈天天，现在说和我一生一世，江俞轩，我咋这么多年都没有发现你是这样的人？你go吧，离我远点！”陈子昂勃然大怒，冲着江俞轩喊，而且拿着手包打着江俞轩。
<br/>
<br/>        江俞轩任由陈子昂打着他，并且朝陈子昂靠近，手一伸将陈子昂抱在了怀里：“子昂，我知道你心里难受，你打我吧，只要你不难受了你尽管打我，我一直都在你身边，我一直都喜欢着你，只是，只是我被外面的世界蒙蔽了心智，其实我一直都在你身边，不是吗？你看我从来都没有离开过你。”
<br/>
<br/>        陈子昂突然就放声大哭：“江俞轩，我好怕啊，我好怕你们都离开我，求你们别离开我。”
<br/>
<br/>        “子昂，我不会离开你的，永远都不会。”江俞轩紧紧的抱着陈子昂，心疼得不能呼吸，这个女人只有在他的面前是脆弱的，她还是原来的那个陈子昂，她一直都没有变。
<br/>
<br/>        哭了一会，陈子昂推开江俞轩说：“我好了！走吧！”
<br/>
<br/>        “这就好了？”江俞轩感觉有些失落。
<br/>
<br/>        “那要怎么样？我在这里哭个三天三夜？”陈子昂瞪了江俞轩一眼。
<br/>
<br/>        陈子霖已经安排人在机场接他们了。
<br/>
<br/>        陈子昂问江俞轩：“你也要去吗？”
<br/>
<br/>        “我当然要去啊，我不去谁去？”
<br/>
<br/>        “你不回家？”
<br/>
<br/>        “你在哪里哪里就是我的家！”
<br/>
<br/>        “江俞轩。”
<br/>
<br/>        “嗯。”
<br/>
<br/>        “这种话以后还是不要说了吧，我怕我误会太深！”陈子昂突然就语气冰冷。
<br/>
<br/>        “我知道这种时候说这种话不合适，可是我说的是真话！无论你悲伤欢喜我都一直会陪着你！”
<br/>
<br/>        “江俞轩，你忘了你自己是谁了吧？”
<br/>
<br/>        “没有，我一直都记得我是谁，我一直都是你第一次见到的江俞轩！”
<br/>
<br/>        陈子昂长长的叹了口气，闭上嘴不再说话，她觉得，她没有办法和江俞轩再说下去，越说越绕，说多了江俞轩会更误会。
<br/>
<br/>        江俞轩确实对她的帮助很大，但，是有妇之夫，无论他喜欢谁和她陈子昂都没有关系了，他们现在只是朋友关系。
<br/>
<br/>        她刚刚实在时心里压抑得很痛苦，忍不住哭出来了，哭出来感觉自己清醒了很多，清醒了她就回归现实了。
<br/>
<br/>        到了冯睿的家里，院子里都是花圈，冯睿原来在单位也是一个领导，陈子霖和陈子乔忙忙碌碌的招呼着来人。
<br/>
<br/>        陈天天和陈子建还没有从悲伤中回过神，他们不明白为什么冯睿会突然离开，二秀也在忙忙碌碌的穿梭着。
<br/>
<br/>        西城的风俗习惯是人死后会在家里放几天，然后才去火葬，冯睿原本还很年轻，也没有墓地，所以陈子霖还得张罗着给冯睿买墓地。
<br/>
<br/>        陈子霖给陈伟冯打完电话之后就再也没有给陈伟冯打电话了，而陈伟冯一直都没有回家。
<br/>
<br/>        陈子昂和江俞轩回来了，陈子霖心里一松，陈子昂的推广会应该已经成功了。
<br/>
<br/>        陈子霖也不含糊，直接叫上陈子慧、陈子乔和陈子悦、陈子建一起商量给冯睿买墓地的事情。
<br/>
<br/>        “子建，你也别孬着了，现在已经这样了，你们离婚了，你还要过以后的生活，二秀在那里忙来忙去的，人家该你的？”陈子慧骂陈子建。
<br/>
<br/>        “二哥，你是个男人，买墓地你得吭气啊，总不能天天放家里吧？”陈子乔也很生气陈子建的态度。
<br/>
<br/>        “伟冯没有回来吗？”陈子昂没有看见陈伟冯。
<br/>
<br/>        “我打电话了，人家不回来。”陈子霖咬牙切齿。
<br/>
<br/>        “你们没有给二嫂娘家说一声？”陈子昂也没有发现冯睿的娘家人。
<br/>
<br/>        “忙忘了。”陈子霖拍拍脑袋。
