        “小倩啊！你这打我的姑娘，也不提前跟我吱一声，嗯？”
<br/>
<br/>        门外，一个女人的声音，高调地响起。
<br/>
<br/>        唐倩兮不用回过头去看，也知道来人是谁。
<br/>
<br/>        小莹看到自己的上司晚霞姗姗来迟，心里虽然有一丝抱怨，但眼睛里却多了一股希冀。
<br/>
<br/>        她这算是得救了吧！
<br/>
<br/>        唐倩兮瞥了晚霞一眼，双手抱臂，一脸不悦。
<br/>
<br/>        在这“夜惑”里，跟她唐倩兮平级的女人，有五个，但是她和晚霞的关系最好。晚霞和她一样，是这“夜惑”里的“前辈”，和她一起进入“夜惑”的。
<br/>
<br/>        晚霞咧着红唇，笑容明艳，走到唐倩兮的身后，伸出双手，轻轻地搭在了唐倩兮的肩头上，一边给她拿捏按摩，一边安慰道：“小倩，我可是带着法子过来找你的。你要是不消火，我就不讲给你听。”
<br/>
<br/>        “真的？”唐倩兮惊讶地转过身来，喜上眉梢地看着晚霞。
<br/>
<br/>        晚霞微笑着点了点头，然后朝跪坐在地上的小莹看了一眼，接着向唐倩兮抬了抬黛眉。
<br/>
<br/>        唐倩兮会意地斜睨了小莹一眼后，回眸看着晚霞的眼睛，微微一笑：“你若是有法子可以帮我把小谢安然无恙地从十爷那儿带出来，我就放了小莹。”
<br/>
<br/>        “哎呦！我可没说要你放了小莹！小莹擅自离岗，当然该罚。小倩，你说，我该怎么罚她呢？”晚霞故作软绵绵地笑着。
<br/>
<br/>        和晚霞当了这么多年的姐妹，唐倩兮自然是明白晚霞所说的话里的另外一层意思。
<br/>
<br/>        “我刚刚已经打了她一巴掌了！算是给小谢出了气。至于要怎么罚，就随便你了。你把人领回去就是！”唐倩兮不痛不痒地说。
<br/>
<br/>        晚霞会心一笑，接着说道：“我也已经派人去请那位能劝动十爷的小姑娘了！很快，你家的小谢，就能平安无事的回来了。”
<br/>
<br/>        “什么？能劝得动十爷的小姑娘？她是谁？”唐倩兮一脸茫然。
<br/>
<br/>        晚霞抿了抿唇，抬起手，竖起食指来摇了摇：“这件事情了，我得保密。总而言之，小谢，她一定会回到你身边来，便是。”
<br/>
<br/>        “好吧！”唐倩兮虽然对晚霞的话半信半疑，但还是一副“死马当活马医”的心态，只能坐等结果了。
<br/>
<br/>        “啪——啪——”
<br/>
<br/>        一个男人的两巴掌重重地打下来，童筱颖只觉自己的整个脸都在火辣辣的痛，而且还痛得苦不堪言，倏然，一股血腥的咸味涌入口腔，一丝鲜血从她的嘴角处流了出来。
<br/>
<br/>        秋容跪在地上，上半身几乎已经全部匍匐在地面，不停地向坐在沙发上的男人求饶：“十爷，求求你！求求你！不要再打小谢了！你打我吧！”
<br/>
<br/>        “你可真会装！当初你对婴宁下手的时候，怎么就不见你如此心地善良呢？”十爷冷哼。
<br/>
<br/>        一提起“婴宁”，秋容趴在地上，已经泣不成声：“我没有害死婴宁……不是我……真的不是我……要怎样……你才会相信我了！”
<br/>
<br/>        “来人，把‘亚兹’牵上来，然后给那女人打我的那只手，抹上肉汁，拿去喂‘亚兹’！”十爷无视掉秋容的话，冷冷地命令道。
<br/>
<br/>        秋容听到这里，吓得哭声戛然而止，惊慌失措地爬到十爷的腿边，紧紧地抱住了十爷的脚，急不择言地替童筱颖求情：“十爷！十爷！不要！不要这样对小谢！我错了！您、您拿我的手去喂亚兹，好不好？小谢是无辜的！而且，她……她真的是婴宁的那个失散多年的妹妹！您若不信，您可以去查！查清楚了，再处置我，求您！您不能这样虐待婴宁的妹妹，婴宁在天之灵，会不安的啊！”
<br/>
<br/>        “贱女人！你有什么资格在我面前提婴宁！”十爷一声呵斥，他恼火地抬起脚来，并狠狠地往秋容的胸口上踹了一脚，直接将她踹倒在地。
<br/>
<br/>        这边，任由童筱颖如何挣扎，她的身子被人束缚着跪坐在了地上，而她的右手更是被人强制地往一桶飘着油花的肉汁里塞。

<br/>        此时，一个戴着墨镜，穿着西装革履的男人，牵着一只身形庞大近乎一米高的大狼狗，从内屋里走了出来。
<br/>
<br/>        秋容看到那只凶神恶煞的大狼狗的时候，吓得晕了过去。

<br/>        童筱颖则瞪大了眼睛，与狼狗“亚兹”对视。

<br/>        一旁的人把那桶肉汁提走后，大狼狗“亚兹”的黑鼻子动了两下，龇牙咧嘴地吐出了粉红色的大舌头。

<br/>        是朋友！
