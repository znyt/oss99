<br/>　　巫娴的护体玄力，根本挡不住苏莫的攻击，被瞬间撕开。

<br/>　　她想要立刻逃走，但是二十八柄本命灵剑洞穿了她的身体之后，却是瞬间将她包围了起来，让她逃无可逃。
<br/>　　大星宿剑阵笼罩巫娴周身，寒光吞吐，剑气弥漫，只要她稍有异动，便会被瞬间击杀。
<br/>　　“怎么会这样？”巫娴眸中尽是惊骇之意，她败了！败在了一个土著的手中，还败的如此干脆！
<br/>　　唰！

<br/>　　气劲进入巫娴的体内，瞬间化为了封印，封住了对方的丹田。
<br/>　　这种封印名为《九灵印》，乃是苏莫在苍穹神宫所学，比较简单，其实就是一个简易的阵法。
<br/>　　不过，这封印虽然简单，但实力低于他的人，短时间内根本破不可封印。

<br/>　　唰！

<br/>　　擒住了巫娴苏莫心中大定，此女的身份估计不简单，是他保命的资本。
<br/>　　当然，擒拿一个还不够，多擒拿一些巫族天才，才更稳妥。
