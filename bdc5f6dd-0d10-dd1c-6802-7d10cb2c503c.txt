　　上京基地，停机坪上排放的直升机整整齐齐，洛冥从几架直升机的护送下回到上京基地。
<br/>　　确定已经安全抵达停机坪后，上京基地各方面人员才放松下来，经历过差点失去两名最权威老者的事件，之后牧司令和洛冥的出行都成了他们最紧张的事。
<br/>　　所以洛冥离开上京的时候只带了两名飞行员，回来的时候却全程被人护送。
<br/>　　一名年轻的少校走上前迎接刚下直升机的洛冥，敬礼过后便立即报怨道：“洛老，您以后出行能不能先打个招呼？这么一声不吭的就走了，您知不知道我们都差点乱套，好在没有出现意外。”
<br/>　　洛冥看了眼四周将他围的水泄不通的架势，颇为无奈道：“事出有因嘛，而且我去的是狼牙，能有什么危险？”
<br/>　　年轻的少校执着道：“幸亏您去的是狼牙而没有在其他地方乱转，要知道您刚走出上京基地就已经暴露了行踪，眼下虽然南部联盟军已经完成北上任务，但还是有很多作乱分子想在最后关头做出点极端的事。”
<br/>　　“好了，好了，我知道了。”洛冥摆摆手，快步就朝着牧司令所在的地方而去。
<br/>　　年轻少校无奈的摇摇头，这才下达解除贴身保护的命令。
<br/>　　回到上京基地的洛冥一路上神情都有些凝重，当他来到牧司令的办公室后，面上就好似吞了苦瓜一样难看。
<br/>　　牧司令看到这样的洛冥一愣，却只是随意问道：“回来了？”
<br/>　　洛冥点头：“南署基地大尸潮胜利的消息传来了吗？”
<br/>　　牧司令拍了拍桌上的一份文件：“战报都送来了，很详细。”
<br/>　　洛冥坐在桌前沉思，他没有去看那份战报，毕竟亲眼见过大战现场的他，也没有去研究这份战报的必要。
<br/>　　牧司令看到这样神情的洛冥，也猜到了事态恐怕有些不简单，却反其道而行的闲聊起来：“这狼牙战团还真是有意思，因为长官不在便不上报消息，打完仗就直接往地上一躺，集体睡大觉。”
<br/>　　洛冥又喝了口茶，满是皱纹的面上神情略微舒展：“那帮年轻人都随意的很。”
<br/>　　“是啊！”牧司令笑的很开怀，接着问道：“你这次去狼牙，楚涵怎么说？”
<br/>　　洛冥的动作一顿，抬眼直视牧司令：“楚涵怕是一时半会儿无法帮助我们，他在大战中受了伤，不是战报中所说的轻伤需要静养，而是昏迷至今未醒。”
<br/>　　牧司令的笑容僵住，屋内的气氛也一下子冷凝，良久后他才一叹气：“难怪你一进来的脸色这么差。”
<br/>　　“我脸色差并不是因为这个。”洛冥苦笑，继续道：“楚涵虽然昏迷，但据尚九娣说白允儿去看过，告知了尚九娣楚涵一切无碍，过段时间就会醒来。”
<br/>　　牧司令眉头皱了皱：“既然楚涵并无大碍，那是因为什么？”
<br/>　　“因为循环论。”洛冥说到这里身躯有些轻微的颤抖，嘴唇更是发白：“事态可能比我们所想的更严重，并且当下说不定已经发生了！”
<br/>　　牧司令握笔的手顿了顿，思索了一会儿后道：“你见到了袁熙叶？她说了什么？”
<br/>　　洛冥的面容苍白无比，眼神带着一丝空洞：“她给我们看的那份有关循环论的推理，不过是一个开头而已。”
<br/>　　牧司令瞳孔一缩：“我们眼下遇到过的丧尸或异种，顶多也是可以伪装成·人类，但其他方便和人还是有区别，比方说丧尸的血液颜色和消化系统，还有异种的牙齿眼睛和血液气味。”
<br/>　　说着牧司令皱起眉想了想：“这些本质的东西难道也有可能会改变么？丧尸开始长头发，并不能算吧？”
<br/>　　“我不知道。”洛冥两只手握在一起，道：“但依照袁熙叶的推理到达后期，丧尸和异种会变的和人类毫无区别，包括了你上述所说的一切，不仅如此，可能还会有我们意想不到的情况发生。”
<br/>　　“比方说？”牧司令紧盯着洛冥。
<br/>　　洛冥摇着头，目光涣散：“我不知道，袁熙叶也不知道，她在跟我说这些的时候就好似是在说一件极其寻常的事，也就像是在陈述一个数据，完全看不出她有任何害怕。”
<br/>　　说着洛冥一个哆嗦：“以后这种事你去，我反正不去了，跟这种搞研究的怪物打交道，我心脏受不了。”
<br/>　　屋内再次安静下来，并且接下来很长一段时间里，都是两名老人的相视无言，牧司令在消化洛冥带来的消息，洛冥则是在思考从袁熙叶那里听来的话。
<br/>　　若循环论的发展恰好依照了袁熙叶的推论在进行，那么将来地球上的丧尸、异种和人类，恐怕将会出现呈现出难以分辨的可怕场景。
<br/>　　在洛冥亲自见过了袁熙叶并回到上京基地的这段时间，南部联盟军也在以一种不可思议的速度扩张了人数，并且一路快速的朝着段氏基地进发。
<br/>　　这批队伍在文奇胜的统领下分为了好几批，第一批队伍有一小部分的老兵带队，带着一大批新入队的新兵，以简化版的狼牙训练模式，在路途上边快速前进边训练。
<br/>　　剩下的部队，则是一路征集新兵，一旦新兵人数到达一个预定值，文奇胜便会循环复制第一批率先出发队伍的模式。
<br/>　　如此这般的以此类推，南部联盟军就这么分成了好几股，分批前后在路途上行走。
<br/>　　在第一批南部联盟军队伍到达之前，大批从世界各地赶来的猎者，早已在这处基地及周边聚集了起来，他们一边守着这个基地，一边时不时召集几个人一起出动，去猎杀附近游走的丧尸。
<br/>　　在南署基地的大尸潮战役过后，这里的边境地带就不断出现游走的丧尸，就像是城市里人群并不密集的地带一样，丧尸漫无目的在四处乱转。
<br/>　　游走的丧尸属于散漫态度，说明在这些丧尸之中并没有一个丧尸头领，甚至也没出现有智慧的超级丧尸。
<br/>　　这一现象很奇怪，因为这些游走的丧尸数量若是聚集起来可不少，绝对超过了两百万。
<br/>　　而当这一现象出现在段氏基地以南的华夏边境时，就不是奇怪，而是诡异！
