        接着，他轻轻地给她盖好被子，自己却不得不从床上下来，把这张床让给她连星儿。
<br/>
<br/>        小夜灯的照射下，连星儿的脸蛋看起来吹弹可破，睫毛又长又翘，在眼睑上投下一片阴影，睡颜就像一个漂亮的洋娃娃。
<br/>
<br/>        明司寒猛然间回过神来后，踏上拖鞋，立即离开了房间。
<br/>
<br/>        这种情况下，他不可能去睡连星儿的房间，所以，他只能一个人睡客厅的沙发了。
<br/>
<br/>        上半夜，他辗转难眠，好在到了下半夜，他困了，才沉沉地睡了过去。
<br/>
<br/>        等到他醒来后，明司寒发现自己身上多了条毯子，但套房里早已没了连星儿的身影。
<br/>
<br/>        昨晚是他有史以来第一次那样辗转难眠……
<br/>
<br/>        都是被连星儿那丫头害的！
<br/>
<br/>        明司寒从沙发上坐了起来，只觉有些头晕脑胀。
<br/>
<br/>        而且，不是说好了他带她游玩北城的吗？
<br/>
<br/>        结果她竟然扔下他，自己跑出去玩了。
<br/>
<br/>        明司寒暗自下定决心，他再管那丫头的事就不姓“明”。
<br/>
<br/>        但很快，他就被自己打脸了。
<br/>
<br/>        傍晚时分，北城派出所的一个电话，将他叫了过去。
<br/>
<br/>        明司寒赶到派出所的时候，只见连星儿正双手抱臂，傲气十足地扬起下巴，站在两个民警和一个身材高大魁梧，却顶着两只熊猫眼的男人身旁，理直气壮地说道：“他不摸我屁股，我能打他吗？”
<br/>
<br/>        “谁摸你了？你有证据吗？”魁梧男龇牙咧嘴地质问。
<br/>
<br/>        民警调出了当时的监控画面，只看到这男子一直子人群里游荡，特别爱站在女生的身后。
<br/>
<br/>        画面里，好几个女生都回头瞪了这魁梧男一眼，然后不了了之地走了或者往其他地方躲。
<br/>
<br/>        只是到了连星儿这里，只见她转身瞪向这魁梧男时，魁梧男嘴里像是说了一句什么话，一脸横肉的笑容似乎是惹恼了连星儿。
<br/>
<br/>        因为监控是俯视角度，看不清连星儿对这魁梧男做了什么，只见魁梧男突然面部扭曲地弓着身子。
<br/>
<br/>        连星儿又趁机一个抬腿，勾住魁梧男的脖子，直接将他撂倒在地。
<br/>
<br/>        紧接着，连星儿直接坐在这魁梧男身上不停地给他甩耳光。
<br/>
<br/>        面对着魁梧男的狡辩，连星儿毫不示弱地回答：“当然有！去检验科验下我裤子上的指纹，不就一清二楚了吗？”
<br/>
<br/>        顿时，魁梧男的气势就弱了。
<br/>
<br/>        验指纹这事很麻烦，但并不是不可以验。
<br/>
<br/>        魁梧男或许没料到还有这招，一脸心虚地不敢再反驳连星儿的话。
<br/>
<br/>        民警见状，没好气地对魁梧男说道：“你真的没摸这小姑娘吗？”
<br/>
<br/>        “我就摸了一下，哪知道这女生会武功啊！”魁梧男面部扭曲地抱怨道，只觉得两只眼睛到现在都还很疼。
<br/>
<br/>        “那你跟我们去那边做下笔录！”民警说着随后将魁梧男带走了。
<br/>
<br/>        连星儿看着魁梧男跟着民警离去的背影，噘嘴道：“敢摸我，活腻了是吧？”
<br/>
<br/>        明司寒在民警那儿签字时，与连星儿的关系上随手写成了“男友”。
<br/>
<br/>        民警看到后，忍不住调侃了他一句：“你女朋友挺能打的。”
<br/>
<br/>        “……”明司寒语塞。
<br/>
<br/>        其实，起初他们民警赶到现场时，也是大跌眼镜。
<br/>
<br/>        连星儿将那魁梧大汉打趴在了地上，众人在一旁围观，还有几个女生起哄说“打得好”，其他人没有一个人敢上前阻拦。
<br/>
<br/>        而且，连星儿专揍那魁梧男的脸。
<br/>
<br/>        好像不把人家揍到爹妈都不认识了，就誓不收手一样。
<br/>
<br/>        除了把那魁梧男揍的鼻青脸肿以外，他的两只眼睛也跟着遭了殃，好在连星儿没把人家揍到失明，不然麻烦大了。
<br/>
<br/>        但毕竟是连星儿动手打了人，还不止打一下，有防卫过当的嫌疑，所以，得有人过来保释她。
<br/>
<br/>        “你为什么报我的手机号码？”离开派出所后，明司寒问道。
<br/>
<br/>        连星儿随口回答：“因为你离我近啊！总不可能让我叫家长吧？更不可能去把我哥从部队里喊出来。”
<br/>
<br/>        “……”明司寒再次语塞。
<br/>
<br/>        他到底招惹了个什么样的女孩子啊？
<br/>
<br/>        “昨天晚上……”
<br/>
<br/>        “我以后睡觉一定会反锁房门！”明司寒直接打断了连星儿的话。
<br/>
<br/>        连星儿眨巴着眼睛看着明司寒，微微摇了摇头，一边感慨着一边接着把话给说完：“你梦游喜欢睡沙发，我不拦着你。只是，你梦游为什么要拉上我？”
<br/>
<br/>        “？！”明司寒一脸懵。
<br/>
<br/>        “算了，不知者无罪。我不跟你计较了。”连星儿随后摆摆手，一脸淡定地撸顺自己乱糟糟的马尾。
<br/>
<br/>        明司寒嘴角抽搐了一下。
<br/>
<br/>        到底是谁在梦游？
