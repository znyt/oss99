休息室里，夏夕绾站在厨房里剥虾。
<br/>
<br/>        这个休息室相当的奢华，餐厅厨房一应俱全，只不过陆寒霆没有在这里做过饭，顶多就是睡觉休息，公司里有他的私人大厨，他不需要自己动手，当然，他也不会做饭，从来没有下过厨房。
<br/>
<br/>        陆寒霆伸手从后面抱住了她盈盈一握的纤腰，俊脸蹭在她散发香气的长发上，“干什么呢？”
<br/>
<br/>        “剥虾啊，我今天回来的时候正好路过超市，买了一点菜，虾剥好了，我给你炒一个虾仁滑蛋，我还蒸了米饭。”
<br/>
<br/>        这么一说陆寒霆还真觉得肚子饿了，在幽兰苑吃的那一顿味同爵蜡，他将薄唇蹭在她温软干净的小脸蛋上，心里像荡漾的一片轻舟，柔柔软软的，原来她去超市买菜给他做饭了。
<br/>
<br/>        “陆太太，你还会下厨？”
<br/>
<br/>        “当然了，我厨艺可是很好的，一定能满足你的胃，”说着夏夕绾扭头看了他一眼，“陆先生，能娶到我这样的太太，你是不是很有福啊？”
<br/>
<br/>        陆寒霆掐了一下她细软的腰窝，她凹下去的腰肢曲线就像是瓷花瓶的细口，让人爱不释手，“要是陆太太在某些方面也能满足我，跟我一起翻云覆雨，那就更好了。”
<br/>
<br/>        “…”
<br/>
<br/>        夏夕绾直接将一个虾仁塞到了他的嘴巴里吃吧你！
<br/>
<br/>        陆寒霆吃下了虾仁，摘走了她手上的手套，“我来剥虾。”
<br/>
<br/>        堂堂陆氏大总裁亲手剥虾，估计是头一次吧。
<br/>
<br/>        “好啊，陆先生，那虾仁就交给你了。”
<br/>
<br/>        很快，一顿简单的晚餐就搞好了，陆寒霆拿了筷子和勺子，夏夕绾从厨房里走出来，“好啦，开饭了。”
<br/>
<br/>        陆寒霆看了一下，夏夕绾做了盖饭，松软的米饭垫底，上面是嫩嫩的虾仁滑蛋，还有用黄油煎过的西兰花，洒了一点细碎的海苔，弄成了心型的形状，看着色香味俱全，让人食欲大开。
<br/>
<br/>        陆寒霆见她还炖了两碗汤，打开盖子，她是一份玉米排骨汤，他的是…
<br/>
<br/>        “你给我炖了什么汤？”陆寒霆问。
<br/>
<br/>        夏夕绾的目光有些闪躲，“就是…补汤啊，你喝喝看，很好喝的。”
<br/>
<br/>        说完夏夕绾就往厨房里跑。
<br/>
<br/>        但是陆寒霆伸出长臂，一把就将她捞进了自己的怀里，“跑什么，鬼鬼祟祟的，你究竟给我炖了什么汤？”
<br/>
<br/>        “那个…”夏夕绾贝齿一咬红唇，嗓音软糯道，“那个不是很晚了嘛，超市要关门了，搞打折，最后还有一份…牛鞭，我就给买回来了。”
<br/>
<br/>        所以，这是牛鞭汤？还是她打折买回来的？
<br/>
<br/>        陆寒霆剑眉上挑，“陆太太，你怎么好意思跑超市里买牛鞭的，也不怕别人笑话你？”
<br/>
<br/>        夏夕绾小脸红红的像个苹果，但嘴上大胆的哼哼，“我给我老公买牛鞭怎么了，谁敢笑话我？”
<br/>
<br/>        其实她没有告诉陆寒霆，她在超市特别怂，拿了牛鞭就跑了，还恨不得戴个面具在脸上，生怕别人知道她买牛鞭了。
