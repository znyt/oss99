　　许飞的事情只是个小插曲。
<br/>　　刘峰在屋内和林峰客气寒暄。
<br/>　　“林峰，久仰大名，今天还是第一次见，我就住在三十七号楼，就在你楼上，以后多走动走动，对了，这几天，组织内可能会举办一次聚会，到时候我们几个干事都要参加。”
<br/>　　“聚会？”林峰问道。
<br/>　　“也没什么，就是大家见个面，彼此认识一下。”刘峰站起身来，“那我就先走了，具体内容到时候会有人来通知我们的。”
<br/>　　许飞的事情只是个小插曲。
<br/>　　到了下午的时候。
<br/>　　平民区忽然不少黑铁战士激动起来了。
<br/>　　“排行榜出现了！高手排行榜！目前根据等级资料最强的一百个人类！”
<br/>　　“握草，不知道有没有我的名字。”
<br/>　　“你个三阶的还说拉到吧，什么时候三阶都可以进入前一百了，那人类就完蛋了。”
<br/>　　许多人忽然都兴奋极了。
<br/>　　林峰一听也是来了兴趣，排行榜？
<br/>　　他也跟着人群来到了一个放着排行榜的地方。
<br/>　　排行榜是一个巨大的LED滚动屏，上面滚动播放着一百个人的名字。
<br/>　　一群人挤在这个滚动屏下面张着嘴巴看。
<br/>　　第一，周华，男，九阶，军部。
<br/>　　第二，寒鸦，男，九阶，战士会。
<br/>　　第三，赵大鸟，男，八阶，天神帮。
<br/>　　第四，李畅，男，七阶，军部。
<br/>　　第五，王浩，男，七阶，军部。
<br/>　　第六，罗敏，女，七阶，战士会。
<br/>　　第七，白骨，男，七阶，骷髅会。
<br/>　　第八，刘春春，女，七阶，流水阁。
<br/>　　第九，叶晓，男，七阶，火神帮。
<br/>　　第十，韩风，男，七阶，战士会。
<br/>　　第十一，范长江，男，七阶，无。
<br/>　　……
<br/>　　“居然这么多七阶的，第一和第二更是九阶的存在。”
<br/>　　林峰震惊了，不过他也知道，这群人八成是属于末世之前就已经厉害的存在了。
<br/>　　不然根本不可能领先这么多。
<br/>　　比如说这个第一的兵王，他可能末世之前就已经是黑铁战士，可能是四五阶的存在，毕竟他每天训练，不断冲刺人体极限，在配合一些基因药物科学训练，比普通人要强大许多。
<br/>　　“还真是人外有人天外有天啊，前十名，大部分都是军部的，也无愧于军部最强了，其次就是战士会，会长是个九阶的存在，这个叫做罗敏的估计就是副会长了，毕竟那个麻子和我说过，战士会内除了会长和副会长最强之外，就是一群干事了，不过没想到她却是个女的，不过这个韩风是谁，莫非是个执事不成，居然有足足七阶，这可有点厉害了。”
<br/>　　“至于其余的什么流水阁，估计都是地下前三的大势力的掌舵人，不过流水阁这个名字起的太邪恶了，老大还是个叫做刘春春的女人。不会是开妓院的吧。”
<br/>　　林峰扫了眼，七阶的高手尤其不少，一直到二十名才出现六阶的。
<br/>　　这也就是说，林峰的真实实力，足够进入前二十了。
<br/>　　当然了，这只是等级上的划分，同级别之中还是有区别的，这个榜单只代表等级不代表真实战斗力。
<br/>　　“不知道这些人之中有几个是异能者的。不过应该有那么几个吧。”
<br/>　　林峰继续往下看，在六十名上下，看到了自己的名字，林峰，男，战士会。
<br/>　　至于陈勾他们则没有上榜。
<br/>　　六阶的都有那么几十号人，五阶的更多，四阶根本没有机会上榜。
<br/>　　“妈的，这么多五阶的？”
<br/>　　“我了个擦，最后一名都是五阶的。这么恐怖？这几天下来，我以为我一个四阶的可以横着走了，天天装逼，现在看来要低调点，这要哪天遇到这群大佬，死都不知道怎么死的。”
<br/>　　“没错，我以后也不敢随意装逼了，大佬太多了。这群家伙简直不是人，也不知道怎么杀丧尸的。”
<br/>　　“还装逼？你怕是死都不知道怎么死的，我一个哥们就装逼，以为自己四阶了，就到处装逼，结果抢了一个异能者的老婆，被那个异能者给大卸八块。”
<br/>　　“凭什么老子要在这个家伙后面，我也是五阶的，真干起来，他肯定不是我对手！我从小就练习搏击！”
<br/>　　许多人都大声议论着。
<br/>　　榜单的出现，让许多人都知道了人类之中的强者名字。
<br/>　　几天下来，每个人都讨论着相关的问题。
<br/>　　值得一提的是，这几天，这个榜单也在不断的刷新，当然了，前三十基本没有变动。
<br/>　　但后面的不断的刷新，一些人会忽然出现一个X，然后名字变成了其余人。
<br/>　　这个X就代表这个人死了，或者变成丧尸了，反正就是消失了。
<br/>　　这也让每个人都更加深刻认识到了这个世界的残酷。
<br/>　　与此同时。
<br/>　　特等区内。
<br/>　　三大家族同时都得到了这一份榜单。
<br/>　　“去，把这上面还没加入势力的人都联系一遍，看能不能拉拢过来。”
<br/>　　几乎在同时。
<br/>　　三大家族都派人出去。
<br/>　　……
<br/>　　而这一天。
<br/>　　林峰收到了通知，要前往战士会的总部聚集了。
<br/>　　战士会的总部也在平民区，不过是一个特别大的房子，整个楼都被战士会给买下来了。
<br/>　　此时此刻。
<br/>　　林峰带着陈勾他们来到了这个大楼。
<br/>　　这里已经聚集了不少人。
<br/>　　“林峰，还真是够巧的。”
<br/>　　刘峰走了过来看到林峰笑道。
<br/>　　对方身边也跟着五个男人。
<br/>　　“嗯。一起进去吧。”
<br/>　　林峰点了点头就和对方一起进入了大楼。
<br/>　　大楼内。装修的还不错。
<br/>　　此时此刻就好像是开晚会一样。聚集了上千人之多。基本上战士会能来的都来了。
<br/>　　看到林峰他们都是目光好奇的扫了过来。
<br/>　　没多久。
<br/>　　一个男人走了出来，这个男人，林峰不知道该怎么形容，长得挺帅，头发有点苍白的感觉。
<br/>　　气势也无比的强大，不怒自威，站在那里，就会情不自禁的注意到他。
<br/>　　“很高兴能够看到我战士会这么多人。先自我介绍一下，我叫寒鸦，战士会的会长。”
<br/>　　“哇，他就是寒鸦？”
<br/>　　“好厉害啊。”
<br/>　　“不知道会长有没有女朋友，好帅啊。”
<br/>　　不少人议论纷纷。
<br/>　　“当然了寒鸦是我的绰号，之前，我是在一个缉毒队里面工作，当然这都是过去的事情了，今天叫大家过来，主要是让彼此大家认识一下。”
<br/>　　寒鸦笑了笑。
<br/>　　“几位干事都上来吧，让大家看看我们战士会的干事，以后也好心里有个数。”
