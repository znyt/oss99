　　第447章
<br/>　　方怒踉跄从地上站起，感觉头晕眼花，周围乌压压一片，觉得每张脸都看着自己，但自己却看不清楚他们脸上的表情。
<br/>　　林少羽从后面刺来一枪，洞穿了他的心脏，黑色的血从他胸膛流出，他晃了三晃，身体噗通一声倒地。
<br/>　　这一幕狠狠的烙印在帝国军心中，只见林少羽横枪立马，血顺着枪身流下来。与之伴随着，是林少羽身后起义军的哇呀呀大叫，如一群见血便狂的野兽。
<br/>　　李行哉看着这一幕轻轻点头，表情倒是没有多少动容。
<br/>　　“无论如何，林少羽都是有资格留在史书上的，不管留下的是清名，还是骂名，史官都没办法抹掉他的存在。”
<br/>　　宋游渠的目光落在林少羽身上，此时此刻，三军人马谁也没办法将目光从林少羽身上移开。
<br/>　　“林少羽可以史册留名，杀了他的人自然也可留下重重一笔，不知三军之中，是否还有人敢与他动手。”宋游渠忽然道。
<br/>　　“这……自然是有的。”李行哉轻声道。
<br/>　　他的声音刚刚落地，就见队伍中冲出一人，不是旁个，正是将军府的柳轻名。
<br/>　　“多年未见，柳轻名似乎长进不小。”李行哉道：“但想对付林少羽，怕还是差了些。”
<br/>　　“殿下莫忘了一件事，起义军人困马乏，又接连战斗，此刻体力怕已撑不了多长时间了。”宋游渠。
<br/>　　“言之有理，”李行哉点点头：“林少羽现在已经是只病虎，但虎死威风在，就看看谁能斩下虎头，而不被虎威所伤吧。”
<br/>　　柳轻名与林少羽作对厮杀在一起，大家都知道，在三军面前能够打败林少羽是大大露脸的事情。军中的年轻人都有一腔血勇在，想着鳌里夺尊，扬名立万。
<br/>　　而林少羽毕竟已经力疲，一枪砸死方怒，看似威风，但这样的招数太过消耗体力。林少羽能用一次，却未必能用第二次。
<br/>　　他现在和柳轻名交手，用的是花枪的架门。两人正在对枪，只见枪来枪往，密不透风。林家破浪枪施展开来，一枪更比一枪凶险。最后长枪直进，直冲柳轻名面门。柳轻名一枪被挑落马下，林少羽紧跟着一枪狠狠摔在他脑袋上，摔得他脑袋碎裂而死。
<br/>　　“轻名！”
<br/>　　一员大将猛吼一声，手持一杆大枪冲出阵来：“狗贼，竟敢杀我孩儿，给我去死！”
<br/>　　这员老将正是柳轻名的父亲，戍守京州的大将柳弄山。他今年已经年过五旬，女儿生了很多，但儿子就这一个。
<br/>　　此刻见到柳轻名被杀，老将发怒，自己冲了出来。
<br/>　　可他毕竟年老体弱，更不是林少羽的对手，林少羽斜刺一枪，要了他的性命。
<br/>　　“嗷！”
<br/>　　起义军发出一阵鬼哭狼嚎的声音，用力捶打着胸脯，庆祝林少羽的胜利。
<br/>　　李行哉啧啧叹息，多年不见，林少羽不仅本事大了，心肠也变得够狠。这一点，倒是和程大雷不太一样，程大雷虽然面目凶恶，但李行哉看得出来，他其实并没有视苍生为鱼肉的狠绝。
<br/>　　“谁敢与吾一战！”林少羽马上大喝。
<br/>　　帝国军当真无人敢答言，眨眼之间，林少羽已经击败帝国三员猛将。柳轻名父子二人皆死在他的枪下，柳家一门的香火算是绝了。
<br/>　　当真让他如此耀武扬威么，最后就算能杀了林少羽，帝国军的面子可真就一点也剩不下了。
<br/>　　李行哉手握长枪，心中战意又起。
<br/>　　咦，好像帝国军本就没剩下多少体面了。
<br/>　　心中正想着，突然听得耳边战鼓捶响，帝国军发出一声声欢呼。李行哉一怔，见帝国军中又有一人冲出去，欢呼声正是因他引起的。
<br/>　　出阵的是李乐天。
<br/>　　这需要极大的勇气，虽然林少羽已经力疲，但杀威不减。面对这一幕，许多成名多年的帝国猛将都犹豫不决，李乐天竟敢冒险冲出去。林少羽可不会因为他的身份而手下留情，真如果他被一枪杀了，帝国可就没第二个大皇子了。
<br/>　　李行哉轻轻叹息，自己在某些方面，还真是不如自己这位大哥。
<br/>　　这是李行哉自己的评价，然而在帝国上层，李行哉哪里是比不上李乐天，二者根本是一个天上一个地下。
<br/>　　李乐天一直是个很优秀的人，琴棋书画，弓马武艺，任何一样拿起来，都能玩得不错。他算是完美继承了皇族血统的人，但不知怎地，李乐天这样优秀，明帝还是不如何看重对方。
<br/>　　明帝最看重的人，是帝国的废物皇子李行哉。如果李行哉稍微表现得争气些，怕明帝就会为他铺好一座长通龙椅之路。
<br/>　　幸运的是，明帝临死之前，总算做了一件明白事，把天子剑交给了李乐天。
<br/>　　李乐天手持天子剑，死守长安城，虽然最后长安城被攻破，但他的表现大家都看在眼里。如今军中很多少壮派支持他，就是因为这个原因。
<br/>　　李乐天冲马到阵前，勒马停下，手中长枪一抖，也没有废话，一枪向林少羽刺了过去。
<br/>　　“看枪！”
<br/>　　枪是李家盘龙枪，八打盘龙枪，李乐天一共学会三招，分别是：青龙甩尾，离龙吐火，黑龙砸水。
<br/>　　当年帝国开国武帝，能用这枪打下整个天下，盘龙枪自然威力无穷。甩，刺，砸三招，如果学得精深，也足够一辈子受用了。
<br/>　　盘龙枪对破浪枪。
<br/>　　李乐天抢攻三招，林少羽毕竟有些力疲，竟没有找到还手的机会。李乐天每刺出一枪，帝国军就发出一声声欢呼。毕竟，对于一个皇子，不能要求太多，他能和林少羽打过七八个回合还不落败，已经算是相当不错。
<br/>　　扬州牧李星的脸沉下来，变得极为难看。这一声声欢呼，就是人心呐。只要李乐天不死，帝国军大半人心都被他收入囊中。
<br/>　　现在他甚至盼着林少羽出枪更快些，能够一枪将李乐天杀了，自己也好少一个竞争对手。
<br/>　　但说也奇怪，二人打得不可开交，但李乐天还未有落败的迹象。
<br/>　　可那一声声欢呼，就实在令李星有些心焦了。
<br/>(本章完)
