　　苏莫又呼唤了阎魔鬼圣几声，对方依旧不理他。
<br/>　　无奈的摇了摇头，苏莫便不在呼唤对方，自己独自沉吟了起来。
<br/>　　天劫一事，他懒得去考虑，毕竟对他没有什么影响，他要考虑的首先是目前面对的困局。
<br/>　　沉吟了良久之后，他还是觉得去内府火元峰，比加入其它派系要好的多。
<br/>　　而且，到时候有火元圣师撑腰，别说太史轩可尤天韩，就算是整个太史世家一脉，都很难动他了。
<br/>　　而且，这样也能更好的给予聂乾和韩盈庇护。
<br/>　　只不过，在与冷江一战之前，木璃导师说了，他只要一个月的考虑时间，不然便没有机会了。
<br/>　　现在一月之期早已过去，不知道火元圣师还会不会收留他！
<br/>　　当然，还是要去试一试才能知道。
<br/>　　随即，苏莫便离开了自己的阁楼，向木璃导师的住处走去。
<br/>　　此事，还需要经过木璃导师的媒介，他也不好直接去火元峰。
<br/>　　不多时，苏莫便来到了木璃导师居住的院落门前，他立刻敲了敲门。
<br/>　　不过，院门敲响了，其内却并没有任何声音，木璃导师并不在。
<br/>　　苏莫无奈，只好在门前等待了起来。
<br/>　　这一等就是三个时辰，三个时辰之后，木璃导师终于回来了！
<br/>　　嗖！
<br/>　　白衣飘飘的木璃，从远处飞来，落在了苏莫的身边。
<br/>　　“你来做什么？”木璃冷冷的问道。
<br/>　　“那个……木璃导师，我想问一下，火元圣师前辈，他还愿意收留我吗？”苏莫面现尴尬之色，吞吞吐吐的问道。
<br/>　　之前拒绝了对方，现在再回头，真的是有些丢脸。
<br/>　　“考虑清楚了？”木璃面无表情，淡淡的问道。
<br/>　　“考虑清楚了！”苏莫点了点头。
<br/>　　“跟我来吧！”木璃轻言一声，便转身向远处飞去。
<br/>　　苏莫立刻跟了上去。
<br/>　　很快，两人便来到了进入内府的大门之前，进入了内府之中。
<br/>　　再一次的进入内府，苏莫依旧感到非常的震撼，数不尽的山峰漂浮在浩瀚的星空中，充满着磅礴的气势。
<br/>　　嗖！嗖！
<br/>　　木璃和苏莫一前一后，向火元圣师所在的火元峰飞去。
<br/>　　很快，两人便落在了火元峰之上。
<br/>　　偌大的火元峰，比当初天灵宗九峰中的任何一峰都要庞大，但其上人烟稀少，安静无比。
<br/>　　木璃带着苏莫，再次来到了那座中央主殿之前。
<br/>　　随即，两人就这样站在门前，静静的等待。
<br/>　　“进来吧！”片刻之后，殿内响起了火元圣师苍老的声音。
<br/>　　而后，木璃推开殿门，带着苏莫走进了大殿之中。
<br/>　　大殿之中，身形微胖、满头白发的火元圣师，依旧闭目盘坐在蒲团之上，与苏莫上次见其之时，没有任何的改变。
<br/>　　“导师，苏莫来了！”走到火元圣师的身前，木璃躬身行礼。
<br/>　　“见过前辈！”苏莫同样行礼，他现在有求于人，礼数可不能少。
<br/>　　满脸皱纹的火元圣师睁开了眼眸，深邃而又沧桑的眼眸，静静的凝视着苏莫。
<br/>　　“非要走头无路，才愿意来老夫这里？”火元圣师淡淡的问道。
<br/>　　呃~~
<br/>　　苏莫闻言顿时有些尴尬，看来对方对他的事很清楚啊，知道他是无奈之下才来到此地。
<br/>　　“承蒙前辈看重，以前是晚辈唐突了！“苏莫抱拳说道。
<br/>　　“数千年以来，你是唯一一个拒绝过老夫的学府弟子！”火元圣师道。
<br/>　　从来都是别人三顾茅庐来找他，可苏莫倒好，让他三顾茅庐去相请。
<br/>　　苏莫尴尬的笑了笑，不知如何接话。
<br/>　　少倾，火元圣师略一沉吟，道：“自己在山上建处住所吧！”
<br/>　　听闻此言，苏莫顿时心中一喜，没想到对方这么容易就同意了！
<br/>　　他本来还以为对方或许会刁难他呢！或者是让他认错之类的，没想到如此大度。
<br/>　　“多谢前辈！”
<br/>　　苏莫立刻抱拳，随即试探性的说道：“前辈，晚辈有两位要好的朋友，能不能来和晚辈同住？”
<br/>　　火元圣师闻言，略显花白的眉头微皱了起来。
<br/>　　苏莫见此，立刻心中一沉，若是对方不同意的话，他倒是安全了，但聂乾和韩盈就麻烦了。
<br/>　　“可以！”少倾之后，火元圣师眉头舒展，微微颌首。
<br/>　　“多谢前辈！”苏莫顿时大喜，对方同意，那他和聂乾韩盈都无忧了。
<br/>　　他们三人住在火元峰，名义上就是火元圣师的人，太史轩和尤天韩就算再张狂，应该也不敢明目张胆的对方他们吧！
<br/>　　“去吧！”火元圣师随意挥了挥手，下了逐客令。
<br/>　　“晚辈告辞！”苏莫向对方躬身行了一礼，便转身走出了大殿。
<br/>　　待苏莫离去之后，木璃看了一眼火元圣师，略显不忿的说道：“导师，这个苏莫之前如此不识时务，您就这么同意了？”
<br/>　　木璃心中有些闷气，之前三番两次邀苏莫来火元峰，对方都拒绝了。
<br/>　　如此的不给情面，而现在火元圣师居然这么容易，就收留了对方！
<br/>　　木璃对于火元圣师非常的尊敬，所以对苏莫这个不给火元圣师面子的人，自然有些恼怒。
<br/>　　若是她的话，绝对不会让苏莫留在火元峰，自己去承受错误去吧！
<br/>　　“年轻人嘛！难免心高气傲！”火元圣师笑着摇了摇头道。
<br/>　　“真是太便宜他了！”木璃低声嘀咕道。
<br/>　　“他的天赋，远超你数倍，无论是武道还是炼器！”火元圣师说道，他之所以同意收留苏莫，也是看中了苏莫的天赋。
<br/>　　他以前并不知苏莫还拥有特殊体质，不然的话，肯定会直接收为亲传弟子。
<br/>　　他虽然是炼器师，要找炼器天赋超绝的传人，但武道天赋也是重中之重。
<br/>　　试问，没有超强的武道天赋，就不能成为强者，那还如何成为炼器圣师？
<br/>　　木璃闻言，顿时俏脸一僵，眸中露出不服之色，问道：“导师，他有这么厉害吗？”
<br/>　　木璃有些质疑火元圣师的话，或者说是不服气苏莫的天赋。
<br/>　　她的天赋，无论是武道还是炼器，都是绝顶天才，苏莫比他好数倍？这怎么可能？
<br/>　　先不说炼器天赋，就说武道天赋吧！
<br/>　　苏莫的确不凡，拥有特殊体质，越级战斗能力逆天，但她也不差多少。
<br/>　　她年纪五十余岁，便达到了武尊境五重的境界，曾经身为学府弟子之时，也是差点成为圣子的人物。
<br/>　　苏莫比他天赋好数倍？
<br/>　　这未免太夸张了吧！
<br/>　　火元圣师并未回答木璃的话，而是叹了轻叹一声，摇头道：“可惜，他志不在炼器，真是可惜啊！”
