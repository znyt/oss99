　　天穹之上，虚空之中，苏莫和海明江快速飞行。
<br/>　　他们已经飞行了几日，天海城马上就要到了。
<br/>　　“海阁主，阴阳神宫的实力如何？”苏莫向海明江问道。
<br/>　　“阴阳神宫实力不弱，虽然不如人族九大势力，但差距也不算太大，宫主阳鼎天也是准帝修为！”海明江说道。
<br/>　　天涯海阁和阴阳神宫，同处于天海城，算是老邻居了，海明江对于阴阳神宫的实力，还算是比较清楚的。
<br/>　　“阴阳神宫参与了灭你们天涯海阁总部，想必阳鼎天应该还在天海城！”苏莫沉声道。
<br/>　　“极有可能！”海明江点了点头。
<br/>　　“阳鼎天的实力怎么样？”苏莫问道。
<br/>　　“阳鼎天此人，年轻之时是有名的美男子，不知祸害了多少女人，其修炼双修功法，一身修为也都是在女人的肚皮上换来的，但其实力却是不弱，隐隐比老朽还要稍强一分！”海明江道。
<br/>　　“比你还要强？”苏莫闻言眉头微皱，如此的话，他们想杀阳鼎天，估计还有些困难。
<br/>　　毕竟，准帝强者不是那么好杀的，若是对方一心想逃，真的很难杀死对方。
<br/>　　“我们千万不能大意，阴阳神宫既然敢和黄泉魔宗联手，估计也做好了与我们为敌的准备，所以，一切要小心行事！”海明江道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“哎！可惜我天涯海阁一下子被打残了，不然，以我们的情报能力，能轻易的查出对手的任何风吹草动！”海明江叹道。
<br/>　　虽然天涯海阁现在还有大量的分阁，但总部统领的中枢被灭，短时间内根本就无法运转情报系统。
<br/>　　不然的话，他们也不会像现在这样，束手束脚。
<br/>　　“或许这就是他们得目的吧！”
<br/>　　苏莫摇了摇头，道：“总之我们小心行事，先看看情况再说。”
<br/>　　“嗯！”海明江点头。
<br/>　　闲聊之中，不多时，两人便临近了天海城。
<br/>　　远远的，望着天海城那高大雄伟、延绵近千里的城墙，苏莫和海明江均是停下了身形。
<br/>　　“不对劲啊！”海明江皱眉说道。
<br/>　　“的确不对劲！”苏莫点了点头，此刻的天海城除了人气没有以往那般繁华之外，居然没有什么变化。
<br/>　　城门敞开，有人进城有人出城，仿佛没有什么事情发生一般。
<br/>　　这完全不正常！
<br/>　　要知道，天涯海阁总部被灭，傻子都知道苏莫一方，肯定会回来报复。
<br/>　　而阴阳神宫居然一副没有事情的样子，完全没有任何防备。
<br/>　　若是阴阳神宫，如同殷家一般，开启防御大阵，如临大敌的模样，这才是正常的表现。
<br/>　　“若是老朽所料不错的话，天海城内必定布下了无穷杀机，根本不怕我们来攻击！”海明江面色凝重的说道。
<br/>　　“那我们怎么办？回去？”苏莫沉声问道。
<br/>　　对于他来说，就算白跑一趟也无妨，反正他现在资源无数，修炼已经不成问题，待他突破到武皇境四重，到时候基本上没有人是他的对手了。
<br/>　　那个时候，再收拾黄泉魔宗、阴阳神宫和北堂世家也不迟。
<br/>　　海明江目露沉吟之色，他知道天海城中，可能有危险，但他还是想进去查探一番。
<br/>　　毕竟，他和阴阳神宫这个仇，实在是太大了。
<br/>　　“我们进城看看吧！”
<br/>　　海明江声音低沉，继续道：“以我们两人的实力，就算有两三名准帝，也无法拿下我们。”
<br/>　　苏莫点了点头，的确，以他们两人的实力，根本不需要有太多惧怕！
<br/>　　“为了安全起见，我们还是改变一番吧！”苏莫沉声道。
<br/>　　“嗯！”海明江点头，随即两人均是改变了一番面貌，又换了一身衣衫，便向天海城而去。
<br/>　　改变容貌，只要稍稍改变面部肌肉便可以了，这对于真灵境以上武者，都是轻而易举的小事，更遑论两人这种强者。
<br/>　　不多时，两人便进入了天海城中，他们先是去了天涯海阁总部所在之地，远远的看了一番。
<br/>　　天涯海阁的总部，整个内城已经被夷为了平地，彻底的不存在了。
<br/>　　这让海明江心中的怒火，蹭蹭的上涨，心中的杀意几乎抑制不住了。
<br/>　　“走，我们去阴阳神宫！”海明江转身向阴阳神宫的方向而去。
<br/>　　“海阁主莫要激动，小心行事！”苏莫见此，急忙追上去提醒了一句。
<br/>　　看对方的模样，真像是要直接杀进阴阳神宫一般。
<br/>　　两人的速度很快，一步数十丈，不多时便接近了阴阳神宫。
<br/>　　远远的，站在距离阴阳神宫数千丈之外的街道上，两人暗暗打量了起来。
<br/>　　阴阳神宫的所在，同样是一座城中之城，城墙高大，占地方圆百里。
<br/>　　阴阳神宫所在的内城大门之前，有十数名守卫把守，大门前进进出出的阴阳神宫弟子不少。
<br/>　　“这是什么情况？”苏莫眉头微皱，这阴阳神宫，真是有恃无恐啊！
<br/>　　难道他们就那么有自信，能抵御一切攻击？
<br/>　　又或者说，黄泉魔宗有准帝强者在阴阳神宫？
<br/>　　这个情况倒是有很大的可能，因为阴阳神宫和黄泉魔宗一起灭了天涯海阁的总部，黄泉魔宗的强者，倒是极有可能留在阴阳神宫。
<br/>　　“不知道他们玩什么把戏？”海明江亦是摇了摇头，心中想不明白。
<br/>　　一时间，两人都有些踌躇了起来，不知道是该直接开杀，还是再详细查探一番。
<br/>　　“要不……我们试探一番如何？”苏莫面露沉吟之色。
<br/>　　“如何试探？”海明江问道。
<br/>　　“我们这样……！”
<br/>　　随即，苏莫将心中的想法，和海明江诉说了一番，对方当即就同意了。
<br/>　　而后，两人立刻便远离了阴阳神宫数百里，来到了天海城的另一边。
<br/>　　随后，苏莫和海明江找了一家客栈，进入了客栈的房间中。
<br/>　　进入客栈之后，苏莫身上立刻玄力涌动，一具魔影分身幻化而出。
<br/>　　“去吧！”苏莫挥了挥手，魔影分身一闪之间，便消失在了房间的窗口处。
<br/>　　苏莫是打算让魔影分身，去试探一番阴阳神宫的虚实。
<br/>　　至于怎么试探？
<br/>　　当然是自爆，魔影分身直接冲进阴阳神宫自爆，将整个阴阳神宫炸为平地。
<br/>　　如此的话，阴阳神宫到底有什么底牌，便能彻底的显露出来了。
<br/>(本章完)
