　　现场瞬间冷凝的气氛让四周温度好似降到冰点，除了已经朝后方撤离的罗家死士没有听到这番对话外，其他人全部好似双脚被束缚了一样无法动弹。
<br/>　　丧尸王三个字，意义太过于重大！
<br/>　　此时的木天依旧带着询问的眼神看向白允儿，似乎是在等她回答。
<br/>　　白允儿双眼微迷，左脚轻轻向旁侧微移，她一边调整着随时可能战斗的姿势，一边轻描淡写的开口：“丧尸不过是一副行尸走肉的躯体，你是丧尸王又怎样，难道就不是丧尸了？”
<br/>　　紧接着，她也不给对方说话的时间，嘴角一勾冷笑道：“既然是丧尸，那便是这星球上最低劣的物种，甚至连生命都算不上？花花草草也比你高贵。”
<br/>　　一连串的冷嘲热讽，直逼所有丧尸最忌惮的问题，同样也是木天最不满意自己的地方。
<br/>　　当白允儿的话音落下后，现场更是一片死寂，狼牙的人及白家死士一个个大气都不敢出，心中都替白允儿捏了把汗。
<br/>　　高姿态不说，竟然还如此羞辱，先不管丧尸王在生物圈到底算不算生命的一种，但光是白允儿这一番挑衅的话，就是要把事情闹到最严重的地步，让双方都没个台阶下。
<br/>　　在这末世三年众势力都还未成型，高战力者更不是无敌的情况下，就这么直接与丧尸王对上，这个白允儿实在是……
<br/>　　就在众人都心惊胆战，生怕一个不好两人就会再次开打，战的惊天动地之时——
<br/>　　“我倒是认为丧尸也是生命物种之一。”忽然一个略带艳·丽的嗓音出现。
<br/>　　众人大惊之间，只见尚九娣竟然靠近了那一处地带，她衣服凌·乱身上带着血渍和污垢，却偏偏站的笔直也一点不怕死的与白允儿和丧尸王，呈现了三角站位。
<br/>　　震惊，一下子涌上所有人心头，尚九娣怎么跑过去了？！
<br/>　　白家的不少死士都面上浮现出冷笑，暗道这个女人真是自寻死路，这样也好，分担了白允儿的危险。
<br/>　　白允儿更是因为将重点都放在身后的木屋和眼前的丧尸王身上，尚九娣的忽然出声让她下意识的眉头紧皱，但却很快便舒展而开，静静的站在原地没有反应。
<br/>　　顾凉晨却吓到差点失声，他想要上前阻止，却猛然心中一顿反应过来。
<br/>　　先前白允儿的那一段段话说出口他就觉得不对，说废话什么的这根本不是白允儿的画风，所以当下让白允儿做出这番反常举动的原因只有一个：
<br/>　　那就是拖延时间！
<br/>　　顿时，顾凉晨便想到了木屋之内可不只有楚涵，还有另外一个至关重要的女人——
<br/>　　步纱！
<br/>　　他想要上前的脚步收了回去，警惕的站在不远处静待。
<br/>　　白允儿的反常推理出来之后，尚九娣的行为举止也不难猜测。
<br/>　　她的确是不要命了，但却是为了配合白允儿拖延时间！
<br/>　　尚九娣在站定之后，便朝着丧尸王一笑，看也不看白允儿一眼的开口道：“全球几十亿的丧尸只有一个丧尸王，今天竟然出现在狼牙基地，真是幸会！”
<br/>　　带有身份地位的套路化开场白，在其他场合或许极其普通，在此时的木天心中却极其受用，因为这代表着他被当做人类看待。
<br/>　　更何况……
<br/>　　眼前的两个女人，无论是白允儿还是尚九娣，木天都早就认识。
<br/>　　“尚九娣，好久不见。”他扭过头，像是多年不见的好友打招呼，凶煞的面容上甚至带上了一丝笑意。
<br/>　　“好久，不见？”尚九娣一下子愣住，望着丧尸王完全陌生的面孔，眼神满是疑惑。
<br/>　　他们认识？
<br/>　　之前见过？
<br/>　　丧尸王是谁？
<br/>　　不仅尚九娣一下子冒出这三个疑问，在场的狼牙成员也同样如此，更别说少数来自情报部的成员们，当场就开始调动自己大脑里的全部情报信息进行对比。
<br/>　　如此诡异的对话，显然预示着丧尸王的身份，不同寻常！
<br/>　　而这一点至关重要，说不定就是他们这次能找到应对方案的中心！
<br/>　　只是木天并没有要继续说下去的意思，他的目光从尚九娣身上转移，再次看向白允儿刚想要开口——
<br/>　　莎莎！
<br/>　　忽的一阵轻微不可查的声音，响起在木天想要开口的瞬间，紧接着他便目光一寒看向了白允儿的身后。
<br/>　　那间木屋！
<br/>　　白允儿同样察觉到了这一可疑声音，她微微偏头侧脸扭转，用余光注意着自己身后，同时双眸寒光四射眉宇锋利又上扬。
<br/>　　杀机在她的心中荡漾，因为声音并不是来自木屋之内，而是木屋后方的地带。
<br/>　　是那群罗家暗杀者，他们没有离开基地，而是绕了一个大圈后选择从后方直取目标首级！
<br/>　　那里还有部分狼牙成员在防守，但显然不可能是那些死士的对手！
<br/>　　尚九娣并没有如此敏锐的感知察觉到罗家死士的行动，但她清晰的看到了眼前两人的神色变化和目光方向，顿时一颗心再次提起。
<br/>　　发生了什么？
<br/>　　“几只虫子，也想跟我抢东西？”丧尸王一下子怒从心起，比起白允儿弯弯绕绕的理论性挑衅，罗家死士直截了当的行动，才是真正让他不能忍受的行为。
<br/>　　他迈开步伐，无视了眼前的两个女人，就想要直接跨过去进入木屋，强大的波动更是一层层从他体内荡开，一点点加强，不断扩大。
<br/>　　白允儿咬了咬牙，前后夹击的状况让她举步艰难，这时候若退，丧尸王必定直接进入木屋，可不退，罗家的死士便会进入木屋。
<br/>　　无论哪一种，都不行！
<br/>　　就在白允儿眉头越拧越紧，眼角的冷意也越发森寒，打算殊死一战的时候——
<br/>　　刷刷刷！
<br/>　　忽然好几道身影从后方窜出，从不同的角度不约而同的朝丧尸王攻去，他们是白家死士，抓·住了这一重要时间点，爆发了每个人的全力一击！
<br/>　　与此同时一个声音更是焦急喊出口：“大小姐！你跑！”
<br/>　　紧接着下一秒——
<br/>　　几十道攻击在那深坑处爆发，与丧尸王周身散发的能力波动相撞，犹如高速对冲的巨大爆破！
<br/>　　轰！
