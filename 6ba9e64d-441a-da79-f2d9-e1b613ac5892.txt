　　李浩然表情不变，但眼神中充满了轻蔑，“我不杀你，因你不值得死在我手中，回去告诉星月奴，让她亲自来。”
<br/>　　说完，放下了手掌。
<br/>　　鲲鹏张了张嘴，似乎还想说什么，最后觉得还是别说了，拱了拱手，灰溜溜地下山去了。
<br/>　　“祖师威武！”山河真人由衷地赞了一声，内心十分激动，数十年来，自从人间法术界被法术公会掌控，还没有什么人敢这么痛快地击败过法术公会的人。而且只一招……
<br/>　　山河真人试探问道：“祖师的法力，最近是否有所恢复？”
<br/>　　李浩然道：“我不斩三尸，是想从轮回中证道，我每轮回一世，法力积累，但受身体所限，并不能完全发挥，如今为了寻找徐福和叶少阳，也只好提前释放一部分，将来再多轮回几世罢了。”
<br/>　　山河真人听不懂其中的奥妙，想了一下说道：“祖师，你是担心法术公会与您为敌？”
<br/>　　“法术公会，并不是好对付的。”李浩然甩了甩袖子，“但你切记一点，人间法术之神妙，并不输给轩辕氏！不必妄自菲薄，人间，只有我们自己有资格管！若有人觊觎，我辈修士，何惧一战！”
<br/>　　山河真人趴在地上，长跪不起。
<br/>　　人间，老郭的棺材铺里，谢雨晴，还有瓜瓜等捉鬼联盟的几个成员都在。
<br/>　　“什么！老大刚走，他们就要开会选出什么新的首席弟子，这算什么意思！”瓜瓜跳上桌子，嚷了起来。
<br/>　　老郭耸了耸肩，道：“小苏同学这次找我去，就是跟我说商量这件事怎么办，他们显然是故意的。”
<br/>　　谢雨晴道：“你们法术界的事我不懂，可是除了少阳，没人配得上什么首席弟子吧？”
<br/>　　“谁都知道，大帝册封小师弟为人间第一天师，别说是什么第一弟子了，就算在整个法术界，上下三代，也找不出两个能跟小师弟抗衡的。”
<br/>　　瓜瓜说道：“那他们选这个什么人间第一弟子，不是故意打老大的脸吗？”
<br/>　　“何止打小师弟的脸，也是打茅山的脸。”老郭沉吟道，“不过，他们这么做，也是出于自己的考虑，毕竟冷玉……咱们都知道的。法术界几大门派生怕少阳做出过激的举动，到时候整个法术界都受到牵连。”
<br/>　　大伙无话可说。
<br/>　　“这个不是重点，他们给小苏下了通知，让小苏以代掌教的名义，宣布废除小师弟掌教的身份，逐出师门，免得将来因为转世鬼童，将整个法术界拖入万劫不复之地……嗯，这是他们的原话。”老郭愤然说道。
<br/>　　“岂有此理！”瓜瓜跳上桌子，大声骂道，“他们凭什么来管老大的事，凭什么来管茅山的事！”
<br/>　　老郭说：“凭少阳不在人间了。”
<br/>　　人走茶凉。
<br/>　　叶少阳再强，毕竟已经是过去式了，他被徐福带走的消息，早就在法术界传开了，谁也不知道他会不会回来。如果他还在，就算是面对眼前的局面，整个法术界敢于跳出来指责他的也不多，毕竟谁也不想当这个出头鸟，得罪叶少阳，得罪茅山。
<br/>　　“现在少阳走了，外面对于他的负面讨论，简直铺天盖地，除了几大门派真的是为法术界着想之外，其余那些王八蛋，全是幸灾乐祸的。”老郭哼了一声，“少阳在的时候，这些人一个个老老实实的，现在真是墙倒众人推。”
<br/>　　小青冷冷说道：“墙还没倒呢，老大还是会回来的。”
<br/>　　小白道：“就算不看我们的面子，道风大大的面子也不给吗？他们这么对老大，不怕道风大大找他们麻烦？”
<br/>　　老郭道：“道风名义上已经不是茅山的人，甚至都不是人……嗯，不是人类，人间的事，他没资格管。不然阴司一定会干涉，再说他现在全力对付太阴山，根本分不出精力来应付人间这种事情。而且……我听说这背后有法术公会的力量。”
<br/>　　小青一惊，说道：“法术公会要干涉人间的事情了？”
<br/>　　“天劫将至，三界大乱。虽然这什么狗屁天劫我也不知道在哪……不过他们是这么说的，人间缺乏能够领军的人物，法术公会可是有机会插手了。”
<br/>　　小青道：“老大以前表示什么应劫之人吗？”
<br/>　　“那是以前，现在他人都不在这个世界了，还说什么，谁都不知道他是否还会回来。”
<br/>　　小青咬了咬嘴唇，恨恨说道：“真是山中无老虎猴子称大王了。”
<br/>　　小白转头看了美华一眼，拉了一下她的袖子说道：“御姐你怎么一直不说话，魂不守舍的，想什么呢？”
<br/>　　“我在想老大……”
<br/>　　“哦，你想他想到这个程度啊？”
<br/>　　美华迷迷糊糊地点了点头，随即回过神来，瞪了她一眼，“乱说什么，我是在想……怎么说呢，我最近几天，总是梦见老大，感觉很不真切，好像是前世一样，我在民国……应该是民国，见着老大了，跟他一起去对付什么人……”
<br/>　　小白没等她说完就打断道：“你胡说什么，你是鬼域之魂，不用睡觉，做什么梦啊。”
<br/>　　“不是做梦……我形容的不对，是我在吐纳的时候，脑海中产生的片段而……这几天一直都是，仿佛是经历过的，很奇怪，有些场景都很真实……”
<br/>　　老郭本来没兴趣听她们胡扯，听到这里，忍不住心中一动，说道：“你记得什么，有没有什么特别的细节？”
<br/>　　美华想了一下，摇摇头道：“就跟做梦一样，每次吐纳结束，清醒过来之后，就记不清楚了。”
<br/>　　小白道：“你确定不是你太思念老大了，相思成灾。”
<br/>　　“别胡说！”美华瞪了她一眼。
<br/>　　老郭起身走到她面前，已经是一脸严肃，问道：“之前你经历过这种事情没有，我是说，在吐纳的时候，脑海中想到过什么人，或者场景。”
<br/>　　“没有的，从来都没有过，我吐纳的时候心情平静，都是什么都不想的。所以最近几天，我觉得很古怪……”
<br/>(本章完)
