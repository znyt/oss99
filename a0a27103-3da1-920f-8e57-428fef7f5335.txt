　　现在楚风眠踏入了九劫古帝，已经不再是五行刀圣可以对付的了的了。
<br/>　　这五行刀圣甚至都感觉到，他若是在跟楚风眠交手，这一次要死的人，可是他了。
<br/>　　所以这五行刀圣，索性将楚风眠的秘密都给爆了出来，好借助其他人的手，来杀楚风眠。
<br/>　　毕竟三大势力想要剿灭圣龙秘境，已经是明面上的事了，现在他们自然不愿意在看着圣龙秘境，再出一个耀眼的天才。
<br/>　　果然在听到五行刀圣的话后，无数三大势力一方的老古董，看向楚风眠的眼神都变的，蕴含着杀意。
<br/>　　许多老古董的心中，都对楚风眠起了杀心，除了杀心，更有贪婪。
<br/>　　楚风眠现在只是九劫古帝的境界，但是身上的气息俨然已经是不在半步天人境强者之下。
<br/>　　这就代表着，楚风眠的身上绝对是有着惊天奇遇。
<br/>　　“逆天大劫，此人现在的实力都这么可怕吗？如果让他在度过了逆天大劫，那此人该是多么恐怖？只怕是整个大罗天星域都无人是他的对手了吧。”
<br/>　　一名齐林山的半步天人境老古董，目光看向楚风眠冰冷的开口道。
<br/>　　他心中绝不希望，大罗天星域在出现一个如此逆天的人物。
<br/>　　“度过逆天大劫？那有那么容易！这逆天大劫的危险，要比起寻常天劫还要危险百倍，哪怕是在逆天的人物，都可能死在这逆天大劫之下，这小子未必能度过这逆天大劫。”
<br/>　　“曾经无数逆天的人物，可都是死在了这逆天大劫之下，在古籍之中记载的那些死在逆天大劫之下的人物，比起这小子强大的可要多得多。”
<br/>　　“这小子就算是度过了逆天大劫，我们也可以出手，杀了这小子！此人身上肯定怀有逆天之物，不然不可能会引起逆天大劫。”
<br/>　　“不错，就让这小子去渡劫，等到他渡劫结束，我们出手去直接抢夺那逆天之物！”
<br/>　　许许多多的强者，隐藏在周围的星空之中，互相交谈着，他们的目光始终看向楚风眠。
<br/>　　这些前来的老古董之中，不少人都对楚风眠打起了趁火打劫的主意。
<br/>　　这些老古董的想法，楚风眠也是一清二楚，那一道道充满贪婪，杀意的目光，楚风眠感受的一清二楚。
<br/>　　不过现在楚风眠却没有去在乎那些老古董。
<br/>　　在楚风眠的头顶之上的这逆天大劫，才是楚风眠首当其冲的威胁。
<br/>　　逆天大劫，只有是在一尊武者，行逆天之事，令天理难容的时候，才会出现。
<br/>　　现在楚风眠居然是试图要炼化这一缕太初之气，便是引来了这逆天大劫。
<br/>　　太初之气，不是人力可以抗衡的，这是属于天地一部分的力量，属于本源的力量。
<br/>　　就像是一个人，居然是想要掌控天道的力量，自然会引来天道的嫉妒，降下惩罚的天劫。
<br/>　　曾经楚风眠哪怕是炼化天劫，得到造化之力，凝聚本命剑元的时候，都未曾引来过这种逆天大劫。
<br/>　　这太初之气，要比起造化之力，还要令天理难容。
<br/>　　楚风眠现在清晰的感觉到，这逆天大劫已经是在楚风眠的头顶之上，蓄力待发。
<br/>　　只要是楚风眠敢炼化了这一缕太初之气，这劫雷，便是就会无情的落下，诛杀楚风眠。
<br/>　　当然现在楚风眠要是放弃这一缕太初之气，这逆天大劫也会随之消失。
<br/>　　不过想要让楚风眠退避，天道也做不到。
<br/>　　修行一道，本就是逆天之路，不然天道也不会降下天劫要阻拦武者晋升。
<br/>　　这逆天大劫越是厉害，便是也就代表，这一缕太初之气越是厉害。
<br/>　　楚风眠要炼化这一缕太初之气，天道都要嫉妒。
<br/>　　“吞噬！”
<br/>　　楚风眠长啸一声，一把将那一缕太初之气，直接吞噬到了身躯之中去。
<br/>　　逆天大劫又如何？
<br/>　　楚风眠二世为人，本就逆天而行，降下逆天大劫楚风眠也全然不怕。
<br/>　　轰隆隆！
<br/>　　当楚风眠选择将那一缕太初之气吞噬到了身躯之中去的一刻。
<br/>　　天道都仿佛是被激怒了一般。
<br/>　　楚风眠全然无视了天道的警告，要来继续吞噬炼化这一缕太初之气。
<br/>　　在楚风眠头顶之上凝聚的无数劫云，其中咆哮的无数劫雷，都被激怒了，一道道劫雷轰然落下，夹杂毁灭之力降临，要将楚风眠彻底毁灭。
<br/>　　“逆天大劫开始了！”
<br/>　　“好强横的力量，这简直是比起九劫古帝的最终大劫，都要强横数十倍。”
<br/>　　“这就是逆天大劫？”
<br/>　　“你们看着其中的劫雷，都已经是蕴含毁灭之力了，这是毁灭之雷！比起寻常劫雷要更加的恐怖！”
<br/>　　这一道道降临下来的劫雷，已经不是寻常紫色劫雷了，而是黑色，漆黑无比的劫雷，毁灭大劫。
<br/>　　据说是只有在那些千年难遇的妖孽，在渡劫的时候，才会出现，而且最多也就会出现数道。
<br/>　　可眼前这降临的数百道劫雷，全然都这种漆黑无比的毁灭之雷！
<br/>　　哪怕是在妖孽的天才，都没有过这种待遇。
<br/>　　这还只是逆天大劫的开始，就如此的惊天动地。
<br/>　　这数百道毁灭之雷同时降下，哪怕是让一位寻常的半步天人境强者来硬抗，都是不死也要重伤。
<br/>　　面对这数百道降临下来的毁灭劫雷，楚风眠脸色平静无比，他渡劫的劫雷，本就是要比起寻常武者承受的强横的多。
<br/>　　这数百道毁灭之力，在其他人看来，也许无比恐怖，可对现在的楚风眠看来，不值一提！
<br/>　　“剑出！”
<br/>　　楚风眠右手一挥，祖龙至尊剑上便是爆发出了耀眼的剑芒，冲天而起。
<br/>　　一剑之威，跟那数百道降临下来的毁灭之雷，碰撞在了一起，发出了阵阵雷鸣。
<br/>　　在这剑锋横扫之下，这一道道毁灭之雷，都是被打的粉碎。
<br/>　　轻而易举的，一剑将那数百道毁灭之雷，全部横扫。
<br/>　　漫天毁灭之雷，尽是破碎，在空中缓缓消散。
<br/>　　“这？这么简单就化解了那毁灭之雷？”
<br/>　　“这小子的实力，怎么会如此逆天？怪不得天道要降下逆天大劫来，此人的实力，已经不是妖孽可以形容的了！”
