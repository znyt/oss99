　　赤红色刀气席卷而出，宛如九幽地狱。
<br/>　　“天鬼一族的血脉被你继承，当真给你带来了无尽的好处，若非如此，你哪有今日这般实力，只可惜有些浪费了。”
<br/>　　望着席卷而来的赤红色刀气，鬼冥瞥了眼天鬼皇，语气中有嫉妒也有讥讽，轻描淡写间，抱在胸口的长剑一颤，一道锐利剑气疾射而出。
<br/>　　“铿锵……”
<br/>　　刀气与剑气在空中对峙，不断响起清脆刺耳的撞击声，最后气劲耗尽，相互抵消。
<br/>　　“鬼冥就交给本座了，若是鬼厄出现，希望阁下出手相助。”
<br/>　　悄无声息的瞥了眼隐匿于虚空中的慕清霄，天鬼皇向前跨出，持刀而立，刀意冲霄，犀利的双眼中杀意狂涌。
<br/>　　“鬼冥，就冲你的第一句话，本座保证，定让你魂飞魄散，永世不得轮回！”
<br/>　　“有点意思，若是天鬼皇陨落于阴寒域，想必能震慑住其他鬼王吧？待兄长出关，阴寒域就能向外扩张了。”
<br/>　　鬼冥漆黑如墨的双眼中爆出一抹光亮，手持幽冥长剑，声音阴冷沙哑：“既然你执意想死，那便去死吧。”
<br/>　　不再言语，天鬼皇额头上浮现一个神秘纹路，类似于瞳孔，长刀一颤，身形一晃，恐怖的气息便将鬼冥笼罩在其中。
<br/>　　“刀意虽强，但你修为终究落后，靠的不过是血脉而已！”
<br/>　　鬼冥说话的同时，幽冥长剑出鞘，黑光乍现犹如鬼魅，穿过赤红色长刀，瞬间袭向天鬼皇的咽喉！
<br/>　　这一剑似乎跨越空间，诡异无比，难以捕捉轨迹！
<br/>　　“嗤……”
<br/>　　鬼冥目光一凝，嘴角噙着一抹冷笑，猩红的鲜血自幽冥剑上滴落而下。
<br/>　　“不愧号称幽冥鬼剑，好诡异，好快的速度。”
<br/>　　天鬼皇依旧持刀立于上空，望着对面的鬼冥，喃喃自语，在他的咽喉部位，有一道半尺的伤口。
<br/>　　若是刚才，幽冥剑在刺的深一分，那么天鬼皇必将遭到重创。
<br/>　　“奇怪，刚才一剑明明能够刺穿你的咽喉，真是诡异的能力，这就是血脉带来的好处吗？”
<br/>　　阴气一震，幽冥剑上的血液便被驱散，鬼冥饶有兴致的看着天鬼皇，道：“将拥有贵族血脉的天鬼皇击杀，真是令本王欲罢不能。”
<br/>　　“哼，那就要看你有没有这样的本事。”
<br/>　　隐匿于虚空中，慕清霄星眸中闪过一抹诧异。
<br/>　　鬼冥刚才的一剑确实不错，诡异锐利，本能刺穿天鬼皇的咽喉，但在关键时刻，剑尖不知为何偏移了轨迹。
<br/>　　瞥了眼天鬼皇额头上的神秘纹路，慕清霄若有所思。
<br/>　　就在这时，天鬼皇身影一闪，魁梧的身躯跨越空间，一刀横扫而出：“天崩地裂！”
<br/>　　鬼冥不敢怠慢，手起剑落，当下就迎了上去，剑光流转，剑锋轻吟，两道黑色残影瞬间撞在了一起。
<br/>　　“吭~叮……”
<br/>　　无数刀剑相击的声音自两道残影中传出，速度真正的到达了快如闪电，两人掠过之处，空间都会轻微波动一下。
<br/>　　好在慕清霄早已布下空间结界，忘川城中的鬼族后裔根本不知道，两大鬼王在此交战。
<br/>　　此时，天鬼皇与鬼冥，已经交手不下上百招，两者似乎在伯仲之间，谁也奈何不了谁。
<br/>　　天鬼皇的刀大开大合，霸道无双，无处不透着一股帝王般的气势，而鬼冥的招式则诡异难测，绝对是难缠的对手。
<br/>　　天鬼皇一刀不成再出一刀，每一刀都携着恐怖的威势，仿佛要将这片天地都给劈开，刀意无声无息的笼罩着这片空间。
<br/>　　“噗……倒是本王小觑你了。”
<br/>　　手握长剑，鬼冥口中喷出一口深紫色血液，脸色铁青，难看到了极点。
<br/>　　本以为能够轻易将天鬼皇击杀，但交手上百招下来，不仅没有伤到对方，反倒自己落得个遍体鳞伤，大大的出乎意料。
<br/>　　“你到底如何做到的？”
<br/>　　“哼！”
<br/>　　冷哼一声，天鬼皇完全不理会鬼冥，衣襟飘动，刀意冲霄，仿佛整个人都化作魔刀，威势更胜先前百倍不止。
<br/>　　“人刀合一，人剑合一。”
<br/>　　慕清霄有些意外，没想到天鬼皇用刀境界如此深厚，鬼冥在剑道上得了领悟也不差。
<br/>　　瞬间，鬼冥脸色变得狰狞起来，长剑微抬，眼中幽冥之一更浓，顺着长剑，一股气息令人余悸的剑意笼罩在他周身。
<br/>　　“黄泉剑！”
<br/>　　如果说先前交战上百招，两者都是在互相试探对方，那么这一招，绝对是冲着对方的性命去的。
<br/>　　“死气蔓延，诡异的剑。”
<br/>　　慕清霄在剑道上的感悟，远在鬼冥之上，一眼就能看出这一招上蔓延的死气，也只有鬼界才有如此诡异的剑招。

<br/>　　感受到鬼冥长剑上的死气，天鬼皇心头一震，剑招未到，但他已经猜出后者这一剑的恐怖，夺取天地生机
<br/>　　天鬼皇避无可避，浑身已被锁定，陡然在这片空间中，幽冥剑化作一道黑线，朝着天鬼皇疾射而来。
<br/>　　虚空中，慕清霄面色淡然，若是天鬼皇接不下这一剑，那么他的计划就需要有些变动了。
<br/>　　此时，天鬼皇心中一片空白，避无可避，他已经感觉到幽冥剑朝自己的眉心处刺来。
<br/>　　陡然，天鬼皇额头上的蓝色纹路，不知何时已经变成了赤红色，双眼猩红无比，犹如恶魔一般。
<br/>　　“魔掌天下！”
<br/>　　猩红的长刀如同脱缰的野马，化作赤色匹练席卷而出，鬼冥手中的幽冥剑，令得天地色变，在空间凝固的瞬间，一道黑色剑气掠过！

<br/>　　天鬼皇胸前，忽然出现一道极深的剑痕，鲜血自伤口中潺潺流出。
<br/>　　鬼冥脸上的冷笑在一瞬间凝固，幽冥剑自空中坠落，赤黑色的瞳孔中满是不敢置信，最终被绝望与不甘替代。
<br/>　　隐匿与虚空中的慕清霄显现而出，眼中古井无波，道：“可惜了，拥有血脉终究是占据优势的一方。”
