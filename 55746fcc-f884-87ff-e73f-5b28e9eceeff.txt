　　苍穹神殿，宴席之上，苏莫和各大势力的大佬相谈甚欢。
<br/>　　一些和苏莫第一次见面的大佬，也都纷纷介绍自己，并且和苏莫交谈。
<br/>　　灭世剑宗宗主，名叫令狐晋，同样是一名中年人，此人面容冷峻，整个人给人的感觉就是一柄剑，一柄未出鞘的剑。
<br/>　　北海冰宫的宫主，则是一名*****，身姿丰.腴，体态丰.满，人称北海女皇。
<br/>　　北海女皇同样是准帝强者，是北海冰宫玄冰女皇的姐姐。
<br/>　　除了人族的准帝强者之外，还有四名妖族的准帝。
<br/>　　分别是天河水府的府主、南海赤云岛的岛主，万妖山的万妖之主，西海狂鲨一族的族长。
<br/>　　“苏宫主，它日若是有空，可以去我北海做客！”北海女皇微笑着说道。
<br/>　　“呵呵！北海，从未去过，以后若是有机会，定然去走一趟！”苏莫笑道。
<br/>　　“哈哈！还有我西海，苏宫主若是去西海，老夫必定为你准备几名最有姿色的西海美女！”狂鲨族族长大笑道。
<br/>　　狂鲨族的族长，长的五大三粗，丑陋无比，一张大嘴简直要长到耳根处了。
<br/>　　不过，此人虽然相貌丑陋，为人却是豪爽至极。
<br/>　　“呵呵！美女就不必了，若是去了西海，鲨族长能请在下喝几杯美酒，在下就心满意足了！”苏莫笑着说道。
<br/>　　“哈哈！美酒是小事，我西海有最极品的猴儿酒！”鲨族长大笑。
<br/>　　苏莫和一众强者闲谈，相互之间笑声不断，表面上倒是其乐融融。
<br/>　　不多时，终于有人忍不住了，出声旁敲消息。
<br/>　　“苏宫主，贵宫与天涯海阁和帝玄宫结盟，看来是想有所作为啊！”出声之人是灭世剑宗宗主，令狐晋。
<br/>　　苏莫闻言心中明了，此人是要探寻他接下来的打算了。
<br/>　　众人闻听令狐晋之言，均是打起了精神，准备聆听苏莫如何回答。
<br/>　　“令狐宗主，如今的苍穹世界面临危机，无论是你我，还是再坐的诸位，都难以独善其身，我作为苍穹神宫的宫主，自然要想办法壮大自己的实力！”
<br/>　　苏莫面带微笑，他的回答却是模糊不清，让人无法猜测。
<br/>　　“苏宫主，你们人族几大势力与我妖族几大势力之间，一向是井水不犯河水，所以，希望以后苍穹神宫与我天河水府，也能和平相处！”天河水府的府主沉声道。
<br/>　　天河水府，坐落在中洲的天河之内，实力非常强大，人族的这些超级势力，平时都不愿意招惹。
<br/>　　苏莫闻言眼眸微眯，此人的话看似说的颇为和气，但实则是在警告他，不要与天河水府为敌！
<br/>　　苏莫望着天河水府府主，此人长得尖嘴猴腮，脖颈上还有细密的鱼鳞，不知是什么妖兽。
<br/>　　“以后我苍穹神宫，会和所有势力都和睦相处！”
<br/>　　苏莫面上笑意连连，心中却是暗想，待再过一段时间，他有了武帝境的实力，所有势力都要臣服，自然和所有势力都能和平相处。
<br/>　　不过，众人却是没有听明白苏莫的意思，听闻他如此言语，均是微微意外，苏莫难道不想称霸！
<br/>　　当然，众人都是人精，不可能信任苏莫的话。
<br/>　　“宫主！”
<br/>　　“宫主！”
<br/>　　就在此时，大殿之外突然传来急呼之声，随即大量的人影极速飞进了大殿之中。
<br/>　　“嗯？”众人诧异，均是转头看去，苏莫亦是不明所以。
<br/>　　只见金一、石一等傀儡人率先飞进了大殿，几名傀儡人的身后，还跟着大量在外面参加宴席的宾客。
<br/>　　一瞬间，所有人的目光都凝视着石一和金一的身边，因为两人正搀扶着一个人。
<br/>　　此人披头散发，浑身浴血，还断了一条手臂。
<br/>　　此人的面部血肉模糊，已经看不清模样了，其身上气息极为孱弱，显然已经处在了重伤垂死的地步。
<br/>　　“这是……？”众人见此心中一惊，一时间不知道这是什么情况！
<br/>　　“小钱？”
<br/>　　就在此时，一声惊呼陡然响起，只见天涯海阁阁主海明江，闪电般的冲上前去，立刻接住了这名浑身浴血之人。
<br/>　　众人闻言心中一震，此人，居然是天涯海阁的钱副阁主？
<br/>　　苏莫亦是心中大惊，他立刻仔细感受此人气息，的确是钱副阁主不假！
<br/>　　钱副阁主为何会受如此重伤？他不是在坐镇天涯海阁吗？是谁伤了他？
<br/>　　苏莫心中涌现出不好的预感。
<br/>　　“小钱，你怎么样？”海明江惊怒交加，一手抱着重伤的钱副阁主，一手向对方体内注入玄力。
<br/>　　“阁……阁主！”
<br/>　　少倾，钱副阁主缓缓的抬起了头，声音孱弱不已。
<br/>　　“小钱，到底是怎么回事？是谁伤了你？”海明江急声问道。
<br/>　　“黄……黄泉魔宗……黄泉魔宗和阴阳神宫，联手攻击总部，总部已经被灭了！”钱副阁主断断续续的说道。
<br/>　　“什么？”
<br/>　　海明江闻言，顿时身形一颤，瞪大了眼眸，不敢置信的问道：“你说黄泉魔宗和阴阳神宫，联手攻击我天涯海阁的总部？”
<br/>　　整个大殿之中死一般的寂静，钱副阁主的一句话，让得所有人均是心中狂震。
<br/>　　黄泉魔宗居然联合了阴阳神宫，灭了天涯海阁的总部！
<br/>　　这黄泉魔宗，终于露出了獠牙，开始动手了吗？
<br/>　　苏莫闻言脸色瞬间一沉，脸色难看不已，果然是黄泉魔宗。
<br/>　　他还没有对黄泉魔宗出手，黄泉魔宗居然率先动手了。

<br/>　　天涯海阁总部之人，几乎死了九成，剩下的一成均是四散而逃。
<br/>　　这一次，天涯海阁遭到的损失，简直无法估量。

<br/>　　“可恶！可恶啊！”海明江怒了，苍老的脸庞上怒火滔天，眼眸中爆发的杀机，让得整个大殿中的温度，瞬间降到了冰点。
<br/>　　总部被灭，这是海明江不可接受的，总部可是聚集了天涯海阁一大半的力量，更是拥有惊天财富。
<br/>　　如此损失，简直是将天涯海阁打残了！
