　　轰隆隆！！
<br/>　　一阵惊天动地的爆响，猛烈的爆炸震动八荒，一击震退苏莫，巫天御发现远处的状况，脸色铁青。
<br/>　　若是再这样下去，他巫族的试炼族人，就要死光了！
<br/>　　而他，最多压制住苏莫，短时间内根本难以击败苏莫。
<br/>　　“撤！”看到一名名族人惨死，巫天御不甘的大喝了起来，声震数万里。
<br/>　　随即，巫天御微微后退，一众巫族的天才，俱是立刻向他汇聚了过去。
<br/>　　众巫族天才，早就被杀的胆寒，有了撤退之心，现在闻听巫天御之言，个个速度快如疾电。
<br/>　　一直躲避到远处的巫森，也是毫不犹豫，立刻向巫天御汇聚了过去。
<br/>　　“土著，你们活不了！”巫天御怒喝一声，便立刻带着一众巫族天才，狼狈离去，速度快到极致。
<br/>　　苏莫望着一众逃离的巫族之人，并未追逐，巫天御实力太强，他就是追上去也没有多大用处。
<br/>　　一切还是要从长计议！
<br/>　　叹了口气，苏莫扫视一众苍穹神宫之人，发现只剩下了六十余人，陨落了将近四十人。
<br/>　　好在，他在意的一些人，虽然偶有伤势，但并没有陨落。
<br/>　　唰！
<br/>　　光芒一闪，秦不死化为了人形，来到了苏莫的前边。
<br/>　　“小友，你的实力真是让我惊讶！”秦不死目露惊色，满脸的唏嘘之色。
<br/>　　他本以为苏莫的实力，远不如他，没想到却是比他强的多。
<br/>　　“你没事吧！”苏莫瞥了一眼对方，发现对方气息有些虚浮，嘴角还有血迹。
<br/>　　对方之前缠住巫天御，可是受了不轻的伤。
<br/>　　“无妨，没有人能杀死我！”秦不死摇头道，眸中带着自信。
<br/>　　苏莫闻言讶然，真的这么厉害？不死鸟难道真能不死？
<br/>　　“对了，小友，你刚才抓的那名女子，难道身份不凡？”秦不死问道，巫族这么多人，苏莫单单抓那名女子，想必此女身份不简单了。
<br/>　　“应该身份不凡，而且以她的实力，就算身份不高，也是巫族重点培养的天才！”苏莫点头道。
<br/>　　“那就好！”
<br/>　　秦不死闻言点头，略一沉吟道：“为了保险起见，我刚才也擒拿了十名试炼者，镇压在了空间宝器之中！”
<br/>　　“哦？”苏莫有些意外，没想到对方也有空间宝器！
<br/>　　如此最好，有了巫娴和十名巫族天才，到时候把握能更大两分。
<br/>　　随后，苏莫搜索了一番整个战场，找到了三十余名巫族试炼者的尸体，将他们的神丹和储物戒全部收取了。
<br/>　　巫族天才陨落了三百人左右，但大多数人的尸体都被轰成了渣，只有三十余人的尸身保留了下来。
<br/>　　“苏莫，我们接下来怎么办？”众人全部来到了苏莫的身边，司空炎出声问道。
<br/>　　苏莫闻言沉吟了片刻，道：“你们先回我的神图空间养伤吧！接下来我一人足以！”
<br/>　　现在苏莫的目的，基本上已经达到了，唯有没能斩杀或擒拿巫天御救出宏青璇，是一大遗憾！
<br/>　　不过，接下来众人也帮不上什么忙了，所以他便让众人全部进入虚界神图。
<br/>　　众人均是点了点头，他们对于苏莫的实力，都有信心，这些试炼者根本威胁不到苏莫的性命。
<br/>　　“诸位！”
<br/>　　苏莫目光如电，凝望着众人，深深的吸了口气，郑重的道：“待你们再次出来之时，我们已经离开了苍穹世界！”
<br/>　　苏莫声音铿锵有力，眸中带着绝强的自信。
<br/>　　众人闻言均是面带笑容，神情有些激动，想到能离开苍穹世界，去探索外界的广阔天地，没有人不欣喜。
<br/>　　少倾，苏莫将众人全部收进了神图空间中，他的身边只剩下了三人，便是蛮荒三妖。
<br/>　　“苏莫，你为何骗他们？”光头大汉瓮声瓮气的问道，通道之外有巫族尊者镇守，他们能不能活命，还是两说呢！
<br/>　　“不是骗，我只是给他们希望而已！”苏莫摇了摇头，心中暗叹，虽然他不知道自己能不能逃出去，但还是要给其他人希望。
<br/>　　他心中的压力非常大，因为一旦失败，死的人可不是他自己，还有苍穹神宫百万弟子，亲人、朋友、爱人、师尊。
<br/>　　可以说，他接下来的事情，将决定百万人的生死，他自然压力倍增。
<br/>　　他之前甚至想过，将所有人都留下，只身一人去尝试逃离苍穹世界。
<br/>　　但最后还是放弃了这个想法，其一是因为众人都想离开，其二是他也不想将众人留下。
<br/>　　众人留下或许可以活命，但最终还是会化为一杯黄土。
<br/>　　“苏莫，你接下来准备怎么做？“秦不死向苏莫问道。
<br/>　　“我要闭关一段时间，还请三位给我追踪这些逃走巫族天才的行踪。”苏莫沉声说道，他得到来三十余枚神丹，还有那些巫族天才的储物戒，准备尝试冲击武皇境九重境界。
<br/>　　“追踪？”秦不死闻言，眉头微皱道：“苏莫，我们的目的已经达到了，不必追踪了吧！”
<br/>　　在秦不死看来，他们就算追踪到了又如何，根本无法打败巫天御，最多也就多杀一些巫族试炼者而已。
<br/>　　光大大汉和黑衣中年人亦是面带不解之色，难道苏莫闭关，有把握突破修为境界不成？
<br/>　　“前辈，在通道开启之前，我们要杀光或者擒拿所有试炼者！”苏莫沉声说道，这些巫族之人他可不会放过。
<br/>　　“你有把握？”秦不死面带惊疑之色，那巫天御的实力，可是非常的恐怖！
<br/>　　“这次闭关，我有九成的把握，可以更进一步！”苏莫面带郑重之色，三十多枚神丹，均是武皇境九重和准帝级的神丹，还有获得的储物戒，应该差不多了。
<br/>　　秦不死闻言心中一震，苏莫的实力之强，不比那巫天御差多少，若是能突破到武皇境九重，的确有能力解决所有人。
<br/>　　“好，这些巫族之人的行踪，便交给我了！”秦不死点头道。
<br/>　　随即，几人商议了片刻，由秦不死去追踪巫族试炼者的踪迹，苏莫和光头大汉、黑衣中年人，暂回蛮荒圣山等待。
<br/>　　对于秦不死的实力，苏莫还是比较放心的，对方前去追踪，应该不会有多大的危险。
<br/>　　最后，秦不死闪身离去，苏莫与光头大汉以及黑衣中年人一起，向蛮荒圣山而去。
