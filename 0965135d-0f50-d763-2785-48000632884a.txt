　　“冯长老，你这是？”苏莫皱眉问道，他没想到对方居然真的要和他住一起。
<br/>　　“嘿嘿！苏莫，这院落中不是有很多的房间吗？足够我们两人居住了！”冯光嘿嘿说道，小眼睛中笑意黯然。
<br/>　　苏莫闻言无奈，这府邸中这么多的院落和房间，对方居然非要和他住在一起，这还是长老吗？哪有半点长老的威严，活脱脱一个无赖啊！
<br/>　　“好吧！”点了点头，苏莫没有理会对方，自顾自的找了一个房间安顿了下来。
<br/>　　房间中灰尘很厚，显然长时间无人居住了，苏莫衣袖一挥，一股气浪扫过，所有灰尘便都烟消云散。
<br/>　　随即，他在房间中盘算，如何才能见到青璇。
<br/>　　虽说青璇被抹除了记忆，但到底是什么程度，他还不清楚。
<br/>　　是只抹除了部分记忆，还是进入太阴族之前的记忆全部都被抹除了，点滴不剩，这还不好说。
<br/>　　当然，肯定是后者的可能性更大。
<br/>　　“青璇肯定是在内族！”苏莫低声自喃，他想要在太阴城中见到青璇的可能性，基本上可以忽略不计。
<br/>　　那就只能等到会武的时候了，七大势力核心弟子会武，想必青璇会来吧！
<br/>　　虽说此次会武，太阴族之人不会参加，但想必很多人会来观战吧！
<br/>　　青璇现在是太阴族的神女，应该不会错过这次会武。
<br/>　　咚！咚！咚！
<br/>　　就在这时，外面突然响起了沉重的敲门声，苏莫不用想也知道，是冯光来了。
<br/>　　随即，他起身打开了房门，将冯光让进了房中。
<br/>　　“苏莫啊！我们来聊聊！”冯光走进房中，面上笑意迥然。
<br/>　　“聊什么？”苏莫看到对方的表情，就知道对方找他没有好事，这个冯光长老，就是个奇葩！
<br/>　　“哎！老哥我在道宗，非常的落魄，平常也就处理一些跑腿的事情，比如这次会武！”
<br/>　　冯光叹了口气，面上露出落寞之色，继续道：“你和真贤长老关系不一般，那个……能不能在其面前美言几句，让我在宗内换个差事！”
<br/>　　道宗的普通长老，一般都有自己的职务，不同的职务之间，自然有好有次。
<br/>　　比如宗务堂的那名郑长老，虽然平常较忙，但油水丰厚无比，而冯光在宗门内，只能做些杂事。
<br/>　　“冯长老，我只是个小辈，哪有资格要求真贤长老，万一惹得他不喜，那我不就完了！”苏莫摇了摇头说道，他可不会去帮对方。
<br/>　　再说了，他刚加入道宗，承蒙真贤道圣帮助，怎么可能去要求对方这种事情。
<br/>　　最主要的是，他对于这位冯光长老的个人作风，很是鄙视。
<br/>　　“哎！既然你不愿意帮忙，那就算了！”
<br/>　　冯光闻听苏莫之言，轻叹一声，他也知道，现在想要让苏莫帮忙，怕是很困难，毕竟他和苏莫的关系还不熟。
<br/>　　不过，他已经下定决心了，以后较好苏莫，对他有利无害，说不定日后还能靠上真贤道圣这颗大树。
<br/>　　“苏莫啊，会武还有三天的时间，本长老带你到太阴城转转！”冯光轻笑道。
<br/>　　“还是算了吧，我还要静修，冯长老自便吧！”苏莫摇了摇头，拒绝了对方。
<br/>　　在太阴城中又不可能见到青璇，有什么好转的，他现在修炼才是第一要务。
<br/>　　“嘿嘿！苏莫，你有所不知，太阴族中可是有不少好地方，既然来了可不能能错过！”冯光笑着说道，他来到太阴族不下百次，对此城熟悉无比。
<br/>　　“什么好地方？”苏莫淡淡的问道，他并不感兴趣。
<br/>　　“首先推荐你去九重浮屠塔！”冯光道。
<br/>　　“九重浮屠塔，什么地方？”苏莫问道。
<br/>　　“这九重浮屠塔，乃是太阴族的炼器师，模仿阿弥星河的佛道宝器浮屠塔炼制而成，虽然只是仿制品，没有多少玄妙的能力，但是对于武者参悟奥义，有极大的效果。”冯光详细的介绍道。
<br/>　　“哦？”苏莫闻言眼眸一亮，他奥义的参悟严重滞后，若是这九重浮屠塔真有奇效，那倒是可以一去。
<br/>　　“走吧！我估计现在其他的弟子，都已经去了！”冯长老催促道，他也要进入浮屠塔。
<br/>　　此九重浮屠塔，虽然最适合武帝境的武者，但一般的武尊境武者也适用，他的修为才武尊境六重，效果还是很明显的。
<br/>　　一般，他每次来太阴族，都要到九重浮屠塔内待一些时间。
<br/>　　“嗯！”苏莫点了点头，既然有如此宝地，他自然不能错过。
<br/>　　随即，他便和冯光联袂离开了住处，走出了府邸，前往九重浮屠塔的所在地。
<br/>　　九重浮屠塔，在太阴城的北部，塔高千丈有余，就算相隔极远，苏莫也能看的清清楚楚。
<br/>　　“冯长老，你刚才所说的阿弥星河在哪里？”走在路上，苏莫向冯光问道。
<br/>　　“阿弥星河距离明虚星河很远，哪里号称佛星，全部都是修炼佛道之人！”冯光道。
<br/>　　苏莫闻言惊讶，这个世界居然还有佛道，不知和地球上的佛，有无相似之处。
<br/>　　两人大步而行，穿过一条条街道，没过多久便来到了九重浮屠塔的所在之地。
<br/>　　此塔耸立在一片巨大的广场中央，千丈的宝塔，占地方圆百余丈，塔身并无任何的宝光，通体呈灰白之色，古朴大气。
<br/>　　不过，此刻塔门之前的广场上，围拢了很多太阴族之人，嘈杂的声音此起彼伏，好像是里面有人产生了某种冲突。
<br/>　　“是吕罡、唐古和林皇他们！”苏莫一眼便看到，人群的中央，是一众道宗的弟子，不过人数并不多，约莫只有五、六十人的样子。
<br/>　　而在一众道宗的弟子对面，是一众黑人。
<br/>　　不错，是黑人！
<br/>　　这群人人数众多，不下一百余人，个个焦黑如碳，唯有一口牙齿雪白。
<br/>　　这群人都有一个特点，那就是体内蕴含爆炸性的火属性玄力，仿佛整个身体，都隐隐有黑色的火焰在燃烧。
<br/>　　黑炎族！
<br/>　　看到这群人，苏莫脑海中不由的想到了这个种族，这正是七大势力之一，黑炎族之人的特征。
<br/>　　“怎么回事？我们过去看看！”冯光见此脸色微变，他作为道宗的随行长老，自然要保护门下弟子的周全。
<br/>　　嗖！
<br/>　　见到道宗弟子和黑炎族之人起了冲突，他立刻加快速度，向前方飞掠了过去。
<br/>　　苏莫见此，亦是急忙跟了上去。
