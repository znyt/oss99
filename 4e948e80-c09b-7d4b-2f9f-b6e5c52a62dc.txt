　　历代的武斗大会，都是由北域之中的各大宗门轮流担任。
<br/>　　而这一次，正好是轮到了通天阁。
<br/>　　通天阁在北域之中，也是一尊庞然大物，财可通天，在北域的任何角落，甚至就是那虚空海，混元大陆之中，都有着通天阁的势力。
<br/>　　这通天阁，虽然没有自己培养武者，但是各方势力，错综复杂，可以说在整个北域。
<br/>　　也就只有北邙学院，地魔门这两大庞然大物，足以是跟通天阁抗衡，不过却也是轻易不会去得罪通天阁。
<br/>　　毕竟通天阁占据了北域大部分的生意，其中的一些供奉长老，也都是极为强大的存在。
<br/>　　这一届由通天阁负责举办，也的确没有任何人会心生不满。
<br/>　　三个时辰过去。
<br/>　　几乎武王城外的所有武者，如今都进入其中了，如今这武斗会场周围，也是占满了人。
<br/>　　这个时候，那尊通天阁老者，突然站起来，大声的开口道。
<br/>　　“这一次的武斗大会，开始！第一个环节，混战！现在诸位每个人手中的身份牌上，都会出现一个数字，按照其中数字，来分配小组。”
<br/>　　武斗大会的第一关，便是混战。
<br/>　　每年参加武斗大会的人，足足是有着上千人之多，但是上千人的擂台战，的确是极为麻烦。
<br/>　　更何况这人选之中，也的确是有着一些都是浑水摸鱼之人，这第一关混战，便是就跟曾经北邙学院的考核一样，先将大部分的人洗刷下去。
<br/>　　从这十人的混战之中，只有一人，可以晋级。
<br/>　　如此一来，便是足以洗刷下去大部分实力不足的武者。
<br/>　　这混战，楚风眠倒是不担心，以他的实力，度过这混战，绝对是没有问题，他现在的实力，只要不是碰到北域百绝榜上前十的选手，想要晋级都是毫无压力的。
<br/>　　唯有一点让楚风眠有些担心的是，万一是跟寒月漓分到一起的话，情况可就糟糕了。
<br/>　　楚风眠的计划可是要跟寒月漓一起，前往归墟天的，若是提前碰到，则就必须要有一人出局。
<br/>　　“现在开始，分出擂台，按照身份牌上的数字，进入擂台之中！”
<br/>　　通天阁老者，猛然开口道。
<br/>　　只见每一个武者的手中，都是紧握着一个令牌，这令牌便是参加武斗大会资格的象征。
<br/>　　在楚风眠手中的令牌上，缓缓的出现了一个数字，七十七。
<br/>　　这擂台，一共一百个，正好是足以让在场所有的武者，一起战斗。
<br/>　　楚风眠看到自己手中的数字，便是又瞥过头去，看了一眼寒月漓，寒月漓上面的数字是三。
<br/>　　这让楚风眠也是松了一口气，起码不需要在混战中就跟寒月漓碰到了。
<br/>　　“所有人，进入擂台！”
<br/>　　楚风眠刚刚舒了一口气，一声威严的声音，猛然响起。
<br/>　　在场的无数武者，便是纷纷催动身形，全部都落到了擂台之上去。
<br/>　　楚风眠看了一眼寒月漓，嘱咐道。
<br/>　　“小心一点。”
<br/>　　“你也一样。”
<br/>　　寒月漓轻声开口道。
<br/>　　他们二人的身形，便是一下子分开，分别便是前往的不同的擂台之中。
<br/>　　这每一座擂台上，都有着各自的刻印数字，楚风眠一眼便是找到了这第七十七号擂台，一下子站在了其中。
<br/>　　无数武者，纷纷走上擂台，这分组，也是逐渐清晰。
<br/>　　“这百绝榜第十五位的元极，居然是跟百绝榜第十七位的袁海分在了一起。”
<br/>　　一尊围观武者，猛然惊呼道。
<br/>　　就在这第四号擂台上，其中十个人中，居然是有着两位都是北域百绝榜前二十的武者。
<br/>　　这一下子便是吸引了无数人的注意。
<br/>　　北域百绝榜，一共有着一百个名次，在这武斗大会的混战中，很有可能便是会互相遭遇。
<br/>　　不过这种前二十名相遇的，倒是极为少见，一下子吸引了无数人的目光。
<br/>　　这便是真正开场，便是的巅峰对决，像是这样的情况，并不少见。
<br/>　　擂台周围，顿时无数惊呼声，议论声，争吵声纷纷响起，一时间人声鼎沸。
<br/>　　楚风眠的这一组，倒是并没有什么强者，只是有着一尊北域百绝榜第四十一位的武者，正是楚风眠之前所见到过的。
<br/>　　来自北邙圣域之中的核心弟子，古傲寒。
<br/>　　除了古傲寒，这其他八人，也全部都是北邙圣域的种子弟子。
<br/>　　这擂台之中的所有人，居然全部都是来自北邙学院。
<br/>　　“怎么回事？居然是十个北邙学院的人。”
<br/>　　“北邙学院这一次可真是倒霉，居然一开始就在内战。”
<br/>　　一些围观武者，看着擂台上的情况，不少人都是感叹了一声。
<br/>　　他们还没有察觉到几分不对，这种情况，虽然稀少。
<br/>　　不过北邙学院来参加这武斗大会的弟子，人数众多，分在一起，也不是一件大不了的事情。
<br/>　　可是楚风眠自己却是清楚，这绝不可能是一个意外。
<br/>　　看来他招惹上了那燕碧，这一次燕碧便是直接动用手脚，来对付楚风眠了。
<br/>　　“楚风眠，冤家路窄，没想到这里碰到了你！”
<br/>　　古傲寒装出一副极为惊讶的样子开口道。
<br/>　　不过他的语气之中，有的却只有戏虐，如今一看楚风眠便是知道，这古傲寒，显然是故意的。
<br/>　　这其他八名北邙学院的弟子，也全部都是听从古傲寒的命令，一下子便是凝聚法阵，显然是想要一起来对付楚风眠。
<br/>　　古傲寒大笑着开口道。
<br/>　　“碰到我们，只能算是你的运气不好，现在乖乖的滚下台去，念在我们同是北邙学院弟子的份上，就不对你赶尽杀绝了。”
<br/>　　“是吗？那让你们有些失望了，碰到了楚某，你们才是真的运气不好。”
<br/>　　楚风眠不以为然的开口道，嘴角尽是不屑。
<br/>　　“一群蝼蚁，联合起来也不过是蝼蚁罢了，龙党的教训，看来还没有提醒你们，看来你们要跟龙党一样的下场了。”
<br/>　　龙党的教训，便是龙未央陨落，龙党的强者，都被楚风眠一个人所轰杀。
<br/>　　现在的龙党，早已经是名存实亡，从北邙圣域之中除名了。
<br/>　　“大言不惭！”
<br/>　　古傲寒冷喝一声。
<br/>　　“既然你自己不要这个机会，全身而退，那你就死在这擂台上吧！”
<br/>(本章完)
