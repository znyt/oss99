　　第74章
<br/>　　飞虎寨，高飞豹暴跳如雷。
<br/>　　“咱以为咱有脸，可咱的脸到人家那里就是个屁股。”
<br/>　　“是呀，现在不是从前啦，杜老大不用靠着咱们吃饭了，人家眼里可就看不见咱们弟兄了。”
<br/>　　高飞豹咕咚咚灌口酒，嘴里把杜茂十八代祖宗都捎上了。
<br/>　　“住口，别像个娘们似的嘟囔。”
<br/>　　高飞虎喝了一声，他心里本来就不开心，高飞豹这番话恰似火上浇油。
<br/>　　“姓杜的最近是有些狂了。”高飞虎也忍不住骂了一声。江湖人要一个面子，这次他借出面子给程大雷平事，结果他的面子到杜茂那里……用高飞豹的话，真连个屁股都不是。
<br/>　　“大当家，杜茂未必是不给我们面子，可能是冲蛤蟆寨的缘故。”于虬然在一旁道：“蛤蟆寨最近的风头也是不小啊。”
<br/>　　“蛤蟆寨那边什么意思？”
<br/>　　“以程大雷的脾气，估计是要打的。”高飞豹冷哼一声：“人家可没大哥这样的好脾气。”
<br/>　　“打，他拿什么打！杜老大的本事我们也不是不知道，打也是找死。”
<br/>　　“大当家，不妨就让蛤蟆寨碰碰钉子。”于虬然道：“趁这个机会，正好用杜茂杀一杀蛤蟆寨的威风，这次吃了瘪，以后他们就知道老实了。”
<br/>　　……
<br/>　　蛤蟆寨，程大雷几人也正商议着这件事。
<br/>　　随着讨论，一个个问题便浮现出来。
<br/>　　秦蛮是蛤蟆寨第一战力，没有他百人敌的本事，蛤蟆寨不会有今天。可眼下秦蛮受了伤，虽不严重，但高手现争，必是方寸之间。如此，蛤蟆寨已无人能奈何徐神机。
<br/>　　况且，现争满城都是程大雷的通缉令，他是进不了城的。就算想方设法混进城，以他那两下子也没资格往杜茂身边凑。
<br/>　　本来大家听说徐神机的遭遇后，心里都憋了一股火，恨不得去与杜茂火拼。可看着眼下冒出来的问题，大家心里难免有些丧气。
<br/>　　“俗话说的好，忍字头上一把刀，忍得过去是英豪，依我看，忍了吧，忍了罢。”
<br/>　　“君子报仇十年不晚，现在的确还不是和杜茂动手的时候。”
<br/>　　面对强敌，众人无师自通的用上了精神胜利法。话是如此说，心里却始终有几分不痛快的。
<br/>　　纷纷议论声中，程大雷一直保持着沉默。周围的声音不自觉安静下来，大家的视线都落在程大雷身上。
<br/>　　程大雷睁开眼睛：“忍什么，我们是君子么，说什么君子报仇十年不晚，咱们就是一伙山贼，看不顺眼，就打他娘的。”
<br/>　　“可以咱们现在的施礼，的确……”黄三元说到一半，忽然一顿：“大当家莫非已有了办法？”
<br/>　　“都去准备吧！”程大雷一拍桌子，算是结束了今天的讨论：“这一次，我亲自进城！”
<br/>　　程大雷坐在山寨大厅中，秦蛮领过来一个个人与他看。
<br/>　　“大当家，真不用我陪你一起进城么？”秦蛮问。
<br/>　　“你和杜茂照过面，这次不方便再露面。”程大雷一边挑选着这次入城的人手，一边回答秦蛮的问题。
<br/>　　他这次的打算，只要和杜茂照过面的人一概不带。最后只选了黄三元，徐灵儿，林少羽，以及七个柳家的壮汉。
<br/>　　以这些人手，打杜茂是绝对不可能的，众人不知道程大雷心里打的是什么主意，也只好按他的安排行事。
<br/>　　眼下还有一个问题，就是程大雷进城的事情。
<br/>　　他让柳芷把马尾粘在他脸上，装出满脸络腮胡子的模样。对着水盆对照一番，相貌和从前自然大相径庭。可相熟的人，一看眼神就能认出来，程大雷还特意去飞虎寨逛了逛，结果没进门就被人一眼认出。
<br/>　　“程当家，胡子长这么快啊！”
<br/>　　程大雷是落荒而逃，这个法子显然是行不通了，如此进城冒得风险太大。
<br/>　　苦思冥想，程大雷忽然想出一个主意。他让山寨打了一口棺材，外面刷上黑漆，自己躺进棺材中试了试。
<br/>　　徐神机一帮人大呼小叫：“大当家，这万万不行，给山寨招霉头啊！”
<br/>　　“有什么不行的。”程大雷从棺材里跳出来：“破除封建迷信从我做起。”
<br/>　　在一个风和日丽的下午，一队人来到落叶城城门前。
<br/>　　“站住，干嘛的！”持着长矛的守军呵斥道。
<br/>　　一老者走出来，他身上带孝，手里举着白番。
<br/>　　“军爷，我家老爷死在外面，我们拉他回来下葬，入土为安。”
<br/>　　马车上拉着一具棺材，守军刚看见便皱起眉头：“真是晦气，快走！”
<br/>　　“多谢军爷，多谢军爷。”老者忙点头，驱着马想进城。
<br/>　　“慢着！”
<br/>　　一留着两撇小胡子的卫兵突然喊了一声，他目光上下打量着这帮人，最后目光落到那棺材上。
<br/>　　“现在山贼作乱，谁知道你们是不是山贼假扮，想要混入城。”
<br/>　　老者忙道：“军爷开恩，我们都是本分人家。”
<br/>　　“别废话，把棺材打开，我要查看一下。”
<br/>　　众人皆是一激灵，下意识护在棺材前。老者摆手安抚下众人，道：“军爷开恩，老爷死在外面，不能开棺惊动亡者呐。”
<br/>　　说着，一袋铜板已经递了过去。小胡子用手掂了掂，如此脸上才露出笑容。
<br/>　　“死者为大，我们也不想惊动亡者，只是最近山贼太猖狂，不得不小心，好啦，你们进去吧。”
<br/>　　“多谢军爷，多谢军爷。”
<br/>　　老者忙作揖，一伙人如此才混入落叶城。他们刚走，城门守军就围了上来，忙着分钱。
<br/>　　穿过城门大街，拐入小巷，又不知行多久，从后门拐进了湖底捞后院。
<br/>　　“喂，你们干嘛的，赶快出去！”
<br/>　　小蝶看到这帮人拉着棺材拐入院子，以为又是杜茂找人过来捣乱。苏樱闻声也从屋里出来。
<br/>　　咔嚓，咔嚓……
<br/>　　棺材发出古怪声响，小蝶和柳芷错愕的看着这一幕，忽然见一只手从棺材里伸出来。
<br/>　　“妈的，可闷死我了。”
<br/>　　紧跟着一道人影从棺材里跃出来，一跃站在柳芷面前。湖底捞的厨子、小二此刻都弯下腰，口中齐声道：“大当家。”
<br/>　　“收了，收了。”
<br/>　　程大雷摆摆手，郑重其事冲苏樱作了一揖。
<br/>　　“苏姑娘，好久不见。”
<br/>　　
<br/>(本章完)
