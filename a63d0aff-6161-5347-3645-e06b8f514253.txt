　　慕清霄对于草谷的一番警告，可并不是在开玩笑，语气中满是严肃。
<br/>　　《朱子率》是当初，在斗破苍穹位面时，药尘传授给他的，其上记载了斗破位面九成的丹方，更有他一生的炼丹经验。
<br/>　　甚至，关于炼丹术上细节也都叙述的很清楚。
<br/>　　由于仙剑三位面中，炼丹道统几乎断绝，哪怕六界中许多修士都有成为炼丹师的资格，却无人传授。
<br/>　　一旦《朱子率》的存在暴露，六界将会掀起腥风血雨，一本能够让修士成为九品炼丹师的秘典，足以让任何修士趋之若鹜。
<br/>　　暂且不说其他修士，估计蜀山掌门，包裹各位长老都会动心。
<br/>　　草谷性格比同龄成熟许多，将慕清霄的警告记在心上：“长老哥哥，连清微爷爷也不能告诉吗？”
<br/>　　“不错，将《朱子率》传给你，你也算是我半个弟子，记住，是任何人都不能告诉。”
<br/>　　说罢，慕清霄正色道：“若是让我知晓，你私自将上面的东西外传给他人，我会亲自将东西收回来。”
<br/>　　慕清霄所说的收回，可不仅仅是将《朱子率》收回，以他的能力，篡改或者封印记忆在简单不过。
<br/>　　闻言，草谷娇小的身躯一震，脸色微微一白，小鸡啄米般的点着小脑袋。
<br/>　　因为她刚才敏锐的感觉到，慕清霄在说上一句话的时候，语气与平日里的温和完全不同，充满冷酷……
<br/>　　其实，慕清霄如此警告，何尝不是在保护这天真无邪的小丫头。
<br/>　　在强者为尊的世界中，没有同情与怜悯，弱者怀有至宝本就是罪过。
<br/>　　“盘坐下来，这东西也算是我对蜀山派的弥补吧。”
<br/>　　话语刚落，不知何时慕清霄手中就出现一个玉瓶，玉瓶中有紫色灵药流转。
<br/>　　“将它喝下去。”
<br/>　　望着慕清霄递过来的玉瓶，草谷歪了歪小脑袋，但还是乖巧的盘坐下来，接过玉瓶，将其中的灵药一口喝了下去。
<br/>　　“平心静气，运转心法，将灵药炼化掉。”说话的同时，慕清霄食指与中指已经抵在了草谷的玉额上。
<br/>　　这一瓶灵药，乃是他当初在斗破苍穹位面魔兽山脉，紫晶翼狮王的东西中获得的伴生紫晶源。
<br/>　　其中一瓶给了小医仙，两瓶给了彩鳞，现在的这一瓶也是最后一瓶，炼化伴生紫晶源，修士有几率继承紫晶翼狮王的兽火。
<br/>　　在仙剑三位面中，丹火不易凝聚，小丫头现在也不具备那样的能力。
<br/>　　小医仙与彩鳞都没有继承紫晶翼狮王的兽火，至于草谷，那就要看她的造化了。
<br/>　　不过，伴生紫晶源中的能量太过炙热狂暴，根本不是草谷区区一个练气修士能够吸收的。
<br/>　　强行喝下去，只会令她爆体而亡，不过有慕清霄护法就不会出现这样的情况。
<br/>　　当草谷将整瓶伴生紫晶源喝下去后，身躯就变得赤红一片，小脸蛋红彤彤的，甚至有颇高的热气自她嘴中喷出。
<br/>　　旋即，萌萌的小脸上就浮现一抹痛苦之色。
<br/>　　慕清霄双指触及玉额，阴阳灵气不断灌入她的体内，将她的经脉全部护住，引导伴生紫晶源淬炼她的身躯。
<br/>　　随着阴阳灵气护住经脉，草谷小脸上的痛苦瞬间消失。
<br/>　　五心向天，心法运转间，伴生紫晶源渐渐融入她的血肉与骨骼中，同时草谷的气息也渐渐开始攀升。
<br/>　　练气七重。
<br/>　　练气八重。
<br/>　　练气九重。
<br/>　　筑基一重……
<br/>　　在伴生紫晶源融入她体内的同时，小丫头的气息也不断攀升，一举碾碎瓶颈，迈入筑基，最后在筑基二重停了下来。
<br/>　　就在这时，慕清霄嘴角牵起一抹笑意，道：“幸运的小家伙。”
<br/>　　只见一缕紫色的火苗在草谷的玉额上袅袅升起，火焰一出，凉亭中的温度就微微抬高了一些。
<br/>　　在慕清霄的威压下，区区兽火自然不敢放肆，如同乖巧的绵羊与小丫头合二为一，往后也会为她所用。
<br/>　　待能量完全炼化，兽火熄灭，草谷则倒在了慕清霄的腿上，小脸上挂着恬静的笑容，沉沉睡去。
<br/>　　将小丫头放平，检查了下她各处经脉，经脉比先前要宽阔了些，甚至堵塞的经脉也被慕清霄全部打通。
<br/>　　小丫头虽不是灵体，但有炼丹术辅助，修炼速度就算比不上徐长卿这种命运之子，但也绝不会慢太多。
<br/>　　取出一件白袍为她盖上，慕清霄苦笑着摇了摇头，真有种当保姆的感觉。
<br/>　　在过不久后，慕清霄就会离开，将草谷打造成一名炼丹师，并且给她兽火，也算是弥补对蜀山的亏欠。
<br/>　　毕竟，他来到蜀山可是有目的的。
<br/>　　只要中途不出意外，草谷成为九品炼丹师就是板上钉钉的事情，甚至有机会迈入仙品。
<br/>　　不过，这都要看她自己的造化，别人无法左右。
<br/>　　用一些东西来换一名九品，甚至更高的炼丹师，礼尚往来，蜀山怎么也不算亏。
<br/>　　看了眼沉睡的小丫头，慕清霄并没有打扰她，拿出令牌，离开凉亭，他准备去一趟真武殿。
<br/>　　真武殿存着千百年来，蜀山的仙法与收录的仙法，进去一趟，估计能赚到不少的掠夺积分，慕清霄当然不会放过这样的机会。
<br/>　　出了天璇宫，慕清霄就一路向西走去，真武殿就在西侧的天权峰上，由和阳长老掌管。
<br/>　　真武殿，玄气殿，元神殿，律德殿分别由苍古、净明、幽玄、和阳四位长老执掌，坐落在四座山峰上。
<br/>　　约莫十分钟后，慕清霄就来到天权峰下。
<br/>　　天权峰上的真武殿，也是蜀山禁地之地，那里储存着蜀山派与各派的修仙秘法，哪怕是出师弟子都没资格进入其中。
<br/>　　甚至一些没有实权的长老也无法进入其中，慕清霄正好不算在内。
<br/>　　他拥有身份令牌与通行证，整个蜀山就没有他去不得的地方。
<br/>　　来到天权峰广场上，烈阳高照，老远就能看到一座宫殿坐落在灵气袅绕的仙湖旁，宫殿的匾额上有着龙飞凤舞的两字。
<br/>　　天权！
<br/>　　真武殿的存在历史悠久，在蜀山开山祖师太清真人的时代就已经存在，除了妙丹堂，比任何一个殿都要久远。
