　　慕清霄并不理会周围的钟乳，身形最终停在了这支庞大钟乳的低端，也就是连接山洞顶部的地方。
<br/>　　淡淡的荧光自钟乳内渗透而出，将其照映的犹如水晶一般，极其绚丽。
<br/>　　随后，慕清霄从储存空间中取出玉瓶，凝出一道剑气，沿着钟乳底部，轻轻的刺了进去。
<br/>　　锋利的剑气，犹如切豆腐一样，没有受到一丝的阻隔，细微的切割声响起，剑气就刺进了钟乳中。
<br/>　　慕清霄动作很轻，剑气在钟乳内旋转一圈，细微的切割上绕耳不绝。
<br/>　　“咔...”
<br/>　　最后，当剑气旋转一圈后，钟乳碎片落下，其上就出现一个拳头大小的洞口，其中散发出一阵乳白色的光晕。
<br/>　　视线顺着洞口望去，钟乳内部，悬浮着一团翡翠色的粘稠液体，液体犹如有灵性一般，在钟乳中央流淌着。
<br/>　　以钟乳中心范围，紧致有序的流淌，倒不如说是在压缩，这一团翡翠色液体，便是由多年累积而出的精华。
<br/>　　精华之中，蕴含着一股庞大的精纯能量，绝对比地面凹槽中地心淬体乳浓郁了十倍不止。
<br/>　　“如此精纯的能量，是真正的地心淬体乳，地面凹槽中的，都是从这里流逝出去的，相当于稀释过的地心淬体乳。”
<br/>　　说着，慕清霄含笑，道：“运气真是不错，只是对我们来说，作用并不大，但对实力在斗皇以下的，效果极佳。”
<br/>　　整个斗气大陆上，只有极少数的人知道‘地心淬体乳’的事，精华所在是藏于钟乳与大地交界处。
<br/>　　凹槽中稀释过的液体，或许也算是天地灵物的一种保护自身的障眼法吧。
<br/>　　若是一般人，就算碰巧进入这里，恐怕也只会将凹槽中稀释过的取走，而将最宝贵的东西遗留下来。
<br/>　　精纯的地心淬体乳虽包含庞大的能量，但本身却极其的脆弱，只有最为温润的玉器，方才不会将它损坏。
<br/>　　若是使用铁器，或者其他器具，只要稍微沾上一点，这团不知道凝聚了多少年的地心淬体乳，恐怕会当场化作一团毫无作用的液体。
<br/>　　这些信息，都是清楚记载在《朱子率》中，其中，关于大陆上的各种物品，都有相关的记载。
<br/>　　这本犹如炼药师宝库的典籍，特别是对灵药，记载的极其清楚。
<br/>　　回过神来，慕清霄将储存空间中的玉瓶全部取出，随手一卷，中央的精纯地心淬体乳，就一点点的灌进其中。
<br/>　　慕清霄并没有将地心淬体乳全部取走，装满整整十个玉瓶后，其中也只剩下了少量。
<br/>　　经过长年累月的积累，才凝聚成这团精纯的地心淬体乳，若是断其根本，就好比如斩草除根。
<br/>　　如此一来，这根庞大的钟乳恐怕也会奔溃，往后再也不可能凝聚出这种天地灵物。
<br/>　　做人不能太贪心，整整十瓶已经不少，给自己的女人用也完全足够了。
<br/>　　望着玉瓶中，犹如活物的精纯液体，慕清霄微微一笑，便全部装进储存空间中。
<br/>　　此时，他的储存空间中，堆积着各种灵药，淬体液，清灵丹等等......
<br/>　　不管是丹药，或者高阶灵药都有一大堆，随便拿出去一些，都能令无数人眼红。
<br/>　　这些，他自己已经用不上，都是给自己女人准备的，地球上灵气稀薄，几乎没有任何资源可言。
<br/>　　她们修炼起来，速度极慢，但根基却非常牢固，到时候用灵药淬体后，便能令她们的实力飞速提升。
<br/>　　而且，不会留有任何隐患。
<br/>　　灵药收取完毕，慕清霄轻轻的将钟乳碎片合了上去，虽已经不能恢复原本的模样，但也无关紧要。
<br/>　　离开山洞顶部，来到下方，凹槽中稀释过的液体，虽然不如刚才的精华，但也能派上用场。
<br/>　　将早已准备好的数十个玉瓶取出，灵气再次卷，其中稀释过的地心淬体乳就灌入玉瓶中。
<br/>　　将灵液全都装进储存空间中，慕清霄微微一笑，道：“彩鳞，我们走吧，回内院看看情况。”
<br/>　　闻言，彩鳞淡紫色美眸望着他，一脸担忧。
<br/>　　“你吞噬了陨落心炎，继续留在内院，只会徒增危险......”
<br/>　　话语刚落，慕清霄走上前去，替她捋了捋额前的青丝，道：“没事，真要对付我，就不会等到现在了。”
<br/>　　“何出此言？”彩鳞美眸中闪过一抹困惑。
<br/>　　“迦南学院可不像表面上这么简单，藏书阁你应该知道吧？看守藏书阁的两位老者，都是斗宗巅峰强者，说半步斗尊也不为过。”
<br/>　　彩鳞淡紫色瞳孔微微一缩，藏书阁她当然清楚，慕清霄进入其中的时候，她也在场，只不过躲在衣袖中罢了。
<br/>　　当时，她就觉得，看不出两位老者的深浅，没想到居然是斗宗巅峰，甚至一只脚踏进斗尊层次的超级强者！
<br/>　　斗尊强者，哪怕整个斗气大陆，都称得上超级强者的存在。
<br/>　　“所以说，在他们面前，哪怕气息收敛的在完美，都会被看出破绽，更何况，外院还藏着一名二星斗尊。”
<br/>　　见彩鳞冷艳绝美的脸蛋上，露出一副惊愕的表情，慕清霄嘴角也牵起一丝苦笑，终究是自己太弱了。
<br/>　　“陨落心炎的事情，不必担心，在进去前，我就已经想好了退路，不会有危险的。”
<br/>　　...
<br/>　　...
<br/>　　与此同时，内院，天焚炼气塔最底层，岩浆池前，一名老者负手而立，双眼中透着一抹沧桑。
<br/>　　“那小子来历真是神秘，虽取走了陨落心炎，却将异火种子留了下来，以供练气塔运行，倒也算是帮了学院的大忙。”
<br/>　　旋即，老者神色凝重的看了一眼岩浆池，转身看向站在身后的苏长老与庆长老。
<br/>　　“此事无碍，练气塔照常运行，封印也不需要在施加，至于神秘人，随他们去吧。”说着，老者微微一笑，身影就凭空消失。
<br/>　　老者离去，苏长老与庆长老面面相觑，要知道，神秘人如此放肆，入侵学院，打破封印，简直就是在挑衅。
<br/>　　但院长不在学院，老者就是最有话语权的人，难道真就这么算了？
<br/>　　今晚的事情，简直扑朔迷离，令得两人莫名其妙。
<br/>　　瞥了一眼岩浆池，苏长老叹息一声，道：“走吧，既然他都这么说了，那就只能先这样了。”
