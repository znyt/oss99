　　第660章
<br/>　　“开什么门，这些人鱼龙混杂，放进来祸害人怎么办。”
<br/>　　程大雷把阿喜的话堵了回去。目光看着城下的人群，但不开门也不是个办法。城下人越聚越多，人人口中嚷着程大雷的名字，终归要想个解决的法子。
<br/>　　程大雷沉思片刻，已经有了主意，口中道：“开门，让张肥带兵出关。”
<br/>　　阿喜一头雾水，刚才程大雷不是说不许开门么。当然，命令已经下达，他也没有不照办的道理。立刻传令下去，告知张肥，让他的张字军集结，准备出关。
<br/>　　这当然需要一定的时间，这时候城门下聚集的人越来越多，人群一多，智商便成比例下降，胆子却越来越大。
<br/>　　大家都来得不太容易，跋山涉水，千山万里，而这些人中，也是作奸犯科者居多。当然，对于一群饭都吃不饱的人来说，也无所谓善恶。
<br/>　　大家来得这么不容易，程大雷却连门都不开，这明显是不给面子么。
<br/>　　群情激奋，大家吵个不停，委实说，这已经影响到程大雷的名声。当然，是在恶人中的名声。
<br/>　　这时候，张肥的张字军已经集结完毕，在城外诸多人已经嚷得无法无天时，城门突然打开，两队人马涌出城门。
<br/>　　乱哄哄的声音嘎然而至，大家都闭上了嘴巴，哑口无言的盯着这群人马。
<br/>　　人人身穿黑色披甲，神情肃穆，当真是人马精神，为首一员大将身披黑铁甲，座下乌骓马，手持丈八蛇矛，在马上哇呀呀大叫。
<br/>　　“谁，是谁？”
<br/>　　在程大雷手下的大将中，张肥是最暴躁的，他手下的张字军也同样继承了这一点，性格暴躁，无法无天。与总管兵权的刘悲惹了不少麻烦。
<br/>　　不过，无论如何说，都是有些纪律性的。张肥只出动了五千人马，但集结在一起后，当真是莫大的威势。外面的这些流贼乱匪，平时最害怕的便是官兵，而眼前这群兵，明显比官兵更加凶恶。
<br/>　　这时候，程大雷从城头上一跃而下，长袍猎猎而舞，宛若云上之仙。程大雷本身模样不如何，走的是豪放路线，但此刻一跃而下，竟然有几分出尘潇洒的气概。
<br/>　　双脚稳稳落地，双手轻轻一合，摆摆手道：“诸位，诸位，请听我一言，你们不是想见在下么，在下到了。”
<br/>　　程大雷脸上的笑容，多多少少减轻了这些人身上的压力。有几个身强力壮的干笑几声，上前打招呼。
<br/>　　“程当家，我们是从扬州来的。”
<br/>　　“程当家我们是青州云砀山的人。”
<br/>　　“程当家……”
<br/>　　“蛮好，蛮好。”程大雷虚伪的笑着，摆摆手道：“诸位的来意我已经清楚，既然看得起我程某人，大家日后便是兄弟。我已给诸位安排好地方，但不是在这里，而是在朔方城。”
<br/>　　派张字军出城，一就是为了吓唬吓唬对方，让这些人老实些。二也是为了显示蛤蟆城的实力，只有兵强马壮，方才能保护好大家。
<br/>　　当没有能力反抗时，人们往往会选择屈服，所谓人性，从来如此。
<br/>　　程大雷发下话来，大家又是投奔于人，自然而然，没有不听话的道理。关键是张肥这黑脸大汉杵在一边，大家也不敢不听话。
<br/>　　程大雷让这些人列队前往朔方城，由张肥护送。刘悲看好蛤蟆城，但凡再有人来，直接令他们去朔方城，不要给此地添麻烦。
<br/>　　人群络绎不绝，前赴后继，在长路上连成一道线。
<br/>　　程大雷快马加鞭，与徐神机一起回到朔方城。赶忙吩咐和珅，朔方城的人出动，带着干粮到路上接人。
<br/>　　这些人奔波千里，一路风餐露宿，诸多人已不知多少日子没吃过饱饭，连饿带病，莫最后一程路走不完，累死在半路上。
<br/>　　这件事中，要算和珅最是开心。朔方城最缺的便是人手，只要有了人自己才好做事，自己的权力才更有价值。
<br/>　　当然，这群人鱼龙混杂，中间什么人都有，想要真正收付他们的心，还需要一定辛苦。
<br/>　　但和珅最是不怕这种辛苦，反而有几分乐在其中的意思。他马不停蹄，来回奔波，等将这批人接回城后，再安排他们的住宿与饭食。
<br/>　　忙忙碌碌，需要颇多时间，晚上鱼字军和张字军两支队伍巡逻，免得其中一些人生死。
<br/>　　当真是找到十几个不安分的主儿，在分房和饭食的问题上发生矛盾，有某个江北来的大汉，持刀杀人。
<br/>　　被捉住后，在城主府门前，将这十几个人挨个问斩。
<br/>　　入了夏天，天气一日比一日炎热，黄沙滚滚，人头也同样滚滚。程大雷捂着口鼻，坐在城主府中饮茶。
<br/>　　“刚进城便杀人，有些不太地道呐。”
<br/>　　“大当家，这波人来路不正，也是用人之际，才收留他们。对于他们，当是用重典。”和珅道。
<br/>　　和珅的话也不错，他今天杀完人已经对这些人说了：不管你们是什么人，不管你们是什么来路，既然到了这里，便按规矩做事。
<br/>　　假若是程大雷主持这件事，估计也会这样做，但多多少少，心里还有些不舒服。
<br/>　　至于徐神机，他颇有些闭关悟道的意思，坐在那里喝茶，对这件事不是很关心。
<br/>　　三人坐在一起饮茶，程大雷心中忽然一动，想起自己的困扰。
<br/>　　正好今天是个机会，程大雷便在饮茶的间歇，将柳芷的事一五一十说了。
<br/>　　那和珅果真不愧是个人精，程大雷只说了三言两语，他便立刻明白了。
<br/>　　听罢之后，目光看着程大雷，眼神有些奇怪。
<br/>　　“大当家，这件事你不会真的不明白吧，还是……”
<br/>　　和珅在想，程大雷是否在考自己，但仔细想象，这个问题的难度也忒小了些。
<br/>　　程大雷干笑两声，然后问：“究竟是怎么会事？”
<br/>　　如此看来程大雷是当真不明白，和珅仔细想想，觉得这一点还是颇奇怪的。
<br/>　　他简简单单便将问题与程大雷挑明，本不是太复杂的事情，程大雷听了几句，便大概明白过来。
<br/>　　可关键是，目前自己与苏樱之间都是一团浆糊，谁料又添了柳芷这件麻烦事。
<br/>(本章完)
