　　“是在等后面的丧尸大军。”段江伟纠正鲁初雪的话，表情也严肃起来：“亦或者，那只有智慧的丧尸头领，恐怕还在观望那些猎者的战力水平，从而推断这里需要多少数量的丧尸来攻破。”
<br/>　　此话一出，四座皆惊。
<br/>　　“我们这根本不像是在跟丧尸战斗，而是像跟一方战术高超的军队。”一名段氏高层说着，面露震惊：“丧尸的智慧，真的可以这么高？”
<br/>　　段江伟眯起了眼睛：“你们难道忘记了当初倾巢之战时，被屠杀一空的南战区后勤大营？那时候在营地里的可不仅仅是后勤人员，还有来自四大基地的战团人数上万！”
<br/>　　随着段江伟的话音落下，很多人都神情恍惚起来，那一场战役的受害基地虽说大多数最后都叛变，其中一座基地更是被楚涵率领着黑芒直接给掀了，但那次尸潮冲击的事件，却让很多人事后都心惊不已。
<br/>　　这也是人类第一次直观的感受到，丧尸一旦拥有智慧，会是一件多么可怕的事情。
<br/>　　“战前准备继续！”段江伟开始下达命令：“发出消息让那些猎者都回来，小心被埋伏，此外联系上南部联盟军，让他们加快速度过来！”
<br/>　　“是！”鲁初雪正色点头，而后又忽然问道：“狼牙战团？”
<br/>　　段江伟沉默了一秒，问道：“狼牙战团眼下到达了哪里？”
<br/>　　一人起身回答道：“反馈信息来看，他们已经走了一半的路程，再有一周时间就会达到西南方位的预定地点。”
<br/>　　段江伟点点头没有多说，心中却对楚涵再次佩服不已，明明大尸潮一战中就昏迷不醒，狼牙战团却在战役结束后没多久便集体向华夏的最西南方位走，说明楚涵早已布置好狼牙战团一切行动流程，哪怕他人不在，也影响不了大局。
<br/>　　短暂的会议结束，众人都纷纷行动起来，开始向分散在边境线的猎者们传递消息。
<br/>　　鲁初雪在结束了布置之后再次来到段江伟面前，开门见山的问道：“上将，狼牙战团为何要去华夏最西南处？那里都是高原地带，丧尸群既然有智慧型的首领，必然不会选择从那里走，就像第一次大尸潮的地点一样，那些来自南亚的丧尸，都是绕了一个弯来到了这里。”
<br/>　　段江伟摇摇头，道：“都是楚涵的安排，可能他想翻过那座高山直接跟埋骨之地的丧尸开打呢？”
<br/>　　鲁初雪噎住，对段江伟这种随意的回答没放在心上，直接进入埋骨之地面对十几亿的丧尸？
<br/>　　开什么星际玩笑！
<br/>　　在段氏基地的消息发出之时，苏行的建筑工程队也在火速搭建边境线的防御，段氏基地的改造已经完成，他们早已延着这一边境线朝西面移动，一路上路径的所有中、小型基地，甚至文明时代遗留下来的城镇或村庄，都毫无遗漏的在苏行工程队的手中经历了一番改造。
<br/>　　这一批移动的工匠，不断用巧夺天工的技术，将所有可以利用的地带都改造成最有利于人类一方的战区。
<br/>　　紧跟这批工程队其后的，是狼牙后勤部，与段氏基地负责整条战线的后勤部不同，狼牙的后勤部又老高带领，专门负责这一路上的战区资源分配。
<br/>　　他们的行动快的惊人，不断乘坐着狼牙的专属直升机，往返于段氏基地与这些战区之间。
<br/>　　驻扎在段氏基地的狼牙其他部门，更是疯狂制造着战役必需品，各式新型武器，各种储备能量，一大堆不断新制造而出的药剂，就这么源源不断的运输到战线上的一个个点上。
<br/>　　于是在南部联盟军到达之前，狼牙的人就已经将一切安排妥当，一条在华夏最南方，由东向西延伸的战线就这么一点点建造而起。
<br/>　　只是在所有人都忙于各项工作的时候，多则内容大相径庭的消息，逐渐传到了段氏高层的耳中。
<br/>　　鲁初雪在这一相同消息出现次数到达五份的时候，就立即向段江伟汇报。
<br/>　　“上将！前方来报，已有五批猎者小队失联！”她有些紧张，声音带着颤抖：“并且从目前的形式来看，失联的猎者小队数量还在不断增加，只是反馈的消息还未来得及传递过来。”
<br/>　　“已经失联了？！”段江伟瞳孔一缩：“今天上午我们才做过简单会议，消息什么时候发布出去的？有多少猎者在往回赶？”
<br/>　　“会议结束后立即就令人前去传递了，已有百名不到猎者在回程路上。”鲁初雪说着，咬了咬牙不甘心道：“但据猎王说，这次问询而来的猎者数量，有五百人之上。”
<br/>　　“也就是说其余的四百人，不是失联，就是还没传达到消息？”段江伟忽然想到了某个可怕的可能性。
<br/>　　鲁初雪点头，又摇了摇头：“有些猎者小队走的有些远，我们的人寻找需要一些时间，但可以确定的是，这些猎者小队这么久都没找到传递消息，一定有问题！他们并不是狼牙三战队之一的神隐，也不是侦查和反侦察都逆天的侦查队，段氏战团的人出动寻找，没有理由这么久了还没找到。”
<br/>　　说着鲁初雪有些心凉：“最大的可能……”
<br/>　　“就是死了。”段江伟冰冷的声音接上，瞳孔里毫无温度：“或者说是被吃了。”
<br/>　　鲁初雪身子一震，颤抖道：“这些猎者都是自发前来，不是通过猎者联盟接的任务，他们没有收取一分钱，各自形成小队在四周猎杀靠太近的丧尸，方才使得段氏基地周围一片安宁。”
<br/>　　“他们不是军人，更没有做这些的职责和义务，可却……”
<br/>　　段江伟摆了摆手，打断鲁初雪的话：“跟猎王打个招呼，让他有心理准备。”
<br/>　　“是。”鲁初雪不再多留，转身出去传达段江伟的话。
<br/>　　……
<br/>　　在段氏基地周边大批猎者下落不明的时候，远在猎者联盟总部的那处死火山冷潭中，冰冷清澈的潭水则是已经一片血红，周围的空地上不断爆发着战斗，更不断有浑身是血的躯体被抛至这处冷潭里，让这里的潭水血色更浓。
