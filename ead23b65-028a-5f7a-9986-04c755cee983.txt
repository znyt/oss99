        连星儿看了郎安一眼，没好气地接过手机，起身进了屋，并反锁了房门。
<br/>
<br/>        其实，她并不讨厌郎安。
<br/>
<br/>        只是郎安上次绑架她的事情，给她造成了一丝心理阴影。
<br/>
<br/>        大概，就是心有余悸的那种。
<br/>
<br/>        连星儿抱膝坐在床上发起呆来。
<br/>
<br/>        对于明司寒，她忽然有些患得患失的感觉。
<br/>
<br/>        难道是因为异地恋的缘故吗？
<br/>
<br/>        连星儿黯然神伤地愣了许久，直到手机铃声响起，她还以为是明司寒打来的电话，结果让她白开心一场。
<br/>
<br/>        “喂，千凝。”连星儿无精打采地接听电话。
<br/>
<br/>        顾千凝或许是人逢喜事精神爽，完全没有察觉到连星儿那不开心的情绪，而是自顾自地说道：“晚上，我和子晋，请你们吃饭，就是学校外的那家新开的餐馆。”
<br/>
<br/>        “嗯，好。”连星儿深吸了口气，假装很开心地应了声好，随后又跟顾千凝聊了几句，才挂的电话。
<br/>
<br/>        到了约定的时间，连星儿赶到了顾千凝定好的餐馆中。
<br/>
<br/>        在座的不仅仅只有她们几个女生，还有明司寒寝室里的那三个男生。
<br/>
<br/>        当他们齐刷刷地将目光落向连星儿身后时，连星儿也怔愣地回过头去，看到郎安也跟来了，一时间又不知道该怎么解释了。
<br/>
<br/>        “服务员，再加一套碗筷和椅子过来！”顾千凝连忙伸手跟餐馆里的服务员打招呼，这才化解了连星儿的尴尬。
<br/>
<br/>        连星儿入座后，郎安也跟着在连星儿的身旁坐下。
<br/>
<br/>        邢乐游看了看连星儿，又看了看郎安，忍不住调侃道：“咱们寒哥这出国没几天，你连大小姐就开始……唔——”
<br/>
<br/>        小腿上的胫骨传来一阵痛感，直接逼停了邢乐游的话。
<br/>
<br/>        张子晋在桌底下收回自己的脚，并向邢乐游使了使眼色，示意他不要乱说。
<br/>
<br/>        连星儿没好气地瞪了邢乐游一眼，底气十足地说道：“你不用拐着弯来说我，我连星儿可不是那种花心的女孩子。”
<br/>
<br/>        “你确定你不花心？”邢乐游莫名其妙地就跟连星儿杠上了，“你不花心，带着这小白脸来做什么？你——”
<br/>
<br/>        “哐”地一声，一把狼型图腾的匕首直接立在了餐桌上。
<br/>
<br/>        邢乐游的话戛然而止。
<br/>
<br/>        郎安伸手拿回匕首，极其冷酷地说道：“我是大小姐的贴身保镖，你再敢多说大小姐一句不是，试试？”
<br/>
<br/>        邢乐游吓得咽了咽口水。
<br/>
<br/>        张子晋为了缓和气氛，连忙笑着打趣道：“连同学，乐游跟你开玩笑的。”
<br/>
<br/>        “那我就当他是在开玩笑好了！”连星儿落落大方地笑了笑，丝毫不跟邢乐游计较。
<br/>
<br/>        这餐饭，虽然大家有说有笑，但是，连星儿却一直在假装自己很开心。
<br/>
<br/>        饭后，连星儿找了个借口，跟张子晋借了手机，给明司寒打了通电话。
<br/>
<br/>        她是一个人走到餐馆外去打的，郎安依旧跟在她的身后。
<br/>
<br/>        这回电话不仅打通了，而且还被接通了。
<br/>
<br/>        “子晋，你找我？”明司寒的声音，慵懒地从手机听筒里传出。
<br/>
<br/>        连星儿忍不住凶巴巴地喊了声：“明司寒！”
<br/>
<br/>        “星儿？！”或许是那边，明司寒也未料到打电话过来的人是她连星儿，所以，刚开始时，声音显得有些局促。
<br/>
<br/>        “你不接我电话，也不回我电话，是几个意思？”连星儿恼火地质问道。
<br/>
<br/>        明司寒的声音有些清清冷冷地抱怨道：“你打电话不挑时间的吗？你也不看看现在几点？”
<br/>
<br/>        “你什么意思？”连星儿一脸懵。
<br/>
<br/>        明司寒不耐烦地回答道：“我这边，现在是凌晨六点！你不睡觉，我要睡觉的啊！大小姐，你能不能收敛一下你的‘公主病’，打电话也要挑时间啊！”
<br/>
<br/>        “我……”连星儿欲言又止，顿了顿后，接着问道，“你是不是把我的号码拉黑了？为什么后来我给你打电话，提示是关机？”
<br/>
<br/>        “后来手机没电了，充了电后，就忘了……”明司寒云淡风轻地回答。
<br/>
<br/>        连星儿有些难以置信地张着嘴巴，却不知道自己接下来要说什么。
<br/>
<br/>        他们俩才分开不过两天而已，他明司寒怎么突然就跟变了个人似的。
<br/>
<br/>        “明司寒，你到底有没有把我放心上？”连星儿又气又恼，其实心里更多的是抑郁。
<br/>
<br/>        明司寒压低了嗓音，显得极其不耐烦地回答道：“我现在想要继续睡觉，没心情跟你谈情说爱。”
<br/>
<br/>        “你这话是什么意思？”连星儿怔愣。
<br/>
<br/>        明司寒淡然道：“我有起床气！对谁都脾气不好，你也不例外。你要说什么，一切等我睡醒了后再说！就这样，挂了！”
