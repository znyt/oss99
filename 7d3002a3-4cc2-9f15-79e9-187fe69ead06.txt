　　初步掌控蜂巢基地，苏重松一口气。但这只是开始，想赚取本源，想活命，他就必须把病毒血清研究出来。
<br/>　　“红后，基地内水电供应是否安全可靠。”苏重问道。他要整合资源，做好持久战准备。
<br/>　　“蜂巢基地链接一座核电站，供电稳定安全。即便核电站出问题，也有一套地热发电设施备用。虽然只能供给基地半数设备，但也足够你们几人使用。”红后一边说，一边投影出图像给苏重展示。
<br/>　　“水源完全取自底下暗河，有一套完整的净化系统，不用担心水质污染。”
<br/>　　“那食物呢，够我们这些人生活多久？”苏重接着问。
<br/>　　“原本食物储备是一千五百人半个月的分量。有一部分已被污染，不可使用。经过计算，可以支撑你们六人半年生存。如果将基地种植房开启，时间将会大大延长！”
<br/>　　几人长出一口气。有食物有水源，他们几个就能生存下来。电力供应不出问题，蜂巢基地能够运转，他们的生存环境就不会太坏。
<br/>　　“好了，先去清理蜂巢，首先要把舔食者和生化猎犬解决掉！”
<br/>　　舔食者威胁太大！一旦放出来，几人只能躲避起来。
<br/>　　生化猎犬速度快，几人一旦被抓伤，立刻就会变成丧尸。是威胁几人生存的巨大隐患。
<br/>　　好在红后关闭时间不长，舔食者还没来得及苏醒。休眠在集装箱内的它们只能等死。
<br/>　　“红后，给我之处生化猎犬位置。我先去杀了它们！”
<br/>　　“好的先生！”苏重找了个无线耳机带上，手里拿着一个平板。上面明确标识着生化猎犬和丧尸的位置。
<br/>　　苏重有钢钉在手，一路上挥洒不断，丧尸成片到底。跟在后面的几个人，根本来不及出手。只能带着手套帮忙搬开挡路尸体。
<br/>　　很快几人就遭遇丧尸猎犬。肌肉外翻，血肉模糊，眼睛浑浊，和那些活死人一个模样。但意外的，这些猎犬速度极其快速，而且露在外面的血肉仍然新鲜。
<br/>　　它们不是丧尸，是经过丧尸改造过的变异怪物！
<br/>　　见到活人，丧尸犬张开大嘴，猛然飞扑过来。苏重镇定自若，钢钉脱手而出，直接扎进对方脑袋。
<br/>　　苏重微微侧身，三只丧尸猎犬擦身而过，噗通一声落在地上了无声息，苏重瞬间就将他们放倒。
<br/>　　让他意外的是，爱丽丝竟一脚踢死了一头从背后偷袭的猎犬。
<br/>　　“厉害！”苏重赞叹。那一脚凌空飞踢，可不是普通人力气能办到。
<br/>　　而且丧尸猎犬生命力顽强，普通人一脚可踢不死对方！
<br/>　　“看来，生化病毒已经开始对爱丽丝进行改造了。”那位女汉子就要觉醒了。
<br/>　　之后苏重带着几人回到b餐厅，在舔食者沉睡之际，就直接捣烂对方脑袋。之后又清理了其他地方存在的几头奇形怪状的生化生物。苏重就一头扎进一间设备齐全的实验室。
<br/>　　剩下的清扫工作，苏重都交给了爱丽丝。这位女汉子觉醒之后，越发彪悍凌厉。杀起丧尸来那叫一个干净利落。有红后配合关闭门禁，几人对丧尸围追堵截，即便使用简单铁锹，也能轻松杀死丧尸。
<br/>　　基地有焚化炉，这些丧尸全都被几人拖进焚化炉烧毁。基地内丧尸被快速清理。
<br/>　　实验室内，苏重在红后讲解下，快速熟悉试验设备。
<br/>　　“先生，就目前来看，您并不具备生物研究能力，您研制病毒血清的成功率，不足%0.3。”红后一边讲解，一边严肃的泼冷水。
<br/>　　苏重脸色漠然，手上动作从开始的生涩，快速变得熟练起来。
<br/>　　“那你有没有计算我掌握各种仪器的速度？从一窍不通道娴熟操作，我一共用了多少时间？”
<br/>　　红后沉默：“您是我见过接受能力最强的人类。您可能会创造奇迹。”
<br/>　　“奇迹？红后，你不是一直用数据说话的吗？这种不确定性的词汇，你确定不是在调侃？”苏重一边动手一边闲聊。
<br/>　　“我并非简单的程序指令。”红后一板一眼的回答。
<br/>　　“对对，你是最伟大人工智能红后，你又自己的智慧。”
<br/>　　“谢谢夸奖。”
<br/>　　“呵呵，真想把你带走。”苏重忍不住道。
<br/>　　“带走？没有解决病毒，您只能也必然会在这里陪伴我。焚化炉离我并不远。”红后立刻反驳。
<br/>　　苏重笑着摇头，他越发觉得这个人工智能有意思了。
<br/>　　“好了红后，把所有病毒资料都给我放出来，我要开始拼命了！”不拼命就真的要命啦。他已经感觉到身体传来的虚弱感。
<br/>　　他必须在三天内研究出抑制药剂，延缓自身病毒发作时间，不然他就死定了！
<br/>　　海量资料出现在苏重眼前。保护伞对生化病毒，已经研究了数十年，资料庞大的不敢想象！
<br/>　　“摒弃所有生化武器方面的资料，只给我病毒本身还有医疗方向的资料。”他时间有限，没功夫全部研究，只能专精一项，先把小命保住再说。
<br/>　　即便如此，资料依旧成山成海：“看来只能开挂了，希望不会亏死！”
<br/>　　他敢和红后合作，笃定自己能够研究出病毒血清，当然有着自己的依仗。
<br/>　　闭上眼睛，看着技能面板上可怜的2.3单位的本源，苏重狠心直接调动一点本源。
<br/>　　舍不着孩子套不着狼，拼了！
<br/>　　本源涌入体内，在苏重控制下，直接钻进大脑内。
<br/>　　苏重要用本源粗暴的强化大脑，提高推演计算能力！只有这样，他才能在最短时间内完成病毒血清研制！
<br/>　　脑袋一热，然后是剧烈疼痛，就好似烧红烙铁捅入脑海使劲搅动。苏重眼前猛然发黑，一屁股坐在地上。
<br/>　　“先生，您的体温不正常提高，心跳正在加速，您需要帮助吗？”红后罕见的露出担忧神色。
<br/>　　好半晌，苏重才回过神，使劲晃了晃脑袋，然后他就察觉出了不同。
<br/>　　思维更加清晰，注意力显著提高，一点本源下去，苏重大脑得到了改造。强大精神力配合改造后大脑，苏重推演计算能力极速提升。
<br/>　　“红后，显示所有病毒资料。”
<br/>　　眼前光影开始闪烁。
<br/>　　“加快速度！”改造后大脑，让苏重过目不忘，堪比人肉相机！
<br/>　　“继续加快！”
<br/>　　“再快！”
<br/>　　“先生，您真的能看清？目前显示速度，已经超出保护伞最顶尖研究人员的信息接受速度！”
<br/>　　“如果不能超过保护伞研究员，我拿什么去研究血清！继续加快！”苏重满脸潮红，脑海中各种资料不断闪烁，灵感火花碰撞闪耀。那种通透舒爽，让苏重浑身战栗！
