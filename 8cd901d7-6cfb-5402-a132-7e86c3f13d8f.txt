        童筱颖捶了捶发麻的小腿，抹着额头上的汗水，被午后的阳光，晒得哈欠连连。
<br/>
<br/>        就在这时候，电梯门“叮”的一声打开，傅夜沉在一群公司高层的簇拥下，姿态淡然地从电梯里走了出来。
<br/>
<br/>        他所经之处，不断的有着恭敬的“傅总好”响起。
<br/>
<br/>        童筱颖连忙转过头去，只看见大厅里在众人的簇拥下，身穿墨蓝色西装的傅夜沉从正门口走了出来。
<br/>
<br/>        他似乎在与身后的人说着什么工作上的事情，身边的那个女秘书落以柔正埋头握着笔杆子在笔记本上飞快地记录着谈话内容。
<br/>
<br/>        “大哥！”她想大声一点，声音却比想像中小很多，因为坐得太久，又被太阳曝晒了两个小时，她起身的这一刻，发起了黑眼晕。
<br/>
<br/>        童筱颖只觉一片天旋地转，无数颗黑色的光点吞噬了她的视野，脚下的步伐有些不稳。
<br/>
<br/>        不过，她刚刚的那声呼唤，虽然没有引了其他人的注意，却让傅夜沉眸中闪过片刻怔忡。
<br/>
<br/>        傅夜沉像是有心灵感应般，循声望去，看到童筱颖的人时，心中浮泛着喜悦，下一秒，他突然冲出人群，伸手拉住了童筱颖的手腕，将她那摇摇欲坠的身子揽入了怀中。
<br/>
<br/>        在场所有人，都用惊愕的目光，看着这一幕。
<br/>
<br/>        落以柔抬起头来时，也看傻了眼。
<br/>
<br/>        童筱颖甩了甩头，黑眼晕的症状得以缓解后，她才抬起头来，看着这个搂着自己的男人。
<br/>
<br/>        “大哥，你总算出来了。”
<br/>
<br/>        “你等我很久呢？”
<br/>
<br/>        “来给你送午饭，但我进不去。”童筱颖苦着脸说。
<br/>
<br/>        “怎么不给我打电话？”
<br/>
<br/>        “没你号码呀！”
<br/>
<br/>        “你可以给刘婶打电话，让她打给我。”
<br/>
<br/>        “我也没有刘婶的电话号码……”童筱颖有些局促。
<br/>
<br/>        “你还可以让前台打我办公室的座机。”
<br/>
<br/>        “你公司里的员工，一个个都非常的敬职敬业，而且，十分地保护着你，不让你被外界那些动机不纯的女人骚扰，真是勇气可嘉。”童筱颖咧嘴苦笑。
<br/>
<br/>        先是被前台小姐无情地拒绝，又被路经的员工冷漠地嘲讽，再是被保安残酷地“请出”了大门。
<br/>
<br/>        不过，不知者无罪，童筱颖想得很开，也不记仇。
<br/>
<br/>        傅夜沉会心一笑，直接将童筱颖打横抱了起来，转身往大门口走去。
<br/>
<br/>        落以柔和身旁的人，立即识趣地颔首，给傅夜沉让出了一条道。
<br/>
<br/>        郑忠威见状，马上会意地替傅夜沉发号施令，遣散了这群高层，宣布今天下午的子公司的例行检查改到明天下午这个时刻执行。
<br/>
<br/>        傅夜沉抱着童筱颖并未回总裁办，而是直接坐在了大厅的沙发上。
<br/>
<br/>        “你这是给我带了午饭来的？”傅夜沉嘴角微扬，指了指童筱颖怀里抱着的保温饭盒。
<br/>
<br/>        童筱颖这才想起自己来这里的目的，连忙将手里的保温饭盒放到了沙发前的茶几上，并一层一层地打开，微笑着介绍道：“牡蛎汤是刘婶给你炖的，这素炒冰菜、五色摩提、啤酒魔芋烧鸭，是我做的。”
<br/>
<br/>        “正好我饿了。”傅夜沉从保温饭盒上的暗槽里，拿出了配套的筷子，抬眸的无意间，看到落以柔不知何时候在一旁，于是，淡淡地命令道，“让那位前台小姐，帮我太太倒杯温水过来。”
<br/>
<br/>        落以柔不知道是不是自己的错觉，还是的确如此，她觉得“太太”两个字被傅夜沉说得很重，似乎在跟周围所有人提醒着童筱颖的身份。
<br/>
<br/>        “好的，请稍等。”落以柔用颤抖的声音回答道，她丝毫不敢怠慢，加快步伐往前台那边走去。
<br/>
<br/>        此时，童筱颖压根就未注意到落以柔的一举一动，而是一门心思地全在他傅夜沉的身上。
<br/>
<br/>        “好吃吗？”
<br/>
<br/>        “只要是你做的，我都觉得好吃。”
<br/>
<br/>        “正经点嘛！我问认真的！”
<br/>
<br/>        “真的很好吃！”傅夜沉一边咀嚼着口里的食物，嘴角微扬地回答道。
<br/>
<br/>        童筱颖暗自庆幸，好在傅夜沉的喜好跟傅昼景一样。
<br/>
<br/>        傅昼景爱吃的菜，他傅夜沉也爱吃。
<br/>
<br/>        其实，她有问过刘婶，他傅夜沉爱吃什么不爱吃什么，但是刘婶告诉她，只要是她亲手做的，他都会喜欢吃。
<br/>
<br/>        所以，她才不得不照着傅昼景的喜好去给他做菜。
<br/>
<br/>        落以柔和那位前台小姐交代完倒水任务后，又转头看向了傅夜沉那边。
<br/>
<br/>        明明他中午已经吃过她给他从餐厅里打包带上去的午饭了，此刻却为了那个女人装成自己很饿的样子，是为了什么？
<br/>
<br/>        难道是为了讨那个女人的欢心吗？
