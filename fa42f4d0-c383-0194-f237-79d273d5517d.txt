　　“苏莫哥哥，你的意思是，带着我们逃离古周星吗？”夕儿率先问道。
<br/>　　苏莫闻言沉默不语，少倾之后，轻轻的摇了摇头，道“本来我是有这个打算，将苍穹神宫都带在身上，毕竟我随时都可能逃亡，但现在我改变主意了！”
<br/>　　众人没有出声，静听苏莫讲解。
<br/>　　“巫族并不知道你们的存在，只知道我和秦不死两人，所以，你们不会有危险！”苏莫沉声说道。
<br/>　　众人点了点头，当初逃出苍穹世界之时，他们都是在空间宝器之中，没有人知道他们的存在。
<br/>　　“我的想法是，你们就留在这里吧！若是我真的逃离了古周星，那就独自离开！”
<br/>　　苏莫继续说道，他也考虑清楚了，带着苍穹神宫的人一起，万一他遭难了，整个苍穹神宫也就完了。
<br/>　　所以，他不准备带着苍穹神宫了，无论他是否离开古周星，是否离开古灵星河，就让苍穹神宫，安稳的在古周星发展吧！
<br/>　　“苏莫哥哥，你带上我吧！”
<br/>　　夕儿沉声说道，他不想和苏莫分开，若是苏莫真的逃离的话，肯定会离开古灵星河，估计不知道多少年以后，才能再回来了。
<br/>　　“夕儿，别任性，跟着我太危险了！”苏莫摇了摇头。
<br/>　　夕儿闻言，顿时俏脸一苦，她知道苏莫既然做出了决定，她很难说服。
<br/>　　“苏莫，这样也行，我们跟着你也给不了你什么帮助！”闻人天都叹道，他们实力较低，跟着苏莫不仅无法给予帮助，还会给予苏莫压力。
<br/>　　“嗯，以后苍穹神宫的发展，就靠你们了。”苏莫点头。
<br/>　　司空炎沉吟了片刻，开口道：“苏莫，我有一个想法，不知道你同不同意？”
<br/>　　“什么想法？”苏莫问道。
<br/>　　“我们几个天赋都很好，憋屈在苍穹神宫，实在是限制了天赋，修炼速度太慢了！”
<br/>　　司空炎面色郑重，继续道：“我的想法是，我们几个也离开苍穹神宫吧，去加入各方超级势力，以后也能给予你帮助！”
<br/>　　苏莫闻言面现沉吟之色，这个想法倒也不错，如果几人都加入各方大势力，以后的确是能给予他帮助。
<br/>　　“你们的想法呢？”苏莫又看向其他人。
<br/>　　“我同意，我上次出去历练，还碰到了一名高阶武尊，是某个大势力的长老，想要收我为徒，但被我拒绝了！”战乱说道。
<br/>　　“我也同意！”
<br/>　　“这样也行！”
<br/>　　“……”
<br/>　　除了夕儿之外，其他几人纷纷表示赞同，他们早就有加入各方大势力的打算，但没有得到苏莫的认可，便没有行动起来。
<br/>　　“好吧！我也没有意见！”
<br/>　　苏莫点了点头，他也不想限制众人的发展，随即他又补充道：“不过，我有一个要求，那就是你们一定要加入那种真正的超级势力！”
<br/>　　“这没有问题，以我们的天赋，那些超级势力也会抢着收我们的！”司空炎笑着说道。
<br/>　　“嗯，希望你们都能有自己的机遇！”苏莫点头笑道，若是几人都等在各个大势力扎根，对于他来说也是好事。
<br/>　　又和几人闲聊了几句之后，苏莫目光看向夕儿、彩芸、清雅、闻人天都四人，道：“你们四人随我去空间宝器之中，我为你们提升天赋！”
<br/>　　“好！”四人异口同声的说道，她们早已知道司空炎和姜枫然两人，被苏莫赐予了战魂，都变成了双生战魂的天才。
<br/>　　对于此事，她们四人早已心动，但并没有提及。
<br/>　　因为他们知道，苏莫不会单单只赐予司空炎和姜枫然，他们肯定也有。
<br/>　　而后，苏莫意念一动，带着四人进入了玉镯空间之中。
<br/>　　“战乱，没有你的份了！”见苏莫几人消失，司空炎面带调笑的对战乱说道。
<br/>　　“哼！我本来就是双生战魂！”战乱轻哼一声，毫不在意。
<br/>　　……
<br/>　　玉镯空间之中。
<br/>　　夕儿几人乍一进来，便被震惊了，目瞪口呆的望着眼前的一切。
<br/>　　只见此刻，方圆三百里的玉镯空间，几乎快被塞满了。
<br/>　　庞大无比的灵石矿脉，仿佛一座座山峰一般，杂乱无章的堆放在这片空间之中。
<br/>　　而且，不远处还有两男一女，在卖力的开采灵石。
<br/>　　这两男一女，自然便是巫娴和另外两名巫族天才。
<br/>　　“宫主，您哪来的这么大的灵石矿？”清雅小嘴微张，满脸震惊的问道。
<br/>　　“偷的！”苏莫微微一笑，他早就命令巫娴三人给他开采灵石了，三人都是准帝修为，开采的着实不忙，要不了多久便能全部开采完毕。
<br/>　　几人闻言无语，偷的？在哪里能偷到这么大的极品灵石矿脉。
<br/>　　“好了，现在我就为你们融合战魂！”苏莫不愿浪费时间，立刻便行动了起来。
<br/>　　他最先为闻人天都融合了战魂，是一种颇为强大的长刀战魂，因为并没有更适合对方的光属性战魂，他便只能赐予了对方一种器战魂了。
<br/>　　而后又为夕儿融合了一种兽战魂，是一种五彩神鸟，同样颇为强大。
<br/>　　夕儿本来的战魂，是一头青凤，现在加上一头五彩神鸟，两种兽战魂倒是相得益彰。
<br/>　　之后，苏莫又为清雅融合了一种青藤战魂。
<br/>　　清雅本来的战魂是一柄青色长剑，攻击力颇为不俗，苏莫本准备再给对方融合一种长剑战魂，但考虑到清雅的属性天赋，以及攻击和防御的多样性，便给对方融合了青藤战魂。
<br/>　　最后是彩芸，彩芸原本的战魂是刀，一柄黑色的珍袖短刀。
<br/>　　苏莫再次给对方融合了一柄黑色的长刀，威力比对方本来的短刀战魂更加强大。
<br/>　　“彩芸，这个给你！”
<br/>　　一切完事之后，苏莫手掌一番取出了一枚玉简，放在额头上扩印之后，便递向彩芸。
<br/>　　“宫主，这是什么？”彩芸面带疑惑之色。
<br/>　　如今的彩芸，依旧和以往一样，穿着黑色的长袍，将脑袋都盖住了。
<br/>　　这是因为她是异族，头生双角，不想引人注意。
<br/>　　“这是一门魔功，无比强大的魔功，很适合你修炼！”苏莫沉声说道，他给予对方的正是混天魔功。
<br/>　　彩芸是角魔族之人，体内天生便带有魔气，非常适合修炼这本功法。
<br/>　　“哦？”彩芸闻言面现惊疑之色，随即接过玉简立刻便查看了起来。
<br/>　　这一看之下，彩芸娇柔的身躯立刻一震，随即便沉浸在了其中。
<br/>　　“宫主，您给彩芸姐姐的是什么功法？”清雅见此，立刻好奇了起来。
<br/>　　夕儿和闻人天都两人，也是面带好奇之色。
<br/>　　“一种魔功，等级为圣级上品！”苏莫沉声说道。
<br/>　　“什么？”清雅三人闻言，顿时震惊不已，居然是圣级上品的武学！
<br/>　　苏莫身上居然连圣级上品的武学都有，这也太夸张了吧！
<br/>　　这种级别的武学，就算是那些超级大势力中，最核心的弟子，也没有资格修炼吧！
<br/>　　少倾之后，彩芸将混天魔功粗略的看了一遍，意念退出了玉简，面上依旧带着震撼之色。
<br/>　　“彩芸，这魔功非常强大，但修炼难度也是极大，你若要修炼的话就要有心理准备！”苏莫告诫道。
<br/>　　“宫主，这魔功非常适合我，谢谢你！我不会让你失望的！”彩芸收起玉简，贝齿轻咬红唇，郑重的说道，她的眸中带有感动之色。
<br/>　　苏莫能将如此珍贵的功法给她，便是对于她的绝对信任。
<br/>　　“谢就不必了，我作为宫主，帮助你们是应该的！”苏莫摇头笑道。
<br/>　　随后，几人也没有在此多待，便一起离开了玉镯空间。
