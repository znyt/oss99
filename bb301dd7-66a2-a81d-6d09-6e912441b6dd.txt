        起初，季思妍一直在怀疑傅昼景是否真的爱上了童筱颖。
<br/>
<br/>        而现在，她不再怀疑了。
<br/>
<br/>        季思妍不认为傅昼景是真的爱上了童筱颖，倘若，他真的爱上了童筱颖，那么就不会背着童筱颖跟她一次又一次的上床。
<br/>
<br/>        哪怕他现在想要跟她撇清关系，但是他的身体，似乎对她产生了一种依赖性的记忆。
<br/>
<br/>        因为，她只要稍稍暗示一下，他的身体就会对她做出诚实的回应。
<br/>
<br/>        “我还要工作，你请回。”傅昼景微微皱了下眉头，下逐客令。
<br/>
<br/>        季思妍见好就收，红唇微扬，双手轻轻地放开了傅昼景。
<br/>
<br/>        “那我先回去了，你好好考虑一下我所说的话。”季思妍向傅昼景抛了个媚眼，抬起手来，在唇上吹了个飞吻，朝傅昼景送去。
<br/>
<br/>        傅昼景淡定地收回目光，埋头继续工作。
<br/>
<br/>        季思妍离开后，傅昼景心里其实已经开始迷茫了。
<br/>
<br/>        翌日。
<br/>
<br/>        临海大学，女生宿舍。
<br/>
<br/>        天才蒙蒙亮，楼下就传来了一阵又一阵的口哨声。
<br/>
<br/>        童筱颖是被睡在自己对床的蓝筱蝶给摇醒的。
<br/>
<br/>        “筱颖、筱颖，你快醒醒啊！楼下有人跟你表白！”蓝筱蝶激动不已地说道。
<br/>
<br/>        童筱颖揉着惺忪的睡眼，从床上爬了起来，才发现自己居然是最后一个起床的。
<br/>
<br/>        而另外两个室友恭如兰和严恩恩正趴在窗台上，往外眺望着什么新鲜的事情。
<br/>
<br/>        “你快起来去看看啊！”蓝筱蝶催促道。
<br/>
<br/>        童筱颖不得不从床上下来，打着哈欠，趿着拖鞋，和蓝筱蝶一起走到窗户边去“凑热闹”。
<br/>
<br/>        只见楼下的水泥坪里，不知道是谁用玫瑰花的盆栽，摆出了“童筱颖我爱你”这六个字，引得整栋女生宿舍一片哗然。
<br/>
<br/>        “筱颖，你到底是咱们宿舍的头号美女！第一天开学报到，第二天就有男生向你表白！”恭如兰忍俊不禁地调侃。
<br/>
<br/>        童筱颖凑完“热闹”后，微眯着惺忪的睡眼，趿着拖鞋，又重新爬回了床上，好像楼下那用玫瑰花盆栽摆成的告白与她无关一样。
<br/>
<br/>        事实上，童筱颖也确实这么认为，说不定，学校里真有女生跟她同名同姓。
<br/>
<br/>        这种事情，当事人本人都不在意，三个舍友看完热闹也是各忙各的。
<br/>
<br/>        没一会儿，宿舍门被人敲响。
<br/>
<br/>        离门口最近的严恩恩，主动走过去开门，但未见敲门的人。
<br/>
<br/>        当严恩恩准备关门的时候，无意间地垂眸看到地上摆着一篮子的进口水果和一大袋的进口零食。
<br/>
<br/>        上面还放着一张写有“童筱颖我爱你”这六个字的卡片。
<br/>
<br/>        “筱颖、筱颖，真的有人在追你啊！”严恩恩将水果和零食以及那张卡片拿了进来。
<br/>
<br/>        大大咧咧的蓝筱蝶直接夺走了严恩恩手中的吃的，严恩恩忍俊不禁地摇摇头，然后将手里的递给了躺在上铺的童筱颖。
<br/>
<br/>        童筱颖迷迷糊糊地拿着卡片看了看，发现卡片上的字体是电脑打印的，看不出什么所以然来，于是又将卡片还给了严恩恩，呢喃道：“一定送错寝室了！”
<br/>
<br/>        “依我看，就是有人在追求你！”恭如兰走到严恩恩的身旁，替严恩恩接过卡片，好奇地看了看。
<br/>
<br/>        恭如兰看完后，一本正经地分析道：“卡片上的字体确实是电脑打印的，而且没有任何落款人，说明这个人想要保持自己的神秘感！”
<br/>
<br/>        童筱颖被这先是玫瑰花后是送水果送零食的事情一折腾，瞬间睡意全无。
<br/>
<br/>        最重要的一点是，三个室友一边吃着水果和零食，还一边像审问犯人一样，开始审问起童筱颖来。
<br/>
<br/>        “我发誓，我真的已经有……男朋友了！”童筱颖信誓旦旦地说，本来她是想说她已经有老公的，但是她说她有男朋友了，她们都不信，更别提有老公。
<br/>
<br/>        “那你在学校里有认识的异性吗？”恭如兰接着问。
<br/>
<br/>        童筱颖点了点头，如实回答：“有三个认识的异性，其中一个已经有女朋友了，另外两个是宅男！”
<br/>
<br/>        她嘴里所说的这三异性，一个是何明旭，至于他的女朋友，自然是林若琴。另外两个，就是何明旭的室友章海昌和方浩博。
<br/>
<br/>        “那你把另外两个没有女朋友的宅男叫出来问问，不就行了！”恭如兰提议道。
<br/>
<br/>        严恩恩和蓝筱蝶猛点头。
<br/>
<br/>        童筱颖却摇头道：“我认识他们快一年了，他们不可能会做出这种事情。”
<br/>
<br/>        说到这，童筱颖见蓝筱蝶又拆了一包零食，连忙制止道：“筱蝶，别吃了。万一别人搞错对象了，我们岂不是很尴尬！”
