这几天陆寒霆一直惦记着她身上的伤，不能问，不敢问，现在她主动送上了门，他就肆无忌惮了起来，一定要看一下她身上的伤。
<br/>
<br/>        碎花雪纺衫揭了上去露出了她的肌肤，那些被咬破的青紫伤口已经都好了，一点伤痕都没有留下，她的肌肤牛奶白似的，肤如凝脂。
<br/>
<br/>        “不要再看了。”
<br/>
<br/>        这时夏夕绾将他戴着名贵腕表的大手给推了下去，遮住了自己的娇肌，不让他再看。
<br/>
<br/>        陆寒霆抬眸看着她，声线沙哑，“真的都好了？”
<br/>
<br/>        “对啊，伤的又不是很重，我抹点药膏就好了，不过…这里还有点疼。”夏夕绾拉着他的大手放在了自己细嫩的粉颈上。
<br/>
<br/>        陆寒霆看了一下，当时他就着她的血管深深咬下去的，其他地方的伤都好了，就这里还有一道浅痕，隐约可以看到他的齿印，可以想象得出当时他咬的有多狠。
<br/>
<br/>        陆寒霆将俊脸埋在她的粉颈里，贪婪的吸着她身上那股令他着迷的少女体香，薄唇落在那个伤痕上，反复的亲吻，“我很抱歉，对不起…”
<br/>
<br/>        道歉是最苍白的话语，但是此刻他只能反复说着这样的话。
<br/>
<br/>        “恩，这一次就原谅你了，不过你以后不要咬我了。”夏夕绾勾唇笑道。
<br/>
<br/>        陆寒霆薄唇往上，亲她雪白的小耳垂，“不咬了，顶多就是这样，轻轻的咬一下。”
<br/>
<br/>        他咬了一下她的耳垂。
<br/>
<br/>        夏夕绾迅速躲避，两只小手抵着他精硕的胸膛将他给推开，“好痒啊，陆先生，你越亲越…色情了。”
<br/>
<br/>        她青涩的反应分明就是一张白纸，陆寒霆敛了一下俊眸让她还缠在他精硕腰身的两条细腿落在地上，然后扣着她的腰身将她往自己的怀里带，“这样就色情了？”
<br/>
<br/>        他有点看不起她的意思，夏夕绾挑着精致的柳叶眉，不甘示弱的回道，“那天晚上你还不够色情？”
<br/>
<br/>        这时电梯到了，门开了，陆寒霆搂着她走了出去，“如果你真的好奇的话，我可以带你见识一下世面，让你知道什么叫真正的色情。”
<br/>
<br/>        “…”
<br/>
<br/>        夏夕绾觉得不能再跟他讨论这么不健康的话题，“陆先生，我发现你挺喜欢小姑娘的，等以后我老了，你会不会喜欢上其他小姑娘啊？”
<br/>
<br/>        夏夕绾真的发现他喜欢清纯水灵的小姑娘，喜欢粉润俏妍的少女气，这很正常，像他这种权势地位的男人大都都喜欢年纪小的女孩子，女人的年龄和美貌是他们的消费品，他们可以肆意挥霍。
<br/>
<br/>        她现在还小，可是以后会长大，然后慢慢变老的。
<br/>
<br/>        陆寒霆低头看了她一眼，“在你之前，我身边又不是没有小姑娘，我并没有发现我对这一挂的多感兴趣，我不是喜欢小姑娘，而是因为你恰巧是小姑娘，我喜欢你，明白么？”
<br/>
<br/>        夏夕绾心里一甜，果然是霸道总裁啊，连表白都拽的跟个二五八万似的。
